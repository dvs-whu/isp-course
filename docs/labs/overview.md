## 一、实验目标

1. 通过代码实践理解信号处理的核心概念。
2. 掌握常用Python工具完成信号生成、读取、可视化、频谱分析和滤波。
3. 能把实验结果转化为项目报告中的图表、指标和分析。
4. 为项目一和项目二提供可复用的代码基础。

## 实验1：Python信号处理环境与基础（W1，2学时）

### 对应知识点

信号与系统基础、傅里叶分析概述、Python基础。

### 实验内容

1. Jupyter环境搭建与常用库导入。
2. 用NumPy生成基本信号：正弦波、方波、脉冲、含噪信号。
3. 信号的时域可视化。
4. 第一个FFT频谱分析程序。

### 预期产出

- 混合信号的波形图和频谱图。
- 能正确识别主要频率分量。

---

## 实验2：FFT频谱分析与滤波器设计（W4，2学时）

### 对应知识点

傅里叶分析、采样与重建、数字滤波器设计。

### 实验内容

1. 采样率变换实验：下采样与混叠现象演示。
2. 用 `scipy.signal.firwin` 设计FIR滤波器。
3. 用 `scipy.signal.butter` 设计IIR滤波器。
4. 对比FIR/IIR的频率响应、零极点和滤波效果。

### 预期产出

- 混叠现象对比图。
- FIR/IIR频率响应和零极点图。
- 滤波前后SNR对比表。

---

## 实验3：图像去噪与增强（W6，2学时）

### 对应知识点

图像信号处理、二维FFT、图像去噪、PSNR/SSIM。

### 实验内容

1. 对图像做二维FFT并可视化频谱。
2. 设计频域低通滤波器去噪。
3. 对比空间域滤波：均值、中值、高斯。
4. 用PSNR和SSIM评估去噪效果。

### 预期产出

- 原图、频谱、滤波后频谱、滤波后图像。
- 多种方法的PSNR/SSIM对比表。

---

## 实验4：统计信号处理（W7，2学时）

### 对应知识点

随机信号基础、功率谱估计、维纳滤波、自适应滤波。

### 实验内容

1. 计算随机信号的均值、方差和自相关。
2. 对比周期图法与Welch法的功率谱估计。
3. 分析白噪声与色噪声的统计特性。
4. 选做：LMS自适应滤波或一维卡尔曼滤波。

### 预期产出

- 自相关图、功率谱图。
- 周期图和Welch方法的对比。
- 简短结果分析。

---

## 实验5：深度学习信号处理（W9，2学时）

### 对应知识点

深度学习基础、CNN、PyTorch训练流程。

### 实验内容

1. PyTorch张量操作和自动求导。
2. 用CNN完成MNIST分类热身。
3. 理解训练循环：前向、损失、反向传播、参数更新。
4. 绘制loss和accuracy曲线。

### 预期产出

- 可运行的CNN训练代码。
- 训练损失和测试准确率曲线。

---

## 实验6：综合实验（W11，2学时）

### 对应知识点

综合应用、事件相机基础、语音增强、STFT。

### 实验内容（二选一）

| 方向 | 内容 |
|------|------|
| A：事件相机数据处理 | 加载事件流、事件帧累积、事件率分析、事件率FFT |
| B：语音增强实战 | STFT、谱减法、iSTFT重构、SNR评估 |

### 预期产出

- 完整的信号处理pipeline代码。
- 可视化结果与定量评估。
