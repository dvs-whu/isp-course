# 智能信号处理

**武汉大学人工智能学院 · 2026年秋季学期 · 本科生必修课**

---

## 课程简介

> 从真实信号出发，先学会观察、变换、滤波和建模，再用AI方法完成更复杂的信号处理任务。

| 项目 | 内容 |
|------|------|
| **主讲教师** | 余磊 教授 |
| **学时** | 52学时（13周 × 4学时/周，40理论 + 12实践） |
| **学分** | 3学分 |
| **先修课程** | 高等数学、线性代数、Python程序设计 |

## 知识体系

### 核心知识（W1-W8，期末考试范围）

- [信号与系统基础](知识点/经典信号处理/信号与系统基础/) — 信号分类、系统性质、卷积
- [傅里叶分析](知识点/经典信号处理/傅里叶分析/) — DFT/FFT、频谱解读、帕塞瓦尔定理
- [采样与重建](知识点/经典信号处理/采样与重建/) — 采样定理、混叠、量化
- [数字滤波器设计](知识点/经典信号处理/数字滤波器设计/) — FIR/IIR、窗函数、频率响应
- [Z变换](知识点/经典信号处理/Z变换/) — 系统函数、零极点、稳定性
- [短时傅里叶变换](知识点/时频分析/短时傅里叶变换/) — 语谱图、时频分辨率
- [小波变换](知识点/时频分析/小波变换/) — 多分辨率分析、连续/离散小波
- [随机信号基础](知识点/统计信号处理/随机信号基础/) — 均值、方差、自相关
- [功率谱估计](知识点/统计信号处理/功率谱估计/) — 周期图、Welch方法
- [维纳滤波](知识点/统计信号处理/维纳滤波/) — 最优线性滤波
- [卡尔曼滤波](知识点/统计信号处理/卡尔曼滤波/) — 状态估计、递推更新
- [自适应滤波](知识点/统计信号处理/自适应滤波/) — LMS、RLS算法

### 应用拓展（W9-W12，项目二会用）

- [深度学习基础](知识点/深度学习/深度学习基础/) — 神经网络、CNN、训练流程
- [展开网络](知识点/深度学习/展开网络/) — 将迭代算法展开为网络层
- [生成模型](知识点/深度学习/生成模型/) — GAN、扩散模型
- [图像去噪](知识点/图像信号处理/图像去噪/) — DnCNN、空间/频域方法
- [图像超分辨率](知识点/图像信号处理/图像超分辨率/) — SRCNN、ESRGAN
- [计算成像](知识点/图像信号处理/计算成像/) — 单像素相机、相位恢复
- [语音增强](知识点/语音信号处理/语音增强/) — 谱减法、深度学习方法
- [语音分离](知识点/语音信号处理/语音分离/) — 盲源分离、掩膜方法
- [声源定位](知识点/语音信号处理/声源定位/) — 波束形成、DOA估计
- [事件相机基础](知识点/事件驱动/事件相机基础/) — 异步像素、事件流表示

| 选题 | 名称 | 核心技术 |
|------|------|----------|
| A | 心电信号滤波与心率检测 | FFT、带通滤波、R波检测 |
| B | 音频信号降噪与采样实验 | FIR/IIR设计、SNR评估 |
| C | 图像频域滤波与时频增强 | 二维FFT、频域滤波 |
| D | 语音信号分析与特征提取 | STFT、MFCC、小波 |
| E | 自适应滤波与信号估计 | 功率谱、维纳、LMS |
