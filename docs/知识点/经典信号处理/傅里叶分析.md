## 2. 通俗入口

> [!tip] 🎯 一句话理解
> 傅里叶变换就是把一个复杂的波形"拆"成一堆不同频率的正弦波——就像把一道混合果汁拆回每种水果的比例。

**一个真实问题**：你听音乐时播放器上跳动的**频谱图**（低音、中音、高音的柱状条），背后就是傅里叶变换在实时工作；**音乐均衡器**让你单独调节某个频率的音量，也是在频域上操作。

**生活类比**：你在一间嘈杂的教室里，同时听到有人说话（低频为主）、粉笔写字（高频为主）、空调嗡嗡声（某个固定频率）。你的耳朵本质上在做"频谱分析"——把混合声波按频率分开。傅里叶变换就是数学上做这件事的工具。

**学这个知识点能解决什么**：
- 从"看不懂"的时域信号中提取出清晰的频率成分
- 在频域设计滤波器比时域更直观
- 为功率谱估计、语音分析、图像处理提供数学工具

## 3. 严谨核心

### 数学对象

傅里叶分析的核心思想：**任意满足一定条件的信号都可以分解为不同频率正弦/余弦分量的叠加。**

信号有**连续/离散**和**周期/非周期**两种属性，组合出四种情况，对应四种变换：

|          | **周期信号**             | **非周期信号**            |
| -------- | -------------------- | -------------------- |
| **连续时间** | 傅里叶级数（FS）：频谱是**离散**的 | 傅里叶变换（FT）：频谱是**连续**的 |
| **离散时间** | DFT（有限长，用 FFT 快速算）   | DTFT：频谱是**连续周期**的    |

> **规律**：时域越"受限"（如周期化、离散化），频域就越"展开"（如离散化、周期化）。这是傅里叶分析对偶性的体现。

| 变换类型 | 时域特点 | 频域特点 | 典型场景 |
|----------|----------|----------|----------|
| FS（傅里叶级数） | 连续、周期 | 离散、非周期 | 分析周期信号的谐波成分 |
| FT（傅里叶变换） | 连续、非周期 | 连续、非周期 | 理论分析，如单个脉冲的频谱 |
| DTFT | 离散、非周期 | 连续、周期（$2\pi$） | 理论上分析数字滤波器频率响应 |
| DFT / FFT | 离散、周期 | 离散、周期 | **实际计算**——计算机上的频谱分析 |

> **实际工作中的真相**：在计算机上能做的只有 DFT（因为计算机只能处理有限长离散数据），而 FFT 只是 DFT 的快速算法，复杂度从 $O(N^2)$ 降到 $O(N\log N)$。

### 核心公式

**连续时间傅里叶变换（FT）：**

$$X(j\omega) = \int_{-\infty}^{\infty} x(t) e^{-j\omega t} \, dt$$

$$x(t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} X(j\omega) e^{j\omega t} \, d\omega$$

> **直觉**：$e^{-j\omega t}$ 是一个以频率 $\omega$ 旋转的"探针"。把信号 $x(t)$ 和这个探针相乘再积分，就是在问："信号里有多少频率为 $\omega$ 的成分？" 对所有 $\omega$ 都问一遍，就得到了完整频谱 $X(j\omega)$。

**离散时间傅里叶变换（DTFT）：**

$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n] e^{-j\omega n}$$

$$x[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\omega}) e^{j\omega n} \, d\omega$$

**离散傅里叶变换（DFT）：**

$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-j \frac{2\pi}{N} kn}, \quad k = 0, 1, \dots, N-1$$

$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] e^{j \frac{2\pi}{N} kn}, \quad n = 0, 1, \dots, N-1$$

**卷积定理**——时域卷积 = 频域相乘：
$$\mathcal{F}\{x(t) * h(t)\} = X(j\omega) \cdot H(j\omega)$$

**帕塞瓦尔定理**——能量守恒：
$$\int_{-\infty}^{\infty} |x(t)|^2 \, dt = \frac{1}{2\pi} \int_{-\infty}^{\infty} |X(j\omega)|^2 \, d\omega$$

### 适用条件和边界

- FT 存在要求信号绝对可积（或使用广义函数）；不满足时需用傅里叶级数或 DTFT
- DFT 隐含周期延拓假设：有限长序列被视为周期信号的一个周期
- 频率分辨率 $\Delta f = f_s / N$，由采样率和数据长度共同决定
- DFT 的"栅栏效应"：只能看到 $N$ 个离散频率点，可能漏掉真实频率

## 4. 方法流程

**输入**：离散时间信号 $x[n]$，长度为 $N$

**处理步骤**：
1. 选择合适的窗函数（矩形窗、汉宁窗、汉明窗等）减少频谱泄漏
2. 对信号加窗：$x_w[n] = x[n] \cdot w[n]$
3. 计算 FFT：$X[k] = \text{FFT}\{x_w[n]\}$，复杂度 $O(N\log N)$
4. 计算幅度谱 $|X[k]|$ 和相位谱 $\angle X[k]$
5. 将频率轴映射为物理频率：$f_k = k \cdot f_s / N$

**输出解释**：
- 幅度谱的峰值对应信号中的主要频率成分
- 相位谱反映各频率分量的初始相位
- 功率谱 $|X[k]|^2$ 描述能量在频率上的分布

**常见参数选择**：
- FFT 长度 $N$：通常取 2 的幂次（如 1024、4096），提高计算效率
- 窗函数选择：汉宁窗适合一般频谱分析，凯泽窗可调节主瓣宽度与旁瓣衰减的折中
- 重叠率：STFT 中常用 50% 或 75% 重叠

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 生成复合信号：50Hz + 120Hz 两个正弦波叠加
fs = 1000  # 采样率 1000 Hz
t = np.arange(0, 1, 1/fs)  # 1秒时长
x = np.sin(2 * np.pi * 50 * t) + 0.5 * np.sin(2 * np.pi * 120 * t)

# 计算 FFT
N = len(t)
X = np.fft.fft(x)
freqs = np.fft.fftfreq(N, 1/fs)

# 绘制时域和频域图
fig, axes = plt.subplots(2, 1, figsize=(10, 6))
axes[0].plot(t[:200], x[:200])
axes[0].set_xlabel('Time (s)')
axes[0].set_ylabel('Amplitude')
axes[0].set_title('时域波形（两个正弦波叠加）')
axes[0].grid(True)

# 只绘制正频率部分的幅度谱
axes[1].plot(freqs[:N//2], 2/N * np.abs(X[:N//2]))
axes[1].set_xlabel('Frequency (Hz)')
axes[1].set_ylabel('Amplitude')
axes[1].set_title('FFT Spectrum: 50Hz and 120Hz Peaks')
axes[1].set_xlim([0, 200])
axes[1].grid(True)

plt.tight_layout()
plt.show()
```

**结果解释**：时域波形看起来是一团乱的曲线，但频谱图上会清楚地看到 **50 Hz** 和 **120 Hz** 处有两个尖峰——这就是傅里叶变换的威力：从"看不懂"的时域信号中提取出清晰的频率成分。50Hz 的峰更高因为其幅度为 1，120Hz 的峰较低因为幅度为 0.5。

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| FFT 就是傅里叶变换 | FFT 只是 DFT 的快速算法；DFT 是 DTFT 的采样版本，DTFT 才是离散信号的"真正"频谱 |
| 频率分辨率只取决于采样率 | 频率分辨率 $\Delta f = f_s/N = 1/T$，取决于观测时长 $T$，单纯提高采样率不改善低频分辨率 |
| 只看幅度谱就够了 | 相位谱包含信号的时间结构信息；丢弃相位只保留幅度，逆变换后波形完全不同 |
| 频谱泄漏是算法 bug | 频谱泄漏是因为有限长观测相当于加了矩形窗；用合适的窗函数可以缓解但不能完全消除 |
| DFT 频率轴总是从 0 到 $f_s/2$ | DFT 输出是周期的，负频率部分对应 $f_s/2$ 到 $f_s$；实际分析时通常只看正频率 |

## 7. 启发问题与创新拓展

**一个反例或边界问题**：
如果信号是非平稳的（如语音、音乐），直接对整段信号做 FFT 会得到什么？——所有时间结构被平均掉，只能看到"整体频率分布"而看不到频率随时间的变化。这就是为什么需要短时傅里叶变换（STFT）和语谱图。

**一个跨学科迁移场景**：
在金融时间序列分析中，股票价格的周期性波动（如季节性、经济周期）可以用 FFT 检测。对收益率序列做频谱分析，峰值频率可能对应经济周期的长度——这是一个将信号处理工具迁移到金融领域的典型场景。

**一个可做成课程项目的小想法**：
设计一个"频谱分析仪"交互工具：用户可以叠加不同频率和幅度的正弦波，实时看到时域波形和频谱图的变化。加入噪声后，观察能否从频谱中识别出信号成分——直观理解信噪比与频谱分辨率的关系。
