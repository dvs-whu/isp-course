## 2. 通俗入口

> [!tip] 🎯 一句话理解
> 数字滤波器就是一个"筛子"，让想要的频率分量通过，把不想要的频率分量挡住。

**一个真实问题**：你录了一段心电信号（ECG），但里面混入了 50Hz 的电源工频干扰。你需要一个"筛子"只挡住 50Hz，让其他频率正常通过——这就是陷波滤波器。

**生活类比**：
- FIR 滤波器像"一次性投票"——只看当前和过去的输入，每个输入按权重投票，投完就忘。稳定可靠但需要更多"投票者"（高阶数）。
- IIR 滤波器像"有记忆的投票"——不仅看当前输入，还参考之前的输出（反馈）。效率高但可能"失控"（不稳定）。

**学这个知识点能解决什么**：
- 心电信号去除基线漂移（高通滤波）和工频干扰（陷波滤波）
- 音频信号降噪（低通/带通滤波）
- 图像去除周期性条纹（二维滤波）

> 💡 **一句话记忆法**：FIR靠"卷积"（稳定但费算力），IIR靠"反馈"（省算力但要小心炸）！

## 3. 严谨核心

### 数学对象

数字滤波器是对离散时间信号进行频率选择性处理的系统，通过差分方程或卷积实现对特定频率分量的增强或抑制。

### 基本假设

- 滤波器是 LTI 系统（线性时不变）
- FIR 滤波器：冲激响应有限长，总是稳定的，可实现精确线性相位
- IIR 滤波器：冲激响应无限长，通常基于模拟滤波器原型设计，效率高但相位非线性

### 数字滤波器设计流程

![[assets/filter_design_flow_ai.png]]

*图：数字滤波器设计的本科主线。先把"想保留什么、抑制什么"写成频率指标，再选择FIR或IIR，最后必须同时检查频率响应、相位/延迟和滤波前后结果。*

### FIR与IIR滤波器对比

| 特性 | FIR滤波器 | IIR滤波器 |
|------|-----------|-----------|
| 冲激响应 | 有限长 | 无限长 |
| 稳定性 | 始终稳定 | 需设计保证（极点在单位圆内） |
| 相位特性 | 可实现精确线性相位 | 相位非线性 |
| 实现结构 | 非递归（无反馈） | 递归（有反馈） |
| 阶数需求 | 通常较高（过渡带窄时） | 通常较低 |
| 计算效率 | 较低（阶数高时） | 较高 |
| 设计方法 | 窗函数法、频率采样法、等波纹法 | 双线性变换法（基于模拟原型） |
| 典型应用 | 通信、音频延迟均衡 | 音频均衡、实时处理 |

### IIR模拟滤波器原型对比

| 原型 | 通带特性 | 阻带特性 | 过渡带 | 阶数效率 |
|------|----------|----------|--------|----------|
| 巴特沃斯 | 最平坦（无纹波） | 单调衰减 | 较宽 | 中等 |
| 切比雪夫I型 | 等纹波 | 单调衰减 | 较窄 | 较高 |
| 切比雪夫II型 | 单调 | 等纹波 | 较窄 | 较高 |
| 椭圆函数 | 等纹波 | 等纹波 | 最窄 | 最高 |

### 核心公式

**FIR滤波器输入输出关系：**
$$y[n] = \sum_{k=0}^{M} h[k] \, x[n-k]$$

**IIR滤波器差分方程：**
$$y[n] = \sum_{k=0}^{M} b_k \, x[n-k] - \sum_{k=1}^{N} a_k \, y[n-k]$$

**传递函数：**
$$H(z) = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 + \sum_{k=1}^{N} a_k z^{-k}}$$

**双线性变换（s域到z域映射）：**
$$s = \frac{2}{T} \cdot \frac{1 - z^{-1}}{1 + z^{-1}}$$

**预畸变校正：**
$$\Omega = \frac{2}{T} \tan\left(\frac{\omega}{2}\right)$$

**巴特沃斯滤波器幅度响应：**
$$|H(j\Omega)|^2 = \frac{1}{1 + (\Omega/\Omega_c)^{2N}}$$

**窗函数法设计FIR：**
$$h[n] = h_d[n] \cdot w[n]$$

**线性相位条件（FIR）：**
$$h[n] = h[M-n], \quad 0 \le n \le M$$
$$\text{群延迟} = \frac{M}{2} \text{ 个采样周期}$$

### 适用条件和边界

- FIR 滤波器在需要线性相位（如图像处理、数据通信）时是首选
- IIR 滤波器在计算资源有限、对相位要求不高时更高效
- 双线性变换会引入频率畸变（高频压缩），需要预畸变校正
- 滤波器阶数越高过渡带越窄，但延迟、计算量和数值误差也越大

## 4. 方法流程

**输入**：技术指标——通带截止频率 $f_p$、阻带截止频率 $f_s$、通带最大纹波 $\delta_p$、阻带最小衰减 $\delta_s$，采样率 $f_s$

**处理步骤**：
1. 确定技术指标（通带/阻带频率、纹波、衰减）
2. 选择滤波器类型：FIR（需要线性相位时）或 IIR（计算效率优先时）
3. 选择设计方法：
   - FIR：窗函数法（简单）、等波纹法（最优）
   - IIR：双线性变换法（基于巴特沃斯/切比雪夫/椭圆原型）
4. 计算滤波器系数 $h[n]$（FIR）或 $b_k, a_k$（IIR）
5. 验证：检查频率响应是否满足指标，检查相位/群延迟

**输出解释**：
- 频率响应曲线 $H(e^{j\omega})$：显示各频率的增益和相位
- 群延迟：显示不同频率成分通过滤波器后的延迟差异

**常见参数选择**：
- 截止频率：根据信号带宽和噪声频率确定
- 阶数 $M$ 或 $N$：过渡带宽度越窄需要越高阶数；经验公式 $M \approx \frac{A_s - 7.95}{2.285 \cdot \Delta\omega}$（$A_s$ 为阻带衰减 dB，$\Delta\omega$ 为过渡带宽度）
- 窗函数：矩形窗（最窄主瓣但最高旁瓣）、汉宁窗（折中）、凯泽窗（可调）

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import signal
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 设计低通 FIR 滤波器（窗函数法）
fs = 500  # 采样率
fc = 50   # 截止频率 50Hz
N_fir = 31
b_fir = signal.firwin(N_fir, fc, fs=fs)

# 设计低通 IIR 滤波器（巴特沃斯）
N_iir = 4
b_iir, a_iir = signal.butter(N_iir, fc, btype='low', fs=fs)

# 计算频率响应
w_fir, h_fir = signal.freqz(b_fir, [1], worN=512, fs=fs)
w_iir, h_iir = signal.freqz(b_iir, a_iir, worN=512, fs=fs)

fig, axes = plt.subplots(2, 1, figsize=(10, 6))
# 幅度响应对比
axes[0].plot(w_fir, 20*np.log10(np.abs(h_fir)), 'b-', label=f'FIR (阶数={N_fir})')
axes[0].plot(w_iir, 20*np.log10(np.abs(h_iir)), 'r-', label=f'Butterworth IIR (阶数={N_iir})')
axes[0].axvline(fc, color='k', linestyle='--', alpha=0.5, label=f'截止频率 {fc}Hz')
axes[0].set_title('Magnitude Response: FIR vs IIR Low-pass')
axes[0].set_xlabel('Frequency (Hz)')
axes[0].set_ylabel('Amplitude (dB)')
axes[0].set_ylim([-80, 5])
axes[0].legend()
axes[0].grid(True)

# 相位响应对比
axes[1].plot(w_fir, np.unwrap(np.angle(h_fir)), 'b-', label='FIR (Linear Phase)')
axes[1].plot(w_iir, np.unwrap(np.angle(h_iir)), 'r-', label='IIR (Non-linear Phase)')
axes[1].set_title('Phase Response Comparison')
axes[1].set_xlabel('Frequency (Hz)')
axes[1].set_ylabel('Phase (rad)')
axes[1].legend()
axes[1].grid(True)

plt.tight_layout()
plt.show()
```

**结果解释**：
- 上图：FIR 和 IIR 低通滤波器在 50Hz 处都有截止，但 FIR 需要 31 阶才能达到与 4 阶 Butterworth IIR 相似的截止陡度
- 下图：FIR 相位是严格线性的（直线），IIR 相位是非线性的（曲线）——这意味着 FIR 不会引入波形失真，而 IIR 会

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| 滤波后更平滑就一定更准确 | 平滑可能同时删掉真实细节，尤其是突变、尖峰和短时事件 |
| 滤波器阶数越高越好 | 高阶数会带来更大延迟、振铃、计算量和数值稳定问题 |
| 只看幅度响应就够了 | 相位响应和群延迟会改变波形位置与形状；零相位滤波（filtfilt）可消除相位失真 |
| IIR滤波器一定不稳定 | IIR是否稳定取决于极点位置；极点在单位圆内时可以稳定 |

## 7. 启发问题与创新拓展

**一个反例或边界问题**：
对心电 R 峰检测，如果用低通滤波器过度平滑，R 峰会被"抹掉"导致漏检。这说明滤波不是越平滑越好——需要根据任务目标选择合适的截止频率和滤波器类型。那么，有没有一种"自适应"的方法，能根据信号局部特征自动调整滤波强度？

**一个跨学科迁移场景**：
在地震学中，地震波信号包含不同频率的地壳运动信息。低频对应深层结构，高频对应浅层细节。用不同截止频率的滤波器"切分"地震信号，可以分别研究不同深度的地壳结构——这就是"多尺度分析"的思想。

**一个可做成课程项目的小想法**：
设计一个"滤波器选择指南"交互工具：给定一段含噪信号（如 ECG + 工频干扰），用户可以尝试不同类型（FIR/IIR）、不同阶数、不同截止频率的滤波器，实时观察滤波效果和 PSNR 变化，直观理解参数选择的权衡。
