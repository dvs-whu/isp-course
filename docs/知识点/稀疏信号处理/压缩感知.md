## 2. 通俗入口

**一个真实问题**：MRI扫描需要20分钟采集大量数据来重建一张图像。如果能只采集10%的数据就重建出完整图像，扫描时间可以从20分钟缩短到2分钟——这对儿童、急诊患者意义巨大。

**生活类比**：想象一幅拼图有1000块，但你知道这幅图90%的区域是纯蓝色天空（稀疏性），只有10%的区域有复杂图案。你不需要看所有1000块拼图就能还原整幅画——只需要看到关键的100块就够了。压缩感知的数学原理与此类似。

**学这个能解决什么**：MRI加速、单像素相机、雷达欠采样恢复、频谱感知——凡是"采样成本高但信号稀疏"的场景，都可以用压缩感知大幅降低采样量。

## 3. 严谨核心

**数学对象**：稀疏信号 $\mathbf{x} \in \mathbb{R}^N$（或 $\mathbf{s} \in \mathbb{R}^N$），测量矩阵 $\mathbf{\Phi} \in \mathbb{R}^{M \times N}$，$M \ll N$

**基本假设**：
- 信号在某个变换域下是 $K$-稀疏的（仅有 $K \ll N$ 个非零系数）
- 测量矩阵满足受限等距性质（RIP）或与稀疏基低相干

压缩感知（Compressed Sensing, CS）是一种利用信号的稀疏性，以远低于Nyquist采样定理要求的采样率对信号进行采集和恢复的理论框架。

核心思想：若信号 $\mathbf{x} \in \mathbb{R}^N$ 在某个变换域下是 $K$-稀疏的（即仅有 $K \ll N$ 个非零系数），则可通过 $M = O(K \log(N/K))$ 次线性测量精确恢复该信号。

**核心公式：**

信号观测模型：

$$\mathbf{y} = \mathbf{\Phi} \mathbf{x} = \mathbf{\Phi} \mathbf{\Psi} \mathbf{s} = \mathbf{A} \mathbf{s}$$

其中 $\mathbf{y} \in \mathbb{R}^M$ 为观测向量，$\mathbf{\Psi}$ 为稀疏基，$\mathbf{s}$ 为稀疏系数，$\mathbf{A} = \mathbf{\Phi}\mathbf{\Psi}$ 为感知矩阵。

$\ell_1$ 最小化恢复（Basis Pursuit）：

$$\min_{\mathbf{s}} \|\mathbf{s}\|_1 \quad \text{s.t.} \quad \mathbf{y} = \mathbf{A}\mathbf{s}$$

含噪情况下的Basis Pursuit Denoising（BPDN）：

$$\min_{\mathbf{s}} \|\mathbf{s}\|_1 \quad \text{s.t.} \quad \|\mathbf{y} - \mathbf{A}\mathbf{s}\|_2 \leq \epsilon$$

或LASSO形式：

$$\min_{\mathbf{s}} \frac{1}{2}\|\mathbf{y} - \mathbf{A}\mathbf{s}\|_2^2 + \lambda \|\mathbf{s}\|_1$$

RIP条件：存在常数 $\delta_K \in (0,1)$ 使得对所有 $K$-稀疏向量 $\mathbf{s}$：

$$(1-\delta_K)\|\mathbf{s}\|_2^2 \leq \|\mathbf{A}\mathbf{s}\|_2^2 \leq (1+\delta_K)\|\mathbf{s}\|_2^2$$

**适用条件和边界**：
- 信号必须在某变换域下稀疏或近似稀疏——这是CS的根本前提
- 测量数 $M \geq C \cdot K \log(N/K)$——理论上界，实际中常需更多测量
- 非稀疏信号（如白噪声）无法用CS恢复
- RIP验证本身是NP-hard问题，实践中用随机矩阵（高斯、伯努利）以高概率满足

## 4. 方法流程

**输入**：欠采样测量 $\mathbf{y} \in \mathbb{R}^M$，测量矩阵 $\mathbf{\Phi}$，稀疏基 $\mathbf{\Psi}$（或直接给感知矩阵 $\mathbf{A}$）

**处理步骤**：
1. 确定信号的稀疏变换域 $\mathbf{\Psi}$（如小波、DCT、傅里叶基）
2. 设计满足RIP的测量矩阵 $\mathbf{\Phi}$（随机高斯、随机伯努利、结构化采样等）
3. 采集欠采样测量 $\mathbf{y} = \mathbf{\Phi}\mathbf{x}$
4. 求解稀疏优化问题（Basis Pursuit / LASSO / OMP等）恢复 $\mathbf{s}^*$
5. 反变换得到原信号估计 $\hat{\mathbf{x}} = \mathbf{\Psi}\mathbf{s}^*$

**输出如何解释**：
- 恢复信号 $\hat{\mathbf{x}}$：在信号稀疏且测量足够时，$\hat{\mathbf{x}} \approx \mathbf{x}$
- 恢复误差界：$\|\hat{\mathbf{x}} - \mathbf{x}\|_2 \leq C \cdot \epsilon$（与噪声水平成正比）

**常见参数如何选择**：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 测量数 $M$ | $M \geq 4K \sim 5K$ | 经验法则，保证可靠恢复 |
| 正则化参数 $\lambda$ | 与噪声水平相关 | 噪声大→$\lambda$ 大 |
| 稀疏基选择 | 信号相关 | 时域稀疏→单位阵；频域稀疏→傅里叶基；图像→小波基 |

**压缩感知重构算法对比：**

| 算法类别 | 代表算法 | 计算复杂度 | 精度 | 所需测量数 | 适用场景 |
| ---------|---------|-----------|------|-----------|---------|
| 凸优化 | Basis Pursuit, LASSO | 高 | 最高 | M ≥ O(K log(N/K)) | 高精度重建 |
| 贪婪算法 | OMP, CoSaMP, SP | 中 | 较高 | M ≥ O(K log N) | 中等规模问题 |
| 近端梯度法 | ISTA, FISTA | 中 | 高 | M ≥ O(K log(N/K)) | 大规模稀疏问题 |
| 消息传递 | AMP, ADMM | 低-中 | 高 | 接近理论下限 | 大规模统计问题 |
| 贝叶斯方法 | BCS, SparseBayes | 高 | 高 | 自适应 | 需要不确定性估计 |

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt

# Compressed Sensing演示：从欠Sampling测量中恢复SparseSignal
np.random.seed(42)
N = 64    # 信号维度
M = 20    # 测量数（远小于N，欠采样）
K = 5     # 稀疏度（非零元素个数）

# 1. 生成K-稀疏信号
x_true = np.zeros(N)
support = np.random.choice(N, K, replace=False)
x_true[support] = np.random.randn(K) * 3

# 2. 随机测量Matrix（满足RIP性质的Probability较高）
Phi = np.random.randn(M, N) / np.sqrt(M)
y = Phi @ x_true  # 欠采样测量 y = Φx

# 3. OMP（正交Matching Pursuit）Algorithm恢复稀疏信号
def omp(Phi, y, K):
    """贪婪Algorithm：每次选最相关的Atom"""
    N = Phi.shape[1]
    x = np.zeros(N)
    residual = y.copy()
    support_set = []
    for k in range(K):
        correlations = np.abs(Phi.T @ residual)
        j = np.argmax(correlations)
        support_set.append(j)
        A_sub = Phi[:, support_set]
        x_sub = np.linalg.lstsq(A_sub, y, rcond=None)[0]
        residual = y - A_sub @ x_sub
    x[support_set] = x_sub
    return x, support_set

# 4. ISTA（Iterative收缩阈值Algorithm）恢复
def ista(Phi, y, N, lam=0.1, max_iter=200):
    """近端Gradient法：梯度下降 + 软阈值"""
    x = np.zeros(N)
    L = np.linalg.norm(Phi)**2
    step = 1.0 / L
    for _ in range(max_iter):
        grad = Phi.T @ (Phi @ x - y)
        x = np.sign(x - step*grad) * np.maximum(np.abs(x - step*grad) - lam*step, 0)
    return x

# 运行两种Algorithm
x_omp, supp_omp = omp(Phi, y, K)
x_ista = ista(Phi, y, N, lam=0.05)

err_omp = np.linalg.norm(x_omp - x_true) / np.linalg.norm(x_true)
err_ista = np.linalg.norm(x_ista - x_true) / np.linalg.norm(x_true)

fig, axes = plt.subplots(1, 3, figsize=(14, 4))
axes[0].stem(range(N), x_true, linefmt='b-', markerfmt='bo', basefmt='k-')
axes[0].set_title(f'真实SparseSignal (K={K})')
axes[0].set_xlabel('分量索引'); axes[0].set_ylabel('幅值')
axes[1].stem(range(N), x_omp, linefmt='r-', markerfmt='ro', basefmt='k-')
axes[1].set_title(f'OMP 恢复 (误差={err_omp:.4f})')
axes[1].set_xlabel('分量索引')
axes[2].stem(range(N), x_ista, linefmt='g-', markerfmt='go', basefmt='k-')
axes[2].set_title(f'ISTA 恢复 (误差={err_ista:.4f})')
axes[2].set_xlabel('分量索引')
plt.suptitle(f'Compressed Sensing：从 M={M} 次测量恢复 N={N} 维Signal', fontsize=13)
plt.tight_layout()
plt.show()
```

**结果解释**：
- 仅用 $M=20$ 次测量就恢复了 $N=64$ 维信号中的 $K=5$ 个非零分量——压缩比约3.2:1
- OMP（贪婪算法）和支持集匹配度高，恢复精度好
- ISTA（凸优化方法）恢复的系数可能有微小残余噪声分量，但整体结构正确
- 两种算法都从远少于信号维度的测量中成功恢复了稀疏信号——这就是压缩感知的核心价值

## 6. 常见误区

1. **压缩感知能压缩任意信号**：CS的前提是信号在某变换域稀疏。白噪声、全频带信号等不满足稀疏性假设，CS无法有效恢复。

2. **测量越多恢复越好**：超过一定阈值后，增加测量数的边际收益递减。关键是 $M$ 与 $K \log(N/K)$ 的关系，而非 $M$ 与 $N$ 的关系。

3. **$l_0$ 最小化和 $l_1$ 最小化等价**：只有在RIP条件满足时（$\delta_{2K} < \sqrt{2} - 1$），$l_1$ 最小化才能精确恢复 $K$-稀疏信号。条件不满足时两者结果可能不同。

4. **随机测量矩阵是唯一选择**：虽然高斯随机矩阵"通用"，但实际应用中常用结构化矩阵（如部分傅里叶矩阵用于MRI）。结构化矩阵可能需要更严格的条件才能满足RIP。

5. **CS等价于简单欠采样**：欠采样（如均匀丢弃采样点）通常会导致混叠伪影。CS通过随机采样+$l_1$ 优化避免了系统性混叠，这是本质区别。

## 7. 启发问题与创新拓展

**一个反例或边界问题**：
考虑信号 $\mathbf{x} = [1, 1, 1, \ldots, 1]^T \in \mathbb{R}^{100}$（所有分量都为1）。它在时域完全不稀疏（$K = N = 100$），但若用差分基 $\mathbf{\Psi}$ 表示（$\mathbf{s} = \mathbf{\Psi}^T \mathbf{x}$），它在差分域只有1个非零分量——$K = 1$。这说明：**稀疏性是相对于表示基而言的**，选对基是CS成功的关键。

**一个跨学科迁移场景**：
在射电天文学中，射电望远镜阵列（如SKA）的天线数量有限，相当于对天空图像做了欠采样。天体源在天空中是稀疏的（几颗亮点），因此压缩感知被用于从有限天线数据中高分辨率重建天空图像。这是CS从理论走向大规模科学设施的典型案例。

**一个可做成课程项目的小想法**：
"从20%采样重建MRI图像"——使用公开的MRI数据集（如fastMRI），模拟径向采样模式（只采集20%的k空间数据），分别用零填充、OMP和ISTA/FISTA三种方法重建图像，比较PSNR和视觉质量。探究不同采样模式（径向vs随机vs均匀欠采样）对重建质量的影响。
