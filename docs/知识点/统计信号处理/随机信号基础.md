## 2. 通俗入口

> [!tip] 🎯 一句话理解
> 随机信号虽然不确定，但它的"平均行为"是有规律的，我们用统计量来描述它。

**一个真实问题**：你用传感器测量室温，每次读数都略有不同（噪声）。但如果你连续测 1000 次取平均，结果会非常接近真实温度——这就是随机信号的"平均规律"。

**生活类比**：掷骰子每次结果不确定，但掷 1000 次后，每个面出现的概率都接近 1/6。随机信号也一样：单次波形不确定，但均值、方差、自相关等统计量可能非常稳定。

**学这个知识点能解决什么**：
- 为维纳滤波、卡尔曼滤波提供数学基础
- 用功率谱密度描述随机信号的频率特性
- 理解白噪声、有色噪声等常见噪声模型

> 💡 **一句话记忆法**：均值定电平，方差看波动，自相关测相似，平稳性保不变。

## 3. 严谨核心

### 数学对象

随机信号（随机过程）是时间的函数，其在任意时刻的取值是一个随机变量。与确定性信号不同，随机信号无法用精确的数学表达式描述，只能通过统计特性来刻画。

**基本概念：**
- **随机过程**：一族随机变量 $\{X(t), t \in T\}$，其中 $T$ 为时间索引集
- **平稳性**：宽平稳（WSS）要求均值为常数，自相关函数仅依赖于时间差；严平稳要求所有有限维分布与时间起点无关
- **各态历经性（遍历性）**：时间平均等于集合平均，即可用单次观测的长时间平均来估计统计量
- **自相关函数**：$R_X(t_1, t_2) = E[X(t_1)X^*(t_2)]$，描述随机过程在不同时刻取值的相关性
- **功率谱密度（PSD）**：自相关函数的傅里叶变换，描述随机信号功率在频域的分布

### 核心公式

**均值函数：**
$$\mu_X(t) = E[X(t)]$$

**自相关函数（宽平稳）：**
$$R_X(\tau) = E[X(t)X^*(t-\tau)]$$

**自协方差函数：**
$$C_X(\tau) = R_X(\tau) - |\mu_X|^2$$

**维纳-辛钦定理（Wiener-Khinchin）：**
$$S_X(f) = \int_{-\infty}^{\infty} R_X(\tau) e^{-j2\pi f\tau} d\tau$$
$$R_X(\tau) = \int_{-\infty}^{\infty} S_X(f) e^{j2\pi f\tau} df$$

**线性系统的输出统计特性：**
$$R_Y(\tau) = R_X(\tau) * h(\tau) * h^*(-\tau)$$
$$S_Y(f) = S_X(f) |H(f)|^2$$

**互相关与互功率谱：**
$$R_{XY}(\tau) = E[X(t)Y^*(t-\tau)]$$
$$S_{XY}(f) = H(f) S_X(f)$$

### 随机信号分析框架

![[assets/random_signal_framework.png]]

*图：随机信号分析框架。随机并不等于"没有规律"，而是从单次波形转向统计规律；均值、方差、自相关和功率谱都要放在平稳性等模型假设下理解。*

### 平稳性分类对比

| 类型 | 条件 | 实用性 | 判定难度 |
| ------|------|--------|----------|
| 严平稳 (SSS) | 所有有限维分布与时间起点无关 | 理论标准 | 极难，需无限数据 |
| 宽平稳 (WSS) | 均值为常数，自相关仅依赖 τ | 工程最常用 | 较易检验 |
| 各态历经 | 时间平均 = 集合平均 | 单次观测即可统计 | 需满足特定条件 |
| 非平稳 | 统计特性随时间变化 | 短时分析方法 | 需时频分析工具 |

### 适用条件和边界

- 宽平稳假设要求均值恒定、自相关仅依赖时延；语音、生理信号通常只在短时间内近似平稳
- 各态历经性允许用单次长时间观测估计统计量，但不是所有随机过程都满足
- 维纳-辛钦定理要求信号宽平稳且自相关函数绝对可积
- 高斯随机过程完全由均值和自相关函数刻画（二阶统计量即足够）

## 4. 方法流程

**输入**：随机信号的一次或多次观测样本 $x_1[n], x_2[n], \ldots$

**处理步骤**：
1. **平稳性检验**：检查信号是否近似宽平稳（分段均值和方差是否大致恒定）
2. **估计均值和方差**：$\hat{\mu} = \frac{1}{N}\sum x[n]$，$\hat{\sigma}^2 = \frac{1}{N}\sum (x[n]-\hat{\mu})^2$
3. **估计自相关函数**：$\hat{R}[k] = \frac{1}{N}\sum_{n=0}^{N-|k|-1} x[n]x[n+k]$
4. **估计功率谱密度**：对自相关做 FFT（间接法），或直接对信号做 $|X[k]|^2/N$（直接法/周期图法）
5. **分析与建模**：根据 PSD 形状判断噪声类型（白噪声 = 平坦谱，有色噪声 = 有峰值或衰减）

**输出解释**：
- 均值 = 信号的平均电平
- 方差 = 信号围绕均值的波动大小（功率）
- 自相关在零滞后处最大（等于功率），远离零时衰减 → 信号有"记忆"
- PSD 的形状反映信号的频率特性

**常见参数选择**：
- 估计样本长度 $N$：越大估计越准，但要求平稳性假设在 $N$ 内成立
- 自相关估计最大时延 $M$：通常取 $N/10$ 到 $N/4$
- 功率谱估计方法：周期图法（简单但方差大）、Welch 法（分段平均，降低方差）

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 生成高斯白噪声
N = 10000
np.random.seed(42)
x = np.random.randn(N)

# 计算统计量
print(f"样本均值: {np.mean(x):.4f} (理论值: 0)")
print(f"样本方差: {np.var(x):.4f} (理论值: 1)")

# 计算归一化自相关函数
max_lag = 100
acf = np.correlate(x, x, mode='full') / (N * np.var(x))
acf = acf[N-1-max_lag : N+max_lag]
lags = np.arange(-max_lag, max_lag+1)

fig, axes = plt.subplots(2, 1, figsize=(10, 6))
# 上图：直方图 vs 理论高斯 PDF
axes[0].hist(x, bins=50, density=True, alpha=0.7, label='样本直方图')
x_pdf = np.linspace(-4, 4, 200)
axes[0].plot(x_pdf, 1/np.sqrt(2*np.pi)*np.exp(-x_pdf**2/2), 'r-', label='理论高斯PDF')
axes[0].set_title('高斯白噪声的幅值分布')
axes[0].set_xlabel('幅值')
axes[0].set_ylabel('PDF')
axes[0].legend()

# 下图：自相关函数
axes[1].plot(lags, acf, 'b-')
axes[1].set_title('归一化自相关函数（零滞后为1，其余接近0→白噪声特征）')
axes[1].set_xlabel('时延 τ')
axes[1].set_ylabel('归一化自相关')
axes[1].grid(True)

plt.tight_layout()
plt.show()
```

**结果解释**：
- 直方图近似高斯分布，验证了大数定律下的分布逼近效果
- 自相关函数在零滞后处为 1，其余接近 0——这正是白噪声的特征：不同时刻的取值几乎不相关
- 样本均值接近 0，样本方差接近 1，验证了统计量估计的一致性

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| 随机信号就是没有规律 | 单次取值不确定，但分布、均值、相关和功率谱可能非常稳定 |
| 均值和方差能描述全部信息 | 均值、方差只是低阶统计量；非高斯信号还可能需要分布形状、高阶统计或模型结构 |
| 白噪声就是幅度很小的噪声 | "白"描述的是频谱近似平坦，不是幅度大小 |
| 平稳性可以默认成立 | 语音、生理信号、机械故障信号常常只在短时间内近似平稳 |
| 最优滤波在任何情况下都最优 | "最优"总是相对于模型假设、评价指标和噪声统计而言 |

## 7. 启发问题与创新拓展

**一个反例或边界问题**：
如果噪声是非平稳的（如时变方差），经典功率谱估计会得到什么？——估计结果是所有时间段的"平均"，无法反映时变特性。这时需要时频分析方法（如短时功率谱、小波分析）。

**一个跨学科迁移场景**：
在金融工程中，股票收益率的波动率（方差）是随时间变化的（波动率聚集现象）。GARCH 模型就是用自回归结构建模时变方差——这与信号处理中的非平稳信号分析思想一致。

**一个可做成课程项目的小想法**：
设计一个"噪声分类器"：给定一段未知噪声信号，通过分析其自相关函数和功率谱密度，判断它是白噪声、粉红噪声还是有色噪声，并估计其统计参数。
