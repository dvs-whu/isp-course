
# Python信号处理速修

> [!tip] 🎯 初学者简化说明
> 这部分是实验和项目的工具入口。目标不是系统学习Python，而是能把一段真实信号读进来、画出来、变换到频域、做简单滤波，并保存结果。
>
> **建议学习时间：** W1实验前后，60-90分钟  
> **本课程要求：** 会用即可，实验和项目会直接用到

---

## 一、必须掌握

| 能力 | 要会做什么 | 常用工具 |
| ------|------------|----------|
| 数组表示 | 把信号表示成一维、二维或三维数组 | `numpy.ndarray` |
| 时间轴和采样率 | 根据采样率生成时间轴，理解每个采样点对应的时间 | `np.arange`、`np.linspace` |
| 可视化 | 画时域波形、频谱、图像、语谱图 | `matplotlib` |
| FFT | 把时域信号变成频域，找到主要频率成分 | `np.fft` |
| 滤波 | 设计并应用低通、高通、带通滤波器 | `scipy.signal` |
| 数据读写 | 读取`.wav`、`.csv`、图片或NumPy数据 | `scipy.io.wavfile`、`pandas`、`imageio` |

## 三、画图先于建模

做任何算法前，先画三类图：

1. 时域波形：看幅值、趋势、突变和噪声。
2. 频谱图：看主要频率、噪声频带和周期结构。
3. 处理前后对比图：看算法是否真的改变了目标部分。

## 四、FFT最小模板


看频谱时要注意：

- 横轴单位是Hz。
- 频率分辨率约为`fs / N`。
- 只看幅度谱不够时，再看相位谱。
- 信号有噪声时，功率谱密度通常比直接FFT更稳定。

## 六、实验和项目中的最低要求

每个Notebook至少包含：

- 数据来源和采样率说明。
- 原始信号可视化。
- 关键参数说明，例如窗口长度、滤波截止频率、模型超参数。
- 处理前后对比图。
- 一段简短结论，说明算法有效在哪里、失败在哪里。
