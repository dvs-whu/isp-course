## 3. 严谨核心

### 定义与概念

图像重建（Image Reconstruction）是指：**从采集到的Primal测量数据，恢复出我们能看懂的图像**。

举个生活中的例子：
- 你去医院做**CT检查**，机器围着你转一圈，采集到的是各个角度的"投影"Data（不是直接的Image）。
- 然后通过数学计算，把Projection数据"还原"成一张清晰的身体断LayerImage。

> [!note] 关键区别
> 图像Reconstruction ≠ 图像增强/Denoising。增强是对已有图像做改善，而重建是从**非图像形式的PrimalData**生成图像。

### 核心概念详解

**1. Projection = 沿直线求和**

想象一块方糖里面藏着果酱。如果我们用X光从某个Angle照射：
- X光穿过果酱时会被吸收更多
- Detection器记录下每一束X光**沿途被吸收的总量**
- 这就是"投影"——沿一条直线，把路径上的值全部加起来

**换Angle照射**，就能得到不同方向的Projection。这就像你围着一个物体拍照，每张照片都是从不同角度看到的"压缩"信息。

**2. 傅里叶切片定理（通俗版）**

这个定理说了一件很神奇的事：

> 从某个Angle拍一张Projection，对它做傅里叶Transform，结果刚好是原Image二维傅里叶变换中**对应角度的一条线**。

也就是说：**每拍一个角度，就"拼上"频域中的一块拼图**。拍满所有Angle，频域的拼图就完整了，再做逆傅里叶Transform就能恢复原图。

**3. Filtering反Projection（FBP）——经典ReconstructionMethod**

FBP的思路可以用三个步骤概括：
- **投影**：从各角度采集"投影值"
- **Filtering**：用"斜坡Filter"修正Data（补偿FrequencyLoss，让Image更清晰）
- **反Projection**：把修正后的数据"涂"回Image对应位置，多个Angle叠加就得到最终图像

> [!info] 为什么需要Filtering？
> 如果直接把ProjectionData"涂"回去（不Filtering），Image会很模糊（像看到一团雾）。滤波器的作用就是把模糊去掉，让细节变清晰。

### 核心公式

**Radon变换（正向投影）：**

$$p(\theta, t) = \int_{-\infty}^{\infty} f(t\cos\theta - s\sin\theta, \; t\sin\theta + s\cos\theta) \, ds$$

**傅里叶切片定理：**

$$P_\theta(\omega) = F(\omega\cos\theta, \; \omega\sin\theta)$$

其中 $P_\theta$ 为角度 $\theta$ 投影的一维傅里叶变换，$F$ 为原图像的二维傅里叶变换。

**滤波反投影（FBP）：**

$$f(x,y) = \int_0^{\pi} \left[ \int_{-\infty}^{\infty} |\omega| P_\theta(\omega) e^{j2\pi\omega t} d\omega \right]_{t=x\cos\theta+y\sin\theta} d\theta$$

**迭代重建：**

$$\hat{f} = \arg\min_f \frac{1}{2}\|Af - p\|_2^2 + \lambda R(f)$$

找到一个图像 $f$，使得它经过"模拟Projection"后最接近真实测量Data $p$，同时Image本身看起来也不要太乱（$R(f)$ 是"平滑性"要求）。

**压缩感知重建：**

$$\hat{f} = \arg\min_f \|Af - p\|_2^2 + \lambda \|\Psi f\|_1$$

利用Image在小波等Transform下的**Sparse性**（大部分系数接近零），用更少的测量数据也能重建出好图像。

### 图像重建方法对比

| 方法 | 基本思路 | 计算复杂度 | 需要先验？ | 适用场景 | 优点 | 缺点 |
|------|---------|-----------|-----------|---------|------|------|
| 滤波反Projection(FBP) | 滤波 + 叠加投影 | 低 | 否 | 完整投影数据CT | 速度快，有解析公式 | 欠Sampling时伪影严重 |
| 代数重建(ART) | 解线性方程组（迭代） | 中 | 可选 | 有限角度CT | 可融入先验知识 | Convergence慢 |
| 统计迭代(SIRT) | 最小化误差（最小二乘） | 中-高 | 正则化 | PET/SPECT | 统计建模准确 | 计算量大 |
| 压缩感知重建 | 稀疏优化 | 高 | 稀疏性 | 欠采样MRI/CT | 大幅减少采样量 | 需满足稀疏条件 |
| 深度学习重建 | TrainingNeural Network直接映射 | 训练高/推理低 | 数据驱动 | 通用 | 速度快、效果好 | 泛化性待验证 |

## 4. 方法流程

**输入是什么：** 投影数据（sinogram）$p(\theta, t)$，其中 $\theta$ 为投影角度，$t$ 为探测器位置。

**处理步骤：**
1. **数据采集**：CT从多个角度采集X光投影；MRI从k空间不同轨迹采样
2. **预处理**：对数变换（CT）、校准、去伪影
3. **重建算法**：
   - 解析方法（FBP）：频域乘斜坡滤波器 → 空域反投影叠加
   - 迭代方法：定义目标函数 → 正则化 → 优化求解
   - 深度学习：训练端到端映射网络
4. **后处理**：窗宽窗位调整、去伪影、边缘增强
5. **质量评估**：与参考图像对比，计算PSNR/SSIM，目视检查伪影和细节

**输出如何解释：** CT重建输出Hounsfield单位（HU）的断层图像，不同组织有标准HU范围；MRI输出任意单位的对比度图像。

**常见参数如何选择：**
- FBP：投影角度数（通常360~1800）、探测器通道数
- 迭代重建：正则化强度 λ、迭代次数（50~200）
- 深度学习：网络架构（U-Net为主流）、训练数据量

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
from scipy.ndimage import zoom

# 创建简单的TestImage（模拟CT断Layer截面）
N = 128
img = np.zeros((N, N))
# 画几个圆形模拟器官截面
y, x = np.ogrid[:N, :N]
for (cy, cx, r, val) in [(50, 50, 30, 1.0), (40, 70, 12, 0.7),
                           (75, 45, 15, 0.5), (60, 80, 8, 0.8)]:
    mask = (x - cx)**2 + (y - cy)**2 <= r**2
    img[mask] = val

# RadonTransform模拟：从多个AngleProjection
n_angles = 180
angles = np.linspace(0, 180, n_angles, endpoint=False)
sinogram = np.zeros((N, n_angles))
center = N // 2
for i, angle in enumerate(angles):
    # 对每个角度旋转图像并求和（模拟投影）
    theta = np.deg2rad(angle)
    rot_x = np.cos(theta) * (x - center) + np.sin(theta) * (y - center)
    rot_y = -np.sin(theta) * (x - center) + np.cos(theta) * (y - center)
    # 双线性插值Sampling
    rx = np.clip(np.round(rot_x + center).astype(int), 0, N-1)
    ry = np.clip(np.round(rot_y + center).astype(int), 0, N-1)
    sinogram[:, i] = img[ry, rx].sum(axis=1)

# 反投影重建（无Filtering → 模糊，有斜坡滤波 → 清晰）
recon_bp = np.zeros((N, N))
recon_fbp = np.zeros((N, N))
# 斜坡滤波器
freqs = np.fft.fftfreq(N)
ramp = np.abs(freqs)

for i, angle in enumerate(angles):
    projection = sinogram[:, i]
    # 滤波反投影：频域乘斜坡滤波器
    proj_fft = np.fft.fft(projection)
    filtered = np.real(np.fft.ifft(proj_fft * ramp))
    # 反投影（涂回去）
    theta = np.deg2rad(angle)
    for row in range(N):
        offset = filtered[row]
        # 沿行方向反投影
        t = row - center
        px = (center + t * np.sin(theta) + np.arange(N) * np.cos(theta)).astype(int)
        py = (center - t * np.cos(theta) + np.arange(N) * np.sin(theta)).astype(int)
        valid = (px >= 0) & (px < N) & (py >= 0) & (py < N)
        recon_bp[py[valid], px[valid]] += projection[row] / n_angles
        recon_fbp[py[valid], px[valid]] += offset / n_angles

# 显示：原图、正弦图、反投影、滤波反投影
fig, axes = plt.subplots(1, 3, figsize=(12, 4))
axes[0].imshow(img, cmap='gray')
axes[0].set_title('PrimalImage（模拟CT截面）')
axes[1].imshow(sinogram, cmap='gray', aspect='auto')
axes[1].set_title('正弦图（ProjectionData）')
axes[1].set_xlabel('ProjectionAngle')
axes[1].set_ylabel('探测器位置')
axes[2].imshow(np.clip(recon_fbp, 0, None), cmap='gray')
axes[2].set_title('Filtering反ProjectionReconstruction结果')
for ax in axes: ax.axis('off')
plt.tight_layout()
plt.show()
```

**代码要点**：
- 正弦图（sinogram）就是投影数据的可视化——横轴是角度，纵轴是探测器位置
- 斜坡滤波器 $|\omega|$ 补偿了低频过多、高频不足的问题
- 不加滤波的反投影会很模糊（可注释掉滤波部分自行对比）

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| 反投影就能直接得到好图像 | 不加滤波的反投影结果非常模糊，必须配合斜坡滤波器 |
| 图像重建和图像去噪是一回事 | 去噪是从含噪图像恢复干净图像；重建是从非图像形式的测量数据生成图像 |
| 深度学习重建一定比FBP好 | 深度方法依赖训练数据分布，遇到未见过的成像条件可能产生幻觉伪影 |
| 投影角度越多重建越好 | 角度越多信息越完整，但实际中受限于辐射剂量（CT）或扫描时间（MRI） |

**公式误解：** 傅里叶切片定理不是说"投影的傅里叶变换等于原图"，而是等于原图二维傅里叶变换在**对应角度的一条径向线**上。

**参数误解：** 正则化参数 λ 不是越大越好——太大会过度平滑丢失细节，太小则噪声和伪影无法抑制。

**适用条件误解：** FBP要求投影数据在0°-180°范围内均匀完整采样；欠采样时FBP会产生严重伪影，需要迭代或深度学习方法。

## 7. 启发问题与创新拓展

**一个反例或边界问题：**
如果只有30个角度的投影数据（而非180个），FBP重建会怎样？——会出现明显的条纹伪影（star artifacts），但迭代方法通过引入图像的稀疏性先验，仍然可能获得可接受的结果。这说明先验知识是克服欠采样的关键。

**一个跨学科迁移场景：**
射电天文中的"事件视界望远镜"（EHT）拍摄黑洞M87*照片时，全球望远阵列只相当于地球上几个采样点，远不够完整重建——但利用图像的稀疏性和正则化先验，仍然重建出了人类第一张黑洞照片。这与医学CT欠采样重建的思路完全一致。
