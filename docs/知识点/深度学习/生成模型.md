## 2. 通俗入口

> [!tip] 🎯 初学者简化说明
> **生成模型是什么？** 教AI"创造"新Data——让它学会数据的规律，然后自己造出看起来真实的新Sample（比如从未存在过的人脸、从未听过的语音）。
>
> **三大主流类型：**
> - **VAE（变分自Coding器）**：先压缩再还原，学会"中间表示"
> - **GAN（生成对抗网络）**：造假者和侦探的对抗游戏
> - **DDPM（扩散Model）**：加Noise→Denoising声，当前最强生成Method
>
> **生活类比帮你理解：**
> - 🎨 **VAE** = 把一幅画压缩成一段文字描述（Coding），再根据描述画一幅新画（解码）。中间那段"描述"就是隐Variable $\mathbf{z}$。Training时既要画得像（Reconstruction），又不能描述得太离谱（KL正则）。
> - 🎭 **GAN** = 一个造假币的人（生成器 $G$）和一个验钞员（判别器 $D$）不断博弈。造假者越造越真，验钞员越验越精，最终假币以假乱真。
> - 📺 **DDPM** = 拿一张清晰照片，逐步加上电视雪花噪点直到变成纯噪声（前向过程），然后训练一个Neural Network学会一步步把噪点去掉还原照片（反向过程）。生成时从纯噪声开始去噪就行。

**典型应用：**
- 图像生成与编辑（Stable Diffusion、DALL·E）
- 数据增强（医学影像、语音数据扩充）
- 语音合成（WaveNet、扩散模型语音合成）
- 异常检测（基于似然的检测）
- Signal恢复（条件生成用于去噪、超分辨率）

## 3. 严谨核心

### 数学对象与基本假设

生成模型旨在学习数据的概率分布 $p(\mathbf{x})$ 或从隐变量 $\mathbf{z}$ 到数据 $\mathbf{x}$ 的映射，从而能够生成与训练数据分布相似的新Sample。深度生成模型是当前人工智能领域最活跃的研究方向之一。

主要类型：
- **变分自Coding器（VAE）**：基于变分推断，学习数据的隐变量表示，优化证据下界（ELBO）
- **生成对抗网络（GAN）**：通过生成器与判别器的对抗博弈学习数据分布
- **扩散模型（DDPM）**：通过逐步添加和去除噪声的过程学习数据分布，当前图像生成的主流方法
- **自Regression模型**：逐像素/逐token生成，如 PixelRNN、GPT 系列

### 核心公式

**VAE — 证据下界（ELBO）：**
$$
| \log p(\mathbf{x}) \geq \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x})}[\log p_\theta(\mathbf{x}|\mathbf{z})] - D_{\text{KL}}(q_\phi(\mathbf{z}|\mathbf{x}) \| p(\mathbf{z}))
$$
其中 $q_\phi$ 为编码器（近似后验），$p_\theta$ 为解码器（似然），$p(\mathbf{z}) = \mathcal{N}(\mathbf{0}, \mathbf{I})$ 为先验。

**GAN — 极小极大博弈：**
$$
\min_G \max_D \; \mathbb{E}_{\mathbf{x} \sim p_{\text{data}}}[\log D(\mathbf{x})] + \mathbb{E}_{\mathbf{z} \sim p_z}[\log(1 - D(G(\mathbf{z})))]
$$
其中 $G$ 为生成器，$D$ 为判别器。

**DDPM — 前向扩散过程：**
$$
q(\mathbf{x}_t | \mathbf{x}_{t-1}) = \mathcal{N}(\mathbf{x}_t; \sqrt{1-\beta_t}\mathbf{x}_{t-1}, \beta_t\mathbf{I})
$$

**DDPM — 反向去噪过程：**
$$
p_\theta(\mathbf{x}_{t-1}|\mathbf{x}_t) = \mathcal{N}(\mathbf{x}_{t-1}; \boldsymbol{\mu}_\theta(\mathbf{x}_t, t), \sigma_t^2\mathbf{I})
$$

**DDPM — 训练损失：**
$$
| \mathcal{L}_{\text{simple}} = \mathbb{E}_{t, \mathbf{x}_0, \boldsymbol{\epsilon}}\left[\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)\|^2\right]
$$
其中 $\boldsymbol{\epsilon}$ 为加入的噪声，$\boldsymbol{\epsilon}_\theta$ 为噪声Prediction网络。

### 适用条件和边界

- VAE：隐空间连续可插值，但生成质量偏模糊；假设后验为高斯分布可能过于简化
- GAN：生成质量高但训练不稳定，模式坍塌是核心问题
- DDPM：生成质量和多样性好，但采样速度慢（需数百步迭代）

## 4. 方法流程

### VAE / GAN / DDPM 生成模型对比

| 特性 | VAE | GAN | DDPM |
|------|-----|-----|------|
| 训练目标 | ELBO (重建+KL) | 极小极大博弈 | 噪声预测 MSE |
| 生成质量 | 较模糊 | 清晰但模式坍塌 | 高质量、多样性好 |
| 训练稳定性 | 稳定 | 不稳定（需平衡G/D） | 稳定 |
| 采样速度 | 快（单次前向） | 快（单次前向） | 慢（需数百步Iterative） |
| 似然Estimation | 有（ELBO下界） | 无 | 有（变分下界） |
| 隐空间 | 连续、可插值 | 无显式隐空间 | 无显式隐空间 |
| 代表应用 | 图像编辑、表示学习 | 图像生成、风格迁移 | 图像生成、语音合成 |

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt

# 真实Data：8个高斯分量组成的环形分布
np.random.seed(42)
n_real = 800
centers = 8
angles = np.linspace(0, 2*np.pi, centers, endpoint=False)
real_data = []
for i in range(n_real):
    k = i % centers
    cx, cy = 2*np.cos(angles[k]), 2*np.sin(angles[k])
    real_data.append([cx + 0.2*np.random.randn(), cy + 0.2*np.random.randn()])
real_data = np.array(real_data)

# 生成器：2维Noise → 2维数据
def generator(z, W1, b1, W2, b2):
    h = np.maximum(0, z @ W1.T + b1)  # ReLU
    return h @ W2.T + b2

# 判别器：2维数据 → 标量Probability
def discriminator(x, W1, b1, W2, b2):
    h = np.maximum(0, x @ W1.T + b1)  # ReLU
    sig = 1 / (1 + np.exp(-(h @ W2.T + b2)))  # Sigmoid
    return sig.flatten()

# 初始化参数
np.random.seed(0)
gW1 = np.random.randn(32, 2) * 0.5; gb1 = np.zeros(32)
gW2 = np.random.randn(2, 32) * 0.5; gb2 = np.zeros(2)
dW1 = np.random.randn(32, 2) * 0.5; db1 = np.zeros(32)
dW2 = np.random.randn(1, 32) * 0.5; db2 = np.zeros(1)

lr = 0.01
batch_size = 64
n_epochs = 300
d_losses, g_losses = [], []

for epoch in range(n_epochs):
    # ---- Training判别器 ----
    idx = np.random.choice(n_real, batch_size)
    real_batch = real_data[idx]
    z = np.random.randn(batch_size, 2)
    fake_batch = generator(z, gW1, gb1, gW2, gb2)
    
    # 判别器前向（真实数据）
    d_real = discriminator(real_batch, dW1, db1, dW2, db2)
    # 判别器前向（假数据）
    d_fake = discriminator(fake_batch, dW1, db1, dW2, db2)
    
    # 判别器Loss：-log(D(real)) - log(1-D(fake))
    d_loss = -np.mean(np.log(d_real + 1e-8) + np.log(1 - d_fake + 1e-8))
    d_losses.append(d_loss)
    
    # 简化GradientUpdate（数值差分近似）
    eps = 1e-4
    # 更新判别器参数（简化：只更新dW2）
    dW2_grad = np.zeros_like(dW2)
    for j in range(dW2.shape[1]):
        dW2_old = dW2[0, j]
        dW2[0, j] = dW2_old + eps
        d_real_p = discriminator(real_batch, dW1, db1, dW2, db2)
        d_fake_p = discriminator(fake_batch, dW1, db1, dW2, db2)
        loss_p = -np.mean(np.log(d_real_p + 1e-8) + np.log(1 - d_fake_p + 1e-8))
        dW2_grad[0, j] = (loss_p - d_loss) / eps
        dW2[0, j] = dW2_old
    dW2 -= lr * dW2_grad * 0.01  # 小步更新

    # ---- 训练生成器 ----
    z = np.random.randn(batch_size, 2)
    fake_batch = generator(z, gW1, gb1, gW2, gb2)
    d_fake = discriminator(fake_batch, dW1, db1, dW2, db2)
    g_loss = -np.mean(np.log(d_fake + 1e-8))
    g_losses.append(g_loss)
    
    # 生成器梯度（简化：更新gW2）
    gW2_grad = np.zeros_like(gW2)
    for j in range(gW2.shape[1]):
        for i in range(gW2.shape[0]):
            gW2_old = gW2[i, j]
            gW2[i, j] = gW2_old + eps
            fake_p = generator(z, gW1, gb1, gW2, gb2)
            d_p = discriminator(fake_p, dW1, db1, dW2, db2)
            loss_p = -np.mean(np.log(d_p + 1e-8))
            gW2_grad[i, j] = (loss_p - g_loss) / eps
            gW2[i, j] = gW2_old
    gW2 -= lr * gW2_grad * 0.01

# 生成最终Sample
z_test = np.random.randn(500, 2)
generated = generator(z_test, gW1, gb1, gW2, gb2)

# 绘图
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
axes[0].scatter(real_data[:, 0], real_data[:, 1], s=5, alpha=0.5, c='blue')
axes[0].set_title('真实Data（环形高斯混合）')
axes[0].set_xlim([-4, 4]); axes[0].set_ylim([-4, 4]); axes[0].set_aspect('equal')

axes[1].scatter(generated[:, 0], generated[:, 1], s=5, alpha=0.5, c='red')
axes[1].set_title('GAN 生成的Sample')
axes[1].set_xlim([-4, 4]); axes[1].set_ylim([-4, 4]); axes[1].set_aspect('equal')

axes[2].plot(d_losses, label='判别器Loss')
axes[2].plot(g_losses, label='生成器Loss')
axes[2].set_xlabel('Training轮次'); axes[2].set_ylabel('Loss')
axes[2].set_title('TrainingLoss曲线'); axes[2].legend()
plt.suptitle('GAN 生成对抗网络演示', fontsize=13)
plt.tight_layout()
plt.show()
```

> [!note] 代码说明
> - **生成器** $G$：Input随机Noise $\mathbf{z} \sim \mathcal{N}(0, I)$，Output 2D 坐标（假Data）
> - **判别器** $D$：输入 2D 坐标，输出"该点为真实数据的Probability"
> - Training过程中 $D$ 和 $G$ 交替Optimization，体现了 $\min_G \max_D$ 的对抗博弈
> - 真实Data为 8 个高斯分量组成的环形分布，训练后生成器Output将逼近该分布

## 6. 常见误区

**公式误解：**
- ELBO不是 $\log p(\mathbf{x})$ 本身，而是它的下界；最大化ELBO不等于最大化似然，但在实践中效果良好
- GAN的 $\min_G \max_D$ 不是"先训D再训G"的顺序问题，而是需要交替优化达到纳什均衡

**参数误解：**
- DDPM的噪声调度 $\{\beta_t\}$ 不是越线性越好，余弦调度在实践中通常优于线性调度
- GAN的判别器不能太强也不能太弱——太强导致生成器梯度消失，太弱导致生成质量低

**适用条件误解：**
- 生成模型不能"无中生有"——它只能生成训练数据分布内的样本，不能生成训练集中完全不存在的概念
- VAE的隐空间不是"语义解耦"的——除非使用特殊的正则化技巧（如β-VAE），否则各维度可能高度耦合

**单位误解：**
- DDPM的步数 $T$ 不是越大越好，过大的 $T$ 只增加计算量而不提升质量；实践中 $T=1000$ 已足够

## 7. 启发问题与创新拓展

**一个反例或边界问题：**
GAN的模式坍塌（Mode Collapse）：生成器可能只学会生成训练数据中的某几个模式，而忽略其他模式。例如，用8个高斯分量训练GAN，生成器可能只生成其中2-3个分量。这揭示了对抗训练的一个根本性缺陷——生成器找到了"骗过"判别器的捷径，而非真正学习了数据分布。

**一个跨学科迁移场景：**
药物分子设计：生成模型可以学习已知药物分子的化学结构分布，然后生成具有类似性质的新分子。VAE的连续隐空间特别适合这种场景——可以在已知有效分子之间插值，探索化学空间中未被发现的候选药物。

**一个可做成课程项目的小想法：**
在项目二中，用GAN或VAE对信号数据做数据增强：先在小数据集上训练生成模型，然后用生成的合成数据扩充训练集，观察分类器的性能提升。对比"不做增强"、"简单噪声增强"和"生成模型增强"三种方案的效果差异。
