## 2. 通俗入口

> [!tip] 🎯 初学者简化说明
> **一句话理解**：神经网络就是一堆"Function套函数"，通过不断调整Parameter让Output逼近正确答案。
> 
> **你需要重点掌握的：**
> 1. Neural Network = 多Layer函数组合，每层做一次"Weighted求和 + 激活"
> 2. Backward Propagation = 链式法则，从Output往回算每个Parameter该调多少
> 3. CNN = Convolution（提取LocalFeature）+ Pooling（压缩）+ Fully Connected（Classification）
> 4. 激活函数的作用是引入非线性，没有它网络再深也只等于一Layer
> 
> **作为进阶了解的（期末不要求推导）：**
> - Gradient消失/爆炸的理论分析
> - 各种Optimization器（Adam等）的完整数学推导
> 
> 💡 **一句话记忆法**：前向算结果，反向调参数，CNN抓特征，激活加非线性。

**典型应用：**
- Image分类与目标Detection（CNN）

- SignalDenoising与增强（自Coding器、展开网络）

## 3. 严谨核心

### 数学对象与基本假设

深度学习是机器学习的一个分支，通过多层非线性Transform的神经网络Model从数据中自动学习层次化的特征表示。与传统机器学习依赖手工特征不同，深度学习能够从Primal数据中逐层提取从低级到高级的抽象特征。

核心要素：
- **神经网络**：由输入层、多个隐藏层和输出层组成的参数化函数逼近器
- **前向传播**：输入数据逐层计算得到输出的过程
- **反向传播**：利用链式法则计算损失函数对各层参数梯度的算法
- **激活函数**：引入非线性变换，如 ReLU、Sigmoid、Tanh 等
- **优化器**：梯度下降及其变体（SGD、Adam 等）用于参数Update

### 核心公式

**前向传播（单层）：**
$$
\mathbf{h} = \sigma(\mathbf{W}\mathbf{x} + \mathbf{b})
$$
其中 $\mathbf{W}$ 为WeightsMatrix，$\mathbf{b}$ 为Bias向量，$\sigma$ 为激活函数。

**损失函数（Mean Squared Error）：**
$$
\mathcal{L} = \frac{1}{N}\sum_{i=1}^{N} \| \hat{\mathbf{y}}_i - \mathbf{y}_i \|^2
$$

**反向传播（链式法则）：**
$$
\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(l)}} = \frac{\partial \mathcal{L}}{\partial \mathbf{h}^{(l)}} \cdot \frac{\partial \mathbf{h}^{(l)}}{\partial \mathbf{W}^{(l)}}
$$

**SGD 参数更新：**
$$
\mathbf{W} \leftarrow \mathbf{W} - \eta \frac{\partial \mathcal{L}}{\partial \mathbf{W}}
$$
其中 $\eta$ 为学习率。

### 常用激活函数对比

| 激活函数 | 公式 | 值域 | 优点 | 缺点 |
|---------|------|------|------|------|
| Sigmoid | $\sigma(x)=\frac{1}{1+e^{-x}}$ | (0,1) | 输出有概率意义 | 梯度消失、非零中心 |
| Tanh | $\tanh(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | (-1,1) | 零中心输出 | 梯度消失 |
| ReLU | $\max(0,x)$ | [0,∞) | 计算简单、缓解梯度消失 | 神经元死亡 |
| Leaky ReLU | $\max(\alpha x, x)$ | (-∞,∞) | 避免神经元死亡 | 需选超参 α |
| GELU | $x\cdot\Phi(x)$ | (-∞,∞) | 平滑非线性、Transformer常用 | 计算稍复杂 |

### 适用条件和边界

- 万能近似定理保证单隐层MLP可以逼近任意连续函数，但不保证可训练性
- 深度模型需要足够的数据量和正则化，否则容易过拟合
- 梯度消失/爆炸限制了网络的有效深度，需要ReLU、残差连接等技术缓解

## 4. 方法流程

**输入是什么：** 标注数据集 $\{(\mathbf{x}_i, \mathbf{y}_i)\}_{i=1}^{N}$，划分为训练集、验证集和测试集。

**处理步骤：**
1. 定义任务（分类/回归/生成）和评价指标
2. 设计网络结构（MLP/CNN/RNN/Transformer）
3. 选择损失函数和优化器
4. 前向传播计算输出 → 计算损失 → 反向传播计算梯度 → 参数更新
5. 验证集监控过拟合，早停或调整超参数

**输出如何解释：** 分类任务输出概率分布，回归任务输出连续值，生成任务输出新样本。

**常见参数如何选择：**
- 学习率 $\eta$：通常从 0.001 开始，配合学习率调度
- Batch size：32-256，越大越稳定但需更多内存
- 隐藏层宽度/深度：从简单模型开始逐步增加
- 正则化：Dropout (0.1-0.5)、权重衰减 (1e-4 ~ 1e-2)

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt

# 生成二ClassificationData（月牙形，非线性可分）
np.random.seed(42)
n_samples = 200
# 类别0：上半圆
theta0 = np.random.uniform(0, np.pi, n_samples//2)
X0 = np.column_stack([np.cos(theta0), np.sin(theta0)]) + 0.1*np.random.randn(n_samples//2, 2)
# 类别1：下半圆（偏移）
theta1 = np.random.uniform(np.pi, 2*np.pi, n_samples//2)
X1 = np.column_stack([np.cos(theta1), np.sin(theta1)]) + np.array([0.5, -0.5]) + 0.1*np.random.randn(n_samples//2, 2)
X = np.vstack([X0, X1])
y = np.array([0]*(n_samples//2) + [1]*(n_samples//2))

# 打乱数据
idx = np.random.permutation(n_samples)
X, y = X[idx], y[idx]

# 两LayerMLP定义
input_dim, hidden_dim, output_dim = 2, 16, 1
# 初始化Weights
W1 = np.random.randn(hidden_dim, input_dim) * 0.5
b1 = np.zeros((hidden_dim, 1))
W2 = np.random.randn(output_dim, hidden_dim) * 0.5
b2 = np.zeros((output_dim, 1))

def relu(z):
    return np.maximum(0, z)

def relu_grad(z):
    return (z > 0).astype(float)

def sigmoid(z):
    return 1 / (1 + np.exp(-np.clip(z, -500, 500)))

# Training
lr = 0.1
losses = []
for epoch in range(200):
    # Forward Propagation
    X_T = X.T  # (2, N)
    z1 = W1 @ X_T + b1       # 隐藏层线性Transform
    a1 = relu(z1)             # 隐藏层激活
    z2 = W2 @ a1 + b2         # Output层
    a2 = sigmoid(z2)          # 输出Probability
    # 交叉熵Loss
    eps = 1e-8
    loss = -np.mean(y*np.log(a2+eps) + (1-y)*np.log(1-a2+eps))
    losses.append(loss)
    # Backward Propagation
    m = n_samples
    dz2 = a2 - y.reshape(1, -1)         # 输出层Gradient
    dW2 = (1/m) * dz2 @ a1.T
    db2 = (1/m) * np.sum(dz2, axis=1, keepdims=True)
    da1 = W2.T @ dz2
    dz1 = da1 * relu_grad(z1)
    dW1 = (1/m) * dz1 @ X_T.T
    db1 = (1/m) * np.sum(dz1, axis=1, keepdims=True)
    # 参数Update
    W2 -= lr * dW2; b2 -= lr * db2
    W1 -= lr * dW1; b1 -= lr * db1

# 绘图
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
# 损失曲线
axes[0].plot(losses, 'b-')
axes[0].set_xlabel('Training轮次'); axes[0].set_ylabel('交叉熵Loss')
axes[0].set_title('TrainingLoss曲线')

# 决策边界
xx, yy = np.meshgrid(np.linspace(-2, 3, 200), np.linspace(-2, 2, 200))
grid = np.column_stack([xx.ravel(), yy.ravel()])
z1_g = relu(W1 @ grid.T + b1)
z2_g = sigmoid(W2 @ z1_g + b2)
zz = z2_g.reshape(xx.shape)
axes[1].contourf(xx, yy, zz, levels=50, cmap='RdBu_r', alpha=0.6)
axes[1].scatter(X[y==0, 0], X[y==0, 1], c='red', s=15, label='Class0')
axes[1].scatter(X[y==1, 0], X[y==1, 1], c='blue', s=15, label='Class1')
axes[1].set_title('决策边界'); axes[1].legend()

# 隐藏Layer激活可视化
z1_vis = relu(W1 @ X.T + b1)
axes[2].imshow(z1_vis[:, :50], aspect='auto', cmap='viridis')
axes[2].set_xlabel('Sample'); axes[2].set_ylabel('隐藏神经元')
axes[2].set_title('隐藏Layer激活模式')
plt.suptitle('两LayerMLP手写数字Classification（NumPy实现）', fontsize=13)
plt.tight_layout()
plt.show()
```

**结果解释：**
- 左图：损失曲线单调下降，说明模型在学习
- 中图：决策边界将两类月牙形数据分开，说明MLP能处理非线性问题
- 右图：隐藏层神经元激活模式，不同神经元对不同区域敏感

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| 深度学习就是把Data丢进Model | 任务定义、数据清洗、划分方式、baseline和评价指标往往比模型结构更决定结果可信度 |
| 训练准确率高就说明模型好 | 训练集好只能说明模型拟合了训练数据，还必须看Validation集、测试集和失败Sample |
| 网络越深越好 | 更深模型需要更多数据、正则化和计算资源，也更难解释 |
| AI方法可以不做经典baseline | 没有baseline就无法判断模型是否真的超过简单Filtering或传统Feature方法 |

**公式误解：** 反向传播不是"从后往前算一遍"，而是链式法则的系统应用，每一层的梯度依赖于后面所有层的梯度。

**参数误解：** 学习率不是越大越好，太大会导致震荡不收敛，太小会训练极慢；batch size不是越大越好，过大会降低泛化能力。

**适用条件误解：** 深度学习不是万能的——数据量小、信噪比高、问题结构明确时，经典方法往往更可靠。

## 7. 启发问题与创新拓展

**一个反例或边界问题：**
如果Training集和Test集来自同一段连续信号切片（如相邻时间窗口），会造成数据泄漏——模型可能学到的是时间相关性而非真实特征，导致测试集上虚高。

**一个跨学科迁移场景：**
医学影像诊断中，深度学习模型可能学到的是"拍摄设备型号"而非"病灶特征"——因为不同医院使用不同设备，而疾病标签与医院高度相关。这提醒我们：模型学到的"特征"可能不是我们期望的物理规律。

**一个可做成课程项目的小想法：**
在项目二中，设计一个"渐进式baseline"实验：先用均值滤波去噪 → 再用维纳滤波 → 最后用简单CNN，对比三者的PSNR/SSIM，观察AI方法在什么条件下真正超越经典方法。这个实验能让学生理解"baseline"的真正意义。

**其他启发问题：**
- 为什么过拟合模型在项目展示中可能看起来很"聪明"？
- 大二课程项目中什么时候应该选择简单MLP或CNN，而不是Transformer？
- ImageDenoising、语音增强或传感器分类中，最小baseline应如何设计？
