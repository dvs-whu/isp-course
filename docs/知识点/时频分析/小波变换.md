## 2. 通俗入口

想象你拿着一个"变焦镜头"看信号——看大趋势时用广角（低频、大尺度），看细节时用长焦（高频、小尺度）。小波变换就是同时用各种焦距拍一组照片，再把信息合在一起。

**一个真实问题**：你收到一段地震波信号，里面既有缓慢的地壳运动（低频），又有瞬间的岩石断裂冲击（高频突变）。用傅里叶变换只能看到全局频率成分，丢失了时间信息；用 STFT 虽然有时间信息，但窗长固定——看低频时时间分辨率太差，看高频时频率分辨率太差。

小波变换的解决方案：
- 分析**低频**时用**长窗**（大尺度）→ 频率分辨率高
- 分析**高频**时用**短窗**（小尺度）→ 时间分辨率高
- 这种"自适应变焦"特性使小波特别适合分析多尺度信号

**生活类比**：

| 类比对象 | 小波变换对应 | 说明 |
|---------|------------|------|
| 广角镜头（拍全景） | 大尺度 $a$（低频） | 看清信号的整体趋势，但细节模糊 |
| 长焦镜头（拍特写） | 小尺度 $a$（高频） | 捕捉信号的局部跳变/突变，但看不到全局 |
| 移动镜头对准不同位置 | 平移参数 $b$ | 在不同时间位置"观察"信号 |
| 同时用多种焦距拍摄 | 多尺度分解 | 一次获得从粗到细的全部信息 |

**为什么 STFT 不如小波？** STFT 像一个固定焦距的相机——远近都用同样的分辨率。小波则像一台自适应变焦相机：远处用广角（低频大窗），近处用长焦（高频小窗），在每个尺度上都获得最合适的分辨率。

> 💡 **一句话记忆法**：小波变换 = 信号界的"自适应变焦相机"，低频用广角、高频用长焦，一次拍出完整时频信息。

## 3. 严谨核心

### 定义与概念

小波变换（Wavelet Transform, WT）是一种多尺度时频分析方法。与 STFT 使用固定窗长不同，小波变换通过将信号与一族由"母小波"经平移和缩放生成的基函数进行内积运算，实现对信号的时频联合分析。

小波变换的核心优势在于其**自适应分辨率特性**：分析高频成分时使用短的时间窗（高时间分辨率、低频率分辨率），分析低频成分时使用长的时间窗（低时间分辨率、高频率分辨率）。这种"变焦"特性使小波变换特别适合分析具有多尺度特征的非平稳信号。

小波变换分为两种主要形式：
- **连续小波变换（CWT）**：用于信号分析与可视化，具有高度冗余性
- **离散小波变换（DWT）**：通过二进采样实现高效计算，广泛用于信号压缩与去噪

多分辨率分析（MRA）理论为 DWT 提供了严格的数学框架和快速算法（Mallat 算法）。

### 核心公式

**连续小波变换（CWT）：**

$$
W(a, b) = \frac{1}{\sqrt{a}} \int_{-\infty}^{+\infty} x(t)\, \psi^*\!\left(\frac{t - b}{a}\right) dt
$$

其中 $a > 0$ 为尺度参数，$b$ 为平移参数，$\psi(t)$ 为母小波函数，$^*$ 表示共轭。

**母小波的容许条件：**

$$
C_\psi = \int_0^{+\infty} \frac{|\Psi(f)|^2}{f}\, df < \infty
$$

其中 $\Psi(f)$ 为母小波 $\psi(t)$ 的傅里叶变换。此条件保证了小波变换的可逆性。

**逆连续小波变换：**

$$
x(t) = \frac{1}{C_\psi} \int_0^{+\infty} \int_{-\infty}^{+\infty} W(a, b)\, \frac{1}{\sqrt{a}}\, \psi\!\left(\frac{t - b}{a}\right)\, db\, \frac{da}{a^2}
$$

**离散小波变换（DWT）——Mallat 递推算法：**

分解（低通与高通滤波）：

$$
c_j[k] = \sum_n h[n - 2k]\, c_{j+1}[n]
$$

$$
d_j[k] = \sum_n g[n - 2k]\, c_{j+1}[n]
$$

重建：

$$
c_{j+1}[n] = \sum_k \tilde{h}[n - 2k]\, c_j[k] + \sum_k \tilde{g}[n - 2k]\, d_j[k]
$$

其中 $c_j$ 为第 $j$ 层近似系数，$d_j$ 为第 $j$ 层细节系数，$h[n]$、$g[n]$ 分别为低通和高通分解滤波器，$\tilde{h}[n]$、$\tilde{g}[n]$ 为对应的重建滤波器。

### 常用母小波对比

| 母小波 | 正交性 | 紧支撑 | 对称性 | 典型应用 |
| --------|-------|--------|-------|---------|
| Haar | ✓ | ✓ | ✓ | 简单去噪、教学示例 |
| Daubechies (dbN) | ✓ | ✓ | ✗ | 通用信号处理、压缩 |
| Symlets (symN) | ✓ | ✓ | 近似对称 | 图像处理 |
| Coiflets (coifN) | ✓ | ✓ | 近似对称 | 数值分析 |
| Morlet | ✗ | ✗ | ✓ | 连续小波变换、时频分析 |
| Mexican Hat | ✗ | ✗ | ✓ | 边缘检测 |
| Biorthogonal | ✗ | ✓ | ✓ | JPEG 2000 图像压缩 |

### 🔢 Haar 小波数值算例（必考）

Haar 小波是最简单的小波，滤波器为：
- 低通分解：$h[n] = \left[\frac{1}{\sqrt{2}},\ \frac{1}{\sqrt{2}}\right]$
- 高通分解：$g[n] = \left[\frac{1}{\sqrt{2}},\ -\frac{1}{\sqrt{2}}\right]$

**手算演示**：对信号 $x = [4,\ 6,\ 10,\ 12,\ 8,\ 6,\ 5,\ 5]$ 做一层 DWT 分解。

第1步——近似系数（低通 + 下采样）：

$$c_1[0] = \frac{1}{\sqrt{2}}(4+6) = \frac{10}{\sqrt{2}} \approx 7.07, \quad c_1[1] = \frac{10+12}{\sqrt{2}} \approx 15.56$$
$$c_1[2] = \frac{8+6}{\sqrt{2}} \approx 9.90, \quad c_1[3] = \frac{5+5}{\sqrt{2}} \approx 7.07$$

第2步——细节系数（高通 + 下采样）：

$$d_1[0] = \frac{4-6}{\sqrt{2}} \approx -1.41, \quad d_1[1] = \frac{10-12}{\sqrt{2}} \approx -1.41$$
$$d_1[2] = \frac{8-6}{\sqrt{2}} \approx 1.41, \quad d_1[3] = \frac{5-5}{\sqrt{2}} = 0$$

**解读**：$c_1 \approx [7.07,\ 15.56,\ 9.90,\ 7.07]$ 是信号的"大趋势"（低频），$d_1 \approx [-1.41,\ -1.41,\ 1.41,\ 0]$ 是"变化快慢"（高频细节）。$d_1[3]=0$ 说明最后两个采样点没有跳变——这正是小波能捕捉信号"突变位置"的体现。

> 💡 **考试提示**：给定信号和 Haar 滤波器，要求写出一层 DWT 分解的近似/细节系数——这是最常见的计算题形式。

### 典型应用

- **信号去噪**：利用小波阈值去噪（Donoho 方法），对含噪信号进行小波分解，对细节系数施加软/硬阈值处理后重建
- **图像压缩**：JPEG 2000 标准采用小波变换（CDF 9/7 小波）替代 DCT，实现更好的压缩性能
- **边缘检测与特征提取**：利用小波模极大值检测信号中的奇异点与边缘
- **故障诊断**：轴承、齿轮等机械部件故障信号的瞬态冲击特征提取
- **医学信号处理**：ECG（心电图）、EEG（脑电图）中的异常波形检测
- **地震信号分析**：地下结构的多尺度特征识别
- **数值求解**：偏微分方程的自适应小波 Galerkin 方法

## 4. 方法流程

### DWT 多分辨率分解结构（Mallat 算法）

**输入**：一维信号 $x[n]$，长度为 $N$

**Step 1：选择母小波**
- 根据应用选择合适的小波基（如 Haar、db4、sym8 等）
- 不同小波有不同的支撑长度、对称性和正则性

**Step 2：多层分解**
- 每一层将信号分为低频近似 $c_j$ 和高频细节 $d_j$
- 低通滤波器 $h[n]$ → 下采样 → 近似系数
- 高通滤波器 $g[n]$ → 下采样 → 细节系数
- 对近似系数递归分解，直到达到目标层数

**Step 3：阈值处理（去噪场景）**
- 对细节系数施加阈值：
  - 硬阈值：$d_j[k] = d_j[k]$ if $|d_j[k]| > \lambda$，else 0
  - 软阈值：$d_j[k] = \text{sign}(d_j[k]) \cdot \max(|d_j[k]| - \lambda, 0)$
- 阈值 $\lambda$ 的选择：$\lambda = \sigma\sqrt{2\ln N}$（VisuShrink）

**Step 4：重建**
- 从处理后的系数逆向重建信号
- 上采样 → 重建滤波器 → 求和

**输出**：分解系数 $\{c_J, d_J, d_{J-1}, \ldots, d_1\}$ 或去噪后的重建信号

**参数选择建议**：
- **分解层数**：通常 3~5 层，层数过多会引入边界效应
- **小波基选择**：去噪常用 db4/sym8；压缩常用 CDF 9/7；教学常用 Haar
- **阈值方法**：软阈值更平滑，硬阈值保留更多细节

## 5. Python 最小实验

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 使用 Haar 小波手动实现 DWT 分解（不依赖 pywt）
def haar_dwt_one_level(x):
    """Haar 小波一层分解：低通+下采样得到近似，高通+下采样得到细节"""
    approx = (x[0::2] + x[1::2]) / np.sqrt(2)  # 近似系数（低频趋势）
    detail = (x[0::2] - x[1::2]) / np.sqrt(2)  # 细节系数（高频变化）
    return approx, detail

# 生成测试信号：低频 + 高频 + 突变
N = 256
t = np.linspace(0, 1, N)
x = np.sin(2*np.pi*5*t) + 0.5*np.sin(2*np.pi*50*t)
x[120:130] += 3  # 在第120-130个样本处添加突变

# 三层 Haar DWT 分解
c = x.copy()
levels = 3
details = []
for _ in range(levels):
    c, d = haar_dwt_one_level(c)
    details.append(d)

fig, axes = plt.subplots(levels + 2, 1, figsize=(10, 10))
axes[0].plot(x, 'b-')
axes[0].set_title('Original (Low-freq + High-freq + Burst)')

for i, d in enumerate(details):
    axes[i+1].plot(d, 'r-')
    axes[i+1].set_title(f'细节系数 d{i+1}（第{i+1}层高频信息）')

axes[levels+1].plot(c, 'g-')
axes[levels+1].set_title(f'近似系数 c{levels}（第{levels}层低频趋势）')
axes[levels+1].set_xlabel('样本序号')

plt.tight_layout()
plt.show()
```

**结果解释**：上图展示了三层 Haar DWT 分解。原始信号由 5Hz 低频、50Hz 高频和一个突变组成。d1 捕获最高频细节（50Hz 振荡和突变位置），d2 捕获中频信息，d3 和 c3 保留低频趋势。注意 d1 中突变位置（样本 120-130）处有明显的尖峰——这正是小波的"多尺度显微镜"能力。

## 6. 常见误区

| 误区 | 正确理解 |
|------|----------|
| 小波变换和 STFT 本质相同 | STFT 使用固定窗长，时频分辨率恒定；小波变换使用可变窗长，高频高时间分辨率、低频高频率分辨率 |
| 小波基的选择不重要 | 不同小波基的性质差异很大（正交性、紧支撑、对称性），选择不当会严重影响分解效果 |
| 分解层数越多越好 | 层数过多会引入边界效应，且后面的层包含的信息越来越少；通常 3~5 层足够 |
| CWT 和 DWT 只是离散化的关系 | CWT 是连续尺度/平移的冗余表示，DWT 是二进采样的正交表示，两者在理论和应用上有本质区别 |
| 小波去噪的阈值越大越好 | 阈值过大信号会被过度平滑（丢失细节），过小噪声残留；需要根据噪声水平自适应选择 |
| 小波变换只能处理一维信号 | 小波变换可以推广到二维（图像）、更高维，如 JPEG 2000 使用二维小波变换 |
| Haar 小波太简单没有实用价值 | Haar 小波虽然简单，但在边缘检测、快速计算和教学中仍有重要价值；且它是唯一同时满足正交、紧支撑、对称的小波 |
