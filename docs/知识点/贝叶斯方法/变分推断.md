## 2. 通俗入口

想象你面前有一个形状极其复杂的山谷（真实的后验分布），你想知道这个山谷的形状，但它的数学表达式太复杂了，没法直接算。你的策略是：拿一个形状简单的碗（比如高斯分布），把它放到山谷上，调整碗的位置和大小，让它尽可能贴合山谷的形状。虽然碗不能完美匹配山谷的每一个凹凸，但它给出了一个"最好的简单近似"。

这就是变分推断的核心思想：
- **变分**：在一组简单的候选分布中搜索最优的那个
- **推断**：目标是逼近真实的后验分布
- **KL散度**：衡量两个分布之间的"距离"——我们要让近似分布和真实后验尽可能接近

**为什么需要变分推断？** 贝叶斯推断中的后验 $p(\theta|\mathcal{D}) = \frac{p(\mathcal{D}|\theta)p(\theta)}{p(\mathcal{D})}$ 需要计算分母（证据）$p(\mathcal{D}) = \int p(\mathcal{D}|\theta)p(\theta)d\theta$，这个积分在高维空间中几乎不可能解析计算。MCMC 虽然能近似，但速度慢。变分推断用优化替代采样，速度快得多。

**生活类比**：

| 类比对象 | 变分推断对应 | 说明 |
|---------|------------|------|
| 用简单形状近似复杂形状 | 用简单分布族 $\mathcal{Q}$ 近似复杂后验 | 碗不能完美贴合山谷，但足够好 |
| 调整碗的位置和角度 | 优化变分参数 | 通过梯度下降找到最佳拟合 |
| 衡量贴合程度 | KL 散度 | 越小说明近似越准确 |
| 选择碗的形状 | 选择分布族（高斯、均值场等） | 碗越灵活近似越好，但计算越复杂 |

> 💡 **一句话记忆法**：变分推断 = 用"最优的简单分布"去近似"复杂的后验"，把不可解的积分变成可解的优化。

## 3. 严谨核心

### 定义与概念

变分推断（Variational Inference, VI）是一种将后验推断转化为优化问题的近似推断方法。其核心思想是：在预设的简单分布族 $\mathcal{Q}$ 中，寻找与真实后验 $p(\theta|\mathcal{D})$ 最接近的分布 $q^*(\theta)$，"接近"由 KL 散度衡量。

与 MCMC 相比，变分推断通常更快（Optimization vs Sampling），但可能引入近似偏差。常见策略包括：
- **均值场近似**（Mean-Field）：假设 $q(\theta) = \prod_i q_i(\theta_i)$，各因子独立
- **结构化变分推断**：保留部分变量间的依赖关系
- **黑盒变分推断**（BBVI）：利用蒙特卡洛梯度估计，适用于任意模型

### 核心公式

**目标**：最小化 KL 散度

$$
q^* = \arg\min_{q \in \mathcal{Q}} \text{KL}(q(\theta)\,\|\,p(\theta|\mathcal{D}))
$$

等价于最大化 **证据下界（ELBO）**：

$$
\text{ELBO}(q) = \mathbb{E}_{q(\theta)}\!\left[\log p(\mathcal{D}|\theta)\right] - \text{KL}(q(\theta)\,\|\,p(\theta))
$$

均值场更新（坐标上升）：对第 $j$ 个分量，

$$
\log q_j^*(\theta_j) = \mathbb{E}_{q_{-j}}\!\left[\log p(\mathcal{D},\theta)\right] + \text{const}
$$

### 变分推断与MCMC对比表

| 特性 | 变分推断(VI) | MCMC | 拉普拉斯近似 |
| ------|-------------|------|------------|
| 基本思想 | 优化KL散度 | 采样近似后验 | MAP点处高斯近似 |
| 计算方式 | 优化问题 | 随机采样 | 二阶展开 |
| 速度 | 快(优化) | 慢(需要大量样本) | 快 |
| 精度 | 有偏差(欠估计方差) | 渐近无偏 | 局部近似 |
| 可扩展性 | 高(随机优化) | 低-中 | 高 |
| 不确定性估计 | 偏低 | 准确 | 局部准确 |
| 适用模型 | 大规模、高维 | 中小规模 | MAP附近 |
| 收敛诊断 | ELBO监控 | R̂统计量 | Hessian正定性 |
| 典型工具 | Pyro, Edward, Stan(VI) | Stan, PyMC, emcee | Laplace.py |

### 典型应用

- 大规模贝叶斯推断（替代 MCMC）
- 主题模型（LDA）的推断（Blei et al., 2003）
- 贝叶斯深度学习中的权重后验近似
- 信号处理中的稀疏贝叶斯学习加速
- 高斯混合模型的变分 EM

## 4. 方法流程

**输入**：观测数据 $\mathcal{D}$，先验 $p(\theta)$，似然 $p(\mathcal{D}|\theta)$

**Step 1：选择变分族 $\mathcal{Q}$**
- 最常见：均值场假设 $q(\theta) = \prod_i q_i(\theta_i)$
- 更灵活：Normalizing Flows、混合分布等

**Step 2：定义优化目标**
- 最大化 ELBO（等价于最小化 KL 散度）
- $\text{ELBO} = \mathbb{E}_q[\log p(\mathcal{D},\theta)] - \mathbb{E}_q[\log q(\theta)]$

**Step 3：优化求解**
- **坐标上升法**（均值场）：依次更新每个因子 $q_j$
- **随机梯度上升法**（BBVI）：用蒙特卡洛估计 ELBO 梯度，SGD 更新
- **自然梯度法**：利用 Fisher 信息矩阵加速收敛

**Step 4：收敛判断**
- 监控 ELBO 值变化，当变化量小于阈值时停止
- 检查变分参数是否稳定

**Step 5：后验近似使用**
- 用 $q^*(\theta)$ 替代真实后验进行预测、决策等下游任务

**参数调优建议**：
- 均值场假设过强可能导致方差低估→考虑结构化变分
- 学习率过大→ELBO 震荡；过小→收敛慢
- BBVI 的蒙特卡洛样本数影响梯度估计方差

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import stats
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 变分推断演示：用高斯分布近似非高斯后验
# 真实后验是双峰混合高斯
np.random.seed(42)
# 混合高斯后验：0.4*N(-3, 0.5) + 0.6*N(3, 0.8)
def log_target(x):
    return np.log(0.4 * stats.norm.pdf(x, -3, 0.5) + 0.6 * stats.norm.pdf(x, 3, 0.8))

# Mean场变分推断：用单个高斯q(x)=N(mu, sigma^2)近似
# 通过最大化ELBO（最小化KL散度）优化参数
mu, log_sigma = 0.0, 0.0  # 变分参数
lr = 0.01
n_samples = 1000
elbo_history = []
x_range = np.linspace(-7, 7, 500)

for step in range(500):
    sigma = np.exp(log_sigma)
    # 从变分分布采样
    eps = np.random.randn(n_samples)
    z = mu + sigma * eps
    # 计算ELBO的蒙特卡洛估计和梯度
    log_q = -0.5 * ((z - mu) / sigma)**2 - log_sigma - 0.5 * np.log(2 * np.pi)
    log_p = np.array([log_target(zi) for zi in z])
    elbo = np.mean(log_p - log_q)
    elbo_history.append(elbo)
    # 简化梯度更新（随机梯度上升）
    dmu = np.mean((log_p - log_q) * (z - mu) / sigma**2)
    dls = np.mean((log_p - log_q) * (((z - mu) / sigma)**2 - 1))
    mu += lr * dmu
    log_sigma += lr * dls

# 可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 5))
target = np.exp(np.array([log_target(xi) for xi in x_range]))
target = target / np.trapz(target, x_range)
q_approx = stats.norm.pdf(x_range, mu, np.exp(log_sigma))
axes[0].plot(x_range, target, 'b-', linewidth=2.5, label='真实后验(双峰)')
axes[0].plot(x_range, q_approx, 'r--', linewidth=2.5, label=f'变分近似 N({mu:.1f},{np.exp(log_sigma):.1f}²)')
axes[0].set_title('变分推断：高斯近似', fontsize=13)
axes[0].set_xlabel('x')
axes[0].set_ylabel('PDF')
axes[0].legend(fontsize=11)
axes[0].grid(True, alpha=0.3)
axes[1].plot(elbo_history, 'g-', linewidth=2)
axes[1].set_title('ELBO收敛曲线', fontsize=13)
axes[1].set_xlabel('Iterations')
axes[1].set_ylabel('ELBO')
axes[1].grid(True, alpha=0.3)
plt.suptitle('变分推断（Mean场近似）演示', fontsize=15)
plt.tight_layout()
plt.show()
```

**结果解释**：左图展示了变分推断的效果——用单个高斯分布（红色虚线）去近似双峰后验（蓝色实线）。由于均值场假设限制了变分族只能是单峰高斯，最终近似会"折中"取两个峰之间的位置，导致方差被高估。右图展示了 ELBO 随迭代单调上升并收敛的过程。这说明：变分推断速度快，但选择合适的变分族至关重要。

## 6. 常见误区

| 误区 | 正确理解 |
|------|----------|
| 变分推断得到的就是真实后验 | 变分推断得到的是近似后验，受限于变分族的选择，通常不能完美匹配真实后验 |
| KL散度越小越好，一定能找到全局最优 | KL散度是非凸的优化问题，可能存在多个局部最优；且变分族本身可能不包含真实后验 |
| 均值场假设总够用 | 均值场假设各分量独立，会低估后验方差；对于多峰后验或强相关的后验，效果很差 |
| 变分推断一定比MCMC差 | 变分推断在大规模数据下更实用（速度快），且在某些场景下精度足够好 |
| ELBO最大化等价于KL最小化 | 等价的前提是 $p(\mathcal{D})$ 固定；ELBO = $\log p(\mathcal{D})$ - KL，最大化 ELBO 等于最小化 KL |
| BBVI不需要调参 | BBVI 需要调学习率、蒙特卡洛样本数等；梯度估计方差大时收敛慢 |
| 变分推断不能处理连续隐变量 | 变分推断天然适合连续隐变量（重参数化技巧），离散隐变量反而需要特殊处理（如 Gumbel-Softmax） |
