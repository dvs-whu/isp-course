    图像超分辨率¶

    1. 学习定位¶

    [!note] 本科导学定位 周次：W10 AI图像处理应用 深度：会用即可 关联实验和项目：项目二B3和图像类拓展项目。需掌握退化模型、插值方法对比、SRCNN基本结构和PSNR/SSIM评价指标。

    2. 通俗入口¶

    [!tip] 🎯 初学者简化说明 一句话理解：图像超分辨率 = 把模糊的小照片变清晰的大照片。

    生活类比：你有一张老照片，分辨率很低，放大后全是马赛克。超分辨率就是用"AI脑补"的方式，把这张照片放大4倍，让模糊的人脸变得清晰可辨——就像电影里FBI放大监控画面一样。

    核心思路（3步走）： 1. 模糊：清晰照片被Camera抖动/对焦不好弄模糊了 2. 缩小：高分辨率照片被缩小（下采样）成低分辨率小图 3. 加噪：Sensor再给图像加上一些噪点

    超分的任务就是逆向还原这3步，从小图"猜"出大图。

    考试重点 ⭐： - 退化Model公式 \(y = (x*k)\downarrow_s + n\) 的含义（知道每一步对应什么物理过程） - 插值Method对比（最近邻 < 双线性 < 双三次） - SRCNN的基本结构（3LayerConvolution，端到端学习映射） - PSNR/SSIM是评价指标

    可跳过的高级内容：Sparse表示优化推导、感知Loss的数学细节、GAN对抗Training原理

    典型应用： - 卫星/遥感图像分辨率增强 - 医学影像超分（提升MRI/CT分辨率） - 老照片/视频修复与增强 - 监控视频放大与增强

    3. 严谨核心¶

    数学对象与基本假设¶

    图像Super Resolution（Image Super-Resolution, SR）是从一张或多张低Resolution（LR）图像重建高分辨率（HR）图像的技术，本质上是一个病态逆问题。

    核心概念： - 退化模型：LR图像由HR图像经过模糊、下Sampling和Noise污染得到 - 单幅超分（SISR）：从单张LR图像恢复HR图像 - 多幅超分：利用多帧LR图像的互补信息 - 学习字典：通过训练建立LR-HR字典对的映射关系 - 深度学习方法：SRCNN、EDSR、ESRGAN等

    核心公式¶

    退化模型：

    💡 直觉理解：高清图 → 模糊 → 缩小 → 加噪 = 低清图。反过来就是超分要做的事。

    \[y = (x * k) \downarrow_s + n\]

    其中 \(x\) 为HR图像，\(k\) 为模糊核，\(\downarrow_s\) 为下采样因子 \(s\)，\(n\) 为噪声。

    超分辨率的优化目标：

    \[\hat{x} = \arg\min_x \|y - DHx\|_2^2 + \lambda \Phi(x)\]

    其中 \(D\) 为下采样算子，\(H\) 为模糊算子，\(\Phi(x)\) 为正则项。

    稀疏表示超分方法：

    \[\min_{\alpha_h, \alpha_l} \|\alpha_h\|_1 \quad \text{s.t.} \quad \|y_l - D_l \alpha_l\|_2 \le \epsilon, \quad \|\alpha_h - \alpha_l\|_2 \le \delta\]

    SRCNN的映射：

    \[F(y) = W_3 * \sigma(W_2 * \sigma(W_1 * y + b_1) + b_2) + b_3\]

    感知损失（Perceptual Loss）：

    \[\mathcal{L}_{percep} = \|\phi(x) - \phi(\hat{x})\|_2^2\]

    其中 \(\phi\) 为预训练网络的Feature提取函数。

    PSNR和SSIM用于评价超分质量：

    \[SSIM(x,\hat{x}) = \frac{(2\mu_x\mu_{\hat{x}}+C_1)(2\sigma_{x\hat{x}}+C_2)}{(\mu_x^2+\mu_{\hat{x}}^2+C_1)(\sigma_x^2+\sigma_{\hat{x}}^2+C_2)}\]

    适用条件和边界¶

    • 退化模型假设模糊核和噪声类型已知或可估计，实际场景中退化往往更复杂
    • 超分倍率越大，病态性越强，重建难度呈指数增长
    • 传统方法（稀疏表示）在小倍率下有效，大倍率需深度学习

    4. 方法流程¶

    输入是什么： 低分辨率图像 \(y\)（或一组多帧LR图像），退化参数（下采样因子 \(s\)、模糊核类型等）。

    处理步骤： 1. 预处理：上采样LR图像到目标尺寸（插值预处理，或网络内部上采样） 2. 特征提取：通过卷积层提取LR图像特征 3. 非线性映射：学习LR特征到HR特征的映射关系 4. 重建：从HR特征恢复高分辨率图像 5. 后处理：残差学习（输出残差加回插值结果）

    输出如何解释： 超分结果 \(\hat{x}\) 是对HR图像的估计，用PSNR/SSIM衡量与真实HR的相似度。

    常见参数如何选择： - 上采样因子 \(s\)：常用 ×2、×4、×8 - 网络深度：SRCNN(3层) < EDSR(16残差块) < RCAN(20残差块+注意力) - 损失函数：L2损失 → 感知损失 → GAN损失（质量递增但稳定性递减）

    方法对比¶

    方法 类型 Input 核心思想 上Sampling方式 PSNR (Set5) 特点
    Bicubic 传统插值 单帧 双三次插值 直接插值 ~28dB 无需Training，基线方法
    SRCNN CNN 单帧 3LayerCNN映射 预插值 ~30.5dB 开创性深度SR
    ESPCN CNN 单帧 亚像素Convolution 亚像素重排 ~31dB 高效上采样
    EDSR CNN 单帧 残差网络+大模型 亚像素重排 ~32.5dB 去BN层，性能强
    RCAN CNN 单帧 通道注意力残差 亚像素重排 ~32.8dB 注意力机制
    ESRGAN GAN 单帧 对抗训练+感知Loss 亚像素重排 主观质量高 视觉效果好
    SwinIR Transformer 单帧 窗口自注意力 亚像素重排 ~33.0dB 当前SOTA之一
    DBPN CNN 单帧 Iterative上下采样 深度反馈 ~32.5dB 错误反馈机制

    深度学习方法补充¶

    EDSR 残差块：

    \[ \mathbf{F}_{\text{out}} = \mathbf{F}_{\text{in}} + \mathcal{F}_\theta(\mathbf{F}_{\text{in}}) \]

    其中 \(\mathcal{F}_\theta\) 为两层卷积（无 BN）+ ReLU 的残差映射。

    5. Python最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib scipy - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib", "scipy"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
from scipy.ndimage import zoom

# 创建高ResolutionTestImage（含边缘和纹理）
N_hr = 64
img_hr = np.zeros((N_hr, N_hr))
# 方块和条纹图案
img_hr[10:25, 10:50] = 1.0
img_hr[30:55, 15:40] = 0.7
for i in range(N_hr):
    for j in range(N_hr):
        if (i-48)**2 + (j-50)**2 < 10**2:
            img_hr[i, j] = 0.5 + 0.3*np.sin(0.8*i)*np.cos(0.8*j)

# 下Sampling因子
scale = 4
img_lr = zoom(img_hr, 1/scale, order=3)  # 低分辨率图像

# 三种上采样方法对比
img_nn = zoom(img_lr, scale, order=0)     # 最近邻插值
img_bilinear = zoom(img_lr, scale, order=1)  # 双线性插值
img_bicubic = zoom(img_lr, scale, order=3)   # 双三次插值

# 计算PSNR
def psnr(ref, img):
    mse = np.mean((ref - img) ** 2)
    return 10 * np.log10(1.0 / mse) if mse > 0 else float('inf')

# 显示结果
fig, axes = plt.subplots(1, 4, figsize=(14, 4))
titles = ['高Resolution原图', f'最近邻插值\nPSNR={psnr(img_hr, img_nn):.1f}dB',
          f'双线性插值\nPSNR={psnr(img_hr, img_bilinear):.1f}dB',
          f'双三次插值\nPSNR={psnr(img_hr, img_bicubic):.1f}dB']
images = [img_hr, img_nn, img_bilinear, img_bicubic]
for ax, title, img in zip(axes, titles, images):
    ax.imshow(img, cmap='gray', vmin=0, vmax=1)
    ax.set_title(title, fontsize=10)
    ax.axis('off')
plt.suptitle('ImageSuper Resolution：插值Method对比', fontsize=13)
plt.tight_layout()
plt.show()

    结果解释： - 同一张小图，不同插值方法放大后的清晰度差异显著 - 最近邻插值有明显锯齿，双线性插值较模糊，双三次插值效果最好 - 深度学习超分（如SRCNN）效果远超这些传统方法 - PSNR值越高表示与原图越接近

    6. 常见误区¶

    公式误解： - 退化模型 \(y = (x*k)\downarrow_s + n\) 中的顺序很重要：先模糊再下采样，不是先下采样再模糊 - PSNR高不等于视觉效果好——GAN生成的结果PSNR可能较低但主观质量更高

    参数误解： - 上采样倍率不是越大越好——×8超分的可靠性远低于×2超分 - 网络参数量与性能不成正比——EDSR去掉了BN层反而性能更好

    适用条件误解： - 超分辨率不能"凭空创造"不存在的信息——它是在统计先验下做最优猜测 - 训练数据的退化模型如果与实际退化不匹配，超分效果会显著下降

    单位误解： - PSNR的单位是dB，每增加3dB意味着MSE减半；但人眼感知的"清晰度提升"与PSNR不是线性关系

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 如果一张图像完全由随机噪声组成（无任何结构），超分辨率算法会"生成"什么？答案是它仍然会输出看起来有结构的图像——因为它在"脑补"统计先验中的模式。这揭示了超分的一个根本局限：它是在做"最可能的猜测"，而非"真实的恢复"。

    一个跨学科迁移场景： 天文望远镜图像增强：地面望远镜受大气湍流影响，拍摄的星体图像模糊且分辨率低。超分辨率技术可以结合大气退化模型，从多帧短曝光图像中重建出接近太空望远镜质量的高分辨率图像。这是超分从"消费级应用"到"科学研究"的典型迁移。

    一个可做成课程项目的小想法： 设计一个"超分方法公平对比"实验：用同一张图像，分别用Bicubic、SRCNN（用预训练权重）和EDSR进行×4超分，对比PSNR、SSIM和运行时间。特别关注：在图像的平坦区域、边缘区域和纹理区域，三种方法的优劣是否一致？
