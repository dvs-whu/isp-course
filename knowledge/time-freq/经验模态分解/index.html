    经验模态分解（EMD）¶

    1. 学习定位¶

    [!info] 📌 学习定位 周次：W10-W11（非平稳信号分析模块） 深度：了解/选读 — 不纳入期末笔试范围 关联实验：项目一D选做拓展、项目二时序信号方向 预备知识：傅里叶变换、小波变换、Hilbert变换基础概念

    2. 通俗入口¶

    [!tip] 🎯 初学者简化说明 一句话理解：EMD像筛沙子一样，把复杂信号自适应地拆成若干个不同频率的本征模态函数（IMF），不需要预设基函数。

    一个真实问题：你在工厂录到一台电机的振动信号，里面混着轴承故障、齿轮啮合、电机旋转等多种频率成分。传统傅里叶变换需要你预设频率基，但你根本不知道里面有什么频率——EMD能自动帮你"筛"出来。

    生活类比：想象你面前有一碗混合了粗沙、细沙和面粉的粉末。EMD就像一个自动筛子：第一遍筛出最大的颗粒（最高频IMF），第二遍筛出中等颗粒（次高频IMF），……直到最后只剩下最细的粉末（残余趋势项）。整个过程不需要你事先知道粉末的成分。

    学这个能解决什么：对非平稳、非线性信号（如机械振动、地震波、心电信号）进行自适应多尺度分解，提取隐藏在不同频率层中的特征信息。

    💡 一句话记忆法：EMD = 信号界的"自动筛子"，不管你是什么信号，我都能从粗到细一层层筛出来。

    你需要重点掌握的： 1. EMD的核心：通过反复"筛分"（sifting），从高频到低频逐层提取IMF分量 2. 每个IMF必须满足两个条件：极值点数与过零点数相差不超过1；上下包络线均值为零 3. EMD是自适应的——基函数由信号本身决定，这是它与傅里叶变换、小波变换的本质区别

    作为进阶了解的（期末不要求推导）： - Hilbert变换的数学公式及Hilbert谱的构建过程 - 模态混叠（mode mixing）问题及其改进方法（EEMD、CEEMD等）

    3. 严谨核心¶

    3.1 数学对象¶

    经验模态分解（Empirical Mode Decomposition, EMD）是由 Norden E. Huang 等人于1998年提出的一种自适应信号分解方法。与小波变换等需要预设基函数的方法不同，EMD 完全由数据驱动，将复杂信号自适应地分解为有限个本征模态函数（Intrinsic Mode Function, IMF）的叠加。

    3.2 基本假设与IMF条件¶

    本征模态函数（IMF） 需满足两个条件： 1. 在整个数据段上，极值点个数与过零点个数相等或相差不超过1 2. 在任意时刻，由局部极大值定义的上包络线与由局部极小值定义的下包络线的均值为零

    3.3 核心公式¶

    EMD 筛选过程：

    给定信号 \(x(t)\)，第 \(i\) 个 IMF \(c_i(t)\) 的提取过程：

    1. 令 \(h_0(t) = x(t)\)（或 \(h_0(t) = r_{i-1}(t)\)，即上一轮的残差）
    2. 计算 \(h_{k-1}(t)\) 的上下包络 \(u_{k-1}(t)\) 和 \(l_{k-1}(t)\)（通过极值点三次样条插值）
    3. 计算包络均值：\(m_{k-1}(t) = \frac{u_{k-1}(t) + l_{k-1}(t)}{2}\)
    4. 更新：\(h_k(t) = h_{k-1}(t) - m_{k-1}(t)\)
    5. 检查 \(h_k(t)\) 是否满足 IMF 条件；若不满足，重复步骤2-4
    6. 满足后令 \(c_i(t) = h_k(t)\)

    信号分解结果：

    \[ x(t) = \sum_{i=1}^{N} c_i(t) + r_N(t) \]

    其中 \(c_i(t)\) 为第 \(i\) 个 IMF，\(r_N(t)\) 为最终残余趋势项。

    3.4 Hilbert 谱分析¶

    对每个 IMF \(c_i(t)\) 进行 Hilbert 变换：

    \[ \hat{c}_i(t) = \frac{1}{\pi} \, \text{P.V.} \int_{-\infty}^{+\infty} \frac{c_i(\tau)}{t - \tau}\, d\tau \]

    解析信号：

    \[ z_i(t) = c_i(t) + j\hat{c}_i(t) = a_i(t)\, e^{j\phi_i(t)} \]

    瞬时幅值与瞬时频率：

    \[ a_i(t) = \sqrt{c_i^2(t) + \hat{c}_i^2(t)}, \qquad \omega_i(t) = \frac{d\phi_i(t)}{dt} \]

    Hilbert 谱：

    \[ H(\omega, t) = \operatorname{Re} \sum_{i=1}^{N} a_i(t)\, e^{j\int \omega_i(t)\, dt} \]

    3.5 适用条件与边界¶

    • EMD适用于非线性、非平稳信号的自适应分解
    • 端点效应：信号两端的包络线估计不准确，需要端点延拓处理
    • 模态混叠：当信号中存在频率相近的成分时，可能被分到同一个IMF中
    • EMD缺乏严格的数学理论基础（"经验"的由来），分解结果可能不唯一

    3.6 EMD 及其改进方法对比¶

    方法 模态混叠 残余噪声 计算量 完备性 提出年份
    EMD 严重 无 低 ✓ 1998
    EEMD 有效抑制 有（白噪声残余） 高（多次集成） ✓ 2009
    CEEMDAN 进一步抑制 极少 中 ✓ 2011
    VMD 无 无 中 ✓ 2014

    集合经验模态分解（EEMD） 是 EMD 的改进版本，通过向信号中添加白噪声并进行多次 EMD 后取平均，有效抑制了 EMD 中的模态混叠（mode mixing）问题。

    完备集合经验模态分解（CEEMDAN） 进一步改进了 EEMD，在每个分解阶段添加特定的自适应噪声，减少了残余噪声和计算量。

    Hilbert-Huang 变换（HHT） 将 EMD 与 Hilbert 谱分析相结合：对每个 IMF 进行 Hilbert 变换，得到瞬时频率和瞬时幅值，从而构建完整的时频-能量分布（Hilbert 谱）。

    典型应用¶

    • 机械故障诊断：轴承、齿轮箱振动信号的非平稳特征提取，尤其是变速工况下的故障检测
    • 海洋与气象数据分析：海浪、洋流、气候变化信号的多尺度分析（HHT 的原始应用场景）
    • 地震信号处理：地震波的去噪与分层特征提取
    • 生物医学信号：心电信号（ECG）、脑电信号（EEG）的自适应分析
    • 结构健康监测：桥梁、建筑物振动信号的模态参数识别
    • 金融时间序列：股票价格等非线性非平稳序列的趋势分解与周期分析
    • 语音信号处理：语音增强、基频估计

    4. 方法流程¶

    输入：一维时间信号 \(x(t)\)（连续或离散采样）

    处理步骤： 1. 找出信号的所有极大值点和极小值点 2. 分别用三次样条插值拟合上包络线 \(u(t)\) 和下包络线 \(l(t)\) 3. 计算包络均值 \(m(t) = (u(t) + l(t))/2\) 4. 用原信号减去均值：\(h(t) = x(t) - m(t)\) 5. 检查 \(h(t)\) 是否满足IMF两个条件；不满足则回到步骤1继续筛分 6. 满足后提取为一个IMF \(c_i(t)\)，将残差 \(r(t) = x(t) - c_i(t)\) 作为新信号 7. 重复步骤1-6，直到残余项为单调函数或常值

    输出：\(N\) 个IMF分量 \(c_1(t), c_2(t), \ldots, c_N(t)\) + 残余趋势项 \(r_N(t)\)

    常见参数选择： - 筛分停止准则：连续两次筛分结果的标准差 \(SD < 0.2 \sim 0.3\) - 最大筛分次数：通常设为 10~100 次，避免过度筛分 - IMF个数：由信号本身决定，通常取前几个包含主要信息的IMF - 包络插值方法：三次样条（最常用），线性插值（简化版本）

    5. Python最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

def simple_emd(x, n_imfs=3, max_sifts=10):
    """简化版 EMD：通过筛分过程提取 IMF 分量"""
    residue = x.copy()
    imfs = []
    t_idx = np.arange(len(x))
    for _ in range(n_imfs):
        h = residue.copy()
        for _ in range(max_sifts):
            # 寻找极值点（包含首尾以保证包络完整）
            max_idx = np.concatenate([[0],
                np.where((h[1:-1] > h[:-2]) & (h[1:-1] > h[2:]))[0] + 1,
                [len(h)-1]])
            min_idx = np.concatenate([[0],
                np.where((h[1:-1] < h[:-2]) & (h[1:-1] < h[2:]))[0] + 1,
                [len(h)-1]])
            # 线性插值求上下包络
            upper = np.interp(t_idx, max_idx, h[max_idx])
            lower = np.interp(t_idx, min_idx, h[min_idx])
            h = h - (upper + lower) / 2  # 减去包络均值（筛分）
        imfs.append(h)
        residue = residue - h
    return imfs, residue

# 生成非平稳信号
np.random.seed(42)
N = 500
t = np.linspace(0, 5, N)
x = (np.sin(2*np.pi*2*t) + 0.5*np.sin(2*np.pi*8*t) +
     0.3*np.sin(2*np.pi*30*t) + 0.2*np.random.randn(N))

# EMD 分解
imfs, residue = simple_emd(x, n_imfs=3)

fig, axes = plt.subplots(len(imfs) + 2, 1, figsize=(10, 8))
axes[0].plot(t, x, 'b-')
axes[0].set_title('Original (Multi-frequency Superposition)')

for i, imf in enumerate(imfs):
    axes[i+1].plot(t, imf, 'r-')
    axes[i+1].set_title(f'IMF {i+1}（从高频到低频逐层提取）')

axes[-1].plot(t, residue, 'g-')
axes[-1].set_title('残余趋势项')
axes[-1].set_xlabel('Time (s)')

plt.tight_layout()
plt.show()

    结果解释：上图展示了EMD的分解过程。原始信号由2Hz、8Hz、30Hz三个正弦分量加噪声组成。IMF1主要捕获最高频（30Hz）成分，IMF2捕获中频（8Hz）成分，IMF3捕获低频（2Hz）成分，残余项接近零（噪声被分散到各IMF中）。这体现了EMD从高频到低频逐层"筛分"的自适应特性。

    6. 常见误区¶

    公式误解： - ❌ "EMD和小波变换一样，都有固定的基函数" → ✅ EMD的基函数完全由信号本身决定，这是它与小波变换的本质区别 - ❌ "IMF就是正弦波" → ✅ IMF可以是任意波形，只要满足两个IMF条件即可

    参数误解： - ❌ "筛分次数越多越好" → ✅ 过度筛分会导致IMF变成恒定幅值的调频信号，失去物理意义。应使用SD准则及时停止 - ❌ "IMF个数越多分解越精细" → ✅ 后面的IMF通常包含很少有意义的信息，应关注前几个主要IMF

    适用条件误解： - ❌ "EMD可以处理任何信号" → ✅ EMD对含有间歇性噪声（intermittency）的信号容易产生模态混叠 - ❌ "EMD分解结果是唯一的" → ✅ 由于端点效应和筛分准则的不同，分解结果可能存在差异 - ❌ "EMD有严格的数学证明" → ✅ EMD是"经验性的"方法，缺乏完整的数学理论框架，这也是后续EEMD、VMD等方法被提出的原因

    单位/概念误解： - ❌ "瞬时频率一定是正的" → ✅ 通过Hilbert变换计算的瞬时频率可能出现负值，这在物理上不一定有明确意义，是HHT方法的一个争议点

    7. 启发问题与创新拓展¶

    一个反例或边界问题：如果信号中两个正弦分量的频率非常接近（如49Hz和51Hz），EMD能否将它们分到不同的IMF中？尝试用代码验证，并思考什么情况下EMD会失败（模态混叠）。

    一个跨学科迁移场景：在金融领域，股票价格序列是非线性非平稳的。可以用EMD将价格序列分解为若干IMF，分别代表日内波动、周度趋势、月度周期等不同时间尺度的成分，再对每个成分单独建模预测。

    一个可做成课程项目的小想法：实现一个简易的EMD去噪系统——对含噪信号做EMD分解，判断哪些IMF主要包含噪声（如通过相关系数或能量比），将其去除后重构信号。与小波去噪方法做对比实验。
