    时频分布（Time-Frequency Distribution）¶

    [!tip] 🎯 初学者简化说明 时频分布的核心思想非常直观：把信号的能量画在时间-频率平面上，这样我们就能同时看到"在什么时刻、什么频率成分最强"。对初学者来说，最需要掌握的时频分布是频谱图（即 [[短时傅里叶变换|STFT]] 的模平方），它就像给信号拍一张"时间-频率的照片"📸。

    你需要重点掌握的： 1. 频谱图 = STFT 的模平方，是基础阶段最实用、最常用的时频分布 2. 交叉项是什么、为什么会出现（用两个简单正弦波就能理解） 3. 不同分布各有优缺点，频谱图计算简单、没有交叉项，但分辨率偏低

    作为进阶了解的（期末不要求掌握推导）： - Wigner-Ville 分布的双线性积分公式——知道它"聚集性最高但有交叉项"即可 - Cohen 类统一框架——知道"所有时频分布可以通过选不同核函数统一描述"即可

    💡 一句话记忆法：频谱图是信号的"时频照片"，Wigner-Ville 是信号的"时频高清照"（但有伪影）。

    2. 通俗入口¶

    一个真实问题¶

    你录了一段鸟叫的音频。鸟的叫声频率随时间变化——一开始是低沉的"咕咕"声，然后变成高频的"啾啾"声。如果只看整个录音的频谱（FFT），你只能看到"300Hz和2000Hz都有能量"，但不知道哪个先出现、哪个后出现。时频分布就是解决"什么时刻、什么频率最强"这个问题的工具。

    一个生活类比¶

    频谱图就像给信号拍一张时间-频率的"照片"📸。就像我们用相机拍风景——照片能告诉我们"哪里有什么"，频谱图能告诉我们"什么时刻、什么频率成分有多强"。而Wigner-Ville分布就像用更高分辨率的相机拍照，但照片上会出现"鬼影"（交叉项）。

    学这个能解决什么？¶

    领域 应用 说明
    📡 雷达信号 线性调频信号检测 雷达发射的信号频率随时间变化，时频分布能清晰展示这种变化
    🏥 生物医学 脑电/心电信号分析 脑电信号的频率成分随时间快速变化，需要时频分析
    🔧 机械诊断 轴承故障检测 变速工况下振动信号的频率随转速变化
    🗣️ 语音处理 基频追踪 语音的音调随时间变化，时频图是语音分析的基础工具
    🌊 地震勘探 地震波分析 地震反射信号的频率随传播时间变化
    🎵 音频处理 音乐转录 识别音乐中不同时间点的音符频率

    3. 严谨核心¶

    数学对象¶

    时频分布（Time-Frequency Distribution, TFD）是一类方法，它们的目标都是把信号能量表示为时间和频率的函数 \(E(t, f)\)。频谱图是其中最简单、最常用的一种。

    基本假设¶

    • 信号可以分解为多个时变频率分量
    • 对于频谱图：假设信号在窗函数宽度内近似平稳
    • 对于Wigner-Ville分布：信号为解析信号（需先做Hilbert变换）

    ![[assets/time_freq_distribution_ai.png]]

    图：时频分析方法对比——STFT/小波变换/Wigner-Ville的优缺点

    核心公式¶

    频谱图（本科重点 ⭐）

    频谱图就是 [[短时傅里叶变换|STFT]] 的模平方：

    \[ | \text{Spectrogram}(t, f) = |S(t, f)|^2 = \left|\int_{-\infty}^{+\infty} x(\tau)\, w(\tau - t)\, e^{-j2\pi f \tau}\, d\tau\right|^2 \]

    其中 \(w(t)\) 是窗函数（如汉宁窗、高斯窗等）。

    直观理解： 1. 在时间 \(t\) 附近，用窗函数 \(w\) 截取一段信号 2. 对这段信号做傅里叶变换，得到该时刻的频率成分 3. 取模平方，得到该时刻各频率的能量 4. 对每个时刻重复，拼成一张"时间-频率能量图"

    Wigner-Ville 分布（进阶了解 🔬）

    ⚠️ 入门说明：以下公式仅作参考，期末不要求推导。只需记住：WVD 是一种"更高分辨率但有交叉项"的时频分布。

    \[ W(t, f) = \int_{-\infty}^{+\infty} x\!\left(t + \frac{\tau}{2}\right) x^*\!\left(t - \frac{\tau}{2}\right) e^{-j2\pi f \tau}\, d\tau \]

    直观理解：WVD 用信号自身的"左半段"和"右半段"做相关运算，不需要外部窗函数，因此分辨率最高。但正因为没有窗函数的"约束"，多分量信号会产生交叉项。

    Cohen 类统一表示（进阶了解 🔬）

    💡 入门说明：这一段只需要了解概念，不需要掌握公式推导。

    所有常用的二次型时频分布都可以统一到一个叫 Cohen 类 的框架下。你可以把它理解为一个"万能模板"：

    • 选定不同的 核函数 \(\phi(\tau, \nu)\)，就能得到不同的时频分布
    • 核函数就像一个"滤波器"，用来控制分辨率和交叉项之间的平衡
    • 选得好 → 交叉项少但分辨率低（如频谱图）
    • 选得激进 → 分辨率高但交叉项多（如 Wigner-Ville 分布）
    \[ C(t, f) = \int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} \phi(\tau, \nu)\, A(\tau, \nu)\, e^{j2\pi(\nu t - f \tau)}\, d\nu\, d\tau \]

    其中 \(\phi(\tau, \nu)\) 是核函数，\(A(\tau, \nu)\) 是模糊函数。不同的核函数 \(\phi\) 对应不同的分布：

    • \(\phi = 1\) → Wigner-Ville 分布
    • \(\phi\) = 窗函数的 WVD → 频谱图
    • \(\phi = \exp\!\left(-\frac{(2\pi \nu \tau)^2}{\sigma}\right)\) → Choi-Williams 分布

    频谱图与 WVD 的关系¶

    \[ S(t, f)|^2 = \iint W_x(u, \xi)\, W_w(t - u, f - \xi)\, du\, d\xi \]

    这个公式告诉我们：频谱图等于信号的 WVD 与窗函数 WVD 的二维卷积。换句话说，频谱图是 WVD 经过窗函数"平滑"后的结果——平滑减少了交叉项，但也降低了分辨率。

    常用时频分布对比¶

    分布 分辨率 交叉项 计算难度 本科要求
    频谱图 \(\lvert S(t,f)\rvert^2\) 一般 ✅ 无 ⭐ 低（用 scipy.signal.spectrogram） 必须掌握
    Wigner-Ville (WVD) 最高 ❌ 严重 中 了解概念即可
    伪 Wigner-Ville 高 有改善 中 了解概念即可
    Choi-Williams 高 较好 高 了解即可
    锥形核分布 中 好 高 了解即可

    🔑 核心规律：分辨率和交叉项就像"跷跷板"——分辨率越高，交叉项越严重。频谱图在这两者之间取得了很好的平衡，所以是实际工程中最常用的。

    什么是交叉项？（重要概念）¶

    交叉项是二次型时频分布（如 Wigner-Ville 分布）的固有问题。用一个简单的例子来说明：

    假设信号由两个正弦波叠加：\(x(t) = \cos(2\pi f_1 t) + \cos(2\pi f_2 t)\)

    • 频谱图：会正确显示两条水平线，分别在频率 \(f_1\) 和 \(f_2\) 处 ✅
    • Wigner-Ville 分布：除了两条正确的水平线外，在 \(f_1\) 和 \(f_2\) 的中间位置会出现一条虚假的"干扰线" ❌

    这条虚假的干扰线就是交叉项（cross-term）。它是 Wigner-Ville 分布的双线性结构导致的——两个频率分量相乘，产生了位于它们中间的虚假能量。

    🏷️ 生活类比：交叉项就像拍照时的"鬼影"——明明只有一个人站在那里，照片里却出现了两个重叠的影子。Wigner-Ville 分布虽然"分辨率高"，但容易产生这种"鬼影"。

    适用条件与边界¶

    • 频谱图适用于大多数工程场景，是默认首选
    • Wigner-Ville仅适用于单分量信号或分量间隔很远的信号
    • 所有时频分布都面临Heisenberg不确定性原理的约束：时间分辨率和频率分辨率不能同时任意小
    • 窗函数宽度直接决定频谱图的时间-频率分辨率平衡

    4. 方法流程¶

    输入¶

    • 时域信号 \(x[n]\)，采样率 \(f_s\)
    • 窗函数类型和长度（频谱图）
    • 或核函数类型（Cohen类分布）

    处理步骤（以频谱图/STFT为例）¶

    1. 选择窗函数 \(w[n]\) 及窗长 \(M\)（如汉宁窗，\(M = 256\)）
    2. 滑动窗口：在信号上以步长 \(H\)（hop size）滑动，截取每一段信号
    3. 加窗FFT：对每段信号 \(x[n] \cdot w[n - kH]\) 做FFT
    4. 取模平方：\(|S(kH, f)|^2\) 得到该时刻的功率谱
    5. 拼接：将所有时刻的功率谱按时间排列，形成二维时频图

    输出¶

    • 二维矩阵 \(|S(t, f)|^2\)，横轴为时间，纵轴为频率，颜色表示能量强度
    • 通常以dB刻度显示：\(10\log_{10}|S(t,f)|^2\)

    常见参数选择¶

    参数 选择建议 影响
    窗长 \(M\) 短窗→时间分辨率高；长窗→频率分辨率高 时频分辨率折中
    窗类型 汉宁窗（通用）、高斯窗（最优时频聚集） 旁瓣泄漏 vs 主瓣宽度
    重叠率 通常 50%-75% 图像平滑度
    FFT点数 ≥ \(M\) 频率轴采样密度

    5. Python 最小实验¶

    数据生成¶

    多分量信号：两个正弦波（30Hz + 70Hz），比较频谱图与Wigner-Ville分布。

    关键函数¶

    • scipy.signal.spectrogram() — 计算频谱图
    • 自定义 wigner_ville() 函数 — 计算简化版WVD

    代码与曲线¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib scipy - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib", "scipy"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import signal
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

def wigner_ville(x, fs):
    """简化版 Wigner-Ville 分布计算"""
    N = len(x)
    z = signal.hilbert(x)  # 解析信号
    nfft = 2 * N
    WVD = np.zeros((nfft//2, N))
    for n in range(N):
        tau_max = min(n, N-1-n, nfft//2-1)
        tau = np.arange(-tau_max, tau_max+1)
        r = z[n+tau] * np.conj(z[n-tau])  # 瞬时自相关
        spec = np.fft.fft(r, nfft)[:nfft//2]
        WVD[:len(spec), n] = np.real(spec)
    freqs = np.arange(nfft//2) * fs / nfft
    t_arr = np.arange(N) / fs
    return t_arr, freqs, WVD

# 多分量信号：两个正弦波（30Hz + 70Hz）
fs = 256
N = 256
t_sig = np.arange(N) / fs
x = np.sin(2*np.pi*30*t_sig) + np.sin(2*np.pi*70*t_sig)

# 计算两种时频分布
t_wvd, f_wvd, WVD = wigner_ville(x, fs)
f_stft, t_stft, Sxx = signal.spectrogram(x, fs=fs, nperseg=32, noverlap=24)

fig, axes = plt.subplots(1, 2, figsize=(12, 5))
axes[0].pcolormesh(t_stft, f_stft, 10*np.log10(Sxx+1e-10), shading='gouraud', cmap='jet')
axes[0].set_title('STFT Spectrogram: No Cross-terms')
axes[0].set_xlabel('Time (s)')
axes[0].set_ylabel('Frequency (Hz)')

axes[1].pcolormesh(t_wvd, f_wvd, WVD, shading='gouraud', cmap='jet')
axes[1].set_title('Wigner-Ville 分布：高分辨率但30Hz与70Hz间有交叉项')
axes[1].set_xlabel('Time (s)')
axes[1].set_ylabel('Frequency (Hz)')

plt.tight_layout()
plt.show()

    结果解释¶

    • 频谱图（左图）：30Hz和70Hz处显示两条清晰的水平线，没有交叉项，但频率轴的线条稍显"模糊"（分辨率一般）
    • Wigner-Ville分布（右图）：30Hz和70Hz的线条更锐利（分辨率高），但在30Hz和70Hz之间可以看到明显的交叉项能量
    • 结论：频谱图没有交叉项且计算简单，是工程首选；WVD分辨率高但有伪影，仅适合单分量信号

    6. 常见误区¶

    公式误解¶

    • ❌ "Wigner-Ville分布是信号的能量谱" — WVD不一定是处处非负的，它可以取负值，因此不能严格解释为"能量分布"
    • ❌ "频谱图分辨率低是因为算法不好" — 分辨率受限于窗函数长度（Heisenberg不确定性原理），不是算法问题

    单位误解¶

    • ❌ "时频图的颜色表示信号的振幅" — 颜色表示的是功率（模的平方），不是振幅（模）
    • ❌ "频率轴可以直接用Hz，和采样率无关" — 频率轴范围为 \([0, f_s/2]\)，必须根据采样率正确设置

    参数误解¶

    • ❌ "窗长选越长越好" — 窗长越长频率分辨率越好，但时间分辨率越差。对于快速变化的信号（如语音的辅音），短窗更合适
    • ❌ "重叠率越高越准" — 重叠率不影响频谱估计的准确性，只影响时频图的平滑度和计算量

    适用条件误解¶

    • ❌ "对任何信号都可以用Wigner-Ville分布" — WVD只适合单分量信号。多分量信号必须用频谱图或其他抑制交叉项的分布
    • ❌ "时频分布可以同时获得任意高的时间和频率分辨率" — 受Heisenberg不确定性原理限制，两者不可兼得

    7. 启发问题与创新拓展¶

    一个反例或边界问题¶

    Chirp信号的时频分析：考虑一个频率线性增长的信号（线性调频/chirp）：\(x(t) = \cos(2\pi(f_0 t + \frac{k}{2}t^2))\)。频谱图需要在"看到频率变化"（短窗）和"精确测量频率"（长窗）之间做取舍。而WVD对chirp信号恰好表现为一条完美的直线——这说明WVD对单分量非平稳信号有独特优势。这个反例说明"没有万能的时频分布"。

    一个跨学科迁移场景¶

    金融时间序列的时频分析：股票收益率序列可以看作非平稳信号。用短窗口的频谱图分析股价，可以观察到"在金融危机期间，高频波动（日内波动）增强，而低频趋势（周/月周期）减弱"。这种时频视角比单纯的频域分析更有信息量。

    一个可做成课程项目的小想法¶

    "鸟类叫声识别器"：录制不同鸟的叫声，用频谱图提取时频特征，然后用简单的模板匹配或机器学习分类器识别鸟的种类。这个项目把时频分析与模式识别结合，既有工程价值又有生态学趣味。
