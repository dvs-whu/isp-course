    信号与系统基础¶

    1. 学习定位¶

    [!note] 📍 学习导航 - 所属模块：经典信号处理 → 信号表示与系统分析 - 周次：W1 - 学习深度：必须掌握，支撑后续傅里叶、采样、滤波和统计信号处理 - 先修知识：微积分（积分与求和）、复数基础 - 关联实验：所有实验的前置基础 - 关联项目：项目一A/B/C/D/E（所有项目都要先完成信号表示、采样率说明、时域可视化和基本统计描述） - 核心问题：如何把真实世界中的声音、图像、心电、传感器数据表示成可计算的信号？系统如何把输入变成输出？ - 严谨边界：本课重点理解信号分类、系统性质、LTI系统、卷积和相关的含义，不要求完整连续时间泛函分析推导 - 启发思考：同一段数据换采样率、换单位、换观察窗口后，还是同一个信号问题吗？

    2. 通俗入口¶

    [!tip] 🎯 一句话理解 信号就是随时间变化的物理量，系统就是对信号做变换的"黑盒子"，而卷积就是用一个滑动窗口对信号做加权求和。

    一个真实问题：你用手机录一段语音，手机麦克风采集到的是空气压力随时间变化的连续波形。这个波形就是"信号"。手机内部的软件对这个波形做降噪、压缩、编码——这些操作的数学模型就是"系统"。

    生活类比：卷积就像拖把拖地——把一个函数倒过来，沿着另一个函数滑动积分。每滑动一个位置，就计算一次加权求和，最终得到输出信号。

    学这个知识点能解决什么： - 理解信号如何从物理世界进入数字世界 - 理解 LTI 系统为什么可以用冲激响应完全表征 - 为傅里叶分析、滤波器设计、采样定理打下基础

    3. 严谨核心¶

    数学对象¶

    信号是携带信息的函数，通常表示为时间的函数 \(x(t)\)（连续时间）或 \(x[n]\)（离散时间）。系统是对信号进行变换或处理的数学模型，将输入信号映射为输出信号。

    基本信号类型： - 连续时间信号：定义在连续时间轴上的函数 \(x(t)\)，如正弦信号、指数信号 - 离散时间信号（序列）：定义在整数时刻的函数 \(x[n]\)，如单位脉冲序列、单位阶跃序列

    基本假设¶

    LTI（线性时不变）系统是信号与系统理论的核心研究对象，由其冲激响应完全表征。

    系统的基本性质： - 线性：满足叠加性与齐次性 - 时不变性：输入时移导致输出等量时移 - 因果性：输出仅依赖当前及过去的输入 - 稳定性：有界输入产生有界输出（BIBO）

    LTI系统处理流程¶

    ![[assets/lti_system_modern_ai.png]]

    系统性质对比¶

    性质 数学条件 物理含义 判断方法
    线性 \(T[ax_1+bx_2]=aT[x_1]+bT[x_2]\) 满足叠加性与齐次性 两步验证法
    时不变性 \(T[x(t-t_0)] = y(t-t_0)\) 输入时移导致等量输出时移 检查参数是否随时间变化
    因果性 \(h(t)=0, \; t<0\) 输出仅依赖当前及过去输入 检查冲激响应
    稳定性 \(\int h(t) dt < \infty\) 有界输入产生有界输出（BIBO） 绝对可积/可和
    记忆性 \(h(t)=k\delta(t)\) 输出仅依赖当前输入 冲激响应是否为冲激

    核心公式¶

    卷积（连续时间）： $\(y(t) = (x * h)(t) = \int_{-\infty}^{\infty} x(\tau) h(t - \tau) \, d\tau\)$

    卷积（离散时间）： $\(y[n] = (x * h)[n] = \sum_{k=-\infty}^{\infty} x[k] h[n - k]\)$

    LTI系统的特征函数： $\(e^{st} \to H(s) e^{st}, \quad z^n \to H(z) z^n\)$

    其中 \(H(s)\) 为系统函数（拉普拉斯变换域），\(H(z)\) 为传递函数（Z变换域）。

    BIBO稳定性条件： $\(\int_{-\infty}^{\infty} |h(t)| \, dt < \infty \quad \text{或} \quad \sum_{n=-\infty}^{\infty} |h[n]| < \infty\)$

    适用条件和边界¶

    • LTI 理论适用于线性、时不变系统；非线性或时变系统需要其他方法
    • 卷积公式假设系统为 LTI，否则不能用冲激响应表征
    • 连续卷积积分要求信号绝对可积（理论条件），实际中用离散近似

    4. 方法流程¶

    输入：输入信号 \(x[n]\)（或 \(x(t)\)）和 LTI 系统的冲激响应 \(h[n]\)（或 \(h(t)\)）

    处理步骤： 1. 确认系统是否满足 LTI 条件（线性 + 时不变） 2. 获取系统的冲激响应 \(h[n]\)（通过测量或设计） 3. 计算卷积 \(y[n] = x[n] * h[n]\)：翻转 \(h\)、滑动、逐点乘积求和 4. 对于长信号，可用 FFT 实现快速卷积：\(Y = X \cdot H\)（频域相乘再逆变换）

    输出解释：输出 \(y[n]\) 是输入信号经系统变换后的结果。例如：平滑（低通滤波）、锐化（高通滤波）、延迟等。

    常见参数选择： - 冲激响应长度 \(M\)：决定系统的记忆长度和计算量 - 对于 FIR 滤波器，\(M\) 越大频率选择性越强，但延迟和计算量也越大

    5. Python最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 定义输入信号：阶跃信号
n = np.arange(0, 50)
x = np.heaviside(n - 5, 1)  # 从 n=5 开始的阶跃信号

# 定义 LTI 系统的冲激响应：移动平均滤波器 (长度=10)
h_len = 10
h = np.ones(h_len) / h_len  # 归一化矩形脉冲

# 卷积计算（LTI 系统的核心运算）
y = np.convolve(x, h, mode='full')

# 绘制三个子图：输入、冲激响应、输出
fig, axes = plt.subplots(3, 1, figsize=(10, 8))
axes[0].stem(n, x, linefmt='b-', markerfmt='bo', basefmt='k-')
axes[0].set_title('Input x[n]: Step Signal')
axes[0].set_xlabel('样本序号 n')

axes[1].stem(np.arange(h_len), h, linefmt='g-', markerfmt='go', basefmt='k-')
axes[1].set_title('Impulse Response h[n]: Moving Average (Rect Window)')
axes[1].set_xlabel('样本序号 n')

n_out = np.arange(len(y))
axes[2].stem(n_out, y, linefmt='r-', markerfmt='ro', basefmt='k-')
axes[2].set_title('Output y[n] = x[n] * h[n]: Convolution Smoothing')
axes[2].set_xlabel('样本序号 n')

plt.tight_layout()
plt.show()

    结果解释：移动平均滤波器是最简单的 LTI 系统之一，其冲激响应为矩形脉冲。卷积结果展示了阶跃信号经滤波后的平滑过渡过程——原来陡峭的跳变被"抹"成了渐变斜坡。

    6. 常见误区¶

    易错点 正确理解
    卷积就是简单的乘法 卷积是翻转-滑动-求和，不是逐点相乘；频域相乘才是对应时域卷积
    LTI 系统可以处理任何信号 LTI 只适用于线性时不变系统；实际系统（如功放饱和、时变信道）可能不满足
    冲激响应长就一定好 冲激响应越长延迟越大，实时系统需要在性能和延迟之间折中
    因果性不重要 非因果系统在物理上不可实时实现；但离线处理可用非因果滤波

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 如果系统是时变的（如移动通信中的无线信道），LTI 系统的卷积模型就不再适用。这时需要用什么数学工具？（提示：线性时变系统、时频分析）

    一个跨学科迁移场景： 医学 CT 成像中的"投影-重建"过程本质上是连续函数的线性变换。理解 LTI 系统的框架后，可以类比理解 CT 中 Radon 变换的线性性质——系统矩阵 \(A\) 类似于冲激响应，重建就是求逆问题。

    一个可做成课程项目的小想法： 用 Python 实现一个"系统辨识"实验：给定一个未知 LTI 系统（黑盒子），输入冲激信号，观察输出即为冲激响应 \(h[n]\)；然后用卷积预测任意输入下的输出，验证 LTI 性质。
