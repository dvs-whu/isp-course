    采样与重建¶

    1. 学习定位¶

    [!note] 📍 学习导航 - 所属模块：经典信号处理 → 连续与离散的桥梁 - 周次：W3 - 学习深度：必须掌握，支撑数字化、频谱分析、功率谱估计和滤波实验 - 先修知识：信号与系统基础（频域概念）、傅里叶分析（频谱概念） - 关联实验：实验2（频谱分析与滤波） - 关联项目：项目一B/D（可穿戴设备、语音采集、环境监测和低功耗传感项目都需要合理选择采样率） - 核心问题：连续信号进入计算机前，怎样采样才不会把高频信息误认为低频？ - 严谨边界：重点掌握带限假设、奈奎斯特条件、混叠、抗混叠滤波和量化直觉；sinc重建和SQNR推导作为了解 - 启发思考：现实信号很少严格带限，工程上为什么仍能使用采样定理？

    详见[[13周教学计划|13周教学计划]] | [[课程关系图谱]]

    2. 通俗入口¶

    [!tip] 🎯 一句话理解 采样就是用固定时间间隔"拍照"连续信号，只要拍照足够快（采样率 > 2倍最高频率），就能完美还原原始信号。

    一个真实问题：你用手机录音，手机每秒对声压"拍照" 44100 次（44.1kHz 采样率）。为什么是 44100 而不是 1000？因为人耳能听到最高约 20kHz 的声音，根据奈奎斯特定理，采样率必须 > 2×20kHz = 40kHz。

    生活类比——车轮倒转：电影里马车轮看起来在倒转，就是因为帧率（采样率）不够快，高速旋转的轮辐被"误认"为慢速倒转——这就是混叠！

    💡 一句话记忆法：采样定理一句话——"每秒至少抓拍最高频率的两倍次数，信号就不会'认错脸'（混叠）！"

    学这个知识点能解决什么： - 正确选择 ADC 采样率，避免信息丢失 - 设计抗混叠滤波器，保证数字化质量 - 理解音频、图像、传感器数据的数字化过程

    3. 严谨核心¶

    数学对象¶

    采样是将连续时间信号转换为离散时间序列的过程，是数字信号处理的第一步。重建（插值）则是从离散样本恢复连续信号的过程。

    基本假设¶

    奈奎斯特-香农采样定理：对于带限信号（最高频率为 \(f_{\max}\)），当采样率 \(f_s > 2 f_{\max}\) 时，可以由样本精确重建原始信号。\(2 f_{\max}\) 称为奈奎斯特率。

    核心公式¶

    理想采样（脉冲串采样）： $\(x_s(t) = x(t) \cdot \sum_{n=-\infty}^{\infty} \delta(t - nT_s) = \sum_{n=-\infty}^{\infty} x(nT_s) \delta(t - nT_s)\)$

    采样信号的频谱： $\(X_s(j\omega) = \frac{1}{T_s} \sum_{k=-\infty}^{\infty} X\left(j\omega - jk\frac{2\pi}{T_s}\right)\)$

    奈奎斯特采样条件： $\(f_s > 2 f_{\max} \quad \Leftrightarrow \quad \omega_s > 2\omega_{\max}\)$

    理想sinc插值重建： $\(x(t) = \sum_{n=-\infty}^{\infty} x[n] \cdot \text{sinc}\left(\frac{t - nT_s}{T_s}\right)\)$

    其中 \(\text{sinc}(u) = \frac{\sin(\pi u)}{\pi u}\)。

    量化信噪比（SQNR，对B比特均匀量化）： $\(\text{SQNR} \approx 6.02B + 1.76 \quad \text{(dB)}\)$

    采样-量化-重建完整流程¶

    ![[assets/sampling_pipeline_ai.png]]

    图：采样与重建的核心链条。先限带、再采样，是避免混叠的关键；重建只能恢复采样中保留下来的信息，不能补回已经混叠或量化丢失的细节。

    采样相关概念对比¶

    概念 定义 条件/影响 解决方法
    奈奎斯特率 \(2f_{\max}\)（最低采样率） 采样不失真的下界 提高采样率
    混叠（Aliasing） 高频折叠到低频 \(f_s < 2f_{\max}\) 时发生 抗混叠滤波
    量化噪声 幅值离散化误差 \(\text{SQNR} \approx 6.02B + 1.76\) dB 增加量化位数
    sinc插值 理想重建方法 需无限长求和，不可物理实现 近似方法（零阶保持等）
    零阶保持 阶梯近似重建 引入高频分量（sinc衰减） 后接平滑滤波器

    适用条件和边界¶

    • 采样定理严格要求信号带限；实际信号通常不是严格带限的，需要抗混叠滤波
    • sinc 重建需要无限长求和，物理上不可实现；实际用零阶保持、线性插值或高阶插值近似
    • 过采样可以降低对抗混叠滤波器的要求，但增加存储和计算成本
    • 量化假设均匀量化；实际 ADC 可能有非线性失真

    4. 方法流程¶

    输入：连续时间信号 \(x(t)\)（带限，最高频率 \(f_{\max}\)）

    处理步骤： 1. 抗混叠滤波：用截止频率为 \(f_{\max}\) 的低通滤滤器限制信号带宽 2. 采样：以采样率 \(f_s > 2f_{\max}\) 对信号等间隔采样，得到 \(x[n] = x(nT_s)\) 3. 量化：将连续幅值映射为 \(B\) 比特的离散值（引入量化噪声 \(\approx 6.02B + 1.76\) dB） 4. 数字处理：对离散信号进行滤波、分析等操作 5. 重建：从离散样本恢复连续信号（零阶保持 → 平滑滤波器，或 sinc 插值）

    输出解释： - 采样后的离散序列 \(x[n]\) 包含原始信号的全部信息（在满足奈奎斯特条件时） - 重建信号与原始信号的差异来自：混叠（采样率不足）、量化误差、重建方法的近似

    常见参数选择： - 采样率：通常取 \(f_s = (2.5 \sim 4) \times f_{\max}\)，留出工程裕量 - 量化位数：语音 16bit、心电 12-24bit、图像 8-16bit - 抗混叠滤波器：截止频率设为 \(f_s/2\) 以下，过渡带越窄越好

    5. Python最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 连续信号模拟：30Hz 正弦波
f0 = 30  # 信号频率 30Hz
t_cont = np.linspace(0, 0.1, 1000)  # "连续"时间轴
x_cont = np.sin(2 * np.pi * f0 * t_cont)

# 情况1：满足奈奎斯特条件 (fs > 2*f0 = 60Hz)
fs1 = 100  # 采样率 100Hz
t1 = np.arange(0, 0.1, 1/fs1)
x1 = np.sin(2 * np.pi * f0 * t1)

# 情况2：不满足奈奎斯特条件 (fs < 2*f0)，产生混叠
fs2 = 40  # 采样率 40Hz
t2 = np.arange(0, 0.1, 1/fs2)
x2 = np.sin(2 * np.pi * f0 * t2)

fig, axes = plt.subplots(2, 1, figsize=(10, 6))
axes[0].plot(t_cont, x_cont, 'b-', label='连续信号 (30Hz)')
axes[0].stem(t1, x1, linefmt='r-', markerfmt='ro', basefmt='k-', label=f'Sampled fs={fs1}Hz')
axes[0].set_title('正确采样：fs=100Hz > 2×30Hz（满足奈奎斯特条件）')
axes[0].legend()
axes[0].set_xlabel('Time (s)')

axes[1].plot(t_cont, x_cont, 'b-', label='连续信号 (30Hz)')
axes[1].stem(t2, x2, linefmt='r-', markerfmt='ro', basefmt='k-', label=f'Sampled fs={fs2}Hz')
axes[1].set_title('混叠：fs=40Hz < 2×30Hz（高频被误认为低频！）')
axes[1].legend()
axes[1].set_xlabel('Time (s)')

plt.tight_layout()
plt.show()

    结果解释： - 上图：采样率 100Hz > 2×30Hz，采样点能准确描绘 30Hz 正弦波的形状 - 下图：采样率 40Hz < 2×30Hz，采样点连成的曲线看起来像一个低频信号——30Hz 的高频成分被"伪装"成了 10Hz 的低频信号，这就是混叠

    6. 常见误区¶

    易错点 正确理解
    采样率大于 \(2f_{\max}\) 就绝对安全 采样定理依赖带限假设；真实信号通常需要抗混叠滤波和工程裕量
    采样点越多，频率分辨率一定越高 频率分辨率主要受观测时长影响；只提高采样率不一定改善低频分辨率
    重建算法可以恢复所有细节 已经混叠或被量化损失的信息不能凭空恢复
    过采样总是越高越好 过高采样率会增加存储、传输、能耗和计算成本

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 如果传感器噪声包含高频成分，而你没有加抗混叠滤波器就直接采样，这些高频噪声会折叠到信号频带内，变成无法去除的"混叠噪声"。这说明采样前的模拟滤波比采样后的数字滤波更重要——因为混叠一旦发生就不可逆。

    一个跨学科迁移场景： 在天文学中，望远镜的 CCD 传感器对光信号进行空间采样。如果星象的角频率超过奈奎斯特频率（由像素间距决定），就会出现"摩尔纹"——这本质上是空间域的混叠。天文学家通过"抖动采样"（dithering）来缓解这个问题。

    一个可做成课程项目的小想法： 设计一个"混叠可视化器"：用户可以调节采样率和信号频率，实时观察采样后的波形和频谱。当采样率低于奈奎斯特率时，频谱中出现明显的混叠分量——直观理解采样定理的物理含义。
