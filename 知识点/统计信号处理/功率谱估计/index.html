    功率谱估计¶

    🎯 初学者简化说明

    一句话理解：功率谱就是告诉你"信号的能量在各个频率上是怎么分布的"。

    你需要重点掌握的：

    1. 功率谱 = 自相关函数的傅里叶变换，即各频率上的能量分布
    2. 周期图法 = 对信号直接FFT取模平方，简单但方差大
    3. Welch法 = 分段加窗再平均，牺牲分辨率换稳定性（最常用）

    作为进阶了解的（期末不要求推导）：

    • AR模型谱估计与Yule-Walker方程的完整推导
    • Cramér-Rao下界的数学证明

    💡 一句话记忆法：功率谱看频率能量，周期图直接算，Welch分段更稳定。

    关联项目 项目一A/B/E

    🎯 初学者简化说明

    功率谱估计的方法可以分为两大类：

    1. 经典方法（不需要概率论）：直接对信号做FFT，计算频谱的能量分布。Welch方法就是把信号分段FFT然后平均——这就是你在项目中会用的方法。

    2. 参数方法（需要随机过程基础）：假设信号是由某个模型（如AR模型）产生的，先估计模型参数，再推导功率谱。这部分了解概念即可，不要求掌握推导。

    对于大二学生，重点掌握周期图法和Welch方法就够了。

    前置知识：[[随机信号基础]]、[[../经典信号处理/傅里叶分析]]

    2. 通俗入口¶

    一个真实问题¶

    你正在用手机听音乐，突然想："这首歌的低音重还是高音重？"你打开手机上的均衡器，看到一根根高低起伏的柱状图——那些柱子告诉你不同频率上的能量大小。功率谱估计就是从原始音频数据中自动算出这个柱状图的方法。

    一个生活类比¶

    功率谱就像体检报告中的频谱分析。把信号比作一个人的健康数据，功率谱就是告诉你"哪个指标偏高、哪个指标偏低"——它把信号的能量按频率"拆分"后展示出来，让你一眼看出信号中哪些频率成分占主导。

    学这个能解决什么？¶

    • 通信系统：信号带宽分析、干扰检测、频谱监测
    • 语音处理：语音信号的频谱分析，共振峰检测
    • 雷达信号处理：多普勒频移估计、杂波谱分析
    • 生物医学：脑电图（EEG）、心电图（ECG）的频谱分析
    • 振动分析：机械故障诊断中的特征频率识别
    • 地震学：地震信号的频率成分分析

    3. 严谨核心¶

    数学对象¶

    功率谱估计（Power Spectral Density Estimation）是从有限长度的观测数据中估计随机信号的功率谱密度（PSD）的方法。功率谱描述了信号功率在频率轴上的分布，是随机信号分析的核心工具。

    基本假设¶

    • 信号是宽平稳随机过程（均值恒定，自相关函数仅与时间差有关）
    • 观测数据为有限长度 \(N\) 的离散时间序列
    • 对于参数方法，假设信号由特定模型（如AR模型）产生

    核心公式¶

    功率谱的理论定义（维纳-辛钦定理）：

    \[S_{xx}(f) = \sum_{m=-\infty}^{\infty} R_{xx}[m] e^{-j2\pi fm}\]

    其中 \(R_{xx}[m]\) 是自相关函数。

    周期图法：

    \[\hat{S}_{xx}(f) = \frac{1}{N} |X(f)|^2 = \frac{1}{N} \left|\sum_{n=0}^{N-1} x[n] e^{-j2\pi fn}\right|^2\]

    Bartlett方法（分段平均周期图）：

    \[\hat{S}_{xx}(f) = \frac{1}{L}\sum_{i=1}^{L} \hat{S}_i(f)\]

    将 \(N\) 点数据分为 \(L\) 段，每段 \(M\) 点，分别计算周期图后取平均。

    Welch方法（重叠分段加窗平均）：

    \[\hat{S}_{xx}(f) = \frac{1}{LU}\sum_{i=1}^{L} \frac{1}{M}\left|\sum_{n=0}^{M-1} w[n] x_i[n] e^{-j2\pi fn}\right|^2\]

    其中 \(w[n]\) 为窗函数，\(U = \frac{1}{M}\sum_{n=0}^{M-1}|w[n]|^2\) 为归一化因子。

    AR模型谱估计：

    \[\hat{S}_{xx}(f) = \frac{\sigma^2}{|1 + \sum_{k=1}^{p} a_k e^{-j2\pi fk}|^2}\]

    其中 \(a_k\) 为AR模型系数，\(p\) 为模型阶数，\(\sigma^2\) 为激励白噪声方差。

    Yule-Walker方程（AR参数估计）：

    \[\begin{bmatrix} R(0) & R(1) & \cdots & R(p-1) \\ R(1) & R(0) & \cdots & R(p-2) \\ \vdots & \vdots & \ddots & \vdots \\ R(p-1) & R(p-2) & \cdots & R(0) \end{bmatrix} \begin{bmatrix} a_1 \\ a_2 \\ \vdots \\ a_p \end{bmatrix} = -\begin{bmatrix} R(1) \\ R(2) \\ \vdots \\ R(p) \end{bmatrix}\]

    关键概念¶

    • 频率分辨率：区分两个相邻频率分量的能力
    • 谱泄漏：有限长数据加窗导致的频谱展宽
    • 方差-偏差折中：分辨率越高（带宽越窄），方差越大

    适用条件与边界¶

    • 周期图法适用于数据长度足够长、对分辨率要求不高的场景
    • Welch方法适用于需要稳定估计的工程场景（推荐默认方法）
    • AR模型适用于短数据、窄带信号，但需要正确选择阶数 \(p\)
    • 所有方法都要求信号是平稳的；非平稳信号需要使用时频分析方法

    4. 方法流程¶

    两大类方法分类¶

    ![[assets/psd_estimation_methods_ai.png]]

    图：功率谱估计方法对比：经典方法（周期图、Welch）与现代方法（参数模型、多窗口）。

    方法 类型 分辨率 方差 计算量 是否需要模型
    周期图法 非参数 低 (受N限制) 大 O(N log N) 否
    Bartlett 非参数 降低 较小 (平均) O(N log N) 否
    Welch 非参数 降低 小 (重叠+窗) O(N log N) 否
    AR模型 参数 高 (可超FFT) 中等 O(p²N) 是 (阶数p)
    ARMA模型 参数 高 中等 O(p²N) 是 (阶数p,q)

    输入¶

    • 长度为 \(N\) 的离散时间信号 \(x[n]\)
    • 采样率 \(f_s\)（用于将频率轴转换为物理单位 Hz）

    处理步骤（以Welch方法为例）¶

    1. 将信号分为 \(L\) 段，每段 \(M\) 点，相邻段重叠 \(D\) 点（通常 \(D = M/2\)）
    2. 对每段加窗（如汉宁窗）：\(x_i[n] \cdot w[n]\)
    3. 对加窗后的每段做FFT并取模平方，得到各段周期图 \(\hat{S}_i(f)\)
    4. 对所有段的周期图取平均：\(\hat{S}_{xx}(f) = \frac{1}{L}\sum_{i=1}^{L} \hat{S}_i(f)\)

    输出¶

    • 功率谱密度估计 \(\hat{S}_{xx}(f)\)，单位通常为 dB/Hz
    • 频率轴 \(f \in [0, f_s/2]\)

    常见参数选择¶

    参数 选择建议 影响
    段长 \(M\) 越长分辨率越高，但可平均的段数越少 分辨率 vs 方差
    重叠率 通常 50%（Welch推荐） 段数越多方差越小
    窗函数 汉宁窗（通用）、汉明窗、凯泽窗 主瓣宽度 vs 旁瓣衰减
    FFT点数 \(N_{\text{fft}}\) ≥ \(M\)，通常取 \(M\) 的整数倍 频率采样密度（不改变分辨率）

    5. Python 最小实验¶

    数据生成¶

    生成测试信号：50Hz + 120Hz 正弦波 + 白噪声，比较周期图法与Welch方法。

    关键函数¶

    • scipy.signal.periodogram() — 直接计算周期图
    • scipy.signal.welch() — Welch分段平均法

    代码与曲线¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib scipy - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib", "scipy"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import signal
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 生成测试信号：50Hz + 120Hz 正弦波 + 白噪声
fs = 1000
N = 2048
t = np.arange(N) / fs
np.random.seed(42)
x = np.sin(2*np.pi*50*t) + 0.5*np.sin(2*np.pi*120*t) + np.random.randn(N)

# 方法1：周期图法（直接FFT，方差大）
f_per, Pxx_per = signal.periodogram(x, fs=fs)

# 方法2：Welch 方法（分段加窗平均，方差小）
f_welch, Pxx_welch = signal.welch(x, fs=fs, nperseg=256, noverlap=128)

fig, axes = plt.subplots(2, 1, figsize=(10, 6))
axes[0].plot(f_per, 10*np.log10(Pxx_per), 'b-', alpha=0.7)
axes[0].set_title('周期图法：方差大，谱线抖动明显')
axes[0].set_xlabel('Frequency (Hz)')
axes[0].set_ylabel('PSD (dB/Hz)')
axes[0].set_xlim([0, 200])
axes[0].grid(True)

axes[1].plot(f_welch, 10*np.log10(Pxx_welch), 'r-', linewidth=2)
axes[1].set_title('Welch 方法：分段平均后方差显著降低，50Hz和120Hz清晰可见')
axes[1].set_xlabel('Frequency (Hz)')
axes[1].set_ylabel('PSD (dB/Hz)')
axes[1].set_xlim([0, 200])
axes[1].grid(True)

plt.tight_layout()
plt.show()

    结果解释¶

    📝 运行说明：需要 numpy、scipy、matplotlib 库。代码展示了周期图法方差大（谱线抖动明显），而 Welch 方法通过分段平均显著降低方差，但频率分辨率略有下降。参数 nperseg 和 noverlap 可调节分辨率与方差的折中。

    • 周期图法（上图）：可以看到50Hz和120Hz处有峰值，但背景谱线剧烈抖动（方差大），难以区分真实峰与噪声波动
    • Welch方法（下图）：50Hz和120Hz的峰值清晰突出，背景平滑，方差显著降低
    • 代价：Welch的频率分辨率略低于周期图法（因为每段长度 \(M < N\)）

    6. 常见误区¶

    公式误解¶

    • ❌ "周期图法的期望等于真实功率谱" — 周期图是真实PSD的渐近无偏估计，但有限数据长度下偏差不一定小，且方差不随 \(N\) 增大而趋于零（方差约等于真实PSD的平方）
    • ❌ "增加数据长度N就能改善一切" — 增加 \(N\) 可提高分辨率，但周期图法的方差不会因此减小；需要用分段平均（Welch）来降方差

    单位误解¶

    • ❌ "功率谱的单位是瓦特(W)" — PSD的单位是功率/频率（如 W/Hz 或 dB/Hz），不是功率本身。对PSD在某频段上积分才是该频段的功率
    • ❌ "dB表示的是功率的绝对值" — dB是相对比值，10*np.log10(Pxx) 中的Pxx必须是线性单位（不能是已取log的值）

    参数误解¶

    • ❌ "FFT点数越大，分辨率越高" — 频率分辨率由数据长度 \(N/f_s\) 决定，不是FFT点数。补零只能增加频率采样密度，不能提高分辨率
    • ❌ "Welch的nperseg越大越好" — nperseg越大单段分辨率越高，但可平均的段数越少，方差反而增大。需要根据数据长度折中选择

    适用条件误解¶

    • ❌ "对非平稳信号也能直接用Welch" — Welch假设信号在分析窗口内是平稳的。对于频率随时间变化的信号（如chirp），需要使用时频分析方法
    • ❌ "AR模型阶数越高越准" — 阶数过高会导致虚假谱峰（过拟合），阶数过低会丢失细节。通常用AIC/BIC准则选择阶数

    7. 启发问题与创新拓展¶

    一个反例或边界问题¶

    周期图法的"不可靠性"：理论上，周期图法是PSD的有偏估计，且当数据长度 \(N \to \infty\) 时，方差不会趋于零——这是违反直觉的。直觉上"数据越多越准"，但对周期图法来说不完全成立。这个反例推动了Bartlett和Welch等分段平均方法的发展。

    一个跨学科迁移场景¶

    金融市场中的功率谱分析：股票价格的波动可以看作随机信号。对股价序列做功率谱分析，可以发现是否存在"周期性"的市场波动（如季节性、经济周期）。虽然金融信号的非平稳性很强，但用短窗口的Welch方法做局部谱分析仍有一定参考价值。

    一个可做成课程项目的小想法¶

    "音频均衡器的自动调节"：用手机录音一段音乐，用Welch方法计算其功率谱，然后根据频谱特征自动设计一个FIR滤波器，实现"自动均衡"——比如将低频过强的录音自动衰减低频。这个项目串联了功率谱估计 → 滤波器设计 → 信号重建的完整链路。
