    近端算子¶

    📚 拓展知识点说明

    定位：本知识点是本科课程的拓展内容，适合学有余力的同学深入学习。 一句话理解：处理"不可导"Optimization问题的工具，把复杂问题简化为简单Projection。 前置要求：Convex优化基础 与本科课程的关系：展开网络中可学习的近端算子

    1. 学习定位¶

    • 建议周次：W8-W9
    • 学习深度：会用即可（掌握常见近端算子的闭式解和使用场景，不必深入证明收敛性）
    • 关联实验：实验6（稀疏信号恢复）、实验7（展开网络设计）
    • 关联项目：自定义近端算子用于结构化稀疏恢复、展开网络中可学习近端算子
    • 前置知识：[[凸优化基础]]（凸函数、对偶理论）
    • 后续延伸：[[../稀疏信号处理/ISTA与FISTA]]、[[ADMM算法]]、[[../深度学习/展开网络]]

    2. 通俗入口¶

    一个真实问题：你想找一个既接近某个目标点、又满足某种约束（比如"只有少数分量非零"）的解。这就像你既想靠近朋友所在的位置，又必须站在人行道上——你需要"投影"到可行区域。

    生活类比：近端算子就像"带约束的折中"。想象你站在草地上（当前位置 \(\mathbf{v}\)），想走到某个地方去降低某种"代价" \(f(\mathbf{x})\)，但你不能离开人行道（约束）。近端算子就是找到人行道上那个"既不太远、又能降低代价"的最佳位置。\(\lambda\) 参数控制你更在意"靠近原位置"还是"降低代价"。

    学这个能解决什么：压缩感知中的 \(l_1\) 正则化（软阈值）、图像去噪中的全变分正则化、矩阵补全中的低秩约束——这些非光滑优化问题的标准处理工具。

    3. 严谨核心¶

    数学对象：闭真凸函数 \(f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}\)，输入向量 \(\mathbf{v} \in \mathbb{R}^n\)，正则化参数 \(\lambda > 0\)

    基本假设： - \(f\) 是闭、真、凸函数（保证近端算子存在且唯一）

    近端算子（Proximal Operator）是处理非Smooth凸优化问题的核心工具。当目标Function包含不可微的正则化项（如 \(\ell_1\) 范数、核范数）时，传统的GradientMethod无法直接应用，近端算子提供了一种将非光滑部分"近端映射"到Smooth空间的途径。

    近端算子的直观理解：

    近端算子 \(\text{prox}_{\lambda f}(\mathbf{v})\) 是一个折中：既要使Function值 \(f(\mathbf{x})\) 尽量小，又要使 \(\mathbf{x}\) 不要离 \(\mathbf{v}\) 太远。Parameter \(\lambda\) 控制两者的权衡。

    核心公式：

    近端算子定义： $\(\text{prox}_{\lambda f}(\mathbf{v}) = \arg\min_{\mathbf{x}} \left\{ f(\mathbf{x}) + \frac{1}{2\lambda} \|\mathbf{x} - \mathbf{v}\|_2^2 \right\}\)$

    Moreau分解： $\(\mathbf{v} = \text{prox}_{\lambda f}(\mathbf{v}) + \lambda \, \text{prox}_{f^*/\lambda}(\mathbf{v}/\lambda)\)$ 其中 \(f^*\) 为 \(f\) 的共轭函数。

    软阈值算子（\(\ell_1\) 范数的近端算子）：

    对 \(f(\mathbf{x}) = \lambda \|\mathbf{x}\|_1\)： $\([\text{prox}_{\lambda \|\cdot\|_1}(\mathbf{v})]_i = \text{sign}(v_i) \max(|v_i| - \lambda, 0) = S_\lambda(v_i)\)$

    投影算子（集合指示函数的近端算子）：

    对 \(f = \mathcal{I}_C\)（Convex集 \(C\) 的指示函数）： $\(\text{prox}_{\lambda \mathcal{I}_C}(\mathbf{v}) = \Pi_C(\mathbf{v}) = \arg\min_{\mathbf{x} \in C} \|\mathbf{x} - \mathbf{v}\|_2^2\)$

    奇异值阈值算子（核范数的近端算子）：

    对 \(f(\mathbf{X}) = \lambda \|\mathbf{X}\|_*\)（核范数），设 \(\mathbf{X} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^T\)，\(\boldsymbol{\Sigma} = \text{diag}(\sigma_1, \ldots, \sigma_r)\)： $\(\text{prox}_{\lambda \|\cdot\|_*}(\mathbf{X}) = \mathbf{U} \, \text{diag}(\max(\sigma_i - \lambda, 0)) \, \mathbf{V}^T\)$

    近端梯度法（Proximal Gradient Method）：

    对问题 \(\min f(\mathbf{x}) + g(\mathbf{x})\)，其中 \(f\) 光滑，\(g\) 非光滑： $\(\mathbf{x}_{k+1} = \text{prox}_{\alpha g}\left(\mathbf{x}_k - \alpha \nabla f(\mathbf{x}_k)\right)\)$

    此即 ISTA（Iterative Shrinkage-Thresholding Algorithm），加速版本 FISTA： $\(\begin{cases} \mathbf{y}_{k+1} = \text{prox}_{\alpha g}\left(\mathbf{x}_k - \alpha \nabla f(\mathbf{x}_k)\right) \\ t_{k+1} = \frac{1 + \sqrt{1 + 4t_k^2}}{2} \\ \mathbf{x}_{k+1} = \mathbf{y}_{k+1} + \frac{t_k - 1}{t_{k+1}}(\mathbf{y}_{k+1} - \mathbf{y}_k) \end{cases}\)$

    适用条件和边界： - 凸+闭+真：近端算子存在且唯一，非扩张 - 非凸函数：近端算子可能非唯一或不存在，需额外假设（如半代数函数） - 近端算子有闭式解时效率极高，否则需要内嵌优化求解

    4. 方法流程¶

    输入：向量 \(\mathbf{v}\)（当前点），正则化参数 \(\lambda\)，函数 \(f\)

    处理步骤： 1. 确认 \(f\) 的类型（\(l_1\) 范数、核范数、指示函数等） 2. 查表或推导对应的近端算子闭式解 3. 将 \(\mathbf{v}\) 和 \(\lambda\) 代入闭式解公式 4. 若无闭式解，则用内嵌优化算法（如内点法）近似求解

    输出如何解释： - 输出 \(\text{prox}_{\lambda f}(\mathbf{v})\) 是"折中解"——在降低 \(f\) 和靠近 \(\mathbf{v}\) 之间取得平衡 - 对 \(l_1\) 范数：输出是 \(\mathbf{v}\) 的"软阈值"版本，小分量被置零（稀疏化） - 对指示函数：输出是 \(\mathbf{v}\) 到集合 \(C\) 的投影

    常见参数如何选择：

    参数 推荐做法 说明
    \(\lambda\) 与正则化强度一致 \(\lambda\) 大→更稀疏/更约束；\(\lambda\) 小→更接近原输入
    在迭代算法中 \(\lambda/L\) ISTA中使用 \(\lambda/L\)，\(L\) 为Lipschitz常数

    常用近端算子对比：

    正则化项 函数形式 近端算子 闭式解 Output特性 典型应用
    L1范数 \(\lambda\lVert x\rVert_1\) 软阈值 \(S_\lambda(v)\) ✅ \(\mathrm{sign}(v)\max(\lvert v\rvert-\lambda,0)\) 稀疏化 LASSO、压缩感知
    L2范数平方 \(\lambda\lVert x\rVert_2^2\) 邻近缩放 \(v/(1+2\lambda)\) ✅ 整体收缩 RidgeRegression
    核范数 \(\lambda\lVert X\rVert_\ast\) 奇异值阈值 ✅ 低秩化 矩阵补全
    指示函数 IC 0 if x∈C, ∞ 投影算子 ΠC 依C而定 约束满足 约束优化
    全变分(TV) \(\lambda\lVert\nabla x\rVert_1\) 近端Iterative ❌ 需迭代 分段常数 图像去噪
    弹性网络 \(\lambda_1\lVert x\rVert_1+\lambda_2\lVert x\rVert_2^2\) 组合软阈值+缩放 ✅ 稀疏+光滑 特征选择

    核心应用场景： - SparseOptimization（\(\ell_1\) Regularization → 软阈值算子） - Low-rankMatrix恢复（核范数正则化 → 奇异值阈值） - Constraint优化（指示函数 → Projection算子） - 复合优化问题的分裂Method（如ADMM、近端Gradient法）

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib - Pyodide环境：先运行下面的安装代码块

import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# 近端算子可视化：软阈值、Projection、奇异值阈值
fig, axes = plt.subplots(1, 3, figsize=(15, 4.5))

# 左图：L1范数的近端算子（软阈值）
v = np.linspace(-3, 3, 500)
lam = 1.0  # 正则化参数
prox_l1 = np.sign(v) * np.maximum(np.abs(v) - lam, 0)
axes[0].plot(v, v, 'k--', linewidth=1.5, label='恒等映射 (无Regularization)')
axes[0].plot(v, prox_l1, 'r-', linewidth=2.5, label=f'软阈值 λ={lam}')
axes[0].fill_between(v, v, prox_l1, alpha=0.15, color='red')
axes[0].axhline(0, color='gray', linewidth=0.5)
axes[0].axvline(0, color='gray', linewidth=0.5)
axes[0].set_title('L1近端算子（软阈值）', fontsize=13)
axes[0].set_xlabel('Input v', fontsize=12)
axes[0].set_ylabel('prox(v)', fontsize=12)
axes[0].legend(fontsize=10)
axes[0].grid(True, alpha=0.3)
# 标注阈值Interval
axes[0].annotate(f'阈值 ±λ', xy=(lam, 0), fontsize=10, color='red',
                 ha='center', va='top')

# 中图：L2范数的近端算子（缩放）
prox_l2 = v / (1 + 2 * lam)
axes[1].plot(v, v, 'k--', linewidth=1.5, label='恒等映射')
axes[1].plot(v, prox_l2, 'b-', linewidth=2.5, label=f'L2近端 λ={lam}')
axes[1].fill_between(v, v, prox_l2, alpha=0.15, color='blue')
axes[1].set_title('L2范数近端算子（整体收缩）', fontsize=13)
axes[1].set_xlabel('Input v', fontsize=12)
axes[1].set_ylabel('prox(v)', fontsize=12)
axes[1].legend(fontsize=10)
axes[1].grid(True, alpha=0.3)

# 右图：集合Projection算子示意（单位球投影）
theta = np.linspace(0, 2*np.pi, 100)
r_ball = 1.0
# 随机点和它们在单位球上的投影
np.random.seed(42)
points = np.array([[1.5, 0.8], [-0.5, 1.5], [0.3, -1.8], [-1.2, -0.5], [0.6, 0.3]])
axes[2].fill(r_ball * np.cos(theta), r_ball * np.sin(theta), alpha=0.2, color='green')
axes[2].plot(r_ball * np.cos(theta), r_ball * np.sin(theta), 'g-', linewidth=2, label='单位球Constraint')
for p in points:
    norm_p = np.linalg.norm(p)
    if norm_p > r_ball:
        proj = p * r_ball / norm_p
        axes[2].plot([p[0], proj[0]], [p[1], proj[1]], 'r--', alpha=0.6)
        axes[2].scatter(*p, c='red', s=50, zorder=5)
        axes[2].scatter(*proj, c='blue', s=50, marker='^', zorder=5)
    else:
        axes[2].scatter(*p, c='blue', s=50, marker='^', zorder=5)
axes[2].scatter([], [], c='red', s=50, label='Primal点')
axes[2].scatter([], [], c='blue', s=50, marker='^', label='Projection点')
axes[2].set_title('单位球Projection算子', fontsize=13)
axes[2].set_xlabel('x', fontsize=12)
axes[2].set_ylabel('y', fontsize=12)
axes[2].set_xlim([-2.5, 2.5])
axes[2].set_ylim([-2.5, 2.5])
axes[2].set_aspect('equal')
axes[2].legend(fontsize=10)
axes[2].grid(True, alpha=0.3)
plt.suptitle('近端算子可视化', fontsize=15)
plt.tight_layout()
plt.show()

    结果解释： - 左图：\(l_1\) 软阈值将绝对值小于 \(\lambda\) 的分量置零，大于 \(\lambda\) 的分量向零收缩 \(\lambda\)——这就是稀疏化的几何机制 - 中图：\(l_2\) 近端算子对所有分量等比例缩小，不会产生稀疏性——说明为什么稀疏优化要选 \(l_1\) 而非 \(l_2\) - 右图：球外的点被投影到球面上（最近点），球内的点保持不动——这就是约束优化中投影算子的直觉

    6. 常见误区¶

    1. 近端算子就是软阈值：软阈值只是 \(l_1\) 范数的近端算子。不同正则化项有不同的近端算子（如核范数→奇异值阈值，指示函数→投影）。

    2. 近端算子和梯度步是一回事：梯度步是光滑函数的线性近似更新，近端算子是非光滑函数的"邻近"映射。对于光滑函数 \(f\)，\(\text{prox}_{\alpha f}\) 当 \(\alpha \to 0\) 时退化为梯度步，但两者本质不同。

    3. \(\lambda\) 越大近端算子输出越稀疏：对 \(l_1\) 范数的确如此（阈值越大，置零越多），但对一般的 \(f\) 不能一概而论。\(\lambda\) 控制的是"靠近输入点"vs"降低函数值"的权衡。

    4. 近端算子只适用于凸函数：非凸近端算子（如对 SCAD、MCP 等非凸惩罚项）也存在，但可能非唯一。非凸近端梯度法在实践中也有应用，但理论保证较弱。

    5. 近端算子计算一定很快：只有有闭式解的近端算子（如软阈值、奇异值阈值）计算才快。全变分（TV）等正则化的近端算子没有闭式解，需要用迭代方法求解，计算代价可能很高。

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 考虑 \(f(\mathbf{x}) = \|\mathbf{x}\|_0\)（\(l_0\) "范数"，非零分量个数）。这不是凸函数，其"近端算子"是硬阈值（保留最大的 \(k\) 个分量，其余置零）。硬阈值是非唯一的（当分量绝对值相同时），且近端梯度法用硬阈值替代软阈值后，不再有全局收敛保证。这说明近端算子理论对非凸推广需要额外假设。

    一个跨学科迁移场景： 在推荐系统中，用户-物品评分矩阵是稀疏的（大量缺失值）。矩阵补全问题可以建模为核范数正则化优化，其近端算子是奇异值阈值（SVT）。Netflix Prize 的许多顶级算法都用到了这个近端算子。

    一个可做成课程项目的小想法： "可学习的近端算子"——传统近端算子由数学公式定义（如软阈值），但如果将阈值参数 \(\lambda\) 甚至阈值函数的形式都设为可学习参数（用神经网络参数化），嵌入到展开网络中，能否在图像去噪任务上超越传统方法？实现一个简单的LISTA网络并与ISTA对比。
