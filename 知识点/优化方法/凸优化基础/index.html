    凸优化基础¶

    📚 拓展知识点说明

    定位：本知识点是本科课程的拓展内容，适合学有余力的同学深入学习。 一句话理解：凸函数只有一个最低点，所以一定能找到最优解。 前置要求：线性代数、高等数学 与本科课程的关系：维纳滤波就是二次凸优化的特例

    1. 学习定位¶

    • 建议周次：W7-W8
    • 学习深度：会用即可（理解凸性判定和标准问题形式即可，不必深入证明细节）
    • 关联实验：实验4（维纳滤波器设计）、实验6（稀疏信号恢复）
    • 关联项目：带约束的波束赋形优化、投资组合优化建模
    • 前置知识：线性代数（矩阵论、正定性、特征值）、高等数学（多元微积分、极值理论）
    • 后续延伸：[[近端算子]]、[[ADMM算法]]、[[梯度下降与变体]]

    2. 通俗入口¶

    一个真实问题：你要在一块丘陵地形上找到最低点扎营。如果地形只有一个碗形凹坑（凸函数），随便从哪里出发滚下去都能到达最低点。但如果地形有多个凹坑（非凸函数），你可能滚到一个坑底就停了——那个坑不一定是整块地形的最低点。

    生活类比：凸优化就像在一个完美的碗里找最低点——不管球从碗边哪个位置放下，最终都会滚到碗底唯一的最低点。非凸优化则像在一个鸡蛋盒里找最低点——球可能卡在某个局部凹处。

    学这个能解决什么：信号处理中的维纳滤波、卡尔曼滤波、波束赋形等核心问题都可以归结为凸优化问题。理解凸性就能判断一个问题是否有"全局最优解"，以及应该用什么算法来求解。

    3. 严谨核心¶

    数学对象：凸集 \(C \subseteq \mathbb{R}^n\)、凸函数 \(f: \mathbb{R}^n \to \mathbb{R}\)、拉格朗日乘子 \(\boldsymbol{\lambda}, \boldsymbol{\nu}\)

    基本假设： - 目标函数 \(f_0\) 是凸函数 - 不等式约束函数 \(f_i\) 是凸函数 - 等式约束是仿射函数（线性）

    标准凸优化问题形式： $\(\begin{aligned} \min_{\mathbf{x}} \quad & f_0(\mathbf{x}) \\ \text{s.t.} \quad & f_i(\mathbf{x}) \leq 0, \quad i = 1, \ldots, m \\ & a_i^T \mathbf{x} = b_i, \quad i = 1, \ldots, p \end{aligned}\)$ 其中 \(f_0, f_1, \ldots, f_m\) 均为凸函数。

    基本概念： - 凸集：集合 \(C\) 是凸集，当且仅当 \(\forall \mathbf{x}, \mathbf{y} \in C, \forall \theta \in [0,1]\)，有 \(\theta \mathbf{x} + (1-\theta)\mathbf{y} \in C\) - 凸函数：\(f(\theta \mathbf{x} + (1-\theta)\mathbf{y}) \leq \theta f(\mathbf{x}) + (1-\theta)f(\mathbf{y})\) - 仿射集、锥、多面体等特殊凸集 - 共轭函数、对偶理论

    凸优化问题的分类： - 线性规划（LP） - 二次规划（QP） - 二次约束二次规划（QCQP） - 二阶锥规划（SOCP） - 半定规划（SDP） - 几何规划（GP）

    核心公式：

    凸函数的一阶条件： $\(f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^T (\mathbf{y} - \mathbf{x})\)$

    凸函数的二阶条件： $\(\nabla^2 f(\mathbf{x}) \succeq 0\)$ 即 Hessian 矩阵半正定。

    拉格朗日函数： $\(L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) = f_0(\mathbf{x}) + \sum_{i=1}^m \lambda_i f_i(\mathbf{x}) + \sum_{i=1}^p \nu_i (a_i^T \mathbf{x} - b_i)\)$

    拉格朗日对偶函数： $\(g(\boldsymbol{\lambda}, \boldsymbol{\nu}) = \inf_{\mathbf{x}} L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu})\)$

    弱对偶性： $\(d^* \leq p^*\)$ 其中 \(d^* = \sup_{\boldsymbol{\lambda} \succeq 0, \boldsymbol{\nu}} g(\boldsymbol{\lambda}, \boldsymbol{\nu})\) 为对偶最优值，\(p^*\) 为原问题最优值。

    强对偶性（Slater条件）：

    若原问题是凸问题且存在严格可行点，则强对偶成立： $\(d^* = p^*\)$

    KKT条件（最优性必要充分条件）： $\(\begin{cases} f_i(\mathbf{x}^*) \leq 0, \quad i = 1, \ldots, m & \text{(原始可行性)} \\ a_i^T \mathbf{x}^* = b_i, \quad i = 1, \ldots, p & \text{(原始可行性)} \\ \lambda_i^* \geq 0, \quad i = 1, \ldots, m & \text{(对偶可行性)} \\ \lambda_i^* f_i(\mathbf{x}^*) = 0, \quad i = 1, \ldots, m & \text{(互补松弛)} \\ \nabla f_0(\mathbf{x}^*) + \sum_{i=1}^m \lambda_i^* \nabla f_i(\mathbf{x}^*) + \sum_{i=1}^p \nu_i^* a_i = 0 & \text{(驻点条件)} \end{cases}\)$

    适用条件和边界： - 凸+Slater条件：强对偶成立，KKT是充要条件 - 凸但无Slater条件：强对偶不一定成立 - 非凸问题：KKT只是必要条件，不充分

    4. 方法流程¶

    输入：目标函数和约束条件的数学描述

    处理步骤： 1. 问题建模：将实际问题抽象为标准凸优化形式（确定目标函数、约束条件） 2. 凸性验证：检查目标函数和约束是否为凸函数（二阶条件、运算封闭性） 3. 问题分类：确定属于LP/QP/QCQP/SOCP/SDP中的哪一类 4. 选择求解器：根据问题类型选择合适的算法（内点法、梯度法等） 5. 求解与验证：运行求解器，检查KKT条件是否满足

    输出如何解释： - 最优解 \(\mathbf{x}^*\)：满足所有约束的全局最优决策 - 最优值 \(p^* = f_0(\mathbf{x}^*)\)：可达到的最优性能 - 对偶变量 \(\boldsymbol{\lambda}^*\)：约束的"影子价格"，反映约束的紧度

    常见参数如何选择：

    参数 推荐做法 说明
    求解器精度 \(\epsilon\) 1e-6 ~ 1e-8 根据应用需求调整
    最大迭代次数 内点法通常 20~50 步 内点法收敛快
    初始点 可行点或原点 某些求解器需要

    凸优化问题类型对比：

    问题类型 目标函数 约束条件 典型求解器 复杂度 典型应用
    线性规划(LP) 线性 线性等式/不等式 Simplex, Interior Point 多项式 资源分配、调度
    二次规划(QP) 二次凸 线性 Interior Point, Active Set 多项式 SVM、投资组合
    QCQP 二次凸 二次凸 Interior Point 多项式 波束形成
    SOCP 线性 二阶锥约束 Interior Point 多项式 鲁棒优化
    SDP 线性 矩阵不等式 Interior Point 多项式 控制理论、组合优化
    一般凸优化 凸函数 凸函数不等式 内点法、梯度法 依赖结构 信号处理、ML

import numpy as np
import matplotlib

matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False

# ConvexFunction与Non-convex函数可视化对比
x = np.linspace(-3, 5, 300)

fig, axes = plt.subplots(1, 3, figsize=(15, 4.5))
# 左图：凸函数示例
f_convex = x**2 - 2*x + 1  # (x-1)^2
axes[0].plot(x, f_convex, 'b-', linewidth=2.5)
# 切线ValidationConvex性
x0, x1 = -1, 3
for xi in [x0, x1]:
    g = 2*xi - 2  # 导数
    tangent = f_convex[np.argmin(np.abs(x - xi))] + g * (x - xi)
    axes[0].plot(x, tangent, 'r--', alpha=0.6)
# 弦上界
theta = 0.6
x_theta = theta * x0 + (1 - theta) * x1
f_theta = theta * (x0**2 - 2*x0 + 1) + (1 - theta) * (x1**2 - 2*x1 + 1)
axes[0].plot([x0, x1], [x0**2 - 2*x0 + 1, x1**2 - 2*x1 + 1], 'g--', linewidth=1.5, label='弦(线性插值)')
axes[0].scatter([x_theta], [x_theta**2 - 2*x_theta + 1], s=80, c='blue', zorder=5, label='f(θx+(1-θ)y)')
axes[0].scatter([x_theta], [f_theta], s=80, c='green', marker='^', zorder=5, label='θf(x)+(1-θ)f(y)')
axes[0].set_title('ConvexFunction: f(x) = (x-1)²', fontsize=13)
axes[0].set_xlabel('x')
axes[0].set_ylabel('f(x)')
axes[0].legend(fontsize=9)
axes[0].grid(True, alpha=0.3)
# 中图：非ConvexFunction示例
f_nonconvex = np.sin(x) + 0.3 * x  # 非凸函数
axes[1].plot(x, f_nonconvex, 'r-', linewidth=2.5)
axes[1].set_title('非ConvexFunction: f(x) = sin(x) + 0.3x', fontsize=13)
axes[1].set_xlabel('x')
axes[1].set_ylabel('f(x)')
axes[1].grid(True, alpha=0.3)
# 右图：ConstraintOptimization示意（ConvexFeasible Region）
theta_c = np.linspace(0, 2*np.pi, 100)
# 椭圆等高线
xx = np.linspace(-2, 4, 200)
yy = np.linspace(-1, 3, 200)
X, Y = np.meshgrid(xx, yy)
Z = (X - 1)**2 + 2*(Y - 1)**2
axes[2].contour(X, Y, Z, levels=10, cmap='coolwarm', alpha=0.5)
# ConvexConstraint：圆
r = 1.5
cx, cy = 1, 1
x_circle = cx + r * np.cos(theta_c)
y_circle = cy + r * np.sin(theta_c)
axes[2].fill(x_circle, y_circle, alpha=0.2, color='green', label='Feasible Region')
axes[2].plot(x_circle, y_circle, 'g-', linewidth=2)
axes[2].plot(1, 1, 'r*', markersize=15, label='Optimal Solution')
axes[2].set_title('ConstraintConvexOptimization示意', fontsize=13)
axes[2].set_xlabel('x')
axes[2].set_ylabel('y')
axes[2].set_xlim([-2, 4])
axes[2].set_ylim([-1, 3])
axes[2].legend(fontsize=10)
axes[2].set_aspect('equal')
axes[2].grid(True, alpha=0.3)
plt.suptitle('ConvexOptimization基本概念可视化', fontsize=15)
plt.tight_layout()
plt.show()

    结果解释： - 左图展示了凸函数的关键性质：切线总在函数下方（一阶条件），弦上的函数值不小于线性插值（Jensen不等式） - 中图的非凸函数有多个局部极值，说明为什么非凸优化比凸优化困难得多 - 右图展示了带约束凸优化的几何直觉：在凸可行域（绿色圆）上最小化凸目标函数（等高线），最优解在可行域边界上

    6. 常见误区¶

    1. 凸函数一定是"上凸"的：数学中"凸函数"是"下凸"（碗形向上），而非日常语言中的"凸起"形状。\(f(x) = x^2\) 是凸函数，\(f(x) = -x^2\) 是凹函数。

    2. 非线性问题一定是非凸的：二次函数 \(f(x) = x^2\) 是非线性的但也是凸的。凸性取决于Hessian矩阵是否半正定，而非是否线性。

    3. 对偶问题总是等价于原问题：只有在Slater条件成立时（凸问题+严格可行点），强对偶才成立。非凸问题通常只有弱对偶 \(d^* \leq p^*\)，对偶间隙可能很大。

    4. KKT条件对所有优化问题都是充要的：KKT条件对凸问题是充要的（在约束规范下），但对非凸问题只是必要条件。

    5. 凸优化问题一定很容易求解：虽然凸优化理论上是"可解的"，但大规模问题（如百万维的SDP）的计算代价仍然很高。

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 考虑问题 \(\min x^2\) s.t. \(x^2 \leq 0\)。这是一个凸问题，但可行集只有单点 \(\{0\}\)，且不存在严格可行点（Slater条件不满足）。虽然此例中强对偶仍成立，但可以构造更复杂的例子使Slater条件不满足时强对偶失败。这说明Slater条件不是"多余的假设"。

    一个跨学科迁移场景： 在经济学中，消费者效用最大化问题可以建模为凸优化：在预算约束（线性不等式）下最大化凹效用函数（等价于最小化负效用——凸函数）。KKT条件中的对偶变量恰好对应商品的"影子价格"，为价格理论提供了数学基础。

    一个可做成课程项目的小想法： "信号处理中的凸优化建模竞赛"——给定一个实际信号处理问题（如多天线波束赋形、信号去噪、信道均衡），让学生将其建模为标准凸优化形式（LP/QP/SOCP），使用CVXPY求解，并比较不同建模方式（原问题vs对偶问题）的求解效率。
