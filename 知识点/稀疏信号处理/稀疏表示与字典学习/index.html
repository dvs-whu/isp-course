    稀疏表示与字典学习 (Sparse Representation & Dictionary Learning)¶

    1. 学习定位¶

    📚 拓展知识点说明

    定位：本知识点是本科课程的拓展内容，适合学有余力的同学深入学习。 一句话理解：用少量基信号（字典原子）的组合表示复杂信号。 前置要求：线性代数基础（矩阵运算、向量空间）。 与本科课程的关系：与傅里叶分析密切相关——傅里叶基就是最简单的字典，字典学习是其推广。

    前置知识： - 线性代数：矩阵分解（SVD）、向量空间与基 - 优化基础：凸优化、ℓ₁/ℓ₀ 范数约束优化

    相关知识点： - [[压缩感知]] — 稀疏信号模型 - [[ISTA与FISTA]] — 稀疏编码算法 - [[../时频分析/小波变换]] — 解析字典 - [[../深度学习/深度学习基础]] — 特征学习 - [[../图像信号处理/图像去噪]] — 稀疏去噪模型 - [[../经典信号处理/傅里叶分析]] — 傅里叶基作为解析字典

    2. 通俗入口¶

    想象你要用乐高积木拼一座城堡。你有一个巨大的积木库（字典），里面有各种形状的积木（原子）。你不需要用到所有积木——只需要挑出少数几块关键的，就能拼出想要的样子。这就是稀疏表示的核心：用少量"积木"的线性组合来表示一个复杂信号。

    字典学习更进一步：与其用别人给你的积木库，不如从大量城堡照片中自动学习出一套最优的积木。这样学到的积木更贴合你的任务，用更少的积木就能拼得更好。

    生活类比： - 傅里叶变换 = 用标准的"正弦波积木"拼信号（固定的字典） - 字典学习 = 从数据中自动发现更好的"积木"（自适应的字典）

    为什么"过完备"有用？ 如果积木种类太少（字典原子数 = 信号维度），每个信号只能用一种方式表示。如果积木种类很多（原子数 >> 维度），同一个信号就有多种表示方式，你可以从中挑最稀疏的那个。

    3. 严谨核心¶

    定义与概念¶

    稀疏表示是指将信号表示为一组过完备原子（字典元素）的稀疏线性组合。与传统的正交基变换不同，过完备字典提供了更大的灵活性，能更好地适应信号的内在结构。

    字典学习是指从训练数据中自适应地学习一组最优的字典原子，使信号能以最稀疏的方式表示。

    关键概念： - 过完备字典：\(\mathbf{D} \in \mathbb{R}^{n \times K}\)，\(K > n\)，列向量（原子）线性相关 - 稀疏编码：给定字典 \(\mathbf{D}\) 和信号 \(\mathbf{x}\)，求解稀疏系数 \(\boldsymbol{\alpha}\) - K-SVD算法：经典的字典学习算法，交替进行稀疏编码和字典更新 - 相干字典：原子间相干性低的字典 - 结构化字典：具有特定数学结构的字典（如小波基、Gabor框架）

    核心公式¶

    稀疏表示的数学模型：

    \[\mathbf{x} \approx \mathbf{D}\boldsymbol{\alpha}, \quad \|\boldsymbol{\alpha}\|_0 \leq K\]

    其中 \(\mathbf{D} = [\mathbf{d}_1, \mathbf{d}_2, \ldots, \mathbf{d}_K]\) 为字典矩阵，\(\boldsymbol{\alpha}\) 为稀疏系数向量。

    稀疏编码问题（\(\ell_0\) 范数）：

    \[\min_{\boldsymbol{\alpha}} \|\boldsymbol{\alpha}\|_0 \quad \text{s.t.} \quad \|\mathbf{x} - \mathbf{D}\boldsymbol{\alpha}\|_2 \leq \epsilon\]

    松弛为凸问题（\(\ell_1\) 范数）：

    \[\min_{\boldsymbol{\alpha}} \|\boldsymbol{\alpha}\|_1 \quad \text{s.t.} \quad \|\mathbf{x} - \mathbf{D}\boldsymbol{\alpha}\|_2 \leq \epsilon\]

    字典学习的联合优化问题：

    \[\min_{\mathbf{D}, \boldsymbol{\alpha}_i} \sum_{i=1}^{N} \|\mathbf{x}_i - \mathbf{D}\boldsymbol{\alpha}_i\|_2^2 + \lambda \|\boldsymbol{\alpha}_i\|_1 \quad \text{s.t.} \quad \|\mathbf{d}_j\|_2 = 1, \forall j\]

    K-SVD字典更新步骤（对第 \(j\) 列原子）：

    \[\min_{\mathbf{d}_j, \boldsymbol{\alpha}^j_T} \|\mathbf{E}_j - \mathbf{d}_j \boldsymbol{\alpha}^j_T\|_F^2\]

    其中 \(\mathbf{E}_j = \mathbf{X} - \sum_{k \neq j} \mathbf{d}_k \boldsymbol{\alpha}^k_T\) 为残差矩阵。

    字典类型对比¶

    字典类型 代表方法 自适应性 计算复杂度 稀疏表示能力 典型应用
    解析字典 DCT, 小波, Gabor 无 低 中 通用信号处理
    学习字典 K-SVD, MOD 强 高 高 图像去噪、分类
    结构化字典 联合字典, 层次字典 中 中 高 超分辨率、融合
    卷积字典 Convolutional DL 中 高 高 图像/视频处理

    典型应用¶

    • 图像去噪：利用稀疏表示去除噪声，保留信号结构
    • 图像超分辨率：学习高低分辨率字典对，重建高分辨率图像
    • 人脸识别：基于稀疏表示的分类（SRC, Sparse Representation based Classification）
    • 图像压缩：JPEG2000中使用小波稀疏表示
    • 源分离：语音分离、盲源分离
    • 特征提取：字典原子可作为数据的特征表示

    4. 方法流程¶

    K-SVD 字典学习流程图¶

    完整算法流程：

    输入：训练信号集合 \(\{\mathbf{x}_i\}_{i=1}^N\)，字典原子数 \(K\)，稀疏度 \(T_0\)

    Step 1：初始化字典 - 随机选取训练信号作为初始原子，或使用解析字典（DCT、小波） - 列归一化：\(\mathbf{d}_j \leftarrow \mathbf{d}_j / \|\mathbf{d}_j\|_2\)

    Step 2：稀疏编码阶段 对每个训练信号 \(\mathbf{x}_i\)，用 OMP 或 ISTA 求解： $\(\min_{\boldsymbol{\alpha}_i} \|\mathbf{x}_i - \mathbf{D}\boldsymbol{\alpha}_i\|_2^2 \quad \text{s.t.} \quad \|\boldsymbol{\alpha}_i\|_0 \leq T_0\)$

    Step 3：字典更新阶段（K-SVD） 对每个原子 \(j = 1, 2, \ldots, K\)： 1. 找出使用原子 \(j\) 的信号索引集 \(\omega_j = \{i : \alpha_i(j) \neq 0\}\) 2. 计算残差矩阵 \(\mathbf{E}_j = \mathbf{X}_{\omega_j} - \sum_{k \neq j} \mathbf{d}_k \boldsymbol{\alpha}^k_{\omega_j}\) 3. 对 \(\mathbf{E}_j\) 做 SVD 分解：\(\mathbf{E}_j = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^T\) 4. 更新原子：\(\mathbf{d}_j = \mathbf{u}_1\)（第一列左奇异向量） 5. 更新系数：\(\boldsymbol{\alpha}^j_{\omega_j} = \sigma_1 \mathbf{v}_1\)

    Step 4：收敛判断 - 目标函数变化小于阈值，或达到最大迭代次数

    关键参数选择： - \(K/n\)（过完备比）：通常 2~4 倍 - \(T_0\)（稀疏度）：根据信号特性，通常 3~10 - 迭代次数：通常 20~50 次

    5. Python 最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib - Pyodide环境：先运行下面的安装代码块

    # Pyodide环境：安装依赖库
import micropip
await micropip.install(["numpy", "matplotlib"])

    # === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 过完备字典下的稀疏编码演示
np.random.seed(0)
n, K = 20, 60  # 信号维度n=20, 字典原子数K=60（过完备）
D = np.random.randn(n, K)
D = D / np.linalg.norm(D, axis=0)  # 列归一化字典

# 生成稀疏信号：仅3个非零系数
x_true_coeff = np.zeros(K)
x_true_coeff[[5, 20, 45]] = [2.0, -1.5, 1.0]
x = D @ x_true_coeff + 0.05 * np.random.randn(n)  # 含噪信号

# ISTA求解稀疏编码 min ||x - D@alpha||^2 + lam*||alpha||_1
def soft_thresh(v, t):
    return np.sign(v) * np.maximum(np.abs(v) - t, 0)

lam = 0.3
L = np.max(np.linalg.eigvalsh(D.T @ D))
alpha = np.zeros(K)
costs = []
for i in range(300):
    grad = D.T @ (D @ alpha - x)
    alpha = soft_thresh(alpha - grad / L, lam / L)
    cost = 0.5 * np.linalg.norm(x - D @ alpha)**2 + lam * np.linalg.norm(alpha, 1)
    costs.append(cost)

# 可视化：稀疏系数与收敛曲线
fig, axes = plt.subplots(1, 3, figsize=(15, 4))
# 真实稀疏系数
axes[0].stem(range(K), x_true_coeff, linefmt='b-', markerfmt='bo', basefmt='k-')
axes[0].set_title('真实稀疏系数', fontsize=13)
axes[0].set_xlabel('原子索引')
# 恢复的稀疏系数
axes[1].stem(range(K), alpha, linefmt='r-', markerfmt='ro', basefmt='k-')
axes[1].set_title('ISTA恢复的稀疏系数', fontsize=13)
axes[1].set_xlabel('原子索引')
# 收敛曲线
axes[2].semilogy(costs, 'g-', linewidth=2)
axes[2].set_title('Convergence Curve', fontsize=13)
axes[2].set_xlabel('Iterations')
axes[2].set_ylabel('目标函数值 (log)')
axes[2].grid(True, alpha=0.3)
plt.suptitle('过完备字典稀疏编码', fontsize=15, y=1.02)
plt.tight_layout()
plt.show()

    6. 常见误区¶

    误区 正确理解
    字典越大（原子越多）越好 过完备比过大导致计算量剧增、相干性增加，且过拟合风险上升；通常 2~4 倍即可
    ℓ₀ 和 ℓ₁ 稀疏编码效果完全一样 ℓ₁ 是 ℓ₀ 的凸松弛，在满足 RIP/互不相干条件时等价，但实际中可能有差异
    K-SVD 一定能找到全局最优字典 K-SVD 是贪心算法，初始化影响结果，可能陷入局部最优
    学习字典一定比解析字典好 学习字典需要大量训练数据，数据不足时解析字典更稳健
    稀疏表示只对图像有用 稀疏表示适用于任何信号类型：语音、雷达、通信、生物信号等
    字典学习和深度学习是割裂的 字典学习是深度学习的思想先驱——展开的 ISTA 网络就是字典学习的深度化

    7. 启发问题与创新拓展¶

    思考题¶

    1. 为什么 ℓ₁ 范数能代替 ℓ₀ 范数实现稀疏？RIP 条件和互不相干条件分别是什么含义？
    2. K-SVD 中为什么用 SVD 来更新原子？能否用其他矩阵分解方法替代？
    3. 过完备字典的原子之间应该尽量正交还是可以相关？各有什么优劣？
    4. 如何将字典学习与深度学习结合？展开的优化算法网络有什么优势？

    创新拓展方向¶

    • 卷积字典学习：将字典学习推广到卷积结构，适合图像/视频处理
    • 深度展开网络：将 ISTA/ADMM 展开为可学习的神经网络（LISTA、ADMM-Net）
    • 结构化稀疏：Group Sparsity、树结构稀疏、低秩+稀疏联合模型
    • 在线字典学习：流数据场景下的增量字典更新
    • 生成模型视角：VAE/GAN 中的隐空间可视为学习到的"字典"，稀疏编码与隐变量推断的联系
