    图滤波器¶

    1. 学习定位¶

    • 建议周次：W7-W8
    • 学习深度：理解即可（掌握频域滤波与空域多项式滤波的等价关系，会用 Python 实现简单图滤波器）
    • 关联实验：实验5（图信号滤波与去噪）
    • 关联项目：脑网络信号去噪、传感器网络数据平滑
    • 前置知识：[[图傅里叶变换]]（GFT/IGFT）、[[图论基础]]（拉普拉斯矩阵、特征分解）
    • 后续延伸：[[图卷积网络]]（可学习的图滤波）、多分辨率图信号分析

    2. 通俗入口¶

    一个真实问题：在一个城市交通网络中，每个路口有一个交通流量传感器。传感器采集的信号包含真实的交通模式和随机噪声。如何在保留交通趋势的同时去除噪声？

    生活类比：经典信号处理中，低通滤波器通过"抹平"高频抖动来去噪——图滤波器做的是同样的事，只不过"频率"不再由时间周期定义，而是由图的拉普拉斯矩阵特征值定义。小特征值对应"相邻节点变化平缓"的低频模式，大特征值对应"相邻节点剧烈变化"的高频模式。

    学这个能解决什么：图信号去噪与平滑、图神经网络中卷积层的核心组件、脑网络/传感器网络中的信号滤波、多分辨率图信号分析。

    3. 严谨核心¶

    数学对象：图信号 \(\mathbf{x} \in \mathbb{R}^N\)，拉普拉斯矩阵 \(L \in \mathbb{R}^{N \times N}\)，频率响应函数 \(h(\lambda)\)

    基本假设： - 图结构已知（邻接矩阵或拉普拉斯矩阵给定） - 图信号在图频域下具有可区分的频率成分

    图滤波器（Graph Filter） 是对图信号进行频域调制的线性算子。给定频率响应函数 \(h(\lambda)\)，图滤波器在图傅里叶域中对各频率分量进行加权。

    图滤波器可用拉普拉斯矩阵的多项式近似：

    \[H(L) = \sum_{k=0}^{K} h_k L^k\]

    其中 \(h_0, h_1, \dots, h_K\) 为滤波器系数。这种多项式滤波器具有 \(K\)-局部性，即每个顶点的输出仅依赖于其 \(K\)-跳邻域内的信号。

    • 低通滤波器：保留小特征值对应分量（平滑信号）
    • 高通滤波器：保留大特征值对应分量（边缘/突变）
    • 分布式实现：多项式滤波可通过 \(K\) 次局部消息传递实现，适合分布式计算

    核心公式：

    空域表示（多项式形式）：

    \[\mathbf{y} = H(L) \mathbf{x} = \sum_{k=0}^{K} h_k L^k \mathbf{x}\]

    频域表示：

    \[\hat{y}(\lambda_l) = h(\lambda_l) \hat{x}(\lambda_l)\]

    Chebyshev 近似（常用）：

    \[H(L) \approx \sum_{k=0}^{K} \theta_k T_k(\tilde{L})\]

    其中 \(\tilde{L} = \frac{2L}{\lambda_{\max}} - I\) 为缩放后的拉普拉斯，\(T_k\) 为 Chebyshev 多项式。

    图滤波器类型对比：

    滤波器类型 频率响应 \(h(\lambda)\) 保留分量 作用 典型应用
    低通滤波器 小 \(\lambda\) 处增益大，大 \(\lambda\) 衰减 低频（平滑分量） 平滑/去噪 社区检测、信号去噪
    高通滤波器 大 \(\lambda\) 处增益大 高频（边缘/突变） 边缘检测 异常检测
    带通滤波器 中间频段增益大 特定频段 特征选择 多尺度分析
    全通滤波器 所有频率增益相同 全部 相位调整 相位校正

    适用条件和边界： - 图滤波器要求图结构已知且拉普拉斯矩阵可计算 - 多项式滤波的局部性阶数 \(K\) 越大，逼近精度越高但计算量越大 - 频域方法需要完整特征分解 \(O(N^3)\)，大规模图不可行；多项式方法避免特征分解

    4. 方法流程¶

    输入：图信号 \(\mathbf{x} \in \mathbb{R}^N\)，拉普拉斯矩阵 \(L\)，滤波器系数 \(\{h_k\}_{k=0}^K\)（或频率响应 \(h(\lambda)\)）

    处理步骤： 1. 频域方法：先做图傅里叶变换 \(\hat{\mathbf{x}} = U^T \mathbf{x}\)，再逐频率乘以 \(h(\lambda_l)\)，最后逆变换 \(\mathbf{y} = U \hat{\mathbf{y}}\) 2. 空域方法：直接计算 \(\mathbf{y} = \sum_{k=0}^{K} h_k L^k \mathbf{x}\)，通过 \(K\) 次矩阵-向量乘法实现 3. Chebyshev 方法：利用 Chebyshev 多项式递推计算，避免显式计算 \(L^k\)

    输出如何解释： - 滤波后信号 \(\mathbf{y}\)：低通滤波后相邻节点值趋于一致（去噪），高通滤波后突变位置被突出（边缘检测）

    常见参数如何选择：

    参数 推荐值 说明
    滤波器阶数 \(K\) 2~5 \(K\) 越大局部感受野越大，计算量线性增长
    低通衰减系数 \(\alpha\) 1~5 \(h(\lambda) = e^{-\alpha\lambda}\)，\(\alpha\) 越大衰减越快
    采样点数（Chebyshev） 10~30 用于逼近连续频率响应

    5. Python最小实验¶

    运行前准备：本代码需要以下Python库，请确保已安装： - 本地运行：pip install numpy matplotlib

    import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 构建一个含4个顶点的图
N = 4
A = np.array([
    [0, 1, 0.5, 0],
    [1, 0, 1, 0],
    [0.5, 1, 0, 0.8],
    [0, 0, 0.8, 0]
])

# 计算归一化拉普拉斯矩阵
D = np.diag(np.sum(A, axis=1))
D_inv_sqrt = np.diag(1.0 / np.sqrt(np.sum(A, axis=1)))
L_sym = D_inv_sqrt @ (D - A) @ D_inv_sqrt

# 特征分解
eigenvalues, U = np.linalg.eigh(L_sym)
print("特征值（频率）:", np.round(eigenvalues, 4))

# 定义图信号（含噪声）
np.random.seed(42)
x_clean = np.array([1.0, 0.8, 0.2, 0.0])  # 平滑信号
noise = 0.3 * np.random.randn(N)  # 高频噪声
x_noisy = x_clean + noise
print("干净信号:", x_clean)
print("含噪信号:", np.round(x_noisy, 4))

# 图傅里叶变换
x_hat = U.T @ x_noisy
print("图傅里叶系数:", np.round(x_hat, 4))

# 设计低通滤波器：保留低频，衰减高频
# h(λ) = exp(-α * λ)，α越大衰减越快
alpha = 2.0
h = np.exp(-alpha * eigenvalues)
print("滤波器频率响应 h(λ):", np.round(h, 4))

# 应用滤波器：在频域中相乘
y_hat = h * x_hat
print("滤波后的图傅里叶系数:", np.round(y_hat, 4))

# 逆变换得到滤波后的信号
y = U @ y_hat
print("滤波后的信号:", np.round(y, 4))

# 计算去噪效果
print("\n去噪前 MSE:", np.round(np.mean((x_noisy - x_clean)**2), 4))
print("去噪后 MSE:", np.round(np.mean((y - x_clean)**2), 4))

# 可视化
fig, axes = plt.subplots(1, 3, figsize=(15, 4))

# 原始含噪信号
axes[0].bar(range(N), x_noisy, color='steelblue', label='Noisy')
axes[0].bar(range(N), x_clean, color='coral', alpha=0.5, label='Clean')
axes[0].set_xlabel('顶点')
axes[0].set_ylabel('Signal Value')
axes[0].set_title('Original Signal (Noisy vs Clean)')
axes[0].legend()
axes[0].set_xticks(range(N))

# 滤波器频率响应
axes[1].plot(eigenvalues, h, 'o-', color='seagreen', linewidth=2)
axes[1].set_xlabel('特征值（频率）')
axes[1].set_ylabel('Gain')
axes[1].set_title('Low-pass Filter Frequency Response')
axes[1].grid(True)

# 滤波后信号
axes[2].bar(range(N), y, color='seagreen', label='Filtered')
axes[2].bar(range(N), x_clean, color='coral', alpha=0.5, label='Clean')
axes[2].set_xlabel('顶点')
axes[2].set_ylabel('Signal Value')
axes[2].set_title('滤波后信号 vs 干净信号')
axes[2].legend()
axes[2].set_xticks(range(N))

plt.tight_layout()
plt.show()

    运行结果解读：低通滤波器 \(h(\lambda) = e^{-\alpha\lambda}\) 对小特征值（低频）增益接近1，对大特征值（高频）增益接近0。这样可以保留信号的平滑部分，抑制噪声引起的剧烈变化，实现图信号去噪。

    6. 常见误区¶

    1. 图滤波器和经典滤波器完全一样：经典滤波器的"频率"由时间序列的周期性定义，图滤波器的"频率"由图拉普拉斯特征值定义。图的频率不等间距，且取决于图结构本身——同一信号在不同图上可能表现为不同频率。

    2. 多项式滤波阶数 \(K\) 越大越好：\(K\) 增大虽能逼近更复杂的频率响应，但每增加一阶需多一次 \(L\mathbf{x}\) 的矩阵-向量乘法（\(O(|\mathcal{E}|)\)），且过高的阶数可能导致过拟合或数值不稳定。

    3. 频域方法总是优于空域方法：频域方法需要完整特征分解（\(O(N^3)\)），仅适用于小规模图。大规模图必须用多项式空域方法或 Chebyshev 近似。

    4. 图滤波器只能做低通/高通：可以设计任意频率响应，包括带通、带阻、多频段选择等。关键在于选择合适的 \(h(\lambda)\) 或对应的多项式系数。

    5. 归一化拉普拉斯和未归一化拉普拉斯等价：不同定义的拉普拉斯矩阵（组合拉普拉斯、归一化拉普拉斯、随机游走拉普拉斯）特征值范围不同，直接影响滤波器频率响应的设计。

    7. 启发问题与创新拓展¶

    一个反例或边界问题： 考虑一个完全图 \(K_N\)（任意两个节点都相连）。其归一化拉普拉斯矩阵的特征值只有 0 和 \(N/(N-1)\) 两个——图频率只有两个"档位"。此时任何滤波器都只能在这两个频率之间做选择，无法实现精细的频率选择。这说明：图滤波器的频率分辨率取决于图的拓扑结构，稀疏图的频谱更丰富。

    一个跨学科迁移场景： 在脑科学研究中，大脑功能网络可以建模为图（脑区为节点、功能连接为边），fMRI 信号是图上的信号。用图低通滤波器可以提取脑网络中的全局协调模式（低频），用高通滤波器可以检测异常放电（高频突变）。这比传统时间域滤波更有意义，因为它利用了脑网络的空间结构信息。

    一个可做成课程项目的小想法： "图滤波器 vs 空间均值滤波器"——在同一张传感器网络数据上，分别用图低通滤波器（基于网络拓扑）和简单的空间均值滤波器（忽略网络结构，只按地理距离加权）进行去噪。比较两者的去噪效果，探究图结构信息对滤波质量的贡献。
