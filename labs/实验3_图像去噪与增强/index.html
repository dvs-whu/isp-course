    实验3：图像去噪与增强¶

    课程名称：智能信号处理
    实验学时：2学时（W6）
    实验类型：综合性实验

    一、实验目标¶

    1. 掌握二维离散傅里叶变换（2D-FFT）的原理及其在图像处理中的应用
    2. 理解频域滤波的基本原理，能够设计并实现频率域低通滤波器
    3. 学会使用空间域滤波方法（均值、中值、高斯）对图像进行去噪处理
    4. 掌握PSNR和SSIM两种图像质量评价指标的计算方法

    二、实验环境¶

    软件/库 版本要求
    Python ≥ 3.8
    NumPy ≥ 1.21
    SciPy ≥ 1.7
    Matplotlib ≥ 3.5
    scikit-image ≥ 0.19 
    pip install numpy scipy matplotlib scikit-image
