    实验4：统计信号处理¶

    课程名称：智能信号处理
    实验学时：2学时（W7）
    实验类型：综合性实验

    一、实验目标¶

    1. 理解随机信号的基本统计特性，掌握均值、方差、自相关函数的计算方法
    2. 掌握功率谱密度（PSD）估计的两种经典方法：周期图法与Welch法
    3. 理解有色噪声的生成原理及其频谱特性分析

    二、实验环境¶

    软件/库 版本要求
    Python ≥ 3.8
    NumPy ≥ 1.21
    SciPy ≥ 1.7
    Matplotlib ≥ 3.5

    三、实验步骤¶

    任务1：随机信号的统计特性分析¶

    生成随机信号并计算其统计量（均值、方差、自相关函数）。

    import numpy as np
import matplotlib.pyplot as plt

# 参数设置
fs = 1000          # 采样率
N = 1024           # 信号长度
t = np.arange(N) / fs

# 生成高斯白噪声信号
np.random.seed(42)
x = np.random.randn(N)

# TODO: 计算均值和方差
mean_val = np.mean(____)       # 补全参数
var_val = np.var(____)         # 补全参数
print(f"均值: {mean_val:.4f}, 方差: {var_val:.4f}")

# TODO: 计算自相关函数（无偏估计）
def autocorr(x):
    """计算归一化自相关函数"""
    N = len(x)
    x = x - np.mean(x)
    r = np.correlate(x, x, mode='full')
    r = r / (____ * np.var(x))  # 归一化
    return r[N-1:]               # 取单边

r_xx = autocorr(x)
lags = np.arange(len(r_xx))

# 可视化
fig, axes = plt.subplots(3, 1, figsize=(10, 8))
axes[0].plot(t, x); axes[0].set_title('时域波形')
axes[0].set_xlabel('时间 (s)')

# 直方图与理论正态分布对比
axes[1].hist(x, bins=50, density=True, alpha=0.7, label='实际分布')
x_pdf = np.linspace(-4, 4, 200)
axes[1].plot(x_pdf, np.exp(-x_pdf**2/2)/np.sqrt(2*np.pi), 'r-', label='理论N(0,1)')
axes[1].set_title('概率分布'); axes[1].legend()

axes[2].plot(lags[:100], r_xx[:100]); axes[2].set_title('自相关函数')
axes[2].set_xlabel('滞后 (samples)')
plt.tight_layout(); plt.savefig('random_signal.png', dpi=150); plt.show()

    任务3：有色噪声的生成与分析¶

    利用线性滤波器将白噪声转换为有色噪声，分析其频谱特性。

    from scipy.signal import lfilter, butter

# TODO: 生成粉色噪声（1/f噪声）—— 通过低通滤波白噪声
b, a = butter(3, ____, btype='low')  # 3阶低通滤波器，截止频率0.3（归一化）
pink_noise = lfilter(b, a, np.random.randn(N))

# TODO: 生成蓝色噪声（f噪声）—— 通过高通滤波白噪声
b_h, a_h = butter(3, ____, btype='high')  # 截止频率0.3
blue_noise = lfilter(b_h, a_h, np.random.randn(N))

# 计算各噪声的PSD并对比
f_w, Pxx_white = welch(np.random.randn(N), fs=fs, nperseg=256)
f_w, Pxx_pink = welch(pink_noise, fs=fs, nperseg=256)
f_w, Pxx_blue = welch(blue_noise, fs=fs, nperseg=256)

# TODO: 绘制三种噪声的PSD对比图
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 时域波形
t_short = t[:200]
axes[0].plot(t_short, np.random.randn(200), alpha=0.7, label='白噪声')
axes[0].plot(t_short, pink_noise[:200], alpha=0.7, label='粉色噪声')
axes[0].plot(t_short, blue_noise[:200], alpha=0.7, label='蓝色噪声')
axes[0].set_title('时域波形'); axes[0].legend()

# 频域PSD
axes[1].plot(f_w, 10*np.log10(Pxx_white), label='白噪声')
axes[1].plot(f_w, 10*np.log10(Pxx_pink), label='粉色噪声')
axes[1].plot(f_w, 10*np.log10(Pxx_blue), label='蓝色噪声')
axes[1].set_title('PSD对比'); axes[1].set_xlabel('频率 (Hz)')
axes[1].set_ylabel('功率/频率 (dB/Hz)'); axes[1].legend()

plt.tight_layout(); plt.savefig('colored_noise.png', dpi=150); plt.show()

    四、思考题¶

    1. 周期图法的PSD估计方差较大，Welch法是如何通过分段平均来降低方差的？分段数增加会带来什么代价？
    2. 自相关函数与功率谱密度之间是什么数学关系？试用维纳-辛钦定理解释。
    3. 实际工程中，如何根据信号特性选择合适的PSD估计方法和参数（如窗口长度、重叠率）？
