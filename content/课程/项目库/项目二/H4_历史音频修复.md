## 一、项目背景

20 世纪留下了大量珍贵的历史录音——政治演说、戏曲唱片、民间音乐、口述历史等。这些音频资料经过数十年的保存，往往存在严重的退化问题：

### 常见退化类型
- **背景噪声**：唱片嘶嘶声（hiss）、电流嗡嗡声（hum, 50/60Hz）
- **频带限制**：早期设备频响范围窄（约 300Hz-3kHz）
- **非线性失真**：磁带饱和、唱片磨损造成的削波和谐波失真
- **脉冲噪声**：唱片划痕产生的咔嗒声（click/pop）
- **混响**：早期录音环境造成的混响残余

### 信号处理挑战
历史音频修复的核心思想：**将退化音频视为"干净信号 + 噪声"的叠加**，通过信号处理技术估计并去除噪声成分，同时尽可能保留原始信号。

## 二、信号处理建模

### 2.1 退化模型

```
干净语音 s(n)  ──┐
                  ├──→  退化音频 x(n) = h(n)*s(n) + d(n)
信道/媒介 h(n)  ──┘
                        其中:
                        h(n) = 信道冲激响应（频带限制、混响）
                        d(n) = 加性噪声（背景噪声、脉冲噪声）
```

### 2.2 修复流程

```
退化音频 x(n)
  │
  ▼
预处理
  ├── 去直流偏移
  ├── 脉冲噪声检测与插值修复
  └── 预加重
  │
  ▼
噪声估计
  ├── 静音段检测 → 噪声谱估计
  └── 最小值跟踪法
  │
  ▼
降噪算法（任选其一）
  ├── 谱减法 (Spectral Subtraction)
  ├── 维纳滤波 (Wiener Filter)
  └── 自适应滤波 (Adaptive Filter)
  │
  ▼
后处理
  ├── 频带扩展 (Bandwidth Extension)
  ├── 去混响 (Dereverberation)
  └── 动态范围调整
  │
  ▼
修复后音频 ŝ(n)
```

### 2.3 关键数学原理

**谱减法**：
$$|\hat{S}(\omega)|^2 = |X(\omega)|^2 - \alpha |D(\omega)|^2$$
$$\hat{S}(\omega) = \frac{|X(\omega)|^2 - \alpha |D(\omega)|^2}{|X(\omega)|} \cdot X(\omega)$$

**维纳滤波器**：
$$H(\omega) = \frac{P_{ss}(\omega)}{P_{ss}(\omega) + P_{dd}(\omega)} = \frac{S_{xx}(\omega) - S_{dd}(\omega)}{S_{xx}(\omega)}$$

**LMS 自适应滤波器**：
$$w(n+1) = w(n) + \mu \cdot e(n) \cdot x(n)$$
其中 $\mu$ 为步长因子，$e(n) = d(n) - w^T(n)x(n)$

### 4.1 音频加载与预处理

```python
import numpy as np
import librosa
import librosa.display
import soundfile as sf
import matplotlib.pyplot as plt
from matplotlib import font_manager

# 设置中文字体
font_path = '/usr/share/fonts/truetype/noto/NotoSansCJKsc-Regular.otf'
font_prop = font_manager.FontProperties(fname=font_path)
plt.rcParams['font.family'] = font_prop.get_name()
plt.rcParams['axes.unicode_minus'] = False

def load_degraded_audio(audio_path, sr=22050):
    """加载退化音频"""
    y, sr = librosa.load(audio_path, sr=sr)
    
    # 去直流偏移
    y = y - np.mean(y)
    
    # 归一化
    y = y / np.max(np.abs(y))
    
    return y, sr

def detect_and_repair_clicks(y, sr, threshold=3.0):
    """检测并修复脉冲噪声（咔嗒声）"""
    # 计算短时能量
    frame_length = int(0.025 * sr)
    hop_length = int(0.010 * sr)
    energy = librosa.feature.rms(y=y, frame_length=frame_length, hop_length=hop_length)[0]
    
    # 检测异常高能量帧
    mean_energy = np.mean(energy)
    std_energy = np.std(energy)
    click_frames = energy > (mean_energy + threshold * std_energy)
    
    # 标记并插值修复
    y_repaired = y.copy()
    for i, is_click in enumerate(click_frames):
        if is_click:
            start = i * hop_length
            end = min((i + 1) * hop_length, len(y))
            # 线性插值
            if start > 0 and end < len(y):
                y_repaired[start:end] = np.linspace(y[start-1], y[end], end - start)
    
    return y_repaired, click_frames
```

### 4.2 噪声谱估计

```python
def estimate_noise_spectrum(y, sr, n_fft=2048, hop_length=512, 
                           n_noise_frames=10):
    """
    估计噪声频谱
    方法：利用静音段（能量最低的帧）估计噪声功率谱
    """
    # STFT
    S = librosa.stft(y, n_fft=n_fft, hop_length=hop_length)
    magnitude = np.abs(S)
    phase = np.angle(S)
    
    # 计算帧能量
    frame_energy = np.sum(magnitude**2, axis=0)
    
    # 选择能量最低的 n_noise_frames 帧作为噪声段
    noise_frame_indices = np.argsort(frame_energy)[:n_noise_frames]
    
    # 噪声功率谱估计（取平均）
    noise_spectrum = np.mean(magnitude[:, noise_frame_indices]**2, axis=1)
    
    return noise_spectrum, magnitude, phase

def estimate_noise_min_tracking(y, sr, n_fft=2048, hop_length=512,
                                 window_size=30, alpha=0.95):
    """
    最小值跟踪法估计噪声谱
    原理：跟踪每个频率 bin 的功率最小值
    """
    S = librosa.stft(y, n_fft=n_fft, hop_length=hop_length)
    power = np.abs(S)**2
    
    n_freq, n_frames = power.shape
    noise_est = np.zeros((n_freq, n_frames))
    
    # 初始化
    noise_est[:, 0] = power[:, 0]
    min_power = power[:, 0].copy()
    min_counter = np.zeros(n_freq)
    
    for t in range(1, n_frames):
        min_counter += 1
        
        # 比较当前功率与最小值
        update_mask = power[:, t] < min_power
        min_power[update_mask] = power[:, t]
        min_counter[update_mask] = 0
        
        # 超过窗口大小则重新初始化
        reset_mask = min_counter >= window_size
        min_power[reset_mask] = power[:, t][reset_mask]
        min_counter[reset_mask] = 0
        
        noise_est[:, t] = min_power
    
    return noise_est
```

### 4.3 谱减法

```python
def spectral_subtraction(y, sr, n_fft=2048, hop_length=512, 
                         alpha=2.0, beta=0.01):
    """
    谱减法语音增强
    
    参数:
        alpha: 过减因子（越大降噪越强，但可能产生更多音乐噪声）
        beta: 谱下限因子（防止负值，减小音乐噪声）
    """
    # STFT
    S = librosa.stft(y, n_fft=n_fft, hop_length=hop_length)
    magnitude = np.abs(S)
    phase = np.angle(S)
    power = magnitude ** 2
    
    # 估计噪声谱
    noise_spectrum, _, _ = estimate_noise_spectrum(y, sr, n_fft, hop_length)
    
    # 谱减
    enhanced_power = power - alpha * noise_spectrum[:, np.newaxis]
    
    # 应用谱下限
    enhanced_power = np.maximum(enhanced_power, beta * power)
    
    # 重建
    enhanced_magnitude = np.sqrt(enhanced_power)
    enhanced_S = enhanced_magnitude * np.exp(1j * phase)
    enhanced_y = librosa.istft(enhanced_S, hop_length=hop_length)
    
    return enhanced_y

def spectral_subtraction_with_tracking(y, sr, n_fft=2048, hop_length=512,
                                        alpha=2.0, beta=0.01):
    """带噪声跟踪的谱减法（更适合非平稳噪声）"""
    S = librosa.stft(y, n_fft=n_fft, hop_length=hop_length)
    magnitude = np.abs(S)
    phase = np.angle(S)
    power = magnitude ** 2
    
    # 使用最小值跟踪法估计噪声
    noise_est = estimate_noise_min_tracking(y, sr, n_fft, hop_length)
    
    # 逐帧谱减
    enhanced_power = np.maximum(power - alpha * noise_est, beta * power)
    enhanced_magnitude = np.sqrt(enhanced_power)
    
    enhanced_S = enhanced_magnitude * np.exp(1j * phase)
    enhanced_y = librosa.istft(enhanced_S, hop_length=hop_length)
    
    return enhanced_y
```

### 4.4 维纳滤波

```python
def wiener_filter(y, sr, n_fft=2048, hop_length=512):
    """
    维纳滤波语音增强
    
    原理: H(ω) = P_ss(ω) / P_xx(ω) = 1 - P_dd(ω) / P_xx(ω)
    """
    S = librosa.stft(y, n_fft=n_fft, hop_length=hop_length)
    magnitude = np.abs(S)
    phase = np.angle(S)
    power = magnitude ** 2
    
    # 估计噪声功率谱
    noise_spectrum, _, _ = estimate_noise_spectrum(y, sr, n_fft, hop_length)
    
    # 维纳滤波器传递函数
    noise_power = noise_spectrum[:, np.newaxis]
    H = np.maximum(1 - noise_power / (power + 1e-10), 0)
    
    # 应用滤波器
    enhanced_S = H * S
    enhanced_y = librosa.istft(enhanced_S, hop_length=hop_length)
    
    return enhanced_y

def wiener_filter_iterative(y, sr, n_fft=2048, hop_length=512, n_iter=3):
    """迭代维纳滤波（逐步改善估计）"""
    current_y = y.copy()
    
    for i in range(n_iter):
        S = librosa.stft(current_y, n_fft=n_fft, hop_length=hop_length)
        magnitude = np.abs(S)
        phase = np.angle(S)
        power = magnitude ** 2
        
        noise_spectrum, _, _ = estimate_noise_spectrum(current_y, sr, n_fft, hop_length)
        
        # 更新传递函数
        H = np.maximum(1 - noise_spectrum[:, np.newaxis] / (power + 1e-10), 0)
        
        # 使用原始相位重建
        enhanced_power = H * np.abs(librosa.stft(y, n_fft=n_fft, hop_length=hop_length))**2
        enhanced_S = np.sqrt(np.maximum(enhanced_power, 0)) * np.exp(1j * phase)
        current_y = librosa.istft(enhanced_S, hop_length=hop_length)
    
    return current_y
```

### 4.5 自适应滤波

```python
def lms_adaptive_filter(primary, reference, mu=0.01, filter_order=32):
    """
    LMS 自适应滤波器
    
    参数:
        primary: 含噪信号 d(n)
        reference: 噪声参考信号 x(n)
        mu: 步长因子
        filter_order: 滤波器阶数
    """
    n_samples = len(primary)
    w = np.zeros(filter_order)  # 滤波器权重
    output = np.zeros(n_samples)  # 滤波输出
    error = np.zeros(n_samples)   # 误差信号
    
    for n in range(filter_order, n_samples):
        # 参考信号向量
        x_vec = reference[n - filter_order:n][::-1]
        
        # 滤波输出（噪声估计）
        noise_estimate = np.dot(w, x_vec)
        
        # 误差 = 含噪信号 - 噪声估计 = 干净信号估计
        error[n] = primary[n] - noise_estimate
        output[n] = error[n]
        
        # LMS 权重更新
        w = w + 2 * mu * error[n] * x_vec
    
    return output, error, w

def rls_adaptive_filter(primary, reference, filter_order=32, lam=0.99, delta=0.01):
    """
    RLS 自适应滤波器（收敛更快，计算更复杂）
    """
    n_samples = len(primary)
    w = np.zeros(filter_order)
    P = np.eye(filter_order) / delta  # 逆相关矩阵
    output = np.zeros(n_samples)
    
    for n in range(filter_order, n_samples):
        x_vec = reference[n - filter_order:n][::-1].reshape(-1, 1)
        
        # 增益向量
        k = P @ x_vec / (lam + x_vec.T @ P @ x_vec)
        
        # 先验误差
        output[n] = primary[n] - (w @ x_vec).item()
        
        # 权重更新
        w = w + (k.flatten() * output[n])
        
        # 更新逆相关矩阵
        P = (P - k @ x_vec.T @ P) / lam
    
    return output
```

### 4.6 完整修复流程与可视化

```python
def full_restoration_pipeline(audio_path, method='wiener'):
    """完整的历史音频修复流水线"""
    # 加载
    y, sr = load_degraded_audio(audio_path)
    
    # 修复脉冲噪声
    y_repaired, clicks = detect_and_repair_clicks(y, sr)
    
    # 降噪
    if method == 'spectral_subtraction':
        y_enhanced = spectral_subtraction(y_repaired, sr)
    elif method == 'wiener':
        y_enhanced = wiener_filter(y_repaired, sr)
    elif method == 'wiener_iterative':
        y_enhanced = wiener_filter_iterative(y_repaired, sr, n_iter=3)
    else:
        raise ValueError(f"Unknown method: {method}")
    
    # 可视化对比
    fig, axes = plt.subplots(3, 2, figsize=(14, 10))
    
    # 波形对比
    times = np.arange(len(y)) / sr
    axes[0, 0].plot(times, y, color='red', linewidth=0.3)
    axes[0, 0].set_title('退化音频 - 波形', fontproperties=font_prop)
    axes[0, 0].set_xlabel('时间 (s)', fontproperties=font_prop)
    
    times_out = np.arange(len(y_enhanced)) / sr
    axes[0, 1].plot(times_out, y_enhanced, color='green', linewidth=0.3)
    axes[0, 1].set_title('修复后 - 波形', fontproperties=font_prop)
    axes[0, 1].set_xlabel('时间 (s)', fontproperties=font_prop)
    
    # 频谱对比
    S_orig = np.abs(librosa.stft(y))
    S_enh = np.abs(librosa.stft(y_enhanced))
    
    librosa.display.specshow(librosa.amplitude_to_db(S_orig, ref=np.max),
                             sr=sr, hop_length=512, x_axis='time', y_axis='hz',
                             ax=axes[1, 0])
    axes[1, 0].set_title('退化音频 - 频谱', fontproperties=font_prop)
    
    librosa.display.specshow(librosa.amplitude_to_db(S_enh, ref=np.max),
                             sr=sr, hop_length=512, x_axis='time', y_axis='hz',
                             ax=axes[1, 1])
    axes[1, 1].set_title('修复后 - 频谱', fontproperties=font_prop)
    
    # 功率谱密度对比
    f_orig, psd_orig = plt.psd(y, NFFT=2048, Fs=sr, color='red')
    axes[2, 0].cla()
    axes[2, 0].semilogy(f_orig, psd_orig, 'r-', linewidth=1)
    axes[2, 0].set_title('退化音频 - 功率谱', fontproperties=font_prop)
    axes[2, 0].set_xlabel('频率 (Hz)', fontproperties=font_prop)
    
    f_enh, psd_enh = plt.psd(y_enhanced, NFFT=2048, Fs=sr, color='green')
    axes[2, 1].cla()
    axes[2, 1].semilogy(f_enh, psd_enh, 'g-', linewidth=1)
    axes[2, 1].set_title('修复后 - 功率谱', fontproperties=font_prop)
    axes[2, 1].set_xlabel('频率 (Hz)', fontproperties=font_prop)
    
    plt.tight_layout()
    plt.savefig(f'restoration_{method}.png', dpi=150, bbox_inches='tight')
    plt.show()
    
    return y_enhanced, sr
```

### 4.7 客观评估指标

```python
def compute_snr(clean, noisy):
    """信噪比"""
    signal_power = np.mean(clean**2)
    noise_power = np.mean((clean - noisy)**2)
    return 10 * np.log10(signal_power / (noise_power + 1e-10))

def compute_pesq_score(reference, degraded, sr=16000):
    """PESQ 评分（需安装 pesq 库）"""
    try:
        from pesq import pesq
        score = pesq(sr, reference, degraded, 'wb')
        return score
    except ImportError:
        print("请安装 pesq: pip install pesq")
        return None

def compare_methods(y_clean, y_noisy, sr, methods):
    """对比不同修复方法的效果"""
    results = {}
    for name, func in methods.items():
        y_restored = func(y_noisy, sr)
        snr = compute_snr(y_clean, y_restored)
        results[name] = {'snr': snr, 'audio': y_restored}
        print(f"{name}: SNR = {snr:.2f} dB")
    return results
```

- 能实现从破损录音到清晰音频的修复流程
