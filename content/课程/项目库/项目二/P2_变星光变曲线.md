# P2 变星光变曲线

## 一、项目背景

变星（Variable Star）是亮度随时间变化的恒星。亮度变化的原因多种多样：
- **食变星**：双星系统中恒星相互遮掩（如大陵五）
- **脉动变星**：恒星周期性膨胀收缩（如造父变星、天琴座RR型变星）
- **自转变星**：恒星表面不均匀导致旋转时亮度变化

近年来，**凌星法（Transit Method）** 成为发现系外行星的主要方法：当行星从恒星前方经过时，恒星亮度会微微下降（典型深度约0.01-1%）。开普勒（Kepler）和TESS太空望远镜通过连续监测数万颗恒星的光变曲线，已发现数千颗系外行星。

**核心挑战**：从光变曲线中提取周期性信号、去除系统性趋势、检测微弱的凌星信号。

## 二、信号处理建模

### 处理流程

```
原始光变曲线 (flux vs time)
    ↓
去趋势 (Detrending) - 去除系统性漂移
    ↓
异常值剔除 (Sigma Clipping)
    ↓
Lomb-Scargle 周期图 → 估计周期
    ↓
相位折叠 (Phase Folding)
    ↓
凌星信号检测 → 行星参数估计
```

### 关键公式

**Lomb-Scargle 周期图**（不等间隔傅里叶分析）：
$$P(\omega) = \frac{1}{2}\left[\frac{\left(\sum_j (x_j-\bar{x})\cos\omega(t_j-\tau)\right)^2}{\sum_j \cos^2\omega(t_j-\tau)} + \frac{\left(\sum_j (x_j-\bar{x})\sin\omega(t_j-\tau)\right)^2}{\sum_j \sin^2\omega(t_j-\tau)}\right]$$

**凌星深度**：
$$\delta = \left(\frac{R_p}{R_s}\right)^2$$

## 四、数据处理方法

### 4.1 环境准备与数据加载

```python
import numpy as np
import matplotlib.pyplot as plt
from astropy.io import fits
from astropy.timeseries import BoxLeastSquares
from scipy.signal import savgol_filter
from scipy.stats import sigmaclip

# 设置中文字体
plt.rcParams['font.family'] = 'Noto Sans CJK SC'
plt.rcParams['font.sans-serif'] = ['Noto Sans CJK SC']
plt.rcParams['axes.unicode_minus'] = False

# 加载 Kepler 光变曲线（FITS 格式）
def load_kepler_lightcurve(filepath):
    """加载 Kepler 光变曲线 FITS 文件"""
    with fits.open(filepath) as hdul:
        data = hdul[1].data
        time = data['TIME']           # BJD 时间
        flux = data['PDCSAP_FLUX']    # PDC 校正后的流量
        flux_err = data['PDCSAP_FLUX_ERR']
        quality = data['QUALITY']      # 质量标记

    # 移除无效数据
    valid = np.isfinite(flux) & (quality == 0) & np.isfinite(time)
    return time[valid], flux[valid], flux_err[valid]

# 示例：生成模拟光变曲线
np.random.seed(42)
n_points = 5000
time = np.sort(np.random.uniform(0, 30, n_points))  # 30天

# 模拟周期性凌星信号
period = 3.5  # 行星轨道周期（天）
transit_depth = 0.01  # 凌星深度 1%
transit_duration = 0.1  # 凌星持续时间（天）

# 基线亮度 + 噪声
flux = np.ones(n_points) + np.random.normal(0, 0.001, n_points)

# 添加凌星信号
phase = (time % period) / period
transit_mask = (phase > 0.45) & (phase < 0.55)
flux[transit_mask] -= transit_depth * np.sin(np.pi * (phase[transit_mask] - 0.45) / 0.1)**2

# 添加长期趋势
flux += 0.002 * np.sin(2 * np.pi * time / 15)

flux_err = np.full(n_points, 0.001)
```

### 4.2 去趋势与异常值剔除

```python
# 1. 去除长期趋势（Savitzky-Golay 平滑）
trend = savgol_filter(flux, window_length=501, polyorder=3)
flux_detrended = flux / trend

# 2. Sigma clipping 去除异常值
flux_clipped, low, high = sigmaclip(flux_detrended, low=4, high=4)
mask = (flux_detrended >= low) & (flux_detrended <= high)

time_clean = time[mask]
flux_clean = flux_detrended[mask]
flux_err_clean = flux_err[mask] / trend[mask]

# 可视化
fig, axes = plt.subplots(2, 1, figsize=(14, 8))

axes[0].plot(time, flux, 'b.', markersize=1, alpha=0.5, label='原始')
axes[0].plot(time, trend, 'r-', linewidth=2, label='趋势')
axes[0].set_ylabel('相对流量')
axes[0].set_title('原始光变曲线与趋势')
axes[0].legend()

axes[1].plot(time_clean, flux_clean, 'b.', markersize=1, alpha=0.5)
axes[1].set_xlabel('时间 (BJD - 2454833)')
axes[1].set_ylabel('归一化流量')
axes[1].set_title('去趋势后的光变曲线')

plt.tight_layout()
plt.savefig('lightcurve_detrended.png', dpi=150, bbox_inches='tight')
plt.show()
```

### 4.3 Lomb-Scargle 周期图

```python
from astropy.timeseries import LombScargle

# 计算 Lomb-Scargle 周期图
ls = LombScargle(time_clean, flux_clean, flux_err_clean)
frequency, power = ls.autopower(minimum_frequency=0.1,
                                 maximum_frequency=5.0,
                                 samples_per_peak=10)

# 找到最强周期
best_frequency = frequency[np.argmax(power)]
best_period = 1.0 / best_frequency
print(f"最强周期: {best_period:.4f} 天")
print(f"对应频率: {best_frequency:.4f} 天⁻¹")

# 可视化
fig, axes = plt.subplots(2, 1, figsize=(14, 8))

axes[0].plot(1.0 / frequency, power, 'b-', linewidth=0.5)
axes[0].axvline(best_period, color='r', linestyle='--', label=f'最强周期 = {best_period:.3f} 天')
axes[0].set_xlabel('周期 (天)')
axes[0].set_ylabel('Lomb-Scargle 功率')
axes[0].set_title('Lomb-Scargle 周期图')
axes[0].set_xlim(0, 10)
axes[0].legend()

axes[1].plot(frequency, power, 'b-', linewidth=0.5)
axes[1].axvline(best_frequency, color='r', linestyle='--')
axes[1].set_xlabel('频率 (天⁻¹)')
axes[1].set_ylabel('功率')
axes[1].set_title('频率域')

plt.tight_layout()
plt.savefig('lomb_scargle_periodogram.png', dpi=150, bbox_inches='tight')
plt.show()
```

### 4.4 相位折叠

```python
# 相位折叠
phase = (time_clean % best_period) / best_period

# 将相位中心化到凌星中点
phase_shifted = (phase - 0.5 + 1) % 1  # 假设凌星在相位0.5

# 按相位排序
sort_idx = np.argsort(phase_shifted)
phase_sorted = phase_shifted[sort_idx]
flux_sorted = flux_clean[sort_idx]

# 分箱平均
n_bins = 200
bins = np.linspace(0, 1, n_bins + 1)
bin_centers = (bins[:-1] + bins[1:]) / 2
binned_flux = np.zeros(n_bins)
binned_err = np.zeros(n_bins)

for i in range(n_bins):
    in_bin = (phase_sorted >= bins[i]) & (phase_sorted < bins[i+1])
    if np.sum(in_bin) > 0:
        binned_flux[i] = np.mean(flux_sorted[in_bin])
        binned_err[i] = np.std(flux_sorted[in_bin]) / np.sqrt(np.sum(in_bin))

# 可视化
fig, axes = plt.subplots(2, 1, figsize=(14, 8))

# 散点图
axes[0].plot(phase_shifted, flux_clean, 'b.', markersize=1, alpha=0.3)
axes[0].set_xlabel('相位')
axes[0].set_ylabel('归一化流量')
axes[0].set_title(f'相位折叠 (P = {best_period:.4f} 天)')

# 分箱平均
axes[1].errorbar(bin_centers, binned_flux, yerr=binned_err,
                 fmt='o', markersize=2, color='blue', alpha=0.7)
axes[1].set_xlabel('相位')
axes[1].set_ylabel('归一化流量')
axes[1].set_title('分箱平均后的折叠光变曲线')

# 标注凌星
for ax in axes:
    ax.axvspan(0.4, 0.6, alpha=0.2, color='red', label='凌星区间')

plt.tight_layout()
plt.savefig('phase_folded.png', dpi=150, bbox_inches='tight')
plt.show()
```

### 4.5 凌星检测（BLS算法）

```python
# Box Least Squares (BLS) 周期图 - 专门用于凌星检测
from astropy.timeseries import BoxLeastSquares

# BLS 检测
bls = BoxLeastSquares(time_clean, flux_clean)
bls_power = bls.autopower(duration=[0.05, 0.1, 0.15, 0.2],
                           minimum_period=1.0,
                           maximum_period=10.0)

# 最佳拟合参数
best_bls_period = bls_power.period[np.argmax(bls_power.power)]
best_bls_t0 = bls_power.transit_time[np.argmax(bls_power.power)]
best_bls_duration = bls_power.duration[np.argmax(bls_power.power)]
best_bls_depth = bls_power.depth[np.argmax(bls_power.power)]

print(f"BLS 检测结果:")
print(f"  周期: {best_bls_period:.4f} 天")
print(f"  凌星中点: {best_bls_t0:.4f}")
print(f"  持续时间: {best_bls_duration:.4f} 天")
print(f"  凌星深度: {best_bls_depth:.6f}")
print(f"  行星半径比: √{best_bls_depth:.6f} = {np.sqrt(best_bls_depth):.4f}")

# 可视化 BLS 周期图
fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(bls_power.period, bls_power.power, 'b-', linewidth=0.5)
ax.axvline(best_bls_period, color='r', linestyle='--',
           label=f'最佳周期 = {best_bls_period:.3f} 天')
ax.set_xlabel('周期 (天)')
ax.set_ylabel('BLS 功率')
ax.set_title('Box Least Squares 周期图')
ax.legend()
plt.savefig('bls_periodogram.png', dpi=150, bbox_inches='tight')
plt.show()
```
