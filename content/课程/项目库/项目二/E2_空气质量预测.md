> **项目类型**：环境交叉项目 | **难度**：⭐⭐⭐ | **关键词**：时间序列分析、频谱分析、预测模型

## 一、项目背景

空气污染是全球性环境问题，PM2.5、臭氧等污染物直接影响公众健康。准确预测空气质量变化趋势，对于污染预警、交通管制、户外活动规划具有重要意义。

### 核心洞察
空气质量数据本质上是**时间序列信号**，具有以下特征：
- **趋势性**：长期空气质量随环保政策改善呈下降趋势
- **季节性**：冬季供暖期 PM2.5 偏高，夏季臭氧偏高
- **周期性**：日周期（早晚高峰）、周周期（工作日/周末）
- **随机性**：受气象条件（风速、温度、湿度）影响的随机波动

### 信号处理视角
通过频域分析，我们可以：
- 发现隐藏的**周期性模式**（如 24h 日周期、168h 周周期）
- 分离**趋势**、**季节**和**残差**成分
- 利用**相关函数**分析污染物之间的关联性

## 二、信号处理建模

### 2.1 处理流程

```
空气质量时序数据 x(t) [PM2.5, PM10, SO2, NO2, CO, O3, ...]
  │
  ▼
数据预处理
  ├── 缺失值插值（线性/样条）
  ├── 异常值检测与处理
  └── 归一化
  │
  ▼
STL 分解
  ├── 趋势分量 T(t)
  ├── 季节分量 S(t)
  └── 残差分量 R(t)
  │
  ▼
频谱分析
  ├── FFT → 功率谱密度 → 识别主周期
  ├── 自相关函数 → 确认周期性
  └── 互相关函数 → 分析变量间时滞关系
  │
  ▼
预测模型
  ├── ARIMA（线性模型）
  ├── LSTM（深度学习）
  └── Prophet（Facebook 时序模型）
  │
  ▼
空气质量预测值 ŷ(t+1), ŷ(t+2), ...
```

### 2.2 关键数学原理

**STL 分解**（Seasonal and Trend decomposition using Loess）：
$$x(t) = T(t) + S(t) + R(t)$$

**自相关函数**：
$$R_{xx}(\tau) = \frac{1}{N-\tau}\sum_{t=1}^{N-\tau} (x(t)-\bar{x})(x(t+\tau)-\bar{x})$$

**功率谱密度**（PSD）：
$$P(f) = |\text{FFT}\{x(t)\}|^2$$

主周期 $T_0$ 对应功率谱的**最高峰**频率 $f_0$：$T_0 = 1/f_0$

### 4.1 数据加载与预处理

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from matplotlib import font_manager
from scipy import signal, fft
from statsmodels.tsa.seasonal import STL

# 设置中文字体
font_path = '/usr/share/fonts/truetype/noto/NotoSansCJKsc-Regular.otf'
font_prop = font_manager.FontProperties(fname=font_path)
plt.rcParams['font.family'] = font_prop.get_name()
plt.rcParams['axes.unicode_minus'] = False

def load_beijing_pm25(csv_path):
    """加载北京 PM2.5 数据"""
    df = pd.read_csv(csv_path)
    
    # 构建 datetime 索引
    df['datetime'] = pd.to_datetime(df'year', 'month', 'day', 'hour')
    df = df.set_index('datetime')
    
    # 提取 PM2.5 并处理缺失值
    pm25 = df['pm2.5'].copy()
    pm25 = pm25.interpolate(method='linear')  # 线性插值填补缺失
    pm25 = pm25.fillna(method='bfill').fillna(method='ffill')
    
    # 去除异常值（PM2.5 不可能为负）
    pm25 = pm25.clip(lower=0)
    
    return pm25, df

def preprocess_time_series(series, window=24):
    """时序预处理：平滑 + 标准化"""
    # 滑动平均平滑
    series_smooth = series.rolling(window=window, center=True).mean()
    series_smooth = series_smooth.fillna(method='bfill').fillna(method='ffill')
    
    # Z-score 标准化
    mean_val = series_smooth.mean()
    std_val = series_smooth.std()
    series_norm = (series_smooth - mean_val) / std_val
    
    return series_smooth, series_norm, mean_val, std_val
```

### 4.2 STL 分解

```python
def stl_decomposition(series, period=24*7):
    """
    STL 分解：趋势 + 季节 + 残差
    
    period: 季节周期长度
    - 24 = 日周期（逐小时数据）
    - 24*7 = 周周期
    - 24*365 = 年周期
    """
    stl = STL(series, period=period, robust=True)
    result = stl.fit()
    
    fig, axes = plt.subplots(4, 1, figsize=(14, 10))
    
    axes[0].plot(series, color='steelblue', linewidth=0.5)
    axes[0].set_title('原始 PM2.5 序列', fontproperties=font_prop)
    axes[0].set_ylabel('浓度 (μg/m³)', fontproperties=font_prop)
    
    axes[1].plot(result.trend, color='red', linewidth=1.5)
    axes[1].set_title('趋势分量 T(t)', fontproperties=font_prop)
    axes[1].set_ylabel('μg/m³', fontproperties=font_prop)
    
    axes[2].plot(result.seasonal, color='green', linewidth=0.5)
    axes[2].set_title('季节分量 S(t)', fontproperties=font_prop)
    axes[2].set_ylabel('μg/m³', fontproperties=font_prop)
    
    axes[3].plot(result.resid, color='gray', linewidth=0.5)
    axes[3].set_title('残差分量 R(t)', fontproperties=font_prop)
    axes[3].set_ylabel('μg/m³', fontproperties=font_prop)
    axes[3].set_xlabel('时间', fontproperties=font_prop)
    
    plt.tight_layout()
    plt.savefig('stl_decomposition.png', dpi=150, bbox_inches='tight')
    plt.show()
    
    return result
```

### 4.3 频谱分析 — 识别周期性

```python
def spectral_analysis(series, fs=1.0, title="PM2.5"):
    """
    频谱分析：发现隐藏周期
    
    fs: 采样频率（小时数据 fs=1/hour）
    """
    values = series.dropna().values
    values = values - np.mean(values)  # 去均值
    
    n = len(values)
    
    # 方法一：FFT 功率谱
    freqs = np.fft.rfftfreq(n, d=1/fs)
    fft_vals = np.fft.rfft(values)
    power_spectrum = np.abs(fft_vals)**2 / n
    
    # 方法二：Welch 方法（更平滑的 PSD 估计）
    f_welch, psd_welch = signal.welch(values, fs=fs, nperseg=min(1024, n//4))
    
    fig, axes = plt.subplots(1, 2, figsize=(14, 5))
    
    # FFT 功率谱
    # 排除零频率，关注有意义的频率范围
    freq_mask = (freqs > 0) & (freqs < 0.5)  # 排除 Nyquist 以上
    axes[0].semilogy(1/freqs[freq_mask], power_spectrum[freq_mask], 
                     'b-', linewidth=0.8)
    axes[0].set_xlabel('周期 (小时)', fontproperties=font_prop)
    axes[0].set_ylabel('功率谱密度', fontproperties=font_prop)
    axes[0].set_title(f'{title} - 周期图', fontproperties=font_prop)
    axes[0].axvline(x=24, color='red', linestyle='--', label='24h 日周期')
    axes[0].axvline(x=168, color='green', linestyle='--', label='168h 周周期')
    axes[0].legend(prop=font_prop)
    axes[0].grid(True, alpha=0.3)
    
    # Welch PSD
    axes[1].semilogy(1/f_welch[1:], psd_welch[1:], 'r-', linewidth=1)
    axes[1].set_xlabel('周期 (小时)', fontproperties=font_prop)
    axes[1].set_ylabel('PSD', fontproperties=font_prop)
    axes[1].set_title(f'{title} - Welch PSD', fontproperties=font_prop)
    axes[1].grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.savefig(f'spectral_{title}.png', dpi=150, bbox_inches='tight')
    plt.show()
    
    # 找出主周期
    peak_indices = signal.find_peaks(power_spectrum[1:], height=np.percentile(power_spectrum[1:], 95))[0]
    main_periods = 1 / freqs[peak_indices + 1]
    print(f"发现的主要周期（小时）: {main_periods[:5]}")
    
    return freqs, power_spectrum
```

### 4.4 自相关与互相关分析

```python
def autocorrelation_analysis(series, max_lag=24*14):
    """自相关分析：验证周期性"""
    values = series.dropna().values
    values = values - np.mean(values)
    
    # 计算自相关函数
    acf = np.correlate(values, values, mode='full')
    acf = acf[len(values)-1:]  # 取正半部分
    acf = acf / acf[0]  # 归一化
    acf = acf[:max_lag]
    
    fig, axes = plt.subplots(1, 2, figsize=(14, 5))
    
    # 自相关函数
    lags = np.arange(max_lag)
    axes[0].plot(lags, acf, 'b-', linewidth=0.8)
    axes[0].axhline(y=0, color='black', linewidth=0.5)
    # 置信区间 (95%)
    conf = 1.96 / np.sqrt(len(values))
    axes[0].axhline(y=conf, color='red', linestyle='--', alpha=0.5)
    axes[0].axhline(y=-conf, color='red', linestyle='--', alpha=0.5)
    axes[0].set_xlabel('滞后 (小时)', fontproperties=font_prop)
    axes[0].set_ylabel('自相关系数', fontproperties=font_prop)
    axes[0].set_title('PM2.5 自相关函数', fontproperties=font_prop)
    axes[0].grid(True, alpha=0.3)
    
    # 偏自相关（近似）
    from statsmodels.tsa.stattools import pacf
    pacf_vals = pacf(values, nlags=min(48, len(values)//2 - 1))
    axes[1].bar(range(len(pacf_vals)), pacf_vals, alpha=0.7, color='coral')
    axes[1].axhline(y=conf, color='red', linestyle='--', alpha=0.5)
    axes[1].axhline(y=-conf, color='red', linestyle='--', alpha=0.5)
    axes[1].set_xlabel('滞后 (小时)', fontproperties=font_prop)
    axes[1].set_ylabel('偏自相关系数', fontproperties=font_prop)
    axes[1].set_title('PM2.5 偏自相关函数', fontproperties=font_prop)
    
    plt.tight_layout()
    plt.savefig('acf_analysis.png', dpi=150, bbox_inches='tight')
    plt.show()

def cross_correlation(series1, series2, max_lag=48):
    """互相关分析：PM2.5 与风速的时滞关系"""
    s1 = (series1 - series1.mean()) / series1.std()
    s2 = (series2 - series2.mean()) / series2.std()
    
    # 互相关
    ccf = np.correlate(s1.values[:10000], s2.values[:10000], mode='full')
    ccf = ccf / len(s1.values[:10000])
    mid = len(ccf) // 2
    ccf = ccf[mid-max_lag:mid+max_lag+1]
    lags = np.arange(-max_lag, max_lag+1)
    
    fig, ax = plt.subplots(figsize=(10, 4))
    ax.bar(lags, ccf, alpha=0.7, color='steelblue')
    ax.set_xlabel('滞后 (小时)', fontproperties=font_prop)
    ax.set_ylabel('互相关系数', fontproperties=font_prop)
    ax.set_title('PM2.5 与风速的互相关', fontproperties=font_prop)
    ax.axvline(x=0, color='red', linestyle='--')
    
    # 找最大互相关
    peak_lag = lags[np.argmax(np.abs(ccf))]
    print(f"最大互相关滞后: {peak_lag} 小时")
    
    plt.tight_layout()
    plt.savefig('cross_correlation.png', dpi=150, bbox_inches='tight')
    plt.show()
```

### 4.5 LSTM 预测模型

```python
import torch
import torch.nn as nn
from torch.utils.data import Dataset, DataLoader

class AirQualityLSTM(nn.Module):
    """基于 LSTM 的空气质量预测模型"""
    def __init__(self, input_dim=6, hidden_dim=64, num_layers=2, 
                 pred_steps=24, dropout=0.2):
        super(AirQualityLSTM, self).__init__()
        self.hidden_dim = hidden_dim
        self.num_layers = num_layers
        self.pred_steps = pred_steps
        
        self.lstm = nn.LSTM(
            input_dim, hidden_dim, num_layers,
            batch_first=True, dropout=dropout
        )
        
        self.attention = nn.Sequential(
            nn.Linear(hidden_dim, 32),
            nn.Tanh(),
            nn.Linear(32, 1)
        )
        
        self.fc = nn.Sequential(
            nn.Linear(hidden_dim, 32),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(32, pred_steps)
        )
    
    def forward(self, x):
        # x: (batch, seq_len, features)
        lstm_out, _ = self.lstm(x)
        
        # Attention
        attn = self.attention(lstm_out)
        attn = torch.softmax(attn, dim=1)
        context = torch.sum(lstm_out * attn, dim=1)
        
        output = self.fc(context)
        return output

class TimeSeriesDataset(Dataset):
    """时间序列数据集"""
    def __init__(self, data, lookback=168, pred_steps=24):
        self.data = data
        self.lookback = lookback
        self.pred_steps = pred_steps
    
    def __len__(self):
        return len(self.data) - self.lookback - self.pred_steps
    
    def __getitem__(self, idx):
        x = self.data[idx:idx+self.lookback]
        y = self.data[idx+self.lookback:idx+self.lookback+self.pred_steps, 0]  # 预测PM2.5
        return torch.FloatTensor(x), torch.FloatTensor(y)

def train_predictor(series_df, target_col='pm2.5', epochs=50, 
                    lookback=168, pred_steps=24):
    """训练空气质量预测模型"""
    # 准备数据
    features = series_df.values
    mean = features.mean(axis=0)
    std = features.std(axis=0)
    features_norm = (features - mean) / std
    
    # 划分训练/验证集
    split = int(len(features_norm) * 0.8)
    train_data = features_norm[:split]
    val_data = features_norm[split:]
    
    train_set = TimeSeriesDataset(train_data, lookback, pred_steps)
    val_set = TimeSeriesDataset(val_data, lookback, pred_steps)
    
    train_loader = DataLoader(train_set, batch_size=64, shuffle=True)
    val_loader = DataLoader(val_set, batch_size=64)
    
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    model = AirQualityLSTM(
        input_dim=features.shape[1], 
        pred_steps=pred_steps
    ).to(device)
    
    criterion = nn.MSELoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
    scheduler = torch.optim.lr_scheduler.ReduceLROnPlateau(optimizer, patience=5)
    
    for epoch in range(epochs):
        model.train()
        train_loss = 0
        for x_batch, y_batch in train_loader:
            x_batch, y_batch = x_batch.to(device), y_batch.to(device)
            optimizer.zero_grad()
            pred = model(x_batch)
            loss = criterion(pred, y_batch)
            loss.backward()
            torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
            optimizer.step()
            train_loss += loss.item()
        
        model.eval()
        val_loss = 0
        with torch.no_grad():
            for x_batch, y_batch in val_loader:
                x_batch, y_batch = x_batch.to(device), y_batch.to(device)
                pred = model(x_batch)
                val_loss += criterion(pred, y_batch).item()
        
        scheduler.step(val_loss)
        
        if (epoch + 1) % 5 == 0:
            print(f'Epoch [{epoch+1}/{epochs}]  '
                  f'Train Loss: {train_loss/len(train_loader):.4f}  '
                  f'Val Loss: {val_loss/len(val_loader):.4f}')
    
    return model
```
