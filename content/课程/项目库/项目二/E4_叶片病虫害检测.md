> **项目类型**：环境交叉项目 | **难度**：⭐⭐⭐⭐ | **关键词**：高光谱分析、差分光谱、植物病害检测

## 一、项目背景

植物病虫害是全球农业生产的重大威胁，每年造成数十亿美元的经济损失。传统的病害检测依赖人工巡查或实验室鉴定，效率低且难以大面积覆盖。**光谱分析技术**为早期、无损、快速检测提供了新的途径。

### 科学基础
健康植物和患病植物的**光谱反射特性**存在显著差异：
- **健康叶片**：叶绿素在可见光区（400-700nm）强烈吸收，在近红外区（700-1300nm）强烈反射
- **病害叶片**：叶绿素降解导致可见光反射增加，细胞结构破坏导致近红外反射下降
- **不同病害**：具有独特的光谱"指纹"，可通过导数光谱和特征波段区分

### 信号处理视角
叶片光谱本质上是一个**一维信号** $\rho(\lambda)$（反射率随波长变化），信号处理技术在其中的应用：
- **平滑滤波**：去除光谱测量中的高频噪声
- **导数光谱**：通过差分运算突出光谱的细微变化
- **特征波段选择**：利用相关分析和频域特征提取最具区分力的波段

## 二、信号处理建模

### 2.1 处理流程

```
叶片光谱数据 ρ(λ), λ ∈ [400, 2500] nm
  │
  ▼
预处理
  ├── 暗电流校正
  ├── 参考白板校正 → 反射率
  ├── 平滑去噪（SG 滤波）
  └── 连续统去除（Continuum Removal）
  │
  ▼
光谱变换
  ├── 一阶导数 dρ/dλ → 红边位置
  ├── 二阶导数 d²ρ/d²λ → 吸收特征
  └── 对数变换 log(1/ρ) → 吸收系数
  │
  ▼
特征提取
  ├── 红边参数（红边位置、红边幅值、红边面积）
  ├── 吸收特征深度和宽度
  ├── 植被指数（NDVI, PRI, CRI 等）
  └── 小波系数
  │
  ▼
分类器
  ├── SVM（支持向量机）
  ├── Random Forest（随机森林）
  └── 1D-CNN（一维卷积网络）
  │
  ▼
病虫害检测结果 (健康 / 病害类型)
```

### 2.2 关键数学原理

**Savitzky-Golay 平滑滤波器**：
$$\hat{x}_i = \frac{1}{N}\sum_{j=-m}^{m} c_j x_{i+j}$$
通过最小二乘拟合局部多项式实现平滑，保留光谱特征

**一阶导数光谱**：
$$\rho'(\lambda_i) = \frac{\rho(\lambda_{i+1}) - \rho(\lambda_{i-1})}{\lambda_{i+1} - \lambda_{i-1}}$$

**连续统去除**：
$$CR(\lambda) = \frac{\rho(\lambda)}{\rho_{line}(\lambda)}$$
其中 $\rho_{line}$ 是光谱包络线（凸壳）

**红边位置**（REP）：
一阶导数光谱在 680-750nm 区间的最大值对应的波长

```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_leaf_spectrum(health_status='healthy', wavelength=None):
    """
    模拟叶片光谱（用于教学演示）
    
    health_status: 'healthy', 'rust', 'mildew', 'blight'
    """
    if wavelength is None:
        wavelength = np.arange(400, 2501, 1)  # 400-2500nm
    
    # 基础光谱模型
    # 可见光区（400-700nm）：叶绿素吸收
    chlorophyll_absorption = 0.8 * np.exp(-((wavelength - 450)**2) / (2 * 30**2)) + \
                             0.7 * np.exp(-((wavelength - 670)**2) / (2 * 20**2))
    
    # 近红外平台（700-1300nm）：细胞结构反射
    nir_reflection = 0.45 * np.ones_like(wavelength)
    nir_reflection[wavelength < 700] = 0
    nir_reflection[wavelength > 1300] = 0
    
    # 红边过渡（680-750nm）
    red_edge = 0.05 + 0.4 / (1 + np.exp(-0.1 * (wavelength - 720)))
    
    # 水分吸收带（1400nm, 1900nm）
    water_absorption = 0.15 * np.exp(-((wavelength - 1400)**2) / (2 * 40**2)) + \
                       0.2 * np.exp(-((wavelength - 1900)**2) / (2 * 50**2))
    
    # 基础反射率
    rho = 0.05 + (1 - chlorophyll_absorption) * red_edge + nir_reflection - water_absorption
    rho = np.clip(rho, 0, 1)
    
    # 根据病害状态修改
    if health_status == 'rust':
        # 锈病：可见光反射增加，近红外降低
        rho[(wavelength >= 500) & (wavelength <= 700)] += 0.15
        rho[(wavelength >= 700) & (wavelength <= 1300)] -= 0.10
        rho += np.random.normal(0, 0.01, len(wavelength))
    elif health_status == 'mildew':
        # 白粉病：整体反射增加，红边蓝移
        rho += 0.08
        rho[(wavelength >= 700) & (wavelength <= 1300)] -= 0.15
        rho += np.random.normal(0, 0.008, len(wavelength))
    elif health_status == 'blight':
        # 枯萎病：叶绿素严重降解
        rho[(wavelength >= 400) & (wavelength <= 700)] += 0.25
        rho[(wavelength >= 700) & (wavelength <= 1300)] -= 0.20
        rho += np.random.normal(0, 0.012, len(wavelength))
    else:
        # 健康：少量噪声
        rho += np.random.normal(0, 0.005, len(wavelength))
    
    rho = np.clip(rho, 0, 1)
    return wavelength, rho

def load_plantvillage_features(image_path):
    """
    从 PlantVillage 图像提取光谱类特征
    将 RGB 图像的通道统计量和纹理作为"伪光谱"特征
    """
    from PIL import Image
    
    img = Image.open(image_path).convert('RGB')
    img_array = np.array(img, dtype=np.float64) / 255.0
    
    # 通道统计
    features = {}
    for i, channel in enumerate(['R', 'G', 'B']):
        ch = img_array[:, :, i]
        features[f'{channel}_mean'] = np.mean(ch)
        features[f'{channel}_std'] = np.std(ch)
    
    # 植被指数（类比）
    R, G, B = img_array[:,:,0], img_array[:,:,1], img_array[:,:,2]
    features['excess_green'] = 2*G - R - B  # ExG
    features['green_ratio'] = G / (R + G + B + 1e-10)
    
    return features
```

### 4.2 SG 平滑滤波

```python
def sg_smoothing(spectrum, window_length=11, polyorder=3):
    """
    Savitzky-Golay 平滑滤波
    
    优势：在去除噪声的同时保留光谱特征（峰位、峰形）
    参数：
        window_length: 窗口长度（奇数）
        polyorder: 多项式阶数
    """
    smoothed = savgol_filter(spectrum, window_length, polyorder)
    return smoothed

def compare_smoothing_methods(wavelength, noisy_spectrum):
    """对比不同平滑方法的效果"""
    fig, axes = plt.subplots(2, 2, figsize=(14, 8))
    
    # 原始含噪光谱
    axes[0, 0].plot(wavelength, noisy_spectrum, 'b-', linewidth=0.5, alpha=0.7)
    axes[0, 0].set_title('原始含噪光谱', fontproperties=font_prop)
    axes[0, 0].set_xlabel('波长 (nm)', fontproperties=font_prop)
    axes[0, 0].set_ylabel('反射率', fontproperties=font_prop)
    
    # SG 平滑
    sg_smooth = sg_smoothing(noisy_spectrum, window_length=11, polyorder=3)
    axes[0, 1].plot(wavelength, sg_smooth, 'r-', linewidth=1)
    axes[0, 1].set_title('Savitzky-Golay 平滑', fontproperties=font_prop)
    axes[0, 1].set_xlabel('波长 (nm)', fontproperties=font_prop)
    
    # 移动平均
    kernel = np.ones(11) / 11
    ma_smooth = np.convolve(noisy_spectrum, kernel, mode='same')
    axes[1, 0].plot(wavelength, ma_smooth, 'g-', linewidth=1)
    axes[1, 0].set_title('移动平均平滑', fontproperties=font_prop)
    axes[1, 0].set_xlabel('波长 (nm)', fontproperties=font_prop)
    
    # 对比
    axes[1, 1].plot(wavelength, noisy_spectrum, 'b-', linewidth=0.3, alpha=0.5, label='原始')
    axes[1, 1].plot(wavelength, sg_smooth, 'r-', linewidth=1.5, label='SG平滑')
    axes[1, 1].plot(wavelength, ma_smooth, 'g-', linewidth=1.5, label='移动平均')
    axes[1, 1].set_title('方法对比', fontproperties=font_prop)
    axes[1, 1].set_xlabel('波长 (nm)', fontproperties=font_prop)
    axes[1, 1].legend(prop=font_prop)
    
    plt.tight_layout()
    plt.savefig('smoothing_comparison.png', dpi=150, bbox_inches='tight')
    plt.show()
```

### 4.3 导数光谱分析

```python
def compute_derivative_spectra(wavelength, spectrum, orders=[1, 2]):
    """
    计算导数光谱
    
    一阶导数：突出光谱变化速率 → 红边位置
    二阶导数：突出吸收特征 → 病害特征波段
    """
    derivatives = {}
    
    if 1 in orders:
        # 一阶导数
        d1 = np.gradient(spectrum, wavelength)
        d1_smooth = sg_smoothing(d1, window_length=15, polyorder=3)
        derivatives['first'] = d1_smooth
    
    if 2 in orders:
        # 二阶导数
        d2 = np.gradient(np.gradient(spectrum, wavelength), wavelength)
        d2_smooth = sg_smoothing(d2, window_length=21, polyorder=3)
        derivatives['second'] = d2_smooth
    
    return derivatives

def analyze_red_edge(wavelength, spectrum):
    """
    分析红边参数
    
    红边（Red Edge）：680-750nm 区间反射率急剧上升的区域
    是植被健康状态的敏感指标
    """
    # 一阶导数
    d1 = np.gradient(spectrum, wavelength)
    d1_smooth = sg_smoothing(d1, window_length=11, polyorder=3)
    
    # 红边区域
    re_mask = (wavelength >= 680) & (wavelength <= 750)
    re_wavelength = wavelength[re_mask]
    re_d1 = d1_smooth[re_mask]
    
    # 红边位置（一阶导数最大值）
    rep_idx = np.argmax(re_d1)
    red_edge_position = re_wavelength[rep_idx]
    
    # 红边幅值
    red_edge_amplitude = re_d1[rep_idx]
    
    # 红边面积
    red_edge_area = np.trapz(re_d1, re_wavelength)
    
    return {
        'red_edge_position': red_edge_position,
        'red_edge_amplitude': red_edge_amplitude,
        'red_edge_area': red_edge_area
    }

def visualize_derivatives(wavelength, spectrum_dict):
    """可视化不同健康状态的原始光谱和导数光谱"""
    fig, axes = plt.subplots(3, 1, figsize=(12, 10))
    
    colors = {'healthy': 'green', 'rust': 'orange', 'mildew': 'purple', 'blight': 'red'}
    
    for status, (wl, rho) in spectrum_dict.items():
        # 原始光谱
        axes[0].plot(wl, rho, color=colors[status], linewidth=1, label=status)
        
        # 一阶导数
        d1 = compute_derivative_spectra(wl, rho, orders=[1])['first']
        axes[1].plot(wl, d1, color=colors[status], linewidth=1, label=status)
        
        # 二阶导数
        d2 = compute_derivative_spectra(wl, rho, orders=[2])['second']
        axes[2].plot(wl, d2, color=colors[status], linewidth=1, label=status)
    
    axes[0].set_title('原始反射率光谱', fontproperties=font_prop)
    axes[0].set_ylabel('反射率', fontproperties=font_prop)
    axes[0].legend(prop=font_prop)
    axes[0].axvspan(680, 750, alpha=0.1, color='red', label='红边')
    
    axes[1].set_title('一阶导数光谱', fontproperties=font_prop)
    axes[1].set_ylabel('dρ/dλ', fontproperties=font_prop)
```

### 4.4 特征波段选择

```python
from sklearn.feature_selection import mutual_info_classif, f_classif

def select_feature_bands(X, y, wavelength, method='anova', n_features=20):
    """
    特征波段选择
    
    X: 光谱矩阵 (n_samples, n_bands)
    y: 标签 (n_samples,)
    """
    if method == 'anova':
        # 方差分析 F 值
        f_scores, p_values = f_classif(X, y)
        scores = f_scores
    elif method == 'mutual_info':
        # 互信息
        scores = mutual_info_classif(X, y, random_state=42)
    
    # 选择 top-N 特征
    top_indices = np.argsort(scores)[-n_features:]
    top_wavelengths = wavelength[top_indices]
    top_scores = scores[top_indices]
    
    # 可视化
    fig, axes = plt.subplots(2, 1, figsize=(14, 6))
    
    axes[0].bar(range(len(scores)), scores, alpha=0.3, color='blue', label='所有波段')
    axes[0].bar(top_indices, top_scores, alpha=0.8, color='red', label=f'Top-{n_features}')
    axes[0].set_xlabel('波段索引', fontproperties=font_prop)
    axes[0].set_ylabel('F 值' if method == 'anova' else '互信息', fontproperties=font_prop)
    axes[0].set_title('特征波段重要性', fontproperties=font_prop)
    axes[0].legend(prop=font_prop)
    
    # 映射到波长
    axes[1].plot(wavelength, np.mean(X, axis=0), 'k-', linewidth=0.5, label='平均光谱')
    for w in top_wavelengths:
        axes[1].axvline(x=w, color='red', alpha=0.3, linewidth=2)
    axes[1].set_xlabel('波长 (nm)', fontproperties=font_prop)
    axes[1].set_ylabel('反射率', fontproperties=font_prop)
    axes[1].set_title(f'选中的 Top-{n_features} 特征波段', fontproperties=font_prop)
    axes[1].legend(prop=font_prop)
    
    plt.tight_layout()
    plt.savefig('feature_bands.png', dpi=150, bbox_inches='tight')
    plt.show()
    
    return top_indices, top_wavelengths
```

### 4.5 SVM 分类器

```python
from sklearn.svm import SVC
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import cross_val_score, StratifiedKFold
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline
from sklearn.metrics import classification_report, confusion_matrix
import seaborn as sns

def train_disease_classifier(X, y, class_names, method='svm'):
    """
    训练病虫害分类器
    
    X: 特征矩阵 (n_samples, n_features)
    y: 标签
    """
    if method == 'svm':
        clf = Pipeline([
            ('scaler', StandardScaler()),
            ('svm', SVC(kernel='rbf', C=10, gamma='scale', probability=True))
        ])
    elif method == 'rf':
        clf = Pipeline([
            ('scaler', StandardScaler()),
            ('rf', RandomForestClassifier(n_estimators=200, max_depth=20, random_state=42))
        ])
    
    # 交叉验证
    cv = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
    scores = cross_val_score(clf, X, y, cv=cv, scoring='accuracy')
    print(f"5折交叉验证准确率: {scores.mean():.4f} ± {scores.std():.4f}")
    
    # 训练最终模型
    clf.fit(X, y)
    y_pred = clf.predict(X)
    
    # 分类报告
    print("\n分类报告:")
    print(classification_report(y, y_pred, target_names=class_names))
    
    # 混淆矩阵
    cm = confusion_matrix(y, y_pred)
    fig, ax = plt.subplots(figsize=(8, 6))
    im = ax.imshow(cm, interpolation='nearest', cmap='Blues')
    ax.set_title('混淆矩阵', fontproperties=font_prop)

    ax.set_xticklabels(class_names, rotation=45, fontproperties=font_prop)

    # 标注数值
    for i in range(cm.shape[0]):
        for j in range(cm.shape[1]):

    ax.set_ylabel('真实标签', fontproperties=font_prop)
    ax.set_xlabel('预测标签', fontproperties=font_prop)
    
    plt.tight_layout()
    plt.savefig('confusion_matrix.png', dpi=150, bbox_inches='tight')
    plt.show()
    
    return clf
```

### 4.6 从 PlantVillage 图像提取光谱特征

```python
from PIL import Image
import os

def extract_color_vegetation_indices(image_path):
    """从 RGB 图像提取类光谱植被指数"""
    img = Image.open(image_path).convert('RGB')
    img_array = np.array(img, dtype=np.float64) / 255.0
    
    R = img_array[:, :, 0]
    G = img_array[:, :, 1]
    B = img_array[:, :, 2]
    
    indices = {}
    
    # 超绿指数 ExG
    indices['ExG'] = np.mean(2*G - R - B)
    
    # 超红指数 ExR
    indices['ExR'] = np.mean(1.4*R - G)
    
    # 绿色比率
    indices['GR'] = np.mean(G / (R + G + B + 1e-10))
    
    # 色调相关
    # 转换到 HSV
    from matplotlib.colors import rgb_to_hsv
    hsv = rgb_to_hsv(img_array)
    indices['H_mean'] = np.mean(hsv[:, :, 0])
    indices['S_mean'] = np.mean(hsv[:, :, 1])
    indices['V_mean'] = np.mean(hsv[:, :, 2])
    
    # 纹理特征（类比空间频率）
    from scipy.ndimage import sobel
    edges_x = sobel(img_array.mean(axis=2), axis=0)
    edges_y = sobel(img_array.mean(axis=2), axis=1)
    indices['edge_energy'] = np.mean(edges_x**2 + edges_y**2)
    
    return indices

def build_plantvillage_dataset(data_dir, max_per_class=500):
    """从 PlantVillage 构建特征数据集"""
    features_list = []
    labels = []
    class_names = []
    
    class_dirs = sorted(os.listdir(data_dir))
    
    for class_idx, class_name in enumerate(class_dirs):
        class_path = os.path.join(data_dir, class_name)
        if not os.path.isdir(class_path):
            continue
        class_names.append(class_name)
        
        count = 0
        for img_name in os.listdir(class_path):
            if not img_name.lower().endswith(('.jpg', '.jpeg', '.png')):
                continue
            if count >= max_per_class:
                break
            
            img_path = os.path.join(class_path, img_name)
            try:
                feat = extract_color_vegetation_indices(img_path)
                features_list.append(list(feat.values()))
                labels.append(class_idx)
                count += 1
            except Exception as e:
                continue
    
    X = np.array(features_list)
    y = np.array(labels)
    
    return X, y, class_names
```

## 五、涉及知识点
