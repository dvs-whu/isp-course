# B1 脑电信号情绪识别

## 项目背景

情绪是人类认知和行为的重要组成部分，对心理健康、人机交互、智能驾驶等领域具有重要意义。传统的情绪评估方法依赖问卷和行为观察，存在主观性强、实时性差等局限。脑电图（EEG）作为一种非侵入式神经电生理信号，能够直接反映大脑皮层的电活动，具有高时间分辨率（毫秒级）的优势，被认为是情绪识别的理想模态之一。

情感计算（Affective Computing）是人工智能领域的前沿方向，旨在赋予机器感知、理解和表达情感的能力。基于 EEG 的情绪识别系统可应用于：抑郁症早期筛查、注意力缺陷多动症（ADHD）辅助诊断、驾驶员疲劳监测、自适应学习系统等。近年来，深度学习技术的发展使得基于 EEG 的情绪分类准确率显著提升。

本项目旨在通过信号处理与机器学习技术，从多通道脑电信号中提取频域特征，构建自动化的情绪分类模型。学生将掌握 EEG 信号预处理、频谱分析、特征工程及深度学习分类的完整技术流程，理解生物电信号处理与传统工程信号处理的共性与差异。

## 信号处理建模

### EEG 信号数学模型

多通道 EEG 信号可表示为矩阵形式：

$$X(t) = [x_1(t), x_2(t), \ldots, x_C(t)]^T, \quad t = 1, 2, \ldots, N$$

其中 $C$ 为通道数（如 DEAP 数据集的 32 通道），$N$ 为采样点数。

### 频带能量特征提取

对每个通道信号进行短时傅里叶变换（STFT）：

$$X_i(f, t) = \sum_{\tau=0}^{L-1} x_i(\tau) \cdot w(\tau - t) \cdot e^{-j2\pi f\tau}$$

功率谱密度（PSD）为：

$$P_i(f) = |X_i(f)|^2$$

五大经典脑电频带的能量计算：

| 频带 | 频率范围 | 对应脑功能 |
|------|----------|-----------|
| δ（Delta） | 1–4 Hz | 深度睡眠、潜意识 |
| θ（Theta） | 4–8 Hz | 记忆、情绪 |
| α（Alpha） | 8–13 Hz | 放松、安静 |
| β（Beta） | 13–30 Hz | 注意、警觉 |
| γ（Gamma） | 30–50 Hz | 高级认知 |

频带能量定义为：

$$E_{band} = \int_{f_1}^{f_2} P_i(f) \, df = \sum_{f=f_1}^{f_2} P_i(f) \cdot \Delta f$$

### 差分熵特征

差分熵（Differential Entropy, DE）是 EEG 情感识别中广泛使用的特征：

$$DE = \int_{f_1}^{f_2} \frac{1}{2} \log_2(2\pi e \cdot P(f)) \, df$$

对于高斯分布信号，DE 与对数功率谱密度成正比：

$$DE \approx \frac{1}{2} \log_2\left(\frac{1}{f_2-f_1}\int_{f_1}^{f_2} P(f) \, df\right) + \frac{1}{2}\log_2(2\pi e)$$

### 情绪维度模型

DEAP 数据集采用 Russell 的二维情绪模型：

- **效价（Valence）**：不愉悦 ↔ 愉悦，范围 [1, 9]
- **唤醒度（Arousal）**：平静 ↔ 兴奋，范围 [1, 9]

分类任务可将连续评分离散化为二分类（高/低）或多分类。

## 数据来源

### DEAP 数据集

- **全称**：Dataset for Emotion Analysis using Physiological signals
- **网址**：https://www.eecs.qmul.ac.uk/mmv/datasets/deap/
- **规模**：32 名被试，每人 40 个 trials（观看 1 分钟音乐视频）
- **通道**：32 通道 EEG + 8 通道外周生理信号
- **采样率**：原始 512 Hz，预处理后降采样至 128 Hz
- **格式**：`.dat` 文件，Python pickle 格式
- **标签**：Valence, Arousal, Dominance, Liking（1-9 分）
- **下载**：需在官网注册申请，填写使用协议后获取下载链接

### SEED 数据集

- **全称**：Shanghai Jiao Tong University Emotion EEG Dataset
- **网址**：https://bcmi.sjtu.edu.cn/~seed/
- **规模**：15 名被试，每人 3 个 session，每 session 15 个 trials
- **通道**：62 通道 EEG
- **采样率**：200 Hz
- **标签**：正面/中性/负面 三分类
- **格式**：MATLAB `.mat` 文件
- **下载**：官网填写申请表后获取

## 数据处理方法

### 处理流程

```
原始 EEG → 通道选择 → 带通滤波(4-45Hz) → 去伪迹(ICA) →
分段(epoch) → 特征提取(频带能量/DE) → 特征标准化 → 分类器
```

### Step 1：数据加载与预处理

```python
import numpy as np
from scipy.io import loadmat
from scipy.signal import butter, filtfilt
```

- 加载 `.dat` 或 `.mat` 数据
- 选择 32 通道 EEG（排除外周信号通道）
- 带通滤波 4–45 Hz，消除直流漂移和高频噪声

### Step 2：去伪迹

- 使用 ICA（独立成分分析）去除眼电（EOG）和肌电（EMG）伪迹
- 或使用阈值法剔除幅值超过 ±100 μV 的 epoch

### Step 3：频带能量特征提取

- 对每个 epoch（如 4 秒）做 FFT
- 计算五大频带的功率谱能量
- 特征维度：32 通道 × 5 频带 = 160 维

### Step 4：分类

- **传统方法**：SVM / Random Forest / LDA
- **深度学习**：CNN 直接学习频谱图特征，或 LSTM 捕捉时序依赖

def generate_simulated_eeg(n_subjects=5, n_trials=40, n_channels=32,
                           fs=128, duration=4):
    """
    模拟生成多通道 EEG 数据
    返回：(n_subjects*n_trials, n_channels, n_samples)
    """
    n_samples = fs * duration
    n_total = n_subjects * n_trials
    X = np.zeros((n_total, n_channels, n_samples))
    y = np.zeros(n_total, dtype=int)  # 0:低效价, 1:高效价

    t = np.arange(n_samples) / fs
    for i in range(n_total):
        label = np.random.randint(0, 2)
        y[i] = label
        for ch in range(n_channels):
            # 基础脑电节律 + 噪声
            delta = np.sin(2 * np.pi * 2 * t) * np.random.uniform(5, 15)
            theta = np.sin(2 * np.pi * 6 * t) * np.random.uniform(3, 10)
            alpha = np.sin(2 * np.pi * 10 * t) * np.random.uniform(5, 20)
            beta = np.sin(2 * np.pi * 20 * t) * np.random.uniform(2, 8)
            gamma = np.sin(2 * np.pi * 35 * t) * np.random.uniform(1, 5)
            noise = np.random.randn(n_samples) * 3

            # 根据标签调整频带能量比例
            if label == 1:  # 高效价 → alpha 增强
                alpha *= 1.5
                beta *= 1.3
            else:           # 低效价 → theta 增强
                theta *= 1.5
                delta *= 1.3

            X[i, ch] = delta + theta + alpha + beta + gamma + noise

    return X, y, fs

# ============================================================
# 2. 带通滤波器设计
# ============================================================
def bandpass_filter(data, low, high, fs, order=4):
    """Butterworth 带通滤波器"""
    nyq = fs / 2.0
    b, a = butter(order, [low/nyq, high/nyq], btype='band')
    return filtfilt(b, a, data, axis=-1)

# ============================================================
# 3. 频带能量特征提取
# ============================================================
def extract_band_power(data, fs, bands=None):
    """
    提取各频带的功率谱能量
    参数：
        data: (n_epochs, n_channels, n_samples)
        fs: 采样率
        bands: 频带字典 {name: (low, high)}
    返回：
        features: (n_epochs, n_channels * n_bands)
    """
    if bands is None:
        bands = {
            'Delta': (1, 4),
            'Theta': (4, 8),
            'Alpha': (8, 13),
            'Beta':  (13, 30),
            'Gamma': (30, 50)
        }

    n_epochs, n_channels, n_samples = data.shape
    n_bands = len(bands)
    features = np.zeros((n_epochs, n_channels * n_bands))

    for ep in range(n_epochs):
        feat_idx = 0
        for ch in range(n_channels):
            signal = data[ep, ch]
            # 使用 Welch 方法估计 PSD
            freqs, psd = welch(signal, fs=fs, nperseg=min(256, n_samples))
            for band_name, (f_low, f_high) in bands.items():
                mask = (freqs >= f_low) & (freqs <= f_high)
                band_power = np.trapz(psd[mask], freqs[mask])
                features[ep, feat_idx] = np.log(band_power + 1e-10)
                feat_idx += 1

    return features

# ============================================================
# 4. 差分熵特征提取
# ============================================================
def extract_differential_entropy(data, fs, bands=None):
    """
    提取差分熵（DE）特征
    """
    if bands is None:
        bands = {
            'Delta': (1, 4),
            'Theta': (4, 8),
            'Alpha': (8, 13),
            'Beta':  (13, 30),
            'Gamma': (30, 50)
        }

    n_epochs, n_channels, n_samples = data.shape
    n_bands = len(bands)
    features = np.zeros((n_epochs, n_channels * n_bands))

    for ep in range(n_epochs):
        feat_idx = 0
        for ch in range(n_channels):
            signal = data[ep, ch]
            freqs, psd = welch(signal, fs=fs, nperseg=min(256, n_samples))
            for band_name, (f_low, f_high) in bands.items():
                mask = (freqs >= f_low) & (freqs <= f_high)
                band_psd = psd[mask]
                if len(band_psd) > 0:
                    # DE = 0.5 * log2(2*pi*e*sigma^2)
                    mean_power = np.mean(band_psd)
                    de = 0.5 * np.log2(2 * np.pi * np.e * mean_power + 1e-10)
                else:
                    de = 0
                features[ep, feat_idx] = de
                feat_idx += 1

    return features

# 5. CNN 情绪分类模型（PyTorch）
# ============================================================
def build_cnn_model():
    """构建 1D CNN 模型用于 EEG 情绪分类"""
    try:
        import torch
        import torch.nn as nn
        import torch.optim as optim

        class EEGCNN(nn.Module):
            def __init__(self, n_features=160, n_classes=2):
                super(EEGCNN, self).__init__()
                self.conv1 = nn.Sequential(
                    nn.Conv1d(1, 32, kernel_size=5, padding=2),
                    nn.BatchNorm1d(32),
                    nn.ReLU(),
                    nn.MaxPool1d(2),
                    nn.Dropout(0.3)
                )
                self.conv2 = nn.Sequential(
                    nn.Conv1d(32, 64, kernel_size=3, padding=1),
                    nn.BatchNorm1d(64),
                    nn.ReLU(),
                    nn.MaxPool1d(2),
                    nn.Dropout(0.3)
                )
                self.fc = nn.Sequential(
                    nn.Linear(64 * (n_features // 4), 128),
                    nn.ReLU(),
                    nn.Dropout(0.5),
                    nn.Linear(128, n_classes)
                )

            def forward(self, x):
                x = self.conv1(x)
                x = self.conv2(x)
                x = x.view(x.size(0), -1)
                x = self.fc(x)
                return x

        model = EEGCNN()
        print("CNN 模型结构：")
        print(model)
        return model

    except ImportError:
        print("PyTorch 未安装，跳过 CNN 模型构建")
        return None

def main():
    print("=" * 60)

    # 1. 生成模拟数据

    # 5. 分类
    print("\n[5/6] 情绪分类...")

    # 频带能量 + SVM
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(features_bp)
    X_train, X_test, y_train, y_test = train_test_split(
        X_scaled, y, test_size=0.3, random_state=42, stratify=y
    )

    svm_model = SVC(kernel='rbf', C=1.0, gamma='scale')
    svm_model.fit(X_train, y_train)
    y_pred_svm = svm_model.predict(X_test)

    print("\n--- SVM 分类结果（频带能量特征）---")
    print(classification_report(y_test, y_pred_svm,
          target_names=['低效价', '高效价']))

    # Random Forest + DE 特征
    X_de_scaled = StandardScaler().fit_transform(features_de)
    X_train_de, X_test_de, y_train_de, y_test_de = train_test_split(

    )

    rf_model = RandomForestClassifier(n_estimators=100, random_state=42)
    rf_model.fit(X_train_de, y_train_de)
    y_pred_rf = rf_model.predict(X_test_de)

    print("\n--- Random Forest 分类结果（差分熵特征）---")
    print(classification_report(y_test_de, y_pred_rf,

    # 功率谱密度

    freqs, psd_after = welch(X_filtered[0, 0], fs=fs, nperseg=256)
    axes[0, 1].semilogy(freqs, psd_before, 'b-', alpha=0.7, label='原始')

    band_names = ['Delta', 'Theta', 'Alpha', 'Beta', 'Gamma']
    n_channels = 32
    avg_power_label0 = np.mean(features_bp[y==0], axis=0)
    avg_power_label1 = np.mean(features_bp[y==1], axis=0)

1. **跨被试泛化**：用leave-one-subject-out验证模型泛化能力
2. **实时情绪检测**：实现滑动窗口的在线情绪识别
3. **多模态融合**：结合EEG+GSR（皮肤电）信号
4. **注意力可视化**：用Grad-CAM可视化哪些频段对分类最重要

> 💡 这些挑战方向可以作为课程项目的加分项或毕业设计的起点。

## 扩展方向

1. **多模态融合**：结合 EEG + ECG + GSR（皮肤电导）的多模态情绪识别
2. **图神经网络（GNN）**：利用 EEG 通道的空间拓扑关系建模脑功能连接
3. **迁移学习**：利用跨被试、跨数据集的域适应技术解决个体差异问题
4. **实时系统**：基于 OpenBCI 硬件构建实时情绪识别系统
5. **自监督预训练**：利用大量无标签 EEG 数据进行预训练
6. **注意力机制**：引入 Transformer 架构捕捉长程时间依赖
