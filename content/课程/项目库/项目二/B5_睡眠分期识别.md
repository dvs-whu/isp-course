# B5 睡眠分期自动识别

## 项目背景

睡眠是人体最重要的生理活动之一，占据人生约三分之一的时间。睡眠质量直接影响免疫功能、认知能力、情绪调节和代谢健康。世界卫生组织数据显示，全球约 27% 的人存在不同程度的睡眠障碍，包括失眠、睡眠呼吸暂停综合征（OSA）、发作性睡病等。准确的睡眠分期评估是诊断睡眠障碍的基础。

多导睡眠图（Polysomnography, PSG）是睡眠监测的金标准，通过同步记录脑电图（EEG）、眼电图（EOG）、肌电图（EMG）等多种生理信号，由专业的睡眠技师按照 R&K 规则或 AASM 标准进行人工分期。传统的手工分期工作量巨大（每晚 6-8 小时的记录需 2-3 小时标注），且不同技师之间的一致性仅为 75%-85%。

本项目利用 Sleep-EDF 公开数据集，通过 EEG 频谱分析提取睡眠特征，结合随机森林和 LSTM 等机器学习/深度学习方法，实现自动化的睡眠五期分类。学生将掌握睡眠生理学基础、生物信号的时频分析方法以及序列分类的建模技术。

## 信号处理建模

### AASM 睡眠分期标准

美国睡眠医学学会（AASM）将睡眠分为以下阶段：

| 阶段 | 编号 | 特征 EEG 频率 | 说明 |
|------|------|-------------|------|
| Wake（清醒） | W | Alpha (8-13 Hz), Beta (>13 Hz) | 眼动活跃，肌电活跃 |
| N1（浅睡I期） | 1 | Theta (4-7 Hz) | α 逐渐消失，出现低幅混合频率 |
| N2（浅睡II期） | 2 | 睡眠纺锤波 (12-14 Hz), K 复合波 | 纺锤波和 K 复合波为特征标志 |
| N3（深睡期） | 3 | Delta (0.5-2 Hz) | 高幅慢波，>75 μV |
| REM（快速眼动） | R | 低幅混合频率 | 类似清醒，快速眼动，肌张力降低 |

### EEG 频谱特征提取

对 EEG 信号进行短时傅里叶变换（STFT）：

$$S(f, t) = \left|\sum_{\tau=0}^{L-1} x(\tau) \cdot w(\tau - t) \cdot e^{-j2\pi f\tau}\right|^2$$

各频带的相对功率：

$$R_{band} = \frac{P_{band}}{P_{total}}, \quad P_{total} = \sum_{f=0.5}^{f_s/2} P(f)$$

频带功率比值特征：

$$F_{ratio} = \frac{P_\alpha + P_\beta}{P_\delta + P_\theta}$$

清醒时 $F_{ratio}$ 较大，深睡时 $F_{ratio}$ 较小。

### 睡眠特征向量

每个 epoch（30 秒）提取的特征：

1. **频带绝对功率**：$\delta$、$\theta$、$\alpha$、$\sigma$（纺锤波带）、$\beta$ 各频带功率
2. **频带相对功率**：各频带功率占总功率的比例
3. **频带功率比值**：$\delta/\theta$、$\alpha/\theta$、$\theta/\beta$ 等
4. **Hjorth 参数**：
   - Activity：$\sigma^2 = \text{Var}(x)$
   - Mobility：$M = \sqrt{\text{Var}(x') / \text{Var}(x)}$
   - Complexity：$C = M(x') / M(x)$
5. **统计特征**：均值、方差、偏度、峰度、过零率
6. **熵特征**：样本熵、近似熵、谱熵

### LSTM 时序模型

睡眠分期具有强时序依赖性（前一阶段影响后一阶段），LSTM 通过门控机制捕捉这种长期依赖：

$$f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f) \quad \text{(遗忘门)}$$
$$i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i) \quad \text{(输入门)}$$
$$\tilde{C}_t = \tanh(W_C \cdot [h_{t-1}, x_t] + b_C) \quad \text{(候选记忆)}$$
$$C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t \quad \text{(记忆更新)}$$
$$o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o) \quad \text{(输出门)}$$
$$h_t = o_t \odot \tanh(C_t) \quad \text{(隐藏状态)}$$

## 数据处理方法

### 处理流程

```
PSG 原始数据 → 提取 EEG 通道 → 带通滤波(0.5-30Hz) →
分段(30秒 epoch) → 频谱特征提取 → 特征标准化 →
分类(Random Forest / LSTM) → 平滑后处理 → 评估
```

### Step 1：数据加载与预处理

- 使用 MNE-Python 读取 EDF 格式
- 提取 EEG 通道（Fpz-Cz 或 Pz-Oz）
- 带通滤波 0.5–30 Hz
- 按 30 秒分段（与标注对齐）

### Step 2：频谱特征提取

- 每个 epoch 做 FFT 或 Welch PSD 估计
- 计算五大频带的绝对功率和相对功率
- 计算 Hjorth 参数和统计特征

### Step 3：分类

- **Random Forest**：对每个 epoch 独立分类
- **LSTM**：利用前后 epoch 的时序上下文分类
- **序列平滑**：去除过短的阶段（如单个 epoch 的 N1）

# 1. 模拟睡眠 EEG 数据生成

def generate_sleep_eeg(duration_hours=8, fs=100):
    """
    生成模拟的睡眠 EEG 数据（含不同睡眠阶段特征）
    """
    np.random.seed(42)
    epoch_duration = 30  # 秒
    n_epochs = int(duration_hours * 3600 / epoch_duration)
    samples_per_epoch = fs * epoch_duration

    # 生成睡眠阶段序列（简化版真实睡眠结构）
    # 典型睡眠周期：W → N1 → N2 → N3 → N2 → REM，约 90 分钟
    stages = []
    cycle = ['W', 'W', 'N1', 'N2', 'N2', 'N3', 'N3', 'N2', 'N2', 'REM', 'REM']
    cycle_len = len(cycle)

    for i in range(n_epochs):
        # 模拟真实睡眠结构
        hour = i * epoch_duration / 3600
        if hour < 0.1:  # 入睡前清醒
            stages.append('W')
        elif hour < 0.5:  # 入睡期
            stages.append(np.random.choice(['W', 'N1', 'N2'], p=[0.3, 0.3, 0.4]))
        else:
            cycle_idx = int((hour - 0.5) * 60 / 90) % cycle_len
            # 添加噪声
            if np.random.random() < 0.15:
                stages.append(np.random.choice(['W', 'N1', 'N2', 'N3', 'REM']))
            else:
                stages.append(cycle[cycle_idx])

    # 根据阶段生成相应特征的 EEG 信号
    signals = np.zeros(n_epochs * samples_per_epoch)
    t = np.arange(n_epochs * samples_per_epoch) / fs

    for i, stage in enumerate(stages):
        start = i * samples_per_epoch
        end = start + samples_per_epoch
        t_epoch = t[start:end]

        base_noise = 0.5 * np.random.randn(samples_per_epoch)

        if stage == 'W':
            # 清醒：alpha + beta 主导
            alpha = 10 * np.sin(2 * np.pi * 10 * t_epoch + np.random.rand() * 2 * np.pi)
            beta = 5 * np.sin(2 * np.pi * 20 * t_epoch + np.random.rand() * 2 * np.pi)
            signals[start:end] = alpha + beta + base_noise * 2

        elif stage == 'N1':
            # 浅睡 I 期：theta 增加，alpha 减少
            theta = 8 * np.sin(2 * np.pi * 6 * t_epoch + np.random.rand() * 2 * np.pi)
            alpha = 3 * np.sin(2 * np.pi * 9 * t_epoch + np.random.rand() * 2 * np.pi)
            signals[start:end] = theta + alpha + base_noise * 1.5

        elif stage == 'N2':
            # 浅睡 II 期：睡眠纺锤波 + K 复合波
            delta = 4 * np.sin(2 * np.pi * 2 * t_epoch + np.random.rand() * 2 * np.pi)
            spindle = 6 * np.sin(2 * np.pi * 13 * t_epoch + np.random.rand() * 2 * np.pi)
            # K 复合波（低频大波）
            k_idx = samples_per_epoch // 2
            k_wave = np.zeros(samples_per_epoch)
            k_wave[max(0,k_idx-50):min(samples_per_epoch,k_idx+50)] = \
                -20 * np.exp(-np.arange(-50, 50)**2 / 200)
            signals[start:end] = delta + spindle + k_wave + base_noise

        elif stage == 'N3':
            # 深睡期：大振幅 delta 波
            delta = 20 * np.sin(2 * np.pi * 1.5 * t_epoch + np.random.rand() * 2 * np.pi)
            delta2 = 10 * np.sin(2 * np.pi * 0.8 * t_epoch + np.random.rand() * 2 * np.pi)
            signals[start:end] = delta + delta2 + base_noise * 0.8

        elif stage == 'REM':
            # REM：类似清醒但低振幅，无 alpha
            beta = 3 * np.sin(2 * np.pi * 18 * t_epoch + np.random.rand() * 2 * np.pi)
            theta = 4 * np.sin(2 * np.pi * 5 * t_epoch + np.random.rand() * 2 * np.pi)
            signals[start:end] = beta + theta + base_noise

    return signals, stages, fs, n_epochs, samples_per_epoch

# 2. 频谱特征提取
# ============================================================
def extract_sleep_features(signal_epoch, fs):
    """
    从单个 30 秒 epoch 提取睡眠特征
    """
    features = {}

    # --- 频域特征 ---
    freqs, psd = welch(signal_epoch, fs=fs, nperseg=min(256, len(signal_epoch)))

    # 频带定义
    bands = {
        'Delta': (0.5, 4),
        'Theta': (4, 8),
        'Alpha': (8, 13),
        'Sigma': (12, 16),  # 纺锤波带
        'Beta':  (13, 30)
    }

    total_power = np.trapz(psd[(freqs >= 0.5) & (freqs <= 30)],
                           freqs[(freqs >= 0.5) & (freqs <= 30)])

    band_powers = {}
    for band_name, (f_low, f_high) in bands.items():
        mask = (freqs >= f_low) & (freqs <= f_high)
        abs_power = np.trapz(psd[mask], freqs[mask])
        rel_power = abs_power / (total_power + 1e-10)
        band_powers[band_name] = (abs_power, rel_power)
        features[f'{band_name}_abs'] = np.log(abs_power + 1e-10)
        features[f'{band_name}_rel'] = rel_power

    # 频带比值
    d = band_powers['Delta'][0] + 1e-10
    t = band_powers['Theta'][0] + 1e-10
    a = band_powers['Alpha'][0] + 1e-10
    b = band_powers['Beta'][0] + 1e-10

    features['ratio_dt'] = d / t
    features['ratio_at'] = a / t
    features['ratio_ab'] = a / b
    features['ratio_slow_fast'] = (d + t) / (a + b + 1e-10)

    # --- Hjorth 参数 ---
    x = signal_epoch - np.mean(signal_epoch)
    dx = np.diff(x)

    activity = np.var(x)
    mobility = np.sqrt(np.var(dx) / (activity + 1e-10))
    ddx = np.diff(dx)
    complexity = np.sqrt(np.var(ddx) / (np.var(dx) + 1e-10)) / (mobility + 1e-10)

    features['hjorth_activity'] = activity
    features['hjorth_mobility'] = mobility
    features['hjorth_complexity'] = complexity

    # --- 统计特征 ---
    features['mean'] = np.mean(signal_epoch)
    features['std'] = np.std(signal_epoch)
    features['skewness'] = skew(signal_epoch)
    features['kurtosis'] = kurtosis(signal_epoch)

    # 过零率
    zero_crossings = np.sum(np.diff(np.sign(signal_epoch)) != 0)
    features['zero_crossing_rate'] = zero_crossings / len(signal_epoch)

    # --- 谱熵 ---
    psd_norm = psd / (np.sum(psd) + 1e-10)
    spectral_entropy = -np.sum(psd_norm * np.log2(psd_norm + 1e-10))
    features['spectral_entropy'] = spectral_entropy

    return features

def extract_all_features(signals, fs, n_epochs, samples_per_epoch):
    """提取所有 epoch 的特征"""
    all_features = []
    for i in range(n_epochs):
        epoch = signals[i * samples_per_epoch:(i + 1) * samples_per_epoch]
        feat = extract_sleep_features(epoch, fs)
        all_features.append(feat)

    # 转换为 numpy 数组
    feature_names = list(all_features[0].keys())
    X = np.array([[f[name] for name in feature_names] for f in all_features])
    return X, feature_names

# 3. 序列平滑后处理
# ============================================================
def smooth_predictions(predictions, min_length=3):
    """
    去除过短的阶段（最小持续期约束）
    """
    smoothed = predictions.copy()
    n = len(smoothed)
    i = 0
    while i < n:
        j = i
        while j < n and smoothed[j] == smoothed[i]:
            j += 1
        if j - i < min_length:
            # 用前一个有效阶段填充
            fill_val = smoothed[max(0, i - 1)]
            smoothed[i:j] = fill_val
        i = j
    return smoothed

def main():
    print("=" * 60)
    print("B5 睡眠分期自动识别")
    print("=" * 60)

    # 1. 生成模拟数据
    print("\n[1/7] 生成模拟睡眠 EEG 数据...")
    duration_hours = 4  # 4 小时（演示用）
    fs = 100
    signals, stages, fs, n_epochs, samples_per_epoch = generate_sleep_eeg(

    print(f"    时长: {duration_hours} 小时")
    print(f"    采样率: {fs} Hz")
    print(f"    Epoch 数: {n_epochs}")
    print(f"    每个 epoch 采样点: {samples_per_epoch}")

    # 阶段分布
    unique, counts = np.unique(stages, return_counts=True)
    print("    睡眠阶段分布:")
    stage_names_cn = {'W': '清醒', 'N1': 'N1期', 'N2': 'N2期', 'N3': '深睡', 'REM': 'REM'}
    for s, c in zip(unique, counts):
        print(f"      {stage_names_cn.get(s, s)}: {c} epochs ({c/n_epochs*100:.1f}%)")

    # 2. 带通滤波
    print("\n[2/7] EEG 带通滤波...")
    nyq = fs / 2.0
    b, a = butter(4, [0.5/nyq, 30/nyq], btype='band')
    signals_filtered = filtfilt(b, a, signals)
    print("    滤波范围: 0.5-30 Hz")

    # 3. 特征提取
    print("\n[3/7] 提取睡眠特征...")
    X, feature_names = extract_all_features(signals_filtered, fs, n_epochs,
                                             samples_per_epoch)
    print(f"    特征矩阵形状: {X.shape}")
    print(f"    特征数: {len(feature_names)}")
    print(f"    特征列表: {', '.join(feature_names[:8])}...")

    # 4. 标签编码
    label_map = {'W': 0, 'N1': 1, 'N2': 2, 'N3': 3, 'REM': 4}
    y = np.array([label_map[s] for s in stages])

    # 处理 NaN/Inf
    X = np.nan_to_num(X, nan=0, posinf=0, neginf=0)

    # 5. 分类
    print("\n[4/7] 训练 Random Forest 分类器...")
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(X)

    # 时序划分（前 70% 训练，后 30% 测试）
    split_idx = int(0.7 * n_epochs)
    X_train, X_test = X_scaled[:split_idx], X_scaled[split_idx:]
    y_train, y_test = y[:split_idx], y[split_idx:]

    rf = RandomForestClassifier(n_estimators=200, max_depth=20,
                                 random_state=42, n_jobs=-1)
    rf.fit(X_train, y_train)
    y_pred = rf.predict(X_test)

    # 后处理：序列平滑
    y_pred_smooth = smooth_predictions(y_pred, min_length=3)

    stage_labels = ['W', 'N1', 'N2', 'N3', 'REM']
    print("\n--- Random Forest 分类结果（平滑前）---")
    print(classification_report(y_test, y_pred,
          target_names=stage_labels, zero_division=0))

    print("\n--- Random Forest 分类结果（平滑后）---")
    print(classification_report(y_test, y_pred_smooth,
          target_names=stage_labels, zero_division=0))

    kappa = cohen_kappa_score(y_test, y_pred_smooth)
    print(f"Cohen's Kappa: {kappa:.4f}")

    # 6. 特征重要性分析
    print("\n[5/7] 特征重要性分析...")
    importances = rf.feature_importances_
    sorted_idx = np.argsort(importances)[::-1]
    print("    Top 10 重要特征:")
    for i in range(min(10, len(feature_names))):
        idx = sorted_idx[i]
        print(f"      {feature_names[idx]}: {importances[idx]:.4f}")

    # 7. LSTM 模型构建
    print("\n[6/7] LSTM 模型构建...")
    try:
        import torch
        import torch.nn as nn

        class SleepLSTM(nn.Module):
            def __init__(self, input_dim=20, hidden_dim=128,
                         n_layers=2, n_classes=5):
                super(SleepLSTM, self).__init__()
                self.lstm = nn.LSTM(input_dim, hidden_dim, n_layers,
                                    batch_first=True, dropout=0.3,
                                    bidirectional=True)
                self.attention = nn.Sequential(
                    nn.Linear(hidden_dim * 2, 64),
                    nn.Tanh(),
                    nn.Linear(64, 1)
                )
                self.classifier = nn.Sequential(
                    nn.Dropout(0.5),
                    nn.Linear(hidden_dim * 2, 64),
                    nn.ReLU(),
                    nn.Dropout(0.3),
                    nn.Linear(64, n_classes)
                )

            def forward(self, x):
                # x: (batch, seq_len, features)
                lstm_out, _ = self.lstm(x)  # (batch, seq_len, hidden*2)

                # 注意力机制
                attn_weights = self.attention(lstm_out)  # (batch, seq_len, 1)
                attn_weights = torch.softmax(attn_weights, dim=1)
                context = torch.sum(lstm_out * attn_weights, dim=1)

                output = self.classifier(context)
                return output

        model = SleepLSTM(input_dim=len(feature_names))
        total_params = sum(p.numel() for p in model.parameters())
        print(f"    BiLSTM-Attention 参数量: {total_params:,}")
        print("    模型结构：")
        print(model)

    except ImportError:
        print("    PyTorch 未安装，跳过 LSTM 模型构建")

    # 8. 可视化
    print("\n[7/7] 可视化...")
    fig, axes = plt.subplots(4, 1, figsize=(16, 18))

    f, t_spec, Sxx = spectrogram(signals_filtered[:plot_samples],
                                  fs=fs, nperseg=256, noverlap=200)
    axes[0].pcolormesh(t_spec / 60, f, 10 * np.log10(Sxx + 1e-10),

    axes[0].set_ylabel('频率 (Hz)', fontproperties=zh_font)
    axes[0].set_xlabel('时间 (min)', fontproperties=zh_font)
    axes[0].set_title('睡眠 EEG 频谱图', fontproperties=zh_font)
    axes[0].set_ylim(0, 30)
    axes[0].set_xlim(0, plot_duration / 60)

    # 频带功率随时间变化
    epoch_times = np.arange(n_epochs) * 30 / 60  # 转换为分钟
    delta_idx = feature_names.index('Delta_rel')
    theta_idx = feature_names.index('Theta_rel')

    axes[1].plot(epoch_times, X[:, theta_idx], 'g-', lw=1.5, label='Theta')
    axes[1].plot(epoch_times, X[:, alpha_idx], 'r-', lw=1.5, label='Alpha')
    axes[1].set_xlabel('时间 (min)', fontproperties=zh_font)
    axes[1].set_ylabel('相对功率', fontproperties=zh_font)
    axes[1].set_title('各频带相对功率随时间变化', fontproperties=zh_font)
    axes[1].legend(prop=zh_font)
    axes[1].grid(True, alpha=0.3)

    # 真实 vs 预测睡眠分期
    test_times = epoch_times[split_idx:]
    stage_colors = {'W': '#FFD700', 'N1': '#87CEEB', 'N2': '#4169E1',
                    'N3': '#00008B', 'REM': '#FF6347'}

    # 真实标签
    for i in range(len(y_test)):
        s = stage_labels[y_test[i]]
        axes[2].axvspan(test_times[i], test_times[i] + 0.5,
                        facecolor=stage_colors[s], alpha=0.8, edgecolor='none')

    axes[2].set_yticks([])
    axes[2].set_title('真实睡眠分期', fontproperties=zh_font)
    axes[2].set_xlim(test_times[0], test_times[-1])

    # 预测标签
    for i in range(len(y_pred_smooth)):
        s = stage_labels[y_pred_smooth[i]]
        axes[3].axvspan(test_times[i], test_times[i] + 0.5,
                        facecolor=stage_colors[s], alpha=0.8, edgecolor='none')
    axes[3].set_xlabel('时间 (min)', fontproperties=zh_font)
    axes[3].set_yticks([])
    axes[3].set_title('预测睡眠分期 (Random Forest + 平滑)', fontproperties=zh_font)
    axes[3].set_xlim(test_times[0], test_times[-1])

    # 图例
    from matplotlib.patches import Patch
    legend_elements = [Patch(facecolor=stage_colors[s], label=stage_names_cn[s])
                       for s in ['W', 'N1', 'N2', 'N3', 'REM']]
    axes[3].legend(handles=legend_elements, loc='upper right', prop=zh_font)

    plt.tight_layout()
    plt.savefig('B5_sleep_staging_results.png', dpi=150, bbox_inches='tight')
    print("    图表已保存: B5_sleep_staging_results.png")

    # 混淆矩阵
    fig2, ax2 = plt.subplots(1, 1, figsize=(8, 6))
    cm = confusion_matrix(y_test, y_pred_smooth)
    cm_norm = cm.astype(float) / (cm.sum(axis=1, keepdims=True) + 1e-10)
    im = ax2.imshow(cm_norm, interpolation='nearest', cmap='Blues', vmin=0, vmax=1)
    plt.colorbar(im, ax=ax2)
    thresh = 0.5
    for i in range(cm_norm.shape[0]):
        for j in range(cm_norm.shape[1]):
            ax2.text(j, i, f'{cm_norm[i, j]:.2f}\n({cm[i, j]})',
                    ha="center", va="center",
                    color="white" if cm_norm[i, j] > thresh else "black")
    ax2.set_xticks(range(5))
    ax2.set_xticklabels(stage_labels, fontproperties=zh_font)
    ax2.set_yticks(range(5))
    ax2.set_yticklabels(stage_labels, fontproperties=zh_font)
    ax2.set_xlabel('预测阶段', fontproperties=zh_font)
    ax2.set_ylabel('真实阶段', fontproperties=zh_font)
    ax2.set_title('睡眠分期混淆矩阵', fontproperties=zh_font)

    plt.tight_layout()
    plt.savefig('B5_confusion_matrix.png', dpi=150, bbox_inches='tight')
    print("    图表已保存: B5_confusion_matrix.png")

## 扩展方向

1. **端到端深度学习**：直接用原始 EEG 信号训练 CNN-LSTM 模型，省去手工特征提取
2. **多通道融合**：利用多导联 EEG + EOG + EMG 信号联合分析
3. **睡眠事件检测**：检测纺锤波、K 复合波、慢波等微观睡眠事件
4. **睡眠质量评估**：基于分期结果计算睡眠效率、觉醒指数等指标
5. **睡眠呼吸事件**：结合鼻气流和血氧信号检测呼吸暂停
6. **迁移学习**：利用大规模数据预训练模型，适应小样本新数据
7. **可穿戴设备**：基于单通道前额 EEG 的轻量级睡眠监测
