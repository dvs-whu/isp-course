# B4 基因组序列周期性分析

## 项目背景

基因组学是生命科学的核心领域之一，DNA 序列承载着生命活动的全部遗传信息。尽管 DNA 序列由 A、T、C、G 四种碱基组成，看似随机排列，但其中蕴含着丰富的结构特征和统计规律。其中一个引人注目的现象是三碱基周期性（3-base periodicity），即在蛋白质编码区域（外显子）中，每隔 3 个核苷酸位置存在显著的信号周期性，这与遗传密码的三联体（密码子）编码方式直接相关。

这种周期性信号的产生机制与多种生物学因素有关：密码子使用偏好性（codon usage bias）、核小体定位、DNA 弯曲性以及翻译过程中的核糖体运动等。利用数字信号处理技术，特别是离散傅里叶变换（DFT），可以在频域中检测和量化这种周期性信号，从而辅助基因预测和编码区识别。

本项目将 DNA 序列转化为数值信号，应用傅里叶分析方法揭示编码区的三碱基周期性特征。学生将学习生物序列的数值编码方法、谱分析技术在基因组学中的应用，理解信号处理方法如何帮助解决生物学问题。

## 信号处理建模

### DNA 序列数值编码

将 DNA 序列 $S = s_1 s_2 \cdots s_N$（$s_i \in \{A, T, C, G\}$）转化为数值信号的常用方法：

**方法一：二进制指示序列**

为每种碱基创建一个二进制序列：

$$u_A(n) = \begin{cases} 1, & s_n = A \\ 0, & \text{otherwise} \end{cases}, \quad n = 1, 2, \ldots, N$$

类似定义 $u_T(n)$、$u_C(n)$、$u_G(n)$。

**方法二：实数编码（Voss 映射）**

将四种碱基映射到二维实数空间：

| 碱基 | $x$ 值 | $y$ 值 |
|------|--------|--------|
| A    | -1     | -1     |
| T    | -1     | +1     |
| C    | +1     | -1     |
| G    | +1     | +1     |

**方法三：电子电荷编码**

根据碱基的电子结构：

| 碱基 | 编码值 |
|------|--------|
| A    | 0.1260 |
| C    | 0.1340 |
| G    | 0.0806 |
| T    | 0.1335 |

### 三碱基周期性的频域分析

对于长度为 $N$ 的数值序列 $x(n)$，其离散傅里叶变换为：

$$X(k) = \sum_{n=0}^{N-1} x(n) \cdot e^{-j2\pi kn/N}, \quad k = 0, 1, \ldots, N-1$$

功率谱为：

$$P(k) = |X(k)|^2$$

三碱基周期性在频域中的表现为：在频率 $k = N/3$ 处（即归一化频率 $f = 1/3$）出现显著的功率峰值。

**周期性测度指标**：

$$S_{3\text{-period}} = \frac{P(N/3)}{\frac{1}{N/3-1}\sum_{k=1}^{N/3-1} P(k)}$$

该比值越大，表明三碱基周期性越显著，该序列越可能是蛋白质编码区。

### 基因预测中的应用

- 外显子（编码区）：$S_{3\text{-period}} \gg 1$，存在强三碱基周期性
- 内含子（非编码区）：$S_{3\text{-period}} \approx 1$，无明显周期性
- 基因间区：$S_{3\text{-period}} \approx 1$

## 数据处理方法

### 处理流程

```
FASTA 序列 → 数值编码(Voss映射) → 窗口分段 → DFT →
功率谱分析 → 三碱基周期性测度 → 编码区识别 → 可视化
```

### Step 1：序列读取与编码

- 使用 Biopython 读取 FASTA 格式序列
- 将碱基序列转换为数值信号（二进制指示序列或 Voss 映射）

### Step 2：窗口化 DFT 分析

- 选择滑动窗口（如 300 bp，相当于一个典型外显子长度）
- 对每个窗口做 DFT
- 计算 $f = 1/3$ 处的功率谱值

### Step 3：周期性测度计算

- 计算 $S_{3\text{-period}}$ 指标
- 沿基因组滑动窗口，生成周期性信号分布图

### Step 4：编码区预测

- 设定阈值，识别高周期性区域
- 与已知基因注释对比验证

# 1. DNA 序列数值编码

def voss_mapping(sequence):
    """
    Voss 映射：将 DNA 序列转化为 4 个二进制指示序列
    返回：(4, N) 的 numpy 数组
    """
    seq = sequence.upper()
    n = len(seq)
    mapping = {'A': 0, 'T': 1, 'C': 2, 'G': 3}
    result = np.zeros((4, n))

    for i, base in enumerate(seq):
        if base in mapping:
            result[mapping[base], i] = 1.0

    return result

def real_number_encoding(sequence):
    """
    实数编码：将碱基映射到二维实数空间
    返回：(2, N) 的 numpy 数组
    """
    seq = sequence.upper()
    n = len(seq)
    mapping = {
        'A': (-1, -1), 'T': (-1, 1),
        'C': (1, -1),  'G': (1, 1)
    }
    x = np.zeros(n)
    y = np.zeros(n)

    for i, base in enumerate(seq):
        if base in mapping:
            x[i], y[i] = mapping[base]

    return np.vstack([x, y])

def electronic_charge_encoding(sequence):
    """
    电子电荷编码
    """
    seq = sequence.upper()
    charges = {'A': 0.1260, 'C': 0.1340, 'G': 0.0806, 'T': 0.1335}
    return np.array([charges.get(b, 0) for b in seq])

# ============================================================
# 2. 功率谱分析
# ============================================================
def compute_power_spectrum(signal, fs=1.0):
    """计算信号的功率谱"""
    N = len(signal)
    X = fft(signal)
    freqs = fftfreq(N, d=1/fs)
    power = np.abs(X)**2
    return freqs[:N//2], power[:N//2]

def periodicity_measure(power, period=3):
    """
    计算周期性测度 S
    S = P(N/3) / mean(P(1..N/3-1))
    """
    N = len(power) * 2  # 因为 power 只取了前半部分
    k_period = N // period  # N/3 对应的频率索引

    if k_period >= len(power) or k_period < 1:
        return 0

    # 避免使用 k=0（直流分量）
    background = np.mean(power[1:k_period]) if k_period > 1 else 1
    if background == 0:
        background = 1e-10

    return power[k_period] / background

# ============================================================
# 3. 滑动窗口分析
# ============================================================
def sliding_window_analysis(sequence, window_size=300, step=30,
                            encoding='voss'):
    """
    沿序列滑动窗口计算三碱基周期性强度
    """
    n = len(sequence)
    positions = []
    periodicity_values = []

    for start in range(0, n - window_size + 1, step):
        window_seq = sequence[start:start + window_size]

        if encoding == 'voss':
            signals = voss_mapping(window_seq)
            # 计算四个碱基通道的平均功率谱
            avg_power = np.zeros(window_size // 2)
            for ch in range(4):
                freqs, power = compute_power_spectrum(signals[ch])
                avg_power += power
            avg_power /= 4
        elif encoding == 'charge':
            signal = electronic_charge_encoding(window_seq)
            signal = signal - np.mean(signal)  # 去直流
            freqs, power = compute_power_spectrum(signal)
            avg_power = power
        else:
            raise ValueError(f"未知编码方式: {encoding}")

        s3 = periodicity_measure(avg_power, period=3)
        positions.append(start + window_size // 2)
        periodicity_values.append(s3)

    return np.array(positions), np.array(periodicity_values)

def generate_test_sequence(n_exons=3, exon_len=300, intron_len=500):
    """
    生成包含外显子和内含子的测试序列
    外显子：有密码子使用偏好性（三碱基周期性）
    内含子：接近随机序列
    """
    np.random.seed(42)

    # 密码子使用偏好表（模拟大肠杆菌风格）
    codons = []
    amino_acids = 'FFLLSSSSYY**CC*WLLLLPPPPHHQQRRRRIIIMTTTTNNKKSSRRVVVVAAAADDEEGGGG'
    bases = 'TCAG'

    for i in range(0, 64, 1):
        codon = ''
        idx = i
        codon += bases[idx // 16]
        idx = idx % 16
        codon += bases[idx // 4]
        idx = idx % 4
        codon += bases[idx]
        codons.append(codon)

    # 创建偏好密码子列表（高频使用）
    preferred_codons = ['ATG', 'GCT', 'GAA', 'TTC', 'AAA', 'GGT', 'GTG',
                        'ATC', 'CCG', 'ACG', 'TCT', 'CAG', 'CGT', 'GAT',
                        'TTG', 'CTG', 'AAG', 'ACC', 'GAC', 'GCG', 'TGG']

    sequence = ''

    for i in range(n_exons):
        # 内含子（非编码区，接近随机）
        intron = ''.join(np.random.choice(['A', 'T', 'C', 'G'],
                                           size=intron_len))
        sequence += intron

        # 外显子（编码区，有密码子使用偏好性）
        exon = ''
        for _ in range(exon_len // 3):
            if np.random.random() < 0.6:
                # 60% 概率使用偏好密码子
                exon += np.random.choice(preferred_codons)
            else:
                # 40% 概率使用其他密码子
                codon = ''.join(np.random.choice(['A', 'T', 'C', 'G'], size=3))
                exon += codon
        sequence += exon

    # 最后一段内含子
    intron = ''.join(np.random.choice(['A', 'T', 'C', 'G'],
                                       size=intron_len))
    sequence += intron

    # 生成标注信息
    annotations = []
    pos = 0
    for i in range(n_exons):
        pos += intron_len
        annotations.append(('exon', pos, pos + exon_len))
        pos += exon_len
    annotations.append(('intron', pos, pos + intron_len))

    return sequence, annotations

## 📋 学习指南

### 前置知识要求
- ✅ 必备：高等数学（傅里叶变换概念）、Python编程基础
- ✅ 必备：线性代数（向量运算）
- ⚡ 建议：了解基本的生物学概念（DNA、基因、密码子）

### 学习目标
- 理解DNA序列的数值编码方法
- 掌握用FFT检测序列周期性的原理
- 能区分编码区和非编码区的频谱特征差异

### 难度分层

| 层级 | 目标 | 对应任务 |
|------|------|----------|
| ⭐ 基础 | 解析FASTA文件，绘制DNA数值信号 | Step 1 |
| ⭐⭐ 进阶 | 计算FFT频谱，检测3碱基周期性 | Step 2-3 |
| ⭐⭐⭐ 挑战 | 构建完整的基因预测pipeline | Step 4 |

### 🔥 挑战与拔高

1. **多尺度分析**：用小波变换分析不同长度尺度的周期性
2. **蛋白质编码预测**：结合密码子使用偏好性
3. **比较基因组学**：对比不同物种的编码区频谱特征
4. **深度学习方法**：用1D-CNN直接从序列预测编码区

> 💡 这个项目展示了FFT在非传统信号上的应用——DNA序列不是时间信号，但频谱分析同样有效。

## 扩展方向

1. **Gabor 变换**：使用 Gabor 小波进行时频联合分析，同时获得位置和频率信息
2. **小波分析**：利用连续小波变换检测不同尺度上的周期性
3. **机器学习基因预测**：将谱特征与序列特征结合，构建编码区预测分类器
4. **多基因组比较**：比较不同物种基因组的三碱基周期性差异
5. **GC 含量分析**：结合碱基组成分析基因组岛（genomic islands）
6. **RNA 二级结构预测**：利用自相关分析 RNA 序列的配对模式
7. **宏基因组分析**：从环境 DNA 样本中识别编码区

**推荐阅读**：
- Tiwari et al., "Prediction of probable genes by Fourier analysis of genomic sequences," CABIOS, 1997.
- Anastassiou, "Frequency-domain analysis of bimolecular sequences," Bioinformatics, 2000.
- Voss, "Evolution of long-range fractal correlations and 1/f noise in DNA base sequences," Physical Review Letters, 1992.
