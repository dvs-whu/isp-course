# B3 显微细胞图像分割

## 项目背景

细胞图像分割是生物医学图像分析中最基础也是最具挑战性的任务之一。在病理学诊断、药物筛选、细胞生物学研究中，准确地从显微镜图像中识别和分割出每个细胞，是后续定量分析（如细胞计数、形态测量、荧光强度分析）的前提。传统的人工标注方法耗时且主观性强，自动化细胞分割技术的发展对提高分析效率和可重复性具有重要意义。

频域分析在细胞图像处理中扮演着重要角色。由于细胞在显微镜图像中通常呈现为近似圆形或椭圆形的斑块状结构，其空间频率特征具有明显的规律性。利用二维傅里叶变换（2D FFT）可以在频域中识别和去除周期性噪声（如 CCD 摄像头的固定模式噪声）、增强细胞边缘特征、实现图像去噪和平滑。

本项目利用 Broad Bioimage Benchmark Collection（BBBC）提供的公开细胞图像数据集，结合二维频域滤波去噪和 U-Net 语义分割网络，实现细胞的自动分割。学生将深入理解二维傅里叶变换在图像处理中的应用，掌握图像去噪、边缘检测和语义分割的完整技术流程。

## 信号处理建模

### 二维离散傅里叶变换（2D DFT）

对于大小为 $M \times N$ 的数字图像 $f(x, y)$，其二维离散傅里叶变换为：

$$F(u, v) = \sum_{x=0}^{M-1}\sum_{y=0}^{N-1} f(x,y) \cdot e^{-j2\pi\left(\frac{ux}{M} + \frac{vy}{N}\right)}$$

逆变换为：

$$f(x, y) = \frac{1}{MN}\sum_{u=0}^{M-1}\sum_{v=0}^{N-1} F(u,v) \cdot e^{j2\pi\left(\frac{ux}{M} + \frac{vy}{N}\right)}$$

频谱的幅度和相位分别为：

$$|F(u,v)| = \sqrt{Re^2(u,v) + Im^2(u,v)}$$
$$\phi(u,v) = \arctan\frac{Im(u,v)}{Re(u,v)}$$

### 频域滤波去噪

频域滤波的基本思想是设计滤波器函数 $H(u,v)$，对频谱进行修改：

$$G(u,v) = F(u,v) \cdot H(u,v)$$

常用的滤波器类型：

- **理想低通滤波器（ILPF）**：$H(u,v) = \begin{cases} 1, & D(u,v) \leq D_0 \\ 0, & D(u,v) > D_0 \end{cases}$
- **巴特沃斯低通滤波器（BLPF）**：$H(u,v) = \frac{1}{1 + [D(u,v)/D_0]^{2n}}$
- **高斯低通滤波器（GLPF）**：$H(u,v) = e^{-D^2(u,v)/(2D_0^2)}$

其中 $D(u,v) = \sqrt{u^2 + v^2}$ 为频率点到原点的距离，$D_0$ 为截止频率。

### U-Net 语义分割网络

U-Net 是一种经典的编码器-解码器结构，专为生物医学图像分割设计：

1. **编码器（下采样路径）**：逐步提取高层语义特征
   - 每层：Conv 3×3 → ReLU → Conv 3×3 → ReLU → MaxPool 2×2
2. **解码器（上采样路径）**：逐步恢复空间分辨率
   - 每层：Up-Conv 2×2 → 拼接编码器特征 → Conv 3×3 → ReLU → Conv 3×3 → ReLU
3. **跳跃连接（Skip Connection）**：将编码器对应层的特征直接传递给解码器

损失函数通常采用 Dice Loss 或 BCE + Dice Loss 的组合：

$$\mathcal{L}_{Dice} = 1 - \frac{2\sum_i p_i g_i + \epsilon}{\sum_i p_i + \sum_i g_i + \epsilon}$$

## 数据处理方法

### 处理流程

```
原始图像 → 灰度化 → 2D FFT 分析 → 频域去噪 → 空域增强 →
U-Net 训练 → 语义分割 → 后处理(形态学操作) → 细胞计数
```

### Step 1：2D FFT 频域分析

- 对原始细胞图像做 2D FFT
- 可视化频谱，识别噪声频率分量
- 设计低通/带阻滤波器去除高频噪声或周期性干扰

### Step 2：频域滤波去噪

- 巴特沃斯低通滤波器去除高频噪声
- 中值滤波器去除椒盐噪声
- 可选：同态滤波增强图像对比度

### Step 3：空间域增强

- CLAHE（自适应直方图均衡化）增强局部对比度
- 边缘增强（拉普拉斯算子、Sobel 算子）

### Step 4：U-Net 训练与推理

- 数据增强：旋转、翻转、弹性变形
- 训练 U-Net 模型
- 预测分割 mask

### Step 5：后处理

- 形态学操作（开运算、闭运算）平滑边界
- 连通域分析实现细胞实例分割
- 面积过滤去除过小区域

import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft2, ifft2, fftshift, ifftshift
from scipy.ndimage import median_filter, gaussian_filter
from scipy.ndimage import binary_opening, binary_closing, label
from skimage import exposure, filters, morphology, measure
from sklearn.model_selection import train_test_split

# 1. 模拟生成细胞图像
# ============================================================
def generate_synthetic_cells(image_size=256, n_cells=15, noise_level=0.1):
    """
    生成合成的显微镜细胞图像（含标注 mask）
    """
    np.random.seed(42)
    image = np.zeros((image_size, image_size))
    mask = np.zeros((image_size, image_size), dtype=np.uint8)

    for _ in range(n_cells):
        # 随机位置和大小
        cx = np.random.randint(30, image_size - 30)
        cy = np.random.randint(30, image_size - 30)
        radius = np.random.randint(12, 25)

        # 创建圆形细胞
        y, x = np.ogrid[:image_size, :image_size]
        dist = np.sqrt((x - cx)**2 + (y - cy)**2)
        cell = dist <= radius

        # 细胞内部亮度（高斯分布，中心亮）
        intensity = np.exp(-dist**2 / (2 * (radius * 0.6)**2)) * 0.8
        image[cell] += intensity[cell]
        mask[cell] = 1

    # 细胞核（更亮的中心区域）
    for _ in range(n_cells // 2):
        cx = np.random.randint(30, image_size - 30)
        cy = np.random.randint(30, image_size - 30)
        radius = np.random.randint(5, 10)
        y, x = np.ogrid[:image_size, image_size]
        dist = np.sqrt((x - cx)**2 + (y - cy)**2)
        nucleus = dist <= radius
        image[nucleus] += 0.3

    # 添加背景梯度
    y_grid, x_grid = np.mgrid[:image_size, :image_size]
    background = 0.15 * np.sin(2 * np.pi * x_grid / image_size)
    image += background

    # 添加不同类型的噪声
    # 高斯白噪声
    image += noise_level * np.random.randn(image_size, image_size)

    # 周期性条纹噪声（模拟 CCD 固定模式噪声）
    stripe_freq = 30
    stripe = 0.05 * np.sin(2 * np.pi * np.arange(image_size) / stripe_freq)
    image += stripe[np.newaxis, :]

    # 归一化到 [0, 1]
    image = np.clip(image, 0, 1)

    return image, mask

# ============================================================
# 2. 2D FFT 频域分析
# ============================================================
def fft2d_analysis(image):
    """二维 FFT 分析与可视化"""
    # 2D FFT
    F = fft2(image)
    F_shifted = fftshift(F)  # 零频率移到中心

    # 幅度谱（对数尺度）
    magnitude = np.log1p(np.abs(F_shifted))

    # 相位谱
    phase = np.angle(F_shifted)

    return F_shifted, magnitude, phase

# ============================================================
# 3. 频域滤波器
# ============================================================
def ideal_lowpass_filter(shape, D0):
    """理想低通滤波器"""
    M, N = shape
    u = np.arange(M) - M // 2
    v = np.arange(N) - N // 2
    U, V = np.meshgrid(v, u)
    D = np.sqrt(U**2 + V**2)
    H = np.double(D <= D0)
    return H

def butterworth_lowpass_filter(shape, D0, n=2):
    """巴特沃斯低通滤波器"""
    M, N = shape
    u = np.arange(M) - M // 2
    v = np.arange(N) - N // 2
    U, V = np.meshgrid(v, u)
    D = np.sqrt(U**2 + V**2)
    H = 1 / (1 + (D / D0)**(2 * n))
    return H

def gaussian_lowpass_filter(shape, D0):
    """高斯低通滤波器"""
    M, N = shape
    u = np.arange(M) - M // 2
    v = np.arange(N) - N // 2
    U, V = np.meshgrid(v, u)
    D = np.sqrt(U**2 + V**2)
    H = np.exp(-D**2 / (2 * D0**2))
    return H

def bandreject_filter(shape, D0, W):
    """带阻滤波器（去除特定频率的周期性噪声）"""
    M, N = shape
    u = np.arange(M) - M // 2
    v = np.arange(N) - N // 2
    U, V = np.meshgrid(v, u)
    D = np.sqrt(U**2 + V**2)
    H = np.ones_like(D, dtype=float)
    mask = (D >= D0 - W/2) & (D <= D0 + W/2)
    H[mask] = 0
    return H

def frequency_domain_filter(image, H):
    """频域滤波"""
    F = fftshift(fft2(image))
    G = F * H
    g = np.real(ifft2(ifftshift(G)))
    return g

# ============================================================
# 4. 空间域增强
# ============================================================
def enhance_image(image):
    """空间域图像增强"""
    # CLAHE 自适应直方图均衡化
    image_uint8 = (image * 255).astype(np.uint8)
    clahe = exposure.equalize_adapthist(image_uint8 / 255.0, clip_limit=0.03)

    # 中值滤波去椒盐噪声
    denoised = median_filter(clahe, size=3)

    return denoised

# ============================================================
# 5. 简化 U-Net 模型
# ============================================================
def build_simple_unet(input_size=256):
    """构建简化版 U-Net（用于演示）"""
    try:
        import torch
        import torch.nn as nn

        class DoubleConv(nn.Module):
            def __init__(self, in_ch, out_ch):
                super(DoubleConv, self).__init__()
                self.conv = nn.Sequential(
                    nn.Conv2d(in_ch, out_ch, 3, padding=1),
                    nn.BatchNorm2d(out_ch),
                    nn.ReLU(inplace=True),
                    nn.Conv2d(out_ch, out_ch, 3, padding=1),
                    nn.BatchNorm2d(out_ch),
                    nn.ReLU(inplace=True)
                )

            def forward(self, x):
                return self.conv(x)

        class SimpleUNet(nn.Module):
            def __init__(self, in_channels=1, out_channels=1):
                super(SimpleUNet, self).__init__()
                # 编码器
                self.enc1 = DoubleConv(in_channels, 64)
                self.enc2 = DoubleConv(64, 128)
                self.enc3 = DoubleConv(128, 256)
                self.pool = nn.MaxPool2d(2)

                # 瓶颈层
                self.bottleneck = DoubleConv(256, 512)

                # 解码器
                self.up3 = nn.ConvTranspose2d(512, 256, 2, stride=2)
                self.dec3 = DoubleConv(512, 256)
                self.up2 = nn.ConvTranspose2d(256, 128, 2, stride=2)
                self.dec2 = DoubleConv(256, 128)
                self.up1 = nn.ConvTranspose2d(128, 64, 2, stride=2)
                self.dec1 = DoubleConv(128, 64)

                # 输出层
                self.out_conv = nn.Conv2d(64, out_channels, 1)

            def forward(self, x):
                # 编码
                e1 = self.enc1(x)
                e2 = self.enc2(self.pool(e1))
                e3 = self.enc3(self.pool(e2))

                # 瓶颈
                b = self.bottleneck(self.pool(e3))

                # 解码 + 跳跃连接
                d3 = self.dec3(torch.cat([self.up3(b), e3], dim=1))
                d2 = self.dec2(torch.cat([self.up2(d3), e2], dim=1))
                d1 = self.dec1(torch.cat([self.up1(d2), e1], dim=1))

                return torch.sigmoid(self.out_conv(d1))

        model = SimpleUNet()
        total_params = sum(p.numel() for p in model.parameters())
        print(f"U-Net 参数量: {total_params:,}")
        return model

    except ImportError:
        print("PyTorch 未安装，跳过 U-Net 构建")
        return None

# 6. 形态学后处理
# ============================================================
def postprocess_mask(pred_mask, min_area=50):
    """分割结果后处理"""
    # 二值化
    binary = pred_mask > 0.5

    # 形态学操作
    selem = morphology.disk(3)
    binary = binary_opening(binary, selem)
    binary = binary_closing(binary, selem)

    # 连通域分析
    labeled, num_cells = label(binary)

    # 面积过滤
    for region in measure.regionprops(labeled):
        if region.area < min_area:
            labeled[labeled == region.label] = 0

    # 重新标记
    labeled, num_cells = label(labeled > 0)

    return labeled, num_cells

# ============================================================
# 7. Dice 系数评估
# ============================================================
def dice_coefficient(pred, target):
    """计算 Dice 系数"""
    intersection = np.sum(pred * target)
    return 2 * intersection / (np.sum(pred) + np.sum(target) + 1e-8)

# 8. 主流程

def main():
    print("=" * 60)
    print("B3 显微细胞图像分割")
    print("=" * 60)

    # 1. 生成模拟数据
    print("\n[1/7] 生成模拟细胞图像...")
    image, mask = generate_synthetic_cells(image_size=256, n_cells=15,
                                            noise_level=0.1)
    print(f"    图像尺寸: {image.shape}")
    print(f"    细胞像素占比: {np.mean(mask)*100:.1f}%")

    # 2. 2D FFT 分析
    print("\n[2/7] 二维 FFT 频域分析...")
    F_shifted, magnitude, phase = fft2d_analysis(image)

    # 3. 频域去噪
    print("\n[3/7] 频域滤波去噪...")

    # 高斯低通滤波
    D0 = 80
    H_gauss = gaussian_lowpass_filter(image.shape, D0)
    image_glpf = frequency_domain_filter(image, H_gauss)

    # 带阻滤波去除周期性条纹噪声
    H_br = bandreject_filter(image.shape, D0=80, W=10)
    image_br = frequency_domain_filter(image, H_br)

    # 组合：带阻 + 高斯低通
    H_combined = H_gauss * H_br
    image_filtered = frequency_domain_filter(image, H_combined)

    print(f"    高斯低通截止频率: D0={D0}")
    print(f"    带阻滤波中心频率: D0=80, 带宽 W=10")

    # 4. 空间域增强
    print("\n[4/7] 空间域增强...")
    image_enhanced = enhance_image(image_filtered)

    # 5. 简单阈值分割（替代 U-Net 演示）
    print("\n[5/7] 阈值分割（演示）...")
    thresh = filters.threshold_otsu(image_enhanced)
    pred_mask_simple = (image_enhanced > thresh).astype(float)

    # 形态学后处理
    labeled, num_cells = postprocess_mask(pred_mask_simple, min_area=50)
    print(f"    OTSU 阈值: {thresh:.4f}")
    print(f"    检测到细胞区域数: {num_cells}")

    # 评估
    dice = dice_coefficient(pred_mask_simple, mask)
    print(f"    Dice 系数: {dice:.4f}")

    # 6. 构建 U-Net 模型
    print("\n[6/7] 构建 U-Net 模型...")
    model = build_simple_unet(input_size=256)

    # 7. 可视化
    print("\n[7/7] 可视化...")
    fig, axes = plt.subplots(3, 3, figsize=(16, 15))

    # 第一行：原始图像、FFT 幅度谱、相位谱
    axes[0, 0].imshow(image, cmap='gray')
    axes[0, 0].set_title('原始含噪图像', fontproperties=zh_font)
    axes[0, 0].axis('off')

    axes[0, 1].imshow(magnitude, cmap='hot')
    axes[0, 1].set_title('FFT 幅度谱（对数）', fontproperties=zh_font)
    axes[0, 1].axis('off')

    axes[0, 2].imshow(phase, cmap='gray')
    axes[0, 2].set_title('FFT 相位谱', fontproperties=zh_font)
    axes[0, 2].axis('off')

    # 第二行：滤波器、去噪结果、增强结果
    axes[1, 0].imshow(H_gauss, cmap='gray')
    axes[1, 0].set_title('高斯低通滤波器', fontproperties=zh_font)
    axes[1, 0].axis('off')

    axes[1, 1].imshow(image_filtered, cmap='gray')
    axes[1, 1].set_title('频域滤波去噪结果', fontproperties=zh_font)
    axes[1, 1].axis('off')

    axes[1, 2].imshow(image_enhanced, cmap='gray')
    axes[1, 2].set_title('CLAHE 增强结果', fontproperties=zh_font)
    axes[1, 2].axis('off')

    # 第三行：分割结果对比
    axes[2, 0].imshow(mask, cmap='gray')
    axes[2, 0].set_title('真实标注 (Ground Truth)', fontproperties=zh_font)
    axes[2, 0].axis('off')

    axes[2, 1].imshow(pred_mask_simple, cmap='gray')
    axes[2, 1].set_title(f'分割结果 (Dice={dice:.3f})', fontproperties=zh_font)
    axes[2, 1].axis('off')

    # 标注细胞区域
    axes[2, 2].imshow(image_enhanced, cmap='gray')
    axes[2, 2].contour(mask, colors='green', linewidths=1.5, label='真实')
    axes[2, 2].contour(pred_mask_simple, colors='red', linewidths=1,

    axes[2, 2].set_title('分割结果叠加对比', fontproperties=zh_font)

    plt.tight_layout()

    fig2, axes2 = plt.subplots(1, 3, figsize=(15, 5))

    H_blpf = butterworth_lowpass_filter(image.shape, D0=80, n=2)
    H_glpf = gaussian_lowpass_filter(image.shape, D0=80)

    axes2[0].imshow(H_ilpf, cmap='gray')

- [[CLAHE]] — 自适应直方图均衡化
- [[中值滤波]] — 非线性滤波，去椒盐噪声
- [[U-Net]] — 编码器-解码器语义分割网络
- [[跳跃连接]] — 保留空间细节信息
- [[Dice Loss]] — 分割任务常用损失函数
- [[形态学操作]] — 开运算、闭运算，后处理
- [[连通域分析]] — 实例分割与细胞计数
- [[OTSU 阈值]] — 自动阈值选择
- [[图像去噪]] — 空域和频域去噪方法

## 扩展方向

1. **实例分割**：使用 Mask R-CNN 或 Cellpose 实现单细胞级别分割
2. **3D 分割**：扩展到显微镜 z-stack 三维图像分割
3. **弱监督学习**：仅使用点标注或涂鸦标注训练分割模型

5. **细胞追踪**：结合时序信息实现细胞运动轨迹追踪
6. **多任务学习**：同时进行分割、分类和形态学参数测量
