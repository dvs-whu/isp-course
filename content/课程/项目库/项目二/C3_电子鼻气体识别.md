# C3 电子鼻气体识别

## 一、项目背景

电子鼻（Electronic Nose）是一种模拟生物嗅觉系统的气体检测装置，由气敏传感器阵列和模式识别算法组成。每个传感器对不同气体具有不同的响应特性，通过分析传感器阵列的综合响应模式，可以实现对复杂气体的定性识别和定量分析。

电子鼻技术在食品安全检测、环境监测、疾病诊断（呼气分析）、工业过程控制等领域有广泛应用。本项目以金属氧化物半导体（MOS）传感器阵列为对象，利用信号处理和机器学习方法实现对6种气体的自动识别。

**核心挑战**：传感器存在基线漂移（drift），长期使用后灵敏度变化，需要鲁棒的特征提取和分类策略。

## 二、信号处理建模


### 传感器响应模型

单个传感器的响应可以建模为：

$$
y(t) = S \cdot c(t) + n(t)
$$

其中 $S$ 是传感器灵敏度，$c(t)$ 是气体浓度，$n(t)$ 是噪声。

对于 $M$ 个传感器的阵列，时刻 $t$ 的观测为向量：

$$
\mathbf{y}(t) = [y_1(t), y_2(t), \ldots, y_M(t)]^T
$$

### 特征提取

从传感器响应曲线中提取4类特征：

- **统计特征**：均值 $\bar{y} = \frac{1}{N}\sum_{n=1}^N y(n)$，方差，最大值
- **动态特征**：响应斜率 $\frac{dy}{dt}$，上升时间，恢复时间
- **积分特征**：响应曲线下面积 $\int_0^T y(t) dt$
- **频域特征**：FFT频谱的能量分布


### 处理流程

```
传感器原始信号 (16通道时间序列)
    ↓
特征提取 (均值、斜率、曲线下面积、响应峰值)
    ↓
降维 (PCA / t-SNE)
    ↓
分类识别 (KNN / SVM / Random Forest)
    ↓
气体类别输出 (6种气体)
```

### 特征提取策略

对每个传感器通道的响应曲线提取以下特征：
- **稳态特征**：响应阶段均值、最大值
- **动态特征**：响应斜率（上升/下降）、响应时间
- **积分特征**：响应曲线下面积（AUC）
- **差分特征**：最大响应值与基线之差

共 16 传感器 × 8 特征 = 128 维特征向量

### 4.2 特征提取

```python
def extract_features(sensor_response, n_sensors=16, n_timepoints=120):
    """
    从传感器响应中提取特征
    sensor_response: shape (n_timepoints,) 单次测量的传感器响应
    """
    # 重塑为 (n_sensors, n_timepoints)
    response = sensor_response.reshape(n_sensors, n_timepoints)

    features = []
    for i in range(n_sensors):
        signal = response[i, :]
        # 稳态特征
        features.append(np.mean(signal))
        features.append(np.max(signal))
        # 动态特征
        features.append(np.gradient(signal).mean())  # 平均斜率
        features.append(np.argmax(signal))  # 峰值时间
        # 积分特征
        features.append(np.trapz(signal))  # 曲线下面积
        # 差分特征
        features.append(signal[-1] - signal[0])  # 终值-初值

    return np.array(features)
```

### 4.4 分类识别

```python
# 选择合适的主成分数
n_components = np.argmax(np.cumsum(pca_full.explained_variance_ratio_) > 0.95) + 1
pca_final = PCA(n_components=n_components)
X_final = pca_final.fit_transform(X_scaled)

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X_final, y_labels, test_size=0.3, random_state=42, stratify=y_labels
)

# KNN 分类
knn = KNeighborsClassifier(n_neighbors=5, metric='euclidean')
knn.fit(X_train, y_train)
y_pred_knn = knn.predict(X_test)

# SVM 分类
svm = SVC(kernel='rbf', C=10, gamma='scale')
svm.fit(X_train, y_train)
y_pred_svm = svm.predict(X_test)

# 评估
gas_names = ['乙醇', '乙烯', '氨', '丙醛', '一氧化碳', '甲苯']
print("=== KNN 分类报告 ===")
print(classification_report(y_test, y_pred_knn, target_names=gas_names))
print("=== SVM 分类报告 ===")
print(classification_report(y_test, y_pred_svm, target_names=gas_names))

# 混淆矩阵可视化
fig, axes = plt.subplots(1, 2, figsize=(14, 6))
for ax, pred, title in zip(axes, [y_pred_knn, y_pred_svm], ['KNN', 'SVM']):
    cm = confusion_matrix(y_test, pred)
    sns.heatmap(cm, annot=True, fmt='d', cmap='Blues',
                xticklabels=gas_names, yticklabels=gas_names, ax=ax)
    ax.set_xlabel('预测')
    ax.set_ylabel('真实')
    ax.set_title(f'{title} 混淆矩阵')

plt.tight_layout()
plt.savefig('confusion_matrix_gas.png', dpi=150, bbox_inches='tight')
```

### 4.5 传感器漂移分析

```python
# 分析不同批次的漂移现象
def analyze_drift(all_batches):
    """比较不同时间段传感器响应的变化"""
    fig, ax = plt.subplots(figsize=(10, 6))

    for batch_id, batch_data in all_batches.items():
        # 计算同一气体在不同批次的传感器均值
        mean_response = batch_data.mean(axis=0)
        ax.plot(mean_response, label=f'批次 {batch_id}')

    ax.set_xlabel('传感器通道')
    ax.set_ylabel('平均响应值')
    ax.set_title('传感器漂移分析')
    ax.legend()
    plt.savefig('sensor_drift.png', dpi=150, bbox_inches='tight')
    plt.show()
```
