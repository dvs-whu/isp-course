# 项目说明书：引力波信号探测与处理

## 一、项目背景

2015年9月14日，LIGO（激光干涉引力波天文台）首次直接探测到引力波信号GW150914，标志着人类开启了引力波天文学的新纪元。这一信号源自两个黑洞（分别约36和29个太阳质量）的并合，距离地球约13亿光年。

引力波信号极其微弱，其应变幅度约 $h \sim 10^{-21}$，淹没在远大于信号本身的各种噪声中（地震噪声、热噪声、散粒噪声等）。从如此嘈杂的背景中提取出有意义的物理信息，是信号处理领域的经典难题。

本项目引导学生运用经典与现代信号处理技术，亲历从原始数据到物理发现的完整流程，体会信号处理在前沿科学发现中的核心作用。

### 跨学科知识点

| 领域 | 关键概念 |
|------|---------|
| 广义相对论 | 引力波、双星并合、啁啾信号(chirp) |
| 激光干涉测量 | Michelson干涉仪、臂长变化、应变 |
| 数字信号处理 | 功率谱密度、带通滤波、匹配滤波 |
| 时频分析 | 短时傅里叶变换、Q变换、小波分析 |

---

## 二、项目目标

1. 理解引力波探测的基本物理原理与数据特征
2. 掌握噪声分析方法，计算并绘制功率谱密度(PSD)
3. 设计并实现数字滤波器，从原始数据中提取候选信号
4. 实现匹配滤波器，利用模板波形进行信号检测
5. 使用Q变换生成时频图，直观展示引力波信号特征

## 三、数据集

### 主数据源：LIGO Open Science Center (GWOSC)

- **网址**：https://www.gwosc.org/
- **事件**：GW150914
- **数据格式**：HDF5 (.hdf5)
- **采样率**：4096 Hz (strain 数据) 或 16384 Hz (完整数据)
- **探测器**：H1 (Hanford) 和 L1 (Livingston)

### 数据获取方式

```python
# 方式一：使用 gwpy 库直接下载
from gwpy.timeseries import TimeSeries
data_h1 = TimeSeries.fetch_open_data('H1', 1126259446, 1126259478)
data_l1 = TimeSeries.fetch_open_data('L1', 1126259446, 1126259478)

# 方式二：从 GWOSC 网站手动下载
# URL: https://www.gwosc.org/eventapi/html/GW150914/
# 选择 4KHz strain 数据（HDF5 格式）
```

### 辅助数据

- **模板波形**：可使用 `pycbc` 库生成双黑洞并合的理论波形模板
- **注释文件**：GWOSC 提供的事件参数（质量、距离、信噪比等）

## 五、项目阶段与任务

### 阶段一：噪声分析与功率谱估计（第1-2周）

**目标**：理解LIGO噪声特性，掌握功率谱密度估计方法

**任务清单**：

1. **数据加载与探索**
   - 读取 GW150914 的 H1 和 L1 strain 数据（HDF5格式）
   - 绘制原始时域波形，观察数据特征
   - 记录关键元数据：采样率、时间戳、数据长度

2. **功率谱密度(PSD)估计**
   - 使用 Welch 方法（`scipy.signal.welch`）计算 PSD
   - 尝试不同的窗函数（Hann、Blackman等）和分段长度，比较结果
   - 绘制双对数 PSD 图，识别各频段噪声特征

3. **噪声源识别**
   - 标注主要噪声源频段：
     - 低频 (< 10 Hz)：地震噪声
     - 中频 (10-200 Hz)：热噪声 / 悬丝噪声
     - 高频 (> 200 Hz)：散粒噪声（光子计数噪声）
   - 对比 H1 和 L1 的 PSD，讨论差异

4. **自相关分析**
   - 计算噪声的自相关函数
   - 分析噪声的非平稳性特征

**交付物**：
- 噪声分析 Jupyter Notebook
- PSD 图（含噪声源标注）
- 书面分析（500字）

---

### 阶段二：带通滤波器设计与实现（第3-4周）

**目标**：设计数字滤波器，从原始数据中提取感兴趣频段的信号

**任务清单**：

1. **滤波器需求分析**
   - 根据 PSD 分析结果和 GW150914 信号特征，确定通带范围
   - 参考值：35-350 Hz（基于信号频率演化范围）

2. **IIR 滤波器设计**
   - 使用 `scipy.signal.butter` 设计 Butterworth 带通滤波器
   - 计算并绘制滤波器频率响应（幅频、相频特性）
   - 分析滤波器阶数对性能的影响

3. **FIR 滤波器设计**
   - 使用 `scipy.signal.firwin` 或 `scipy.signal.firls` 设计 FIR 滤波器
   - 与 IIR 滤波器进行对比（相位线性度、群延迟、计算复杂度）

4. **滤波效果评估**
   - 分别对 H1 和 L1 数据应用滤波器
   - 绘制滤波前后时域波形对比
   - 计算滤波后的 PSD，验证噪声抑制效果
   - 观察滤波后是否可见 ~0.2 秒的信号瞬态

**交付物**：
- 滤波器设计与实现 Notebook
- 频率响应图、滤波前后对比图
- IIR vs FIR 对比分析

---

### 阶段三：匹配滤波与信号检测（第5-7周）

**目标**：实现匹配滤波器，利用理论模板进行信号检测

**任务清单**：

1. **模板波形获取**
   - 使用 `pycbc` 或 `gwpy` 生成双黑洞并合的理论模板波形
   - 或从 GWOSC 下载预计算的模板
   - 理解模板的物理含义：质量、自旋参数如何影响波形

2. **匹配滤波实现**
   - 在频域实现匹配滤波器：$SNR(f) = \frac{\tilde{s}(f) \cdot \tilde{h}^*(f)}{S_n(f)}$
   - 其中 $\tilde{s}(f)$ 为数据的FFT，$\tilde{h}(f)$ 为模板的FFT，$S_n(f)$ 为PSD
   - 计算信噪比(SNR)时间序列

3. **检测统计量分析**
   - 绘制 SNR 时间序列，定位信号峰值
   - 验证 H1 和 L1 的峰值时间差是否与光速传播一致 (~7 ms)
   - 计算联合 SNR，评估检测显著性

4. **参数估计（选做）**
   - 在质量和自旋参数空间进行网格搜索
   - 找到最大 SNR 对应的最佳拟合参数
   - 与发表的 GW150914 参数进行对比

**交付物**：
- 匹配滤波 Notebook
- SNR 时间序列图
- 最佳拟合模板叠加图

---

### 阶段四：时频分析与 Q 变换（第8-10周）

**目标**：使用时频分析方法可视化引力波信号的频率演化

**任务清单**：

1. **短时傅里叶变换 (STFT)**
   - 实现 STFT，生成频谱图
   - 调整窗长参数，平衡时间分辨率与频率分辨率
   - 观察信号的啁啾（频率随时间升高）特征

2. **Q 变换实现**
   - 理解 Q 变换的原理：恒定 Q 的时频变换
   - 使用 `gwpy` 的 Q 变换功能，或自行实现
   - Q 变换使用对数间隔的频率网格，天然适配引力波信号

3. **时频图优化**
   - 生成高分辨率 Q 变换时频图
   - 绘制等高线形式的 SNR 时频图
   - 标注信号频率演化轨迹（~35 Hz → ~250 Hz，持续 ~0.2 s）

4. **多探测器对比**
   - 分别生成 H1 和 L1 的时频图
   - 讨论两个探测器看到的信号一致性
   - 绘制去噪后的时频图（减去噪声背景）

**交付物**：
- 时频分析 Notebook
- Q 变换时频图（含标注）
- 信号特征总结报告
