# 智能信号处理 — 项目制教学方案（52学时）

> 适用对象：武汉大学人工智能学院 大二本科生
> 总学时：52（理论40 + 实践12）
> 教学模式：全景导览 + 4个项目驱动 + 答辩总结

---

## 一、课程理念

```
传统模式：知识点1 → 知识点2 → ... → 考试（学生被动接收）
项目制模式：全景导览 → 做项目1 → 做项目2 → 做项目3 → 做项目4 → 答辩
          （先见森林）  （在实践中理解理论）  （知识内化）
```

**核心原则：**
1. **理论服务于项目** — 每个知识点的引入都是因为项目需要用到它
2. **知识库自主查阅** — 理论细节在课程知识库中，学生按需学习
3. **跨学科真实场景** — 项目来自生物医学、天体物理、音乐信息检索、计算机视觉
4. **代码先行** — 先跑通代码看到效果，再理解背后的数学

## 三、全景导览（第1-2周，6学时）

### 第1次课（2学时）：信号的世界 — 从模拟到数字

**目标**：建立信号处理的整体直觉

| 时间 | 内容 | 方式 |
|------|------|------|
| 20min | 什么是信号？生活中的信号举例（心电图、音乐、地震、引力波） | 可视化Demo |
| 20min | 信号的时域/频域表示 — 同一个信号两种视角 | Python实时演示 |
| 20min | 采样与数字化 — 为什么CD是44.1kHz？ | 听觉实验 |
| 20min | 滤波器 — 去噪的基本工具 | 交互式滤波Demo |
| 10min | 课程地图预告：我们要做什么项目？ | PPT展示 |

### 第2次课（2学时）：从傅里叶到小波 — 分析信号的工具箱

| 时间 | 内容 | 方式 |
|------|------|------|
| 20min | 傅里叶变换 — 频谱分析的基石 | 3Blue1Brown视频片段 |
| 20min | STFT与小波变换 — 时频分析 | 语谱图可视化 |
| 20min | 统计信号 — 随机性与最优估计 | 噪声消除Demo |
| 20min | 深度学习 + 事件相机 — AI如何改变信号处理 | 案例展示 |
| 10min | 项目分组 + 选题指导 | 讨论 |

### 第3次课（2学时）：Python信号处理实训 + 环境搭建

| 时间 | 内容 | 方式 |
|------|------|------|
| 60min | Jupyter环境搭建 + NumPy/SciPy/Matplotlib基础 | 动手实验 |
| 30min | 第一个信号处理程序：读取音频 → FFT → 频谱图 → 滤波 | 代码演示 |
| 10min | 各项目组确认选题 | 讨论 |

> 🎯 **导览结束后，学生应能**：
> - 说出信号处理的6大模块分别解决什么问题
> - 理解27个知识点之间的关系（看懂知识图谱）
> - 搭建好Python信号处理环境
> - 确定自己的项目选题

## 四、项目详情

### 📱 项目1：心电信号智能分析系统（第3-6周）

**跨学科背景**：生物医学工程
**真实场景**：医院心电监护仪需要自动检测心律异常

#### 知识点覆盖

| 阶段 | 理论知识点 | 实践任务 |
|------|-----------|---------|
| 第3周 | 信号与系统基础、采样与重建 | 读取MIT-BIH心电数据，绘制波形 |
| 第4周 | 傅里叶分析、Z变换 | 对ECG做FFT频谱分析，设计带通滤波器 |
| 第5周 | 数字滤波器设计、自适应滤波 | 去除工频干扰和基线漂移 |
| 第6周 | 深度学习基础 | 用CNN做心律分类，**项目验收** |

#### 项目要求
- 输入：MIT-BIH原始ECG信号（含噪声）
- 处理：滤波去噪 → QRS检测 → 特征提取 → 异常分类
- 输出：心律分类结果 + 可视化报告
- 评分：功能完整性40% + 代码质量30% + 报告质量30%

#### 阶段性理论讲解（8学时）

| 学时 | 内容 | 与项目的关系 |
|------|------|-------------|
| 1-2 | 信号与系统基础 | ECG是什么信号？连续vs离散 |
| 3-4 | 傅里叶分析 + Z变换 | ECG的频谱长什么样？如何分析？ |
| 5-6 | 采样与重建 + 滤波器设计 | 如何去除50Hz工频干扰？ |
| 7-8 | 自适应滤波 + 深度学习入门 | 如何去除基线漂移？如何自动分类？ |

---

### 🎵 项目2：音乐信号智能识别（第7-10周）

**跨学科背景**：音乐信息检索（MIR）× 语音处理
**真实场景**：音乐App自动标注歌曲流派/情绪

#### 知识点覆盖

| 阶段 | 理论知识点 | 实践任务 |
|------|-----------|---------|
| 第7周 | 短时傅里叶变换 | 对音乐做STFT，绘制语谱图 |
| 第8周 | 小波变换、时频分布 | 不同时频方法对比 |
| 第9周 | 语音信号基础、MFCC特征 | 提取音乐MFCC特征 |
| 第10周 | 语音增强/分离概念 | 音乐源分离（人声/伴奏），**项目验收** |

#### 项目要求
- 输入：GTZAN音乐数据集（10个流派，1000首）
- 处理：STFT → 时频特征 → MFCC → 流派/情绪分类
- 扩展：尝试从混合音频中分离人声和伴奏
- 评分：功能完整性40% + 技术深度30% + 报告质量30%

#### 阶段性理论讲解（8学时）

| 学时 | 内容 | 与项目的关系 |
|------|------|-------------|
| 1-2 | STFT | 语谱图：音乐的"指纹" |
| 3-4 | 小波变换 + 时频分布 | 不同时频方法各有什么优劣？ |
| 5-6 | 语音信号基础 | 人声的产生模型、MFCC |
| 7-8 | 语音增强/分离 | 如何分离人声和伴奏？ |

### 🌊 项目3：引力波信号探测（第11-14周）

**跨学科背景**：天体物理学
**真实场景**：LIGO实验如何从噪声中"听到"宇宙的声音

#### 知识点覆盖

| 阶段 | 理论知识点 | 实践任务 |
|------|-----------|---------|
| 第11周 | 随机信号基础 | 理解LIGO噪声特性，功率谱分析 |
| 第12周 | 维纳滤波 | 最优滤波器设计，信噪比分析 |
| 第13周 | 卡尔曼滤波入门 | 状态估计直觉，与维纳滤波对比 |
| 第14周 | 图像重建 + 计算成像 | 时频图（Q-transform）绘制，**项目验收** |

#### 项目要求
- 输入：LIGO开放数据（含真实引力波事件GW150914）
- 处理：噪声分析 → 匹配滤波 → 信噪比计算 → 时频图绘制
- 输出：引力波探测报告 + 时频可视化
- 评分：物理直觉30% + 技术实现40% + 报告质量30%

#### 阶段性理论讲解（8学时）

| 学时 | 内容 | 与项目的关系 |
|------|------|-------------|
| 1-2 | 随机信号基础 | LIGO的噪声是什么性质的？ |
| 3-4 | 维纳滤波 | 如何设计最优滤波器？ |
| 5-6 | 卡尔曼滤波入门 | 动态系统的状态估计 |
| 7-8 | 图像重建 + 计算成像 | 时频图的计算方法 |

### 📷 项目4：事件相机智能感知系统（第15-17周）

**跨学科背景**：计算机视觉 × 神经科学（与DVS-Group研究方向直接相关）
**真实场景**：事件相机如何实现超高速视觉感知

| 阶段 | 理论知识点 | 实践任务 |

| 第17周 | 生成模型、脉冲神经网络 | SNN基础概念，**项目验收** |

#### 项目要求

- 处理：事件流去噪 → 事件表征 → 图像重建/光流估计
- 扩展：尝试SNN模型处理事件数据
- 评分：创新性30% + 技术实现40% + 报告质量30%

#### 阶段性理论讲解（6学时）
