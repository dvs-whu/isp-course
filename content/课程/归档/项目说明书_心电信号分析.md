## 一、项目背景

心电图（Electrocardiogram, ECG）是临床中最常用的无创心脏功能检测手段。通过记录心脏电活动随时间的变化，ECG 能够反映心率、心律及各类心律失常（Arrhythmia）等关键生理信息。传统的心电信号分析依赖医生的肉眼判读，效率低且易受主观因素影响。

近年来，深度学习技术在 ECG 信号分类中取得了突破性进展。本项目将引导学生从**信号处理基础**出发，经历数据探索、滤波去噪、特征工程、深度学习建模的全流程，最终实现一个能自动识别多种心律失常的分类系统。

## 三、数据集

### MIT-BIH Arrhythmia Database

- **来源**：PhysioNet — [https://physionet.org/content/mitdb/1.0.0/](https://physionet.org/content/mitdb/1.0.0/)
- **规模**：48 条半小时双通道 ECG 记录，来自 47 名受试者
- **采样率**：360 Hz
- **标注**：每拍标注，遵循 AAMI 标准，共约 110,000 个心拍
- **心律类别**（AAMI 五类）：
  - **N**：正常心拍（Normal）
  - **S**：室上性异位心拍（Supraventricular ectopic）
  - **V**：室性异位心拍（Ventricular ectopic）
  - **F**：融合心拍（Fusion）
  - **Q**：未知心拍（Unknown）

```python
import wfdb
import numpy as np

# 下载单条记录（示例：100 号记录）
record = wfdb.rdrecord('100', pn_dir='mitdb')
annotation = wfdb.rdann('100', 'atr', pn_dir='mitdb')

# 查看基本信息
print(f"采样率: {record.fs} Hz")
print(f"信号时长: {record.sig_len / record.fs:.1f} 秒")
print(f"通道数: {record.n_sig}")
print(f"导联名称: {record.sig_name}")

# 提取第一个通道的信号
ecg_signal = record.p_signal[:, 0]
print(f"信号形状: {ecg_signal.shape}")
print(f"信号范围: [{ecg_signal.min():.3f}, {ecg_signal.max():.3f}] mV")
```

### 数据可视化（阶段一示例代码）

```python
import matplotlib.pyplot as plt

fig, axes = plt.subplots(3, 1, figsize=(14, 10))

# 图1：完整波形（前 10 秒）
duration = 10  # 秒
samples = duration * record.fs
t = np.arange(samples) / record.fs

axes[0].plot(t, ecg_signal[:samples], 'b-', linewidth=0.5)
axes[0].set_xlabel('时间 (秒)')
axes[0].set_ylabel('幅值 (mV)')
axes[0].set_title('ECG 信号 - 前 10 秒')
axes[0].grid(True, alpha=0.3)

# 标注 R 峰位置
r_peaks = annotation.sample[annotation.symbol == 'N']
r_peaks_10s = r_peaks[r_peaks < samples]
for r in r_peaks_10s:
    axes[0].axvline(x=r/record.fs, color='red', alpha=0.3, linestyle='--')

# 图2：单个心拍（P-QRS-T 波群）
r_idx = r_peaks[10]  # 取第 11 个 R 峰
beat_start = r_idx - 100  # R 峰前约 278ms
beat_end = r_idx + 200    # R 峰后约 556ms
t_beat = np.arange(beat_start, beat_end) / record.fs

axes[1].plot(t_beat, ecg_signal[beat_start:beat_end], 'b-', linewidth=1.5)
axes[1].set_xlabel('时间 (秒)')
axes[1].set_ylabel('幅值 (mV)')
axes[1].set_title('单个心拍波形 (P-QRS-T)')
axes[1].grid(True, alpha=0.3)

# 图3：功率谱密度
from scipy import signal as sig
f, Pxx = sig.welch(ecg_signal, fs=record.fs, nperseg=1024)
axes[2].semilogy(f, Pxx)
axes[2].set_xlabel('频率 (Hz)')
axes[2].set_ylabel('功率谱密度 (mV²/Hz)')
axes[2].set_title('ECG 功率谱密度')
axes[2].set_xlim([0, 50])
axes[2].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('ecg_exploration.png', dpi=150)
plt.show()
```

## 八、常见问题与提示

### Q1: 数据集太大怎么办？
> MIT-BIH 数据集约 250 MB，可以在课程开始时预先下载。如果磁盘空间不足，可以选择 10–15 条记录进行实验。

### Q2: 没有 GPU 怎么训练深度学习模型？
> 推荐使用 Google Colab（免费 GPU）或学校的 GPU 集群。1D-CNN 模型在 CPU 上训练也相对快速（约 30 分钟/50 epochs）。

### Q3: 如何处理类别不平衡问题？
> MIT-BIH 数据集存在严重的类别不平衡（N 类占约 80%）。建议：
> - 使用加权交叉熵损失函数
> - 对少数类进行过采样
> - 使用分层采样划分训练集和测试集

### Q4: 可以使用预训练模型吗？
> 鼓励从零实现模型架构，理解底层原理。可以参考论文中的模型设计，但禁止直接复制代码。
