## 第一阶段：经典信号处理 + 迷你项目（第1-12周）

> 🎯 目标：扎实掌握经典信号处理理论，通过迷你项目动手实践
> 
> 📋 项目一：迷你项目（个人，经典方法，4-5个选题可选）
> - 第2周发布选题，学生自选
> - 第6周中期检查（代码跑通 + 阶段报告）
> - 第12周提交最终报告

### 第1周（理论2学时）
**主题：信号的世界 — 从模拟到数字**

| 时间    | 内容                          | 教学方式       | 知识点     |
| ----- | --------------------------- | ---------- | ------- |
| 20min | 什么是信号？生活中的信号举例（ECG、音乐、事件相机） | 可视化Demo    | 信号与系统基础 |
| 20min | 信号的时域/频域表示                  | Python实时演示 | 傅里叶分析   |
| 20min | 采样与数字化 — 为什么CD是44.1kHz？     | 听觉实验       | 采样与重建   |
| 20min | 滤波器 — 去噪的基本工具               | 交互式Demo    | 数字滤波器设计 |
| 10min | 课程全景：两个项目 + 考核方式介绍          | PPT展示      | —       |

### 第2周（理论2学时 + 实践2学时）
**主题：分析工具全景 + 环境搭建 + 项目一选题**

**理论课**：

| 时间 | 内容 | 教学方式 |
|------|------|---------|
| 20min | 傅里叶变换 — 频谱分析的基石 | 3Blue1Brown视频 |
| 20min | STFT与小波变换 — 时频分析概述 | 语谱图可视化 |
| 20min | 统计信号 — 随机性与最优估计概述 | 噪声消除Demo |
| 20min | 深度学习概述 — AI如何改变信号处理 | 案例展示 |
| 10min | **项目一选题发布**（4-5个迷你项目） | 讨论 |

**实践课（2学时）**：
- 60min：Jupyter环境搭建 + NumPy/SciPy/Matplotlib基础
- 30min：第一个信号处理程序（读取音频 → FFT → 频谱图）
- 10min：确认项目一选题

> ⚠️ 环境搭建是关键卡点。建议提前准备Docker镜像或conda环境文件，学生一键部署。

### 第3周（理论2学时）
**主题：信号与系统基础**

| 内容 | 教学要点 |
|------|---------|
| 连续信号与离散信号 | 采样定理（奈奎斯特）直觉讲解 |
| 线性时不变系统（LTI） | 用"滤波器就是LTI系统"来统一概念 |
| 卷积运算 | 图形化演示卷积过程 |
| 项目一指导 | 课后答疑：迷你项目数据加载 |

### 第4周（理论2学时）
**主题：傅里叶分析**

| 内容 | 教学要点 |
|------|---------|
| 傅里叶级数 → 傅里叶变换 → DFT → FFT | 四个概念的逻辑链条 |
| FFT的计算与频谱解读 | Python演示：不同信号的频谱 |
| 频域滤波思想 | "在频域做乘法 = 在时域做卷积" |
| 功率谱与能量谱 | 直觉理解，不深入推导 |

### 第5周（理论2学时 + 实践2学时）
**主题：采样、量化与重建**

**理论课**：

| 内容 | 教学要点 |
|------|---------|
| 采样定理与混叠 | 听8kHz vs 44.1kHz音频，感受混叠 |
| 量化与信噪比 | 用位深变化演示量化噪声 |
| 上采样与下采样 | 图像缩放中的应用 |
| 信号重建 | 从样本恢复连续信号（插值） |

**实践课（2学时）**：
- 项目一实验：数据加载 + FFT频谱分析 + 采样率变换实验
- 助教巡回指导

### 第6周（理论2学时）
**主题：数字滤波器设计**

| 内容 | 教学要点 |
|------|---------|
| FIR vs IIR 滤波器 | 各自优缺点对比 |
| 窗函数法设计FIR | 汉宁窗、汉明窗、布莱克曼窗 |
| 频率采样法 | 直觉理解 |
| 滤波器性能指标 | 通带、阻带、过渡带、纹波 |

> 📋 **第6周末：项目一中期检查** — 学生提交阶段性代码和简短进展报告

### 第7周（理论2学时）
**主题：图像信号基础**

| 内容 | 教学要点 |
|------|---------|
| 图像作为二维信号 | 从一维到二维的扩展 |
| 二维傅里叶变换 | 图像频谱可视化（频域中心化） |
| 图像频域滤波 | 低通/高通/带通在图像上的效果 |
| 边缘检测 | Sobel、Laplacian = 高通滤波 |

### 第8周（理论2学时 + 实践2学时）
**主题：图像去噪与增强**

**理论课**：

| 内容 | 教学要点 |
|------|---------|
| 空间滤波 vs 频域滤波 | 两种视角的等价性 |
| 均值/中值/高斯滤波 | 不同噪声类型的最优选择 |
| 图像增强 | 直方图均衡化、对比度拉伸 |
| 去噪评估 | PSNR、SSIM指标（简化讲解） |

**实践课（2学时）**：
- 项目一实验：滤波器设计与应用（去噪/增强）
- 助教指导项目一代码

### 第9周（理论2学时）
**主题：语音信号基础**

| 内容 | 教学要点 |
|------|---------|
| 语音产生模型 | 声道 → 激励 → 共鸣，用动画演示 |
| 语音的时域特征 | 短时能量、过零率 |
| 语谱图 | "语音的指纹"，视觉冲击力强 |
| 共振峰与基频 | 直觉理解，不做数学推导 |

### 第10周（理论2学时）
**主题：语音特征提取**

| 内容 | 教学要点 |
|------|---------|
| MFCC特征 | 从物理到数学的桥梁（简化版） |
| 频谱包络 vs 频谱细节 | "远看是包络，近看是细节" |
| 特征可视化 | Python演示MFCC提取全流程 |
| 语音增强基础 | 谱减法的直觉 |

### 第11周（理论2学时 + 实践2学时）
**主题：统计信号处理基础**

**理论课**：

| 内容 | 教学要点 |
|------|---------|
| 随机信号的统计描述 | 用"样本均值/方差"代替"期望/方差"（学生没学概率论） |
| 自相关与互相关 | 用"信号和自己的延迟版本有多像"来直觉化 |
| 功率谱密度 | 自相关的傅里叶变换 |
| 白噪声与色噪声 | 听觉演示 |

**实践课（2学时）**：
- 项目一实验：语音特征提取 + 统计分析
- 助教指导项目一收尾

### 第12周（理论2学时）
**主题：自适应滤波与最优估计**

| 内容 | 教学要点 |
|------|---------|
| 维纳滤波思想 | "最小化误差"的直觉 |
| LMS算法 | 用"下山"的比喻讲解梯度下降 |
| 自适应噪声消除 | 用Python实时演示 |
| 经典方法总结 | 回顾12周的知识图谱 |

> 📋 **第12周末：项目一最终提交** — 个人报告 + 代码 + 3-5分钟展示PPT

### 第13周（理论2学时 + 实践2学时）
**主题：深度学习基础 + 项目二启动**

**理论课**：

| 内容 | 教学要点 |
|------|---------|
| 神经网络基本概念 | 从感知器到多层网络 |
| 反向传播 | 用"链式法则"直觉讲解，不推导 |
| CNN基础 | 卷积层、池化层、全连接层 |
| 深度学习训练 | 损失函数、优化器、过拟合/欠拟合 |

**实践课（2学时）**：
- PyTorch环境配置
- 用CNN做简单图像分类（MNIST/CIFAR-10热身）
- **项目二选题 + 组队**

### 第14周（理论2学时）
**主题：深度学习图像处理**

| 内容 | 教学要点 |
|------|---------|
| 图像去噪（DnCNN等） | 从"滤波去噪"到"学习去噪"的思想转变 |
| 图像超分辨率 | 从插值到SRCNN到Real-ESRGAN |
| 图像分割（U-Net） | 用生物医学图像分割案例 |
| 迁移学习 | "站在巨人肩膀上" |

### 第15周（理论2学时）
**主题：深度学习语音处理**

| 内容 | 教学要点 |
|------|---------|
| 语音增强（神经网络版） | 从谱减法到DNN-based增强 |
| 语音识别概述 | 端到端 vs 传统流水线 |
| 语音合成概述 | TTS的基本思路 |
| 多模态信号处理 | 事件相机+音频等跨模态案例 |

> 📋 **第15周末：项目二中期汇报** — 各组5分钟PPT汇报进展

### 第16周（理论2学时 + 实践2学时）
**主题：前沿方法与项目实战**

**理论课**：

| 内容 | 教学要点 |
|------|---------|
| Transformer在信号处理中的应用 | Vision Transformer、Audio Transformer |
| 生成模型概述 | GAN、Diffusion在信号处理中的应用 |
| 自监督学习 | 用少量标注数据的思路 |
| 事件相机信号处理 | DVS-Group研究方向简介（可邀请博士生Guest Lecture） |

**实践课（2学时）**：
- 项目二实验：模型训练与调参
- 助教巡回指导

### 第17周（理论2学时）
**主题：总结与展望**

| 内容 | 教学要点 |
|------|---------|
| 从经典到AI：信号处理的演进 | 回顾"滤波→特征→端到端"的发展脉络 |
| 信号处理的未来方向 | 基础模型、多模态、神经形态计算 |
| 工程中的信号处理 | 实际工业界怎么用信号处理 |
| 项目二答疑 | 最后一次集体答疑 |
