## 一、项目背景

音乐是人类文化的重要组成部分，而数字音乐的爆炸式增长催生了对自动化音乐分析技术的巨大需求。音乐信息检索（Music Information Retrieval, MIR）是一个融合信号处理、机器学习、音乐学和认知科学的交叉学科，致力于从音频信号中提取有意义的信息。

本项目将以**音乐流派分类**为核心任务，引导学生从音频信号的加载与可视化出发，逐步掌握时频分析、特征工程和分类建模的完整流程。通过本项目，学生将深入理解从一维音频波形到高级语义特征的转换过程。

### 1.1 跨学科意义

| 学科 | 贡献 |
|------|------|
| **信号处理** | 提供时频分析工具（STFT、小波变换、MFCC） |
| **音乐学** | 提供音乐理论基础（节奏、音色、和声） |
| **机器学习** | 提供分类建模方法（传统 ML 与深度学习） |
| **认知科学** | 解释人类如何感知和区分不同音乐类型 |
| **计算机科学** | 提供高效实现与系统集成能力 |

### 1.2 MIR 领域简介

音乐信息检索的典型应用包括：
- 🎵 **音乐流派分类**（本项目）
- 🎤 **歌手识别**
- 🎸 **乐器识别**
- 🎼 **自动转谱**
- 😊 **音乐情感分析**
- 🔄 **翻唱检测**

```python
import librosa

# 加载单个音频文件
file_path = os.path.join(DATA_DIR, "blues", "blues.00000.wav")

print(f"\n音频信息:")

print(f"  信号长度: {len(y)} 样本")

print(f"  数据类型: {y.dtype}")
print(f"  幅值范围: [{y.min():.4f}, {y.max():.4f}]")
```

### 数据可视化（阶段一示例代码）

```python
fig, axes = plt.subplots(4, 1, figsize=(14, 16))

# 图1：波形图
librosa.display.waveshow(y, sr=sr, ax=axes[0], color='blue')
axes[0].set_title('音频波形 - Blues.00000', fontsize=14)
axes[0].set_xlabel('时间 (秒)')
axes[0].set_ylabel('幅值')

# 图2：频谱图（STFT）
D = librosa.stft(y, n_fft=2048, hop_length=512)
S_db = librosa.amplitude_to_db(np.abs(D), ref=np.max)
img = librosa.display.specshow(S_db, sr=sr, hop_length=512, 
                                x_axis='time', y_axis='hz', ax=axes[1])
axes[1].set_title('短时傅里叶变换 (STFT) 频谱图', fontsize=14)
axes[1].set_ylim([0, 8000])
fig.colorbar(img, ax=axes[1], format='%+2.0f dB')

# 图3：梅尔频谱图
S_mel = librosa.feature.melspectrogram(y=y, sr=sr, n_mels=128, fmax=8000)
S_mel_db = librosa.power_to_db(S_mel, ref=np.max)
img2 = librosa.display.specshow(S_mel_db, sr=sr, hop_length=512,
                                 x_axis='time', y_axis='mel', ax=axes[2])
axes[2].set_title('梅尔频谱图 (Mel Spectrogram)', fontsize=14)
axes[2].set_ylim([0, 8000])
fig.colorbar(img2, ax=axes[2], format='%+2.0f dB')

# 图4：MFCC
mfccs = librosa.feature.mfcc(y=y, sr=sr, n_mfcc=13)
img3 = librosa.display.specshow(mfccs, sr=sr, hop_length=512,
                                 x_axis='time', ax=axes[3])
axes[3].set_title('MFCC 特征', fontsize=14)
axes[3].set_ylabel('MFCC 系数')
fig.colorbar(img3, ax=axes[3])

plt.tight_layout()
plt.savefig('audio_exploration.png', dpi=150)
plt.show()

# 不同流派的频谱对比
fig, axes = plt.subplots(2, 5, figsize=(20, 8))
for idx, genre in enumerate(sorted(genres)[:10]):
    row, col = idx // 5, idx % 5
    file_path = os.path.join(DATA_DIR, genre, f"{genre}.00000.wav")
    y_g, sr_g = librosa.load(file_path, sr=22050, duration=30)
    S_g = librosa.feature.melspectrogram(y=y_g, sr=sr_g, n_mels=128)
    S_g_db = librosa.power_to_db(S_g, ref=np.max)
    librosa.display.specshow(S_g_db, sr=sr_g, x_axis='time', 
                              y_axis='mel', ax=axes[row, col])
    axes[row, col].set_title(genre, fontsize=12)
    axes[row, col].set_xticks([])
    axes[row, col].set_yticks([])

plt.suptitle('10 种音乐流派的梅尔频谱图对比', fontsize=16)
plt.tight_layout()
plt.savefig('genre_comparison.png', dpi=150)
plt.show()
```

### Q2: 如何从零实现 MFCC？
> MFCC 提取流程如下，每一步都需要自己实现：
> 1. 预加重滤波器：`y[n] = y[n] - 0.97 * y[n-1]`
> 2. 分帧加窗（帧长 25ms，帧移 10ms）
> 3. 对每帧做 FFT
> 4. 构建梅尔滤波器组（三角滤波器）
> 5. 计算滤波器组能量

```python
import numpy as np

def stft_manual(signal, n_fft=2048, hop_length=512, window='hann'):
    """
    从零实现短时傅里叶变换（STFT）
    
    参数:
        signal: 输入音频信号 (1D array)
        n_fft: FFT 窗口大小
        hop_length: 帧移
        window: 窗函数类型 ('hann', 'hamming', 'rectangular')
    
    返回:
        stft_matrix: 复数 STFT 矩阵 (n_fft//2+1, n_frames)
    """
    # 创建窗函数
    if window == 'hann':
        win = np.hanning(n_fft)
    elif window == 'hamming':
        win = np.hamming(n_fft)
    else:
        win = np.ones(n_fft)
    
    # 计算帧数
    n_frames = 1 + (len(signal) - n_fft) // hop_length
    
    # 初始化 STFT 矩阵
    stft_matrix = np.zeros((n_fft // 2 + 1, n_frames), dtype=complex)
    
    # 逐帧处理
    for i in range(n_frames):
        start = i * hop_length
        frame = signal[start:start + n_fft] * win
        spectrum = np.fft.rfft(frame)
        stft_matrix[:, i] = spectrum
    
    return stft_matrix
```
