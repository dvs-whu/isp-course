## 一、课程设计原则

### 1. 13周作为唯一主线

当前版本只维护13周课程节奏：每周4学时，所有项目、实验、考核节点都按W1-W13编号。学时统计必须保持40理论 + 12实践，其中实践学时只包括6次正式实验课。

### 2. 先讲后练

课程仍然坚持“先讲后用”：先讲当周需要的工具，再通过实验、项目任务或课堂研讨使用这些工具。需要注意的是，项目指导、展示、答疑计入理论/课堂研讨学时，不计入12个实践学时。

### 3. 用项目组织知识

课程不追求把所有公式讲满，而是让学生通过两个项目形成判断力：

| 项目 | 目标 |
|------|------|
| 项目一 | 让学生用经典信号处理方法完成一个小而完整的问题 |
| 项目二 | 让学生比较经典预处理和AI模型在真实任务中的作用 |

## 二、学生基础与教学约束

| 约束        | 风险         | 处理方式                       |
| --------- | ---------- | -------------------------- |
| 概率论可能未修   | 统计信号处理理解困难 | 用样本均值、样本方差、自相关、功率谱的直觉讲法    |
| 编程基础不一致   | 实验进度差异大    | 提供Notebook模板；助教优先帮学生跑通数据加载 |
| 深度学习基础不完整 | 项目二推进慢     | W9用MNIST热身；项目二允许使用成熟模型骨架   |
| 数据集下载困难   | 项目卡住       | 提前准备小样本数据包或合成数据替代          |

## 三、13周执行节奏

| 周次 | 课堂重点 | 助教重点 | 产出 |
|------|----------|----------|------|
| W1 | 信号概述、课程地图、环境搭建 | 检查Python/Jupyter环境，协助项目一选题 | 项目一选题确认 |
| W2 | FFT与频谱分析 | 帮学生完成第一张频谱图 | 项目一频谱分析结果 |
| W3 | 采样、混叠、功率谱 | 检查采样率任务和PSD代码 | 项目一阶段图表 |
| W4 | FIR/IIR滤波器 | 协助调滤波器参数，检查SNR/PSNR等指标 | 实验2结果 |
| W5 | Z变换、零极点、系统分析 | 组织项目一中期检查 | 中期代码和2页进展 |
| W6 | 图像与语音信号处理 | 帮图像/语音方向学生补齐特征和指标 | 实验3结果 |
| W7 | 统计信号与最优估计 | 指导报告结构和图表表达 | 项目一报告初稿 |
| W8 | 经典方法总结 | 收集项目一材料，组织展示 | 项目一提交 |
| W9 | 深度学习基础 | 检查PyTorch环境，协助项目二组队 | 项目二选题和分工 |
| W10 | 图像/语音AI方法 | 检查数据集、预处理和baseline | 项目二数据分析 |
| W11 | 展开网络、生成模型、事件相机概述 | 组织项目二中期汇报 | 中期PPT |
| W12 | 对比实验与结果分析 | 帮学生完成经典vs AI对比和答辩材料 | 项目二提交 |
| W13 | 答辩、回顾、复习 | 记录答辩评分，整理课程反馈 | 项目二答辩 |

## 四、项目管理要求

### 项目一

| 节点 | 要求 |
|------|------|
| W1 | 从5个选题中确定一个，完成数据来源确认 |
| W5 | 代码跑通主流程，提交2页进展说明 |
| W8 | 提交Notebook、3-5页报告、3分钟展示PPT |

项目一只使用经典信号处理方法，不使用深度学习。

### 项目二

| 节点  | 要求                     |
| --- | ---------------------- |
| W9  | 3-4人组队，确定选题、数据集和分工     |
| W11 | 中期汇报：数据、经典预处理、AI模型初步结果 |
| W12 | 提交代码、5-8页报告、答辩PPT      |
| W13 | 15分钟展示 + 提问            |

项目二必须包含经典方法部分和AI模型部分，并做至少一组对比实验。

## 五、实验组织

实验不单独形成另一条课程线，而是嵌入主线：

| 实验 | 周次 | 作用 |
|------|------|------|

| 实验4 | W7 | 支撑项目一E的统计分析 |
| 实验5 | W9 | 支撑项目二的PyTorch基础 |
| 实验6 | W11 | 支撑项目二的综合应用 |

## 六、考核执行

| 考核项 | 比例 | 执行说明 |
|--------|------|----------|
| 期末笔试 | 50% | 只考W1-W8经典信号处理主线 |
| 项目一 | 15% | 个人评分，重点看方法理解、代码可复现、结果解释 |
| 项目二 | 20% | 小组评分，结合个人贡献调整 |
| 过程考核 | 15% | 出勤、互动、小测、实验完成度 |

建议小测安排在W4、W7、W10，每次10-15分钟，题量控制在5道以内。
