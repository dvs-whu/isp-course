```python
import wfdb
# 下载记录 100（推荐用于入门练习）
record = wfdb.rdrecord('100', pn_dir='mitdb', sampto=3600)  # 10秒数据

# 批量下载所有48条记录（可选）
# wfdb.dl_database('mitdb', dl_dir='./data/mitdb')
```

#### 预处理清单

- [x] 基本无需预处理，wfdb直接读取

```python
# 白噪声
noise_white = np.random.randn(len(t)) * 0.5

# 色噪声（低频为主）
from scipy.signal import butter, lfilter
b, a = butter(4, 0.1)
noise_colored = lfilter(b, a, np.random.randn(len(t)))

# 含噪信号
signal_noisy = signal_clean + noise_white

# 卡尔曼滤波：匀速运动目标
# 状态 x = [位置, 速度]
```

## 💡 常见问题

### Q: 下载速度太慢怎么办？
A: 
- 使用 IDM/迅雷等多线程下载工具
- 校园网深夜（0:00-8:00）通常更快
- 大数据集找同学互助拷贝
- Kaggle 数据集可用 `kaggle competitions download -c <比赛名>` 命令行下载

### Q: 数据集太大装不下怎么办？
A:
- 只下载需要的子集（如 STEAD 只下1个chunk，LibriSpeech 只下 dev-clean）
- 项目二只需证明方法有效，不需在完整数据集上跑
- 可以用 `subset` 参数控制加载量

### Q: 注册需要学术邮箱吗？
A:
- PhysioNet：需要有效邮箱注册
- Kaggle：任意邮箱
- Zenodo：直接下载，不强制注册
- IEMOCAP：需要学术邮箱申请

### Q: 数据集应该放在哪里？
A:
```
项目目录/
├── data/          # 数据文件（不要提交到Git）
│   ├── mitdb/     # 选题A
│   ├── audio/     # 选题B/D
│   └── ...
├── notebooks/     # Jupyter Notebook
├── report/        # 报告文档
└── .gitignore     # 添加 data/ 到忽略列表
```
