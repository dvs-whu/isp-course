# 🎓 智能信号处理

**武汉大学人工智能学院 · 2026年秋季学期**

> 从真实信号出发，先学会观察、变换、滤波和建模，再用AI方法完成更复杂的信号处理任务。

## 🎯 学习目标

1. 看到时间序列、图像、语音、事件流等数据，判断它属于什么信号问题
2. 在时域、频域、时频域之间切换，解释信号中的主要结构和噪声来源
3. 用Python完成数据加载、可视化、频谱分析、滤波、特征提取和基础建模
4. 比较经典信号处理方法和AI方法的适用场景、优缺点和工程限制
5. 通过项目把信号处理方法迁移到生物医学、语音、图像、地球物理、生态监测等场景
