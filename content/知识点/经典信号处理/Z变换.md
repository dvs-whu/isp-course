> **关联实验**：项目一B（滤波器分析与设计）
> **预备知识**：离散时间信号基础、傅里叶变换（DTFT）
> **核心问题**：能否通过复平面上的零点和极点，看懂离散系统怎样响应不同频率？

## 2. 通俗入口

> [!tip] 🎯 初学者简化说明
> **一句话理解**：Z变换就像给序列"拍照"，ROC就是照片清晰的范围。

**一个真实问题**：你设计了一个数字滤波器（比如降噪耳机里的），差分方程写好了，但怎么知道它会不会"发散"（输出越来越大直到爆掉）？怎么知道它对不同频率的声音放大还是衰减？Z变换能通过复平面上的零点和极点，一眼看出答案。

**生活类比**：你拍了一张照片（做了Z变换），但只有在一定距离/角度内照片才是清晰的（级数收敛）。如果你站在太远或太近的地方（|z|太大或太小），照片就糊了（级数发散）。**单位圆**是"黄金拍摄距离"——在这个距离上拍出来的照片，就是我们熟悉的频率响应（DTFT）。

**学这个能解决什么**：将离散时域的差分方程转化为代数方程，通过零极点图分析系统稳定性、频率响应、因果性，是数字滤波器设计的数学基础。

本笔记只需要掌握：定义 → 常用变换对 → ROC直觉 → 部分分式展开求逆变换。围线积分等高级内容已省略。

**严谨边界**：重点理解系统函数、单位圆、频率响应、稳定性条件；收敛域的严格分类和留数法作为拓展。

**启发思考**：为什么极点靠近单位圆会让系统响应变强？幅度响应相同，相位响应不同会怎样影响信号？

**项目迁移**：滤波器分析、语音共振峰、预测模型和反馈系统都可用零极点直觉解释。

## 3. 严谨核心

### 3.1 定义

Z变换是离散时间信号与系统分析的核心数学工具，对应于连续时间信号的拉普拉斯变换。它将离散时域的差分方程转化为代数方程，极大简化了LTI系统的分析与设计。

**Z变换定义：**

$$X(z) = \sum_{n=-\infty}^{\infty} x[n] z^{-n}$$

### 3.2 Z变换与傅里叶变换的关系

> [!note] 关键联系
> **Z变换在单位圆上求值 = 离散时间傅里叶变换（DTFT）**
> $$X(e^{j\omega}) = X(z)\big|_{z = e^{j\omega}}$$
>
> 换句话说：Z变换是DTFT的"推广版"。DTFT只能分析频率，Z变换还能分析稳定性和因果性。

| 工具 | 做的事 | 类比 |
|------|--------|------|
| DTFT | 在单位圆上求值，得到频率响应 | 只看一条固定的环形轨道 |
| Z变换 | 在整个复平面上求值 | 拍摄整个空间的全景照片 |

### 3.3 具体数值例子

让我们从一个最简单的序列开始：

**序列**：$x[n] = (0.5)^n u[n] = \{1, \ 0.5, \ 0.25, \ 0.125, \ \ldots\}$

这是一个指数衰减序列（每一步乘以0.5）。

$$X(z) = \sum_{n=0}^{\infty} (0.5)^n z^{-n} = \sum_{n=0}^{\infty} (0.5 z^{-1})^n = \frac{1}{1 - 0.5 z^{-1}}, \quad |z| > 0.5$$

> [!info] 为什么要求 $|z| > 0.5$？
> 这是几何级数 $\sum r^n$ 收敛的条件：$|r| < 1$，即 $|0.5 z^{-1}| < 1$，等价于 $|z| > 0.5$。
> 想象一下：如果 $z$ 太小（靠近原点），$z^{-n}$ 会变得非常大，级数就"爆"了（发散）。

### 3.4 ROC（收敛域）的直觉

> [!example] 🎯 ROC直觉
> **ROC回答的问题**：在复平面上，哪些 $z$ 值能让Z变换的无穷级数"算出一个有限的数"？
>
> **三条经验法则**（初学者记住这些就够了）：
> 1. **因果序列**（右边序列，如 $a^n u[n]$）→ ROC是**某个圆的外部** $|z| > r$
> 2. **反因果序列**（左边序列）→ ROC是**某个圆的内部** $|z| < r$
> 3. **双边序列** → ROC是**环形区域** $r_1 < |z| < r_2$
>
> **稳定判断**：ROC包含单位圆（$|z|=1$）→ 系统稳定！

### 3.5 ROC与序列类型对应关系

| 序列类型 | ROC形状 | 直觉解释 | 稳定性条件 |
|----------|---------|----------|------------|
| 因果序列 $a^n u[n]$ | $\lvert z\rvert > \lvert a\rvert$（圆外） | 序列往右延伸，需要$\lvert z\rvert$足够大才能压住增长 | 当$\lvert a\rvert < 1$时，ROC包含单位圆，系统稳定 |
| 反因果序列 $-a^n u[-n-1]$ | $\lvert z\rvert < \lvert a\rvert$（圆内） | 序列往左延伸，需要$\lvert z\rvert$足够小才能收敛 | 只有当单位圆落在ROC内时稳定 |
| 双边序列 | $r_1 < \lvert z\rvert < r_2$（环形） | 左右两侧序列同时存在，ROC是两个收敛条件的交集 | ROC包含单位圆即稳定 |
| 有限长序列 | 通常为几乎全平面，可能排除$z=0$或$z=\infty$ | 只有有限项求和，所以不会出现无穷级数发散 | 有限长冲激响应系统总是BIBO稳定 |

### 3.6 核心公式

**常用Z变换对（考试必备）**：

| 时域序列 $x[n]$ | Z变换 $X(z)$ | ROC |
|-----------------|--------------|-----|
| $\delta[n]$ | $1$ | 全平面 |
| $u[n]$ | $\dfrac{1}{1 - z^{-1}}$ | $\lvert z\rvert > 1$ |
| $a^n u[n]$ | $\dfrac{1}{1 - az^{-1}}$ | $\lvert z\rvert > \lvert a\rvert$ |
| $-a^n u[-n-1]$ | $\dfrac{1}{1 - az^{-1}}$ | $\lvert z\rvert < \lvert a\rvert$ |
| $n a^n u[n]$ | $\dfrac{az^{-1}}{(1 - az^{-1})^2}$ | $\lvert z\rvert > \lvert a\rvert$ |

> [!warning] 注意
> 同一个代数表达式 $\frac{1}{1-az^{-1}}$ 对应**两种不同的时域序列**，靠ROC区分！这就是为什么ROC如此重要。

### 3.7 逆Z变换方法（基础掌握两种即可）

**方法一：部分分式展开**（最常用）

将 $X(z)$ 展开为简单分式之和，然后查表得到各分式的逆变换。

**示例**：求 $X(z) = \frac{1}{(1-0.5z^{-1})(1-0.8z^{-1})}$ 的逆变换（因果序列）

$$X(z) = \frac{A}{1-0.5z^{-1}} + \frac{B}{1-0.8z^{-1}}$$

通过待定系数法求得 $A, B$，再利用 $a^n u[n] \leftrightarrow \frac{1}{1-az^{-1}}$ 查表。

**方法二：长除法（幂级数展开）**

直接用多项式长除，将 $X(z)$ 展开为 $z^{-n}$ 的幂级数，系数就是 $x[n]$。

### 3.8 卷积定理

$$y[n] = x[n] * h[n] \quad \Leftrightarrow \quad Y(z) = X(z) \cdot H(z)$$

> 这是Z变换最强大的性质之一：**卷积变成了乘法**！

### 3.9 LTI系统传递函数

$$H(z) = \frac{Y(z)}{X(z)} = \frac{\sum_{k=0}^{M} b_k z^{-k}}{\sum_{k=0}^{N} a_k z^{-k}}$$

### 3.10 频率响应

$$H(e^{j\omega}) = H(z)\big|_{z = e^{j\omega}}$$

### 3.11 适用条件与边界

- Z变换适用于离散时间LTI系统的分析
- 级数收敛要求$|z|$在ROC内，否则Z变换无定义
- 频率响应仅在ROC包含单位圆时有意义（否则系统不稳定，DTFT不存在）
- 围线积分法求逆Z变换要求ROC内的闭合路径，本课程不要求掌握

### 典型应用

- **差分方程的代数求解**：将卷积转化为乘法，极大简化计算
- **数字滤波器分析**：通过零极点图判断滤波器类型（低通/高通/带通）
- **稳定性判断**：所有极点在单位圆内 → 系统稳定
- **控制系统**：离散控制器的分析与设计
- **全通滤波器和最小相位系统**分析

## 4. 方法流程

**输入**：离散时间序列 $x[n]$ 或差分方程描述的LTI系统

**处理步骤**：
1. 对序列 $x[n]$ 做Z变换：$X(z) = \sum x[n] z^{-n}$
2. 确定ROC（根据序列的因果性/双边性判断）
3. 如需求逆变换：部分分式展开 → 查表
4. 如分析系统：写出 $H(z) = Y(z)/X(z)$，找出零点和极点
5. 判断稳定性：所有极点是否在单位圆内
6. 求频率响应：令 $z = e^{j\omega}$，得到 $H(e^{j\omega})$

**输出**：系统的零极点分布、频率响应、稳定性判断

**常见参数选择**：
- **分析频率范围**：$[0, \pi]$（归一化频率），对应实际频率 $[0, f_s/2]$
- **零极点精度**：画零极点图时注意标注单位圆作为参考
- **部分分式展开**：对于高阶系统，建议用 `scipy.signal.residuez` 等工具辅助

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import signal
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# 定义 IIR 系统的传递函数 H(z)
# H(z) = (1 - 0.5*z^-1) / (1 - 1.7*z^-1 + 0.72*z^-2)
# 零点: z=0.5, 极点: z=0.8 和 z=0.9（均在单位圆内→稳定）
b = np.array([1, -0.5])
a = np.array([1, -1.7, 0.72])

# 获取零极点
zeros, poles, gain = signal.tf2zpk(b, a)
print(f"零点: {zeros}")
print(f"极点: {poles}")

fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 左图：零极点图
theta = np.linspace(0, 2*np.pi, 100)
axes[0].plot(np.cos(theta), np.sin(theta), 'k--', alpha=0.3, label='单位圆')
axes[0].plot(np.real(zeros), np.imag(zeros), 'bo', markersize=10, label='零点')
axes[0].plot(np.real(poles), np.imag(poles), 'rx', markersize=12, markeredgewidth=2, label='极点')
axes[0].axhline(0, color='k', linewidth=0.3)
axes[0].axvline(0, color='k', linewidth=0.3)
axes[0].set_title('零极点图（极点在单位圆内→系统稳定）')
axes[0].set_xlabel('Real Part')
axes[0].set_ylabel('Imaginary Part')
axes[0].set_aspect('equal')
axes[0].legend()
axes[0].grid(True)

# 右图：频率响应（单位圆上的 Z 变换 = DTFT）
w, h = signal.freqz(b, a, worN=512)
axes[1].plot(w/np.pi, 20*np.log10(np.abs(h)), 'b-')
axes[1].set_title('Frequency Response |H(e^{jω})| (on unit circle)')
axes[1].set_xlabel('Normalized Freq (×π rad/sample)')
axes[1].set_ylabel('Amplitude (dB)')
axes[1].grid(True)

plt.tight_layout()
plt.show()
```

**结果解释**：左图展示了系统的零极点图，极点（红色×）位于单位圆内（z=0.8和z=0.9），说明系统稳定。零点（蓝色○）在z=0.5处。右图是频率响应曲线，可以看出该系统是一个低通滤波器——低频增益高，高频衰减。极点越靠近单位圆，在对应频率处的增益峰值越尖锐。

## 6. 常见误区

**公式误解**：
- ❌ "Z变换和DTFT是完全不同的东西" → ✅ DTFT就是Z变换在单位圆上的特例：$X(e^{j\omega}) = X(z)|_{z=e^{j\omega}}$
- ❌ "同一个$X(z)$表达式只对应一个序列" → ✅ 同一个代数表达式对应不同的序列，必须靠ROC区分。如$\frac{1}{1-0.5z^{-1}}$既对应$(0.5)^n u[n]$（因果），也对应$-(0.5)^n u[-n-1]$（反因果）

**ROC误解**：
- ❌ "ROC可以随便定" → ✅ ROC由序列的支撑（因果/反因果/双边）唯一确定，不是人为选择的
- ❌ "ROC包含极点" → ✅ ROC内不能有极点（否则级数发散），ROC的边界由极点决定

**稳定性误解**：
- ❌ "极点在单位圆外就一定不稳定" → ✅ 稳定性取决于ROC是否包含单位圆，而非极点位置本身。但对于因果系统，极点确实在单位圆内才稳定
- ❌ "频率响应在所有z值上都有意义" → ✅ 频率响应只在单位圆上有定义（即ROC包含单位圆时），否则系统不稳定，DTFT不存在

**单位/概念误解**：
- ❌ "Z变换中的z是实数" → ✅ z是复变量，Z变换是在复平面上的函数
- ❌ "零点处频率响应为零" → ✅ 零点处$H(z)=0$，但频率响应是零点在单位圆上的投影处才为零，如果零点不在单位圆上，频率响应在该角度不为零

## 7. 启发问题与创新拓展

**一个反例或边界问题**：考虑系统$H(z) = \frac{1}{1-z^{-1}}$（即累加器），极点恰好在单位圆上（z=1）。这个系统是稳定还是不稳定？它的频率响应在$\omega=0$处会怎样？（提示：ROC恰好不包含单位圆，系统临界不稳定）

**一个跨学科迁移场景**：在经济学中，ARMA（自回归滑动平均）模型本质上就是一个IIR滤波器。用Z变换分析ARMA模型的零极点，可以判断经济时间序列的稳定性和周期性——极点靠近单位圆意味着经济周期振荡强烈。

**一个可做成课程项目的小想法**：设计一个交互式零极点设计器——用户在复平面上拖动零点和极点，实时观察频率响应曲线和脉冲响应的变化。用Python的matplotlib动画或Streamlit实现，加深对"零极点如何塑造频率响应"的直觉。
