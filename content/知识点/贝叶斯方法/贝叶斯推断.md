## 2. 通俗入口

想象你是一个医生。一个病人来看病，你根据经验（先验知识）初步判断他可能得了感冒。然后你做了一个化验（观测数据），结果显示白细胞偏高。你会更新你的判断——可能不是普通感冒，而是细菌感染。这个"先有判断 + 新证据 → 更新判断"的过程，就是贝叶斯推断。

**贝叶斯思维 vs 频率思维**：
- 频率学派：参数是固定的，数据是随机的。"重复做1000次实验，平均结果是..."
- 贝叶斯学派：参数也有概率分布，我们对参数的信念会随数据更新。"根据目前的证据，我有80%把握认为..."

**一句话总结**：贝叶斯推断 = 先验信念 + 新数据 → 后验信念

**生活中的贝叶斯**：
- 垃圾邮件过滤：P(垃圾|关键词) = P(关键词|垃圾) × P(垃圾) / P(关键词)
- 天气预报：今天的云量更新了你对明天下雨的信念
- 医学诊断：检测结果更新你对患病概率的估计

## 3. 严谨核心

### 定义与概念

贝叶斯推断是一种基于贝叶斯定理的统计推断框架。其核心思想是：在观测到数据后，利用贝叶斯定理将先验信念（先验分布）更新为后验信念（后验分布）。与频率学派不同，贝叶斯方法将模型参数视为随机变量，用概率分布来表达对参数的不确定性。

关键要素：
- **先验分布** $p(\theta)$：观测数据前对参数的信念
- **似然函数** $p(\mathcal{D}|\theta)$：在给定参数下观测数据的概率
- **后验分布** $p(\theta|\mathcal{D})$：结合先验与数据后的参数分布
- **边缘似然（证据）** $p(\mathcal{D})$：归一化常数

### 核心公式

贝叶斯定理：

$$
p(\theta|\mathcal{D}) = \frac{p(\mathcal{D}|\theta)\,p(\theta)}{p(\mathcal{D})}
$$

其中边缘似然为：

$$
p(\mathcal{D}) = \int p(\mathcal{D}|\theta)\,p(\theta)\,d\theta
$$

后验预测分布：

$$
p(x_{\text{new}}|\mathcal{D}) = \int p(x_{\text{new}}|\theta)\,p(\theta|\mathcal{D})\,d\theta
$$

共轭先验：若先验与似然属于共轭族，则后验与先验同族。例如 Beta-Binomial 共轭：

$$
\theta \sim \text{Beta}(\alpha, \beta) \implies \theta|\mathcal{D} \sim \text{Beta}(\alpha + k,\, \beta + n - k)
$$

### 贝叶斯推断方法对比表

| 方法 | 类型 | 精度 | 速度 | 可扩展性 | 适用维度 | 是否无偏 | 典型应用 |
| ------- | ----- | ---- | --- | ---- | ---- | ----- | ----------------------- |
| 共轭先验 | 精确 | 精确 | 极快 | 低 | 任意 | 精确 | Beta-Binomial, Gaussian |
| 变量消元 | 精确 | 精确 | 中 | 低 | 低维 | 精确 | PGM精确推断 |
| 信念传播 | 精确/近似 | 树精确 | 中-快 | 中 | 中 | 树上精确 | 编译码、MAP |
| MH采样 | MCMC | 渐近精确 | 慢 | 低-中 | 中 | 是 | 通用后验采样 |
| Gibbs采样 | MCMC | 渐近精确 | 中 | 中 | 中 | 是 | 分层模型 |
| HMC | MCMC | 渐近精确 | 中 | 中 | 高 | 是 | 连续高维后验 |
| 均值场VI | 变分 | 近似 | 快 | 高 | 高 | 否(偏差) | 大规模贝叶斯 |

### 典型应用

- 参数估计与不确定性量化

- 贝叶斯回归与分类
- 信号检测中的自适应阈值设计

## 4. 方法流程

### 贝叶斯推断方法分类图

**贝叶斯推断的一般流程**：

**Step 1：建立模型**
- 选择似然函数 $p(\mathcal{D}|\theta)$：描述数据如何生成
- 选择先验分布 $p(\theta)$：编码领域知识或无信息先验
- 共轭先验可使推断解析可解；否则需要近似方法

**Step 2：推断后验**
根据问题规模和复杂度选择推断方法：

- **精确推断**（共轭先验、变量消元、信念传播）：适用于小规模、特殊结构问题
- **MCMC 采样**（MH、Gibbs、HMC）：渐近精确，适合中等规模
- **变分推断**（均值场 VI、BBVI）：快速近似，适合大规模问题
- **拉普拉斯近似**：在 MAP 点附近做高斯近似，最快但最粗糙

**Step 3：后验分析**
- 点估计：后验均值、MAP 估计
- 区间估计：可信区间（Credible Interval）
- 后验预测：对新数据的预测分布

**Step 4：模型评估**
- 边缘似然 $p(\mathcal{D})$ 用于模型选择
- 后验预测检查（Posterior Predictive Check）
- 交叉验证

**先验选择建议**：
- 有领域知识 → 信息先验（如 Beta(5,5) 表示"偏向0.5"）
- 无领域知识 → 无信息先验（如 Beta(1,1) 均匀分布，或 Jeffreys 先验）
- 共轭先验 → 解析可解，计算方便

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from scipy import stats
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

# Beta-Binomial共轭贝叶斯推断演示
# 场景：估计硬币正面概率θ
np.random.seed(42)
theta_true = 0.6  # 真实正面概率
n_trials = [0, 10, 50, 200]  # 不同观测数量
data = np.random.binomial(1, theta_true, size=max(n_trials))

# 不同先验参数的对比
priors = [(2, 2), (1, 1), (0.5, 0.5)]  # Beta先验参数
prior_labels = ['Beta(2,2) 弱先验', 'Beta(1,1) 均匀先验', 'Beta(0.5,0.5) 无信息先验']
theta_range = np.linspace(0.01, 0.99, 200)

fig, axes = plt.subplots(2, 2, figsize=(12, 10))
# 左上：不同先验分布
for (a0, b0), label in zip(priors, prior_labels):
    axes[0, 0].plot(theta_range, stats.beta.pdf(theta_range, a0, b0), linewidth=2, label=label)
axes[0, 0].set_title('不同先验分布', fontsize=13)
axes[0, 0].set_xlabel('θ')
axes[0, 0].set_ylabel('PDF')
axes[0, 0].legend(fontsize=10)
axes[0, 0].grid(True, alpha=0.3)
# 其余三个：随着数据增加后验更新
a0, b0 = 2, 2
for idx, n_obs in enumerate([10, 50, 200]):
    ax = axes.flat[idx + 1]
    k = np.sum(data[:n_obs])
    a_post, b_post = a0 + k, b0 + n_obs - k
    ax.plot(theta_range, stats.beta.pdf(theta_range, a0, b0), 'k--', linewidth=1.5, label='先验')
    ax.plot(theta_range, stats.beta.pdf(theta_range, a_post, b_post), 'r-', linewidth=2, label='后验')
    ax.axvline(theta_true, color='g', linestyle=':', linewidth=2, label=f'真实值={theta_true}')
    ax.set_title(f'n={n_obs}次观测 (正面={k}次)', fontsize=13)
    ax.set_xlabel('θ')
    ax.legend(fontsize=10)
    ax.grid(True, alpha=0.3)
plt.suptitle('贝叶斯推断：Beta-Binomial共轭更新', fontsize=15)
plt.tight_layout()
plt.show()
```

## 6. 常见误区

| 误区 | 正确理解 |
|------|----------|
| 贝叶斯推断就是贝叶斯公式 | 贝叶斯公式是基础，但贝叶斯推断是一个完整的框架，包括建模、先验选择、推断、后验分析等 |
| 先验是主观的，所以贝叶斯方法不客观 | 先验可以是无信息的（如均匀分布），且数据量大时先验影响减弱；先验是合理的领域知识编码 |
| 后验分布越集中越好 | 后验集中意味着不确定性小，但如果先验过于强势，可能掩盖数据信息；应检查后验是否合理反映不确定性 |
| 贝叶斯方法计算量大、不实用 | 共轭先验可解析求解；现代 MCMC/VI 方法可处理高维问题；概率编程（PyMC、Stan）大幅降低了使用门槛 |
| 后验均值就是最好的点估计 | 后验均值、中位数、MAP 各有优劣：均值最小化 MSE，MAP 最大化 0-1 损失，应根据损失函数选择 |
| 贝叶斯因子越大模型越好 | 贝叶斯因子受先验影响很大，且对复杂模型有自动的"奥卡姆惩罚"效应 |

### 思考题
1. 先验分布的选择如何影响后验？当数据量很小时和很大时，先验的影响有何不同？

### 创新拓展方向
- **概率编程**：PyMC、Stan、Edward 等框架让贝叶斯建模变得像写程序一样简单
- **贝叶斯深度学习**：给神经网络权值加上先验，量化预测不确定性
- **贝叶斯优化**：用贝叶斯推断指导超参数搜索（高斯过程 + 采集函数）
- **稀疏贝叶斯学习（SBL）**：自动相关性确定（ARD），用于 DOA 估计、压缩感知
- **贝叶斯联邦学习**：分布式场景下的贝叶斯推断，保护隐私的同时量化不确定性
- **因果推断**：贝叶斯网络与因果图模型的结合
