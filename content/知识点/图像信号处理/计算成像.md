## 3. 严谨核心

### 数学对象与基本假设

计算成像（Computational Imaging）是一类将光学系统设计与计算算法深度融合的成像技术。它不再依赖传统光学系统独立完成成像，而是通过联合优化光学编码与后端计算解码来获取传统方法难以获得的图像信息。

核心概念：
- **编码孔径（Coded Aperture）**：通过设计非传统的孔径形状/图案来编码场景信息
- **光场成像（Light Field）**：同时记录光线的空间和角度信息
- **压缩感知成像**：利用稀疏性先验，用远少于奈奎斯特采样的测量重建图像
- **相位恢复**：从强度测量中恢复相位信息
- **单像素成像**：用单个探测器结合空间光调制实现成像

### 核心公式

压缩感知成像的前向模型：

$$y = \Phi x + n$$

其中 $\Phi$ 为测量矩阵（编码矩阵），$x$ 为待重建图像，$y$ 为测量值。

压缩感知重建（$\ell_1$ 最小化）：

$$\hat{x} = \arg\min_x \|x\|_1 \quad \text{s.t.} \quad \|y - \Phi x\|_2 \le \epsilon$$

等价的无约束形式：

$$\hat{x} = \arg\min_x \frac{1}{2}\|y - \Phi x\|_2^2 + \lambda \|x\|_1$$

稀疏性条件——限制等距性质（RIP）：

$$(1-\delta_k)\|x\|_2^2 \le \|\Phi x\|_2^2 \le (1+\delta_k)\|x\|_2^2$$

对所有 $k$-稀疏向量 $x$ 成立，$\delta_k \in (0,1)$。

相位恢复（Phase Retrieval）——强度测量模型：

$$y_m = |\langle \mathbf{a}_m, x \rangle|^2, \quad m = 1, \dots, M$$

Gerchberg-Saxton迭代算法在傅里叶域和实空间域之间交替投影。

光场参数化——双平面参数化：

$$L(u, v, s, t) \quad \text{（4D光场，}(u,v)\text{为角度，}(s,t)\text{为空间）}$$

### 适用条件和边界

- 压缩感知要求信号在某个变换域下是稀疏的，且测量矩阵满足RIP条件
- 实际中RIP条件难以验证，通常用随机矩阵（如高斯矩阵）以高概率满足
- 相位恢复存在相位模糊问题（全局相位不可恢复），需要额外约束

## 4. 方法流程

**输入是什么：** 编码测量值 $y = \Phi x$（压缩感知）或强度测量 $y_m = |\langle \mathbf{a}_m, x \rangle|^2$（相位恢复）。

**处理步骤（压缩感知）：**
1. 设计测量矩阵 $\Phi$（随机矩阵、结构化矩阵）
2. 用 $\Phi$ 对信号进行编码测量，得到 $y$（$M \ll N$ 次测量）
3. 求解 $\ell_1$ 正则化优化问题（ISTA/FISTA算法）
4. 得到稀疏解 $\hat{x}$，即重建结果

**输出如何解释：** 重建结果是对原始信号的近似估计；稀疏性越强、测量越多，重建越精确。

**常见参数如何选择：**
- 测量数 $M$：通常 $M \geq C \cdot k \cdot \log(N/k)$，$k$ 为稀疏度
- 正则化参数 $\lambda$：控制数据保真与稀疏性之间的权衡
- ISTA步长：由Lipschitz常数决定，$L = \|\Phi^T\Phi\|_2$

### 技术对比

| 技术 | 测量方式 | 先验假设 | 重建算法 | 典型应用 | 优势 | 挑战 |
|------|---------|---------|---------|---------|------|------|
| 压缩感知 | 随机投影 | 稀疏性 | L1最小化/ISTA | 单像素相机、MRI | 采样率低 | 测量矩阵设计 |
| 编码孔径 | 特殊孔径编码 | 深度/光谱先验 | 反卷积/优化 | 深度估计、光谱成像 | 单次拍摄 | 编码图案设计 |
| 光场成像 | 微透镜阵列 | 多视角一致性 | 重聚焦/深度估计 | 3D成像、重聚焦 | 角度分辨率 | 空间分辨率损失 |
| 相位恢复 | 强度测量 | 支撑集/稀疏性 | GS迭代/Wirtinger Flow | X射线、全息 | 无需参考光 | 相位模糊 |
| 结构光 | 投射编码图案 | 表面连续性 | 解码+三角测量 | 3D扫描 | 高精度 | 需要投影仪 |

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

def ista_l1(Phi, y, lam=0.1, n_iter=300):
    """ISTA（迭代收缩阈值算法）求解 L1 正则化问题"""
    M, N = Phi.shape
    L = np.linalg.norm(Phi, ord=2)**2  # Lipschitz 常数（步长上限）
    x = np.zeros(N)
    for _ in range(n_iter):
        grad = Phi.T @ (Phi @ x - y)           # 梯度下降方向
        z = x - grad / L                        # 梯度步
        x = np.sign(z) * np.maximum(np.abs(z) - lam/L, 0)  # 软阈值（L1 促稀疏）
    return x

# 创建稀疏信号（64像素中只有4个非零）
np.random.seed(42)
N = 64
x_true = np.zeros(N)
x_true[10], x_true[25], x_true[40], x_true[55] = 1.0, 0.8, 1.2, 0.6

# 随机测量（仅20次，远少于64个像素）
M = 20
Phi = np.random.randn(M, N) / np.sqrt(M)
y = Phi @ x_true  # 测量值

# 压缩感知重建：L1 最小化
x_rec = ista_l1(Phi, y, lam=0.01, n_iter=300)

fig, axes = plt.subplots(1, 3, figsize=(12, 4))
axes[0].stem(x_true, linefmt='b-', markerfmt='bo', basefmt='k-')
axes[0].set_title(f'Original Sparse Signal ({N} pixels)')
axes[1].stem(y, linefmt='r-', markerfmt='ro', basefmt='k-')
axes[1].set_title(f'测量值（仅{M}次，远少于{N}像素）')
axes[2].stem(x_rec, linefmt='g-', markerfmt='go', basefmt='k-')
axes[2].set_title('ISTA 重建结果（L1 最小化恢复稀疏信号）')
for ax in axes:
    ax.set_xlabel('索引')
plt.tight_layout()
plt.show()
```

**结果解释：** 仅用 20 次测量（远少于 64 个像素），L1 最小化就能精确恢复出稀疏图像。这就是"压缩感知"的核心魅力——**测量次数远少于信号维度，只要信号是稀疏的就能重建**。

## 6. 常见误区

**公式误解：**
- $\ell_1$ 最小化不是"找最小的数"，而是促使解稀疏（多数元素为零）；$\ell_0$ 才是真正的稀疏度量但不可解
- RIP条件不是"充分必要条件"，只是充分条件；不满足RIP不代表不能重建

**参数误解：**
- 测量数 $M$ 不是越多越好——超过一定阈值后，增加测量带来的边际收益递减
- 正则化参数 $\lambda$ 不是固定的——需要根据噪声水平和信号稀疏度调整

**适用条件误解：**
- 压缩感知不适用于所有信号——只有在某个变换域下稀疏的信号才能用CS重建
- 相位恢复不能恢复绝对相位——只能恢复到全局相位模糊以内

**单位误解：**
- RIP中的 $\delta_k$ 不是"稀疏度"——它是等距常数，衡量测量矩阵对稀疏信号的保距性

## 7. 启发问题与创新拓展

**一个反例或边界问题：**
如果信号不是稀疏的会怎样？例如，一张自然图像在像素域不稀疏，但在小波域稀疏。如果我们用错误的变换域（如用DCT基去表示小波稀疏的信号），重建质量会急剧下降。这说明：**选择正确的稀疏表示基与选择测量矩阵同样重要**。

**一个跨学科迁移场景：**
MRI（磁共振成像）加速：MRI的k空间采样本质上是傅里叶域测量，采集速度受物理限制。压缩感知MRI通过欠采样k空间 + $\ell_1$ 重建，可以将扫描时间从30分钟缩短到5分钟，对儿科和急诊患者意义重大。这是计算成像从"理论"到"临床"最成功的案例之一。

**一个可做成课程项目的小想法：**
设计一个"单像素相机"实验：用手机屏幕显示随机黑白图案（作为测量矩阵），用一个光敏电阻（作为单探测器）测量透射光强，然后用ISTA算法从少量测量中重建出简单的二值图像。这个实验用不到100元的硬件就能体验压缩感知的完整流程。
