## 2. 通俗入口

> [!tip] 🎯 初学者简化说明
> **一句话理解**：入门时可以把图像去噪理解为"有选择地平滑"——压制随机Noise，但尽量不抹掉边缘和纹理。

> **你需要重点掌握的：**

> 3. 中值滤波：取邻域像素的中间值，对椒盐噪声效果最优且能保留边缘
>
> **作为进阶了解的（期末不要求推导）：**
> - TV（全变分）Regularization的数学原理
> - BM3D等现代图像去噪Algorithm的基本思路
>
> 💡 **一句话记忆法**：均值滤波=大家平均分，高斯滤波=听班长的（中心权重大），中值滤波=去掉最高最低取中间。

## 3. 严谨核心

### 定义与概念

图像去噪（Image Denoising）是从被噪声污染的Observation图像中恢复原始干净图像的过程。去噪是图像处理中最基础也最经典的问题之一。

核心概念：
- **加性噪声模型**：$g(x,y) = f(x,y) + n(x,y)$
- **常见噪声类型**：高斯噪声、椒盐噪声、泊松噪声
- **空域与频域方法**：基于空间邻域或频率滤波
- **非Local自相似性**：图像中存在大量相似的局部块（patch）
- **正则化与变分方法**：将去噪建模为优化问题

### 核心公式

加性噪声模型：

$$g(x,y) = f(x,y) + n(x,y)$$

其中 $f$ 为干净图像，$n$ 为噪声，$g$ 为观测图像。

高斯噪声概率Density：

$$p(n) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{n^2}{2\sigma^2}\right)$$

均值滤波（线性）：

$$\hat{f}(x,y) = \frac{1}{|W|} \sum_{(i,j) \in W} g(i,j)$$

高斯加权滤波：

$$\hat{f}(x,y) = \sum_{(i,j) \in W} w(i,j) \cdot g(i,j), \quad w(i,j) \propto \exp\left(-\frac{(i-x)^2+(j-y)^2}{2\sigma_s^2}\right)$$

双边滤波（Bilateral Filter）：

$$\hat{f}(x,y) = \frac{1}{W_p} \sum_{(i,j) \in W} g(i,j) \cdot G_{\sigma_s}(\|p-q\|) \cdot G_{\sigma_r}(|g(p)-g(q)|)$$

BM3D中的协同滤波核心思想——对相似块堆叠后进行3D变换域阈值处理。

全变分正则化去噪：

$$\hat{f} = \arg\min_f \frac{1}{2}\|g - f\|_2^2 + \lambda \|\nabla f\|_1$$

PSNR评价指标：

$$PSNR = 10 \log_{10} \frac{L^2}{MSE}, \quad MSE = \frac{1}{MN}\sum_{i,j}(f(i,j)-\hat{f}(i,j))^2$$

### 方法对比

| 方法 | 类型 | 核心思想 | 优点 | 缺点 | 典型PSNR提升 |
|------|------|----------|------|------|-------------|
| 均值滤波 | 线性/空域 | 邻域平均 | 计算简单 | 模糊边缘 | 低 |
| 高斯滤波 | 线性/空域 | 加权平均 | 平滑效果好 | 边缘模糊 | 低 |

### 深度学习方法补充

深度学习去噪并不是"自动替代"经典方法，而是在有足够训练数据、噪声分布相对匹配、评价方式清楚的条件下，学习从含噪图像到干净图像或噪声残差的映射。课程项目中建议把均值/高斯/中值/双边滤波作为基础对照，再讨论深度模型是否真的带来更好的细节保真。

**残差学习（DnCNN）**

残差学习的核心思想是让网络学习预测噪声残差，而非直接映射干净图像：

$$
\hat{\mathbf{x}} = \mathbf{y} - \mathcal{R}_\theta(\mathbf{y})
$$

其中 $\mathcal{R}_\theta$ 为网络学习的残差映射（近似噪声）。

**训练损失：**
$$
\mathcal{L}(\theta) = \frac{1}{N}\sum_{i=1}^{N}\|\mathcal{R}_\theta(\mathbf{y}_i) - \mathbf{n}_i\|^2
$$

**残差块结构：**
$$
\mathbf{z}^{(l+1)} = \text{ReLU}\left(\text{BN}\left(\mathbf{W}^{(l)} * \mathbf{z}^{(l)} + \mathbf{b}^{(l)}\right)\right)
$$

**DnCNN 架构**

典型DnCNN可以按下面的流程理解：

| 步骤 | 作用 | 本科理解 |
|------|------|----------|
| 含噪图像 $y$ Input | 提供待去噪观测 | 网络看到的是"Signal + 噪声"的混合结果 |
| 多LayerConvolution与非线性变换 | 提取局部纹理、边缘和噪声模式 | 类似可学习的滤波器组 |
| Output噪声残差 $\mathcal{R}_\theta(y)$ | 预测图像中应被去掉的噪声 | 不是直接画出干净图，而是先估计噪声 |
| 残差相减 $\hat{x}=y-\mathcal{R}_\theta(y)$ | 得到去噪结果 | 保留原图结构，同时减去模型认为的噪声 |

**深度学习方法对比**

| 方法 | 年份 | 网络结构 | 是否盲去噪 | σ=25 PSNR | 特点 |
|------|------|---------|-----------|-----------|------|
| BM3D | 2007 | 非学习 | 否 | ~28.5dB | 传统方法基线 |
| DnCNN | 2017 | 残差CNN | 是 | ~29.2dB | 残差学习、BN |
| FFDNet | 2018 | CNN | 是 | ~29.2dB | 噪声水平图输入 |
| SwinIR | 2021 | Swin Transformer | 是 | ~30.0dB | 窗口注意力 |
| Restormer | 2022 | Transformer | 是 | ~30.3dB | 通道维度注意力 |

## 4. 方法流程

**输入是什么：** 含噪观测图像 $g(x,y)$，噪声类型和水平可以已知（非盲去噪）或未知（盲去噪）。

**处理步骤：**
1. **噪声估计**：估计噪声类型（高斯/椒盐/泊松）和噪声水平 $\sigma$
2. **选择方法**：根据噪声类型和计算预算选择滤波器或学习模型
3. **空域滤波**：对图像逐像素应用均值/高斯/中值/双边滤波器
4. **变换域处理**：将图像分块 → 变换（DCT/小波）→ 阈值/收缩 → 逆变换
5. **深度学习推理**：将含噪图像输入训练好的网络（如DnCNN），直接输出去噪结果或噪声残差
6. **质量评估**：计算 PSNR、SSIM 等定量指标，并目视检查边缘纹理保持情况

**输出如何解释：** 去噪结果 $\hat{f}$ 应在噪声抑制和细节保持之间取得平衡。PSNR 反映整体像素误差，SSIM 反映结构相似性，但都不能完全替代目视评估。

**常见参数如何选择：**
- 滤波器窗口大小：3×3 或 5×5（均值/中值），σ=1.0~2.0（高斯）
- 双边滤波：空间 σ_s=3~10，亮度 σ_r=0.1~0.3
- TV正则化：λ=0.05~0.2（噪声越强 λ 越大）
- DnCNN：σ=25 训练的模型对 σ=15~35 范围泛化较好

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
from scipy.ndimage import uniform_filter, gaussian_filter, median_filter

# 创建TestImage：包含边缘和纹理的合成图像
N = 128
img_clean = np.zeros((N, N))
img_clean[20:50, 20:80] = 1.0      # 矩形区域
img_clean[70:110, 30:60] = 0.7     # 灰度矩形
img_clean[80:100, 80:110] = 0.4    # 低对比度区域
# 添加圆形纹理
for i in range(N):
    for j in range(N):
        if (i-64)**2 + (j-64)**2 < 20**2:
            img_clean[i, j] = 0.5 + 0.2*np.sin(0.5*i)*np.cos(0.5*j)

# 添加高斯Noise
sigma_noise = 0.2
np.random.seed(42)
img_noisy = img_clean + sigma_noise * np.random.randn(N, N)
img_noisy = np.clip(img_noisy, 0, 1)

# 三种DenoisingMethod
img_mean = uniform_filter(img_noisy, size=5)          # MeanFiltering
img_gauss = gaussian_filter(img_noisy, sigma=1.5)     # 高斯滤波
img_median = median_filter(img_noisy, size=5)          # 中值滤波

# 计算PSNR
def psnr(original, processed):
    mse = np.mean((original - processed) ** 2)
    return 10 * np.log10(1.0 / mse) if mse > 0 else float('inf')

# 显示结果
fig, axes = plt.subplots(2, 3, figsize=(10, 7))
titles = ['Primal Image', 'Noisy Image', 'Mean Filtering', '高斯Filtering', '中值Filtering']
images = [img_clean, img_noisy, img_mean, img_gauss, img_median]
for i, (ax, title, img) in enumerate(zip(axes.flat[:5], titles, images)):
    ax.imshow(img, cmap='gray', vmin=0, vmax=1)
    ax.set_title(title, fontsize=11)
    ax.axis('off')
# 最后一格显示PSNR对比
ax = axes.flat[5]
psnr_vals = [psnr(img_clean, img_noisy), psnr(img_clean, img_mean),
             psnr(img_clean, img_gauss), psnr(img_clean, img_median)]
bars = ax.bar(['Noisy', 'Mean', 'Gaussian', 'Median'], psnr_vals, color=['gray','skyblue','salmon','lightgreen'])
ax.set_ylabel('PSNR (dB)')
ax.set_title('Denoising效果对比')
plt.tight_layout()
plt.show()
```

## 6. 常见误区

| 易错点 | 正确理解 |
|--------|----------|
| 去噪就是让Image更平滑 | 去噪要同时考虑噪声抑制和细节保真，过度平滑会抹掉边缘与纹理 |
| PSNR越高，视觉效果一定越好 | PSNR偏向平均误差，可能无法反映纹理、边缘和感知质量 |
| 深度学习方法一定优于经典Filtering | 深度Model依赖TrainingData和噪声Hypothesis，遇到真实噪声Distribution偏移时可能失效 |
| 噪声模型可以随便假设 | 高斯、椒盐、泊松噪声对应不同成因和适用算法 |

**公式误解：** 双边滤波中的亮度权重 $G_{\sigma_r}(|g(p)-g(q)|)$ 不是"越亮权重越大"，而是"亮度差越小权重越大"——相似亮度的像素被保留，差异大的（边缘处）被抑制。

**参数误解：** 中值滤波的窗口不是越大越好——大窗口会抹掉小的纹理细节，且计算量随窗口面积增长。

**适用条件误解：** 经典滤波方法对空间平稳噪声效果好，但对信号依赖型噪声（如泊松噪声）和真实复杂噪声（如手机低光照ISP噪声）效果有限。

**一个反例或边界问题：**
如何判断一张医学图像中的细小结构是真实Signal还是噪声？——如果该结构在不同滤波方法下都保留且空间位置一致，大概率是真实信号；如果仅在特定噪声实现中出现且位置随机，则可能是噪声。
