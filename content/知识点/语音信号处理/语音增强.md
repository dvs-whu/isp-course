> 3. **处理流程**：分帧加窗 → FFT → 噪声估计 → 计算增益 → 增益应用 → IFFT 重建

> **作为进阶了解的（期末不要求推导）：**

>
> 💡 **一句话记忆法**：谱减法="估计噪声再减掉"，维纳Filtering="Minimize误差的最优滤波"

## 3. 严谨核心

### 数学对象与基本假设

语音增强（Speech Enhancement）是指从含噪语音信号中去除或抑制背景噪声，恢复出干净语音信号的技术。其目标是在不引入明显失真的前提下，尽可能提高语音质量和可懂度。

核心概念：
- **加性噪声Model**：$x(n) = s(n) + d(n)$，其中 $s(n)$ 为纯净语音，$d(n)$ 为噪声
- **信噪比（SNR）**：衡量语音质量的基本指标
- **语音存在概率（SPP）**：估计当前帧中语音是否存在的概率
- **先验信噪比**：用于改进增益函数的估计，常用直接Decision法（Decision-Directed）

### 核心公式

频域加性噪声模型：

$$X(\omega) = S(\omega) + D(\omega)$$

谱减法（Spectral Subtraction）：

$$|\hat{S}(\omega)| = \max\left(|X(\omega)| - \alpha |\hat{D}(\omega)|, \beta |X(\omega)|\right)$$

其中 $\alpha$ 为过减因子，$\beta$ 为谱下限因子。

维纳Filter增益函数：

$$H(\omega) = \frac{\Phi_{ss}(\omega)}{\Phi_{ss}(\omega) + \Phi_{dd}(\omega)} = \frac{\xi(\omega)}{\xi(\omega) + 1}$$

其中 $\xi(\omega) = \Phi_{ss}(\omega)/\Phi_{dd}(\omega)$ 为先验信噪比。

先验信噪比的直接判决法（Decision-Directed）估计：

$$\hat{\xi}(\omega, m) = \alpha_d \frac{|\hat{S}(\omega, m-1)|^2}{|\hat{D}(\omega)|^2} + (1-\alpha_d) \max\left(\frac{|X(\omega, m)|^2}{|\hat{D}(\omega)|^2} - 1, 0\right)$$

MMSE短时谱幅度估计（Ephraim-Malah）：

$$|\hat{S}(\omega)| = \frac{\sqrt{\pi}}{2} \cdot \frac{\sqrt{\nu}}{\gamma} \cdot \exp\left(-\frac{\nu}{2}\right) \left[(1+\nu)I_0\left(\frac{\nu}{2}\right) + \nu I_1\left(\frac{\nu}{2}\right)\right] \cdot |X(\omega)|$$

其中 $\nu = \frac{\xi}{1+\xi}\gamma$，$\gamma$ 为后验信噪比，$I_0, I_1$ 为修正贝塞尔函数。

### 适用条件和边界

- 加性噪声模型假设噪声与语音独立叠加，非加性噪声（如混响、乘性噪声）不适用
- 谱减法假设噪声平稳或缓慢变化，快变噪声效果差
- 维纳滤波在最小均方误差意义下最优，但假设噪声功率谱已知

## 4. 方法流程

**输入是什么：** 含噪语音信号 $x(n)$，采样率 $f_s$。

**处理步骤：**
1. 分帧加窗（帧长20-30ms，帧移10-15ms，汉明窗）
2. STFT得到时频表示 $X(\omega, m)$
3. 噪声估计（用前几帧无声段或最小统计量方法）
4. 计算增益函数 $G(\omega, m)$（谱减/维纳/MMSE）
5. 增益应用：$\hat{S}(\omega, m) = G(\omega, m) \cdot X(\omega, m)$
6. ISTFT重建时域信号

**输出如何解释：** 增强后的语音信号，用SNR、PESQ、STOI等指标评价。

**常见参数如何选择：**
- 帧长：256点（8kHz采样率）或512点（16kHz）
- 过减因子 $\alpha$：1.0-2.0，越大去噪越强但失真越大
- 谱下限 $\beta$：0.01-0.1，防止"音乐噪声"
- DD平滑因子 $\alpha_d$：0.95-0.99

### 方法对比

| 方法 | 增益函数 | 是否需要噪声估计 | 计算复杂度 | 音乐噪声 | 语音质量 |
| ------|---------|----------------|-----------|---------|---------|
| 谱减法 | 硬门限减除 | 是 | $O(N\log N)$ | 严重 | 一般 |
| Wiener Filtering | $\xi/(\xi+1)$ | 是 | $O(N\log N)$ | 较轻 | 较好 |
| MMSE-STSA | Bessel 函数修正 | 是 | $O(N\log N)$ | 轻微 | 好 |
| log-MMSE | 对数谱 MMSE | 是 | $O(N\log N)$ | 轻微 | 好 |
| 深度学习（DNN） | 神经网络估计掩膜 | 否（隐式学习） | 取决于网络 | 无 | 优 |

### 深度学习方法补充

**理想比率掩模（IRM）：**
$$
| \text{IRM}(m, k) = \frac{|S(m,k)|^\beta}{|S(m,k)|^\beta + |N(m,k)|^\beta}
$$

**掩模应用：**
$$
\hat{S}(m, k) = M_\theta(m, k) \cdot X(m, k)
$$

**SI-SNR（尺度不变信噪比）：**
$$
| \text{SI-SNR} = 10\log_{10}\frac{\|\mathbf{s}_{\text{target}}\|^2}{\|\mathbf{e}_{\text{noise}}\|^2}
$$
| 其中 $\mathbf{s}_{\text{target}} = \frac{\langle \hat{\mathbf{s}}, \mathbf{s}\rangle}{\|\mathbf{s}\|^2}\mathbf{s}$。

```python
# === 环境预加载 ===
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
matplotlib.rcParams['font.sans-serif'] = ['SimHei', 'Arial Unicode MS', 'DejaVu Sans']
matplotlib.rcParams['axes.unicode_minus'] = False

import numpy as np
import matplotlib.pyplot as plt
from scipy.signal import stft, istft

# 生成模拟语音Signal
fs = 8000
duration = 0.5
t = np.arange(0, duration, 1/fs)
# 纯净语音：两个Frequency分量模拟元音
clean = 0.5*np.sin(2*np.pi*200*t) + 0.3*np.sin(2*np.pi*500*t)
# 加性高斯白Noise
np.random.seed(0)
noise = 0.3 * np.random.randn(len(t))
noisy = clean + noise

# STFTParameter
nperseg = 256
noverlap = 128
f_stft, t_stft, Zxx_noisy = stft(noisy, fs=fs, nperseg=nperseg, noverlap=noverlap)
_, _, Zxx_clean = stft(clean, fs=fs, nperseg=nperseg, noverlap=noverlap)

# 谱减法
mag_noisy = np.abs(Zxx_noisy)
phase_noisy = np.angle(Zxx_noisy)
# 用前5帧Estimation噪声功率谱（Hypothesis前5帧只有噪声）
noise_est = np.mean(mag_noisy[:, :5]**2, axis=1, keepdims=True)
# 谱减：Amplitude谱减去噪声估计，取下限防止负值
alpha = 1.5  # 过减因子
beta = 0.05  # 谱下限因子
mag_enhanced = np.maximum(mag_noisy**2 - alpha * noise_est, beta * mag_noisy**2)
mag_enhanced = np.sqrt(mag_enhanced)

# 维纳FilteringGain
xi = np.maximum(mag_enhanced**2 / (noise_est + 1e-10), 0)  # 先验SNR
H_wiener = xi / (xi + 1)  # 维纳增益
mag_wiener = H_wiener * mag_noisy

# Reconstruction信号
Zxx_enhanced = mag_enhanced * np.exp(1j * phase_noisy)
Zxx_wiener = mag_wiener * np.exp(1j * phase_noisy)
_, enhanced = istft(Zxx_enhanced, fs=fs, nperseg=nperseg, noverlap=noverlap)
_, wiener_out = istft(Zxx_wiener, fs=fs, nperseg=nperseg, noverlap=noverlap)

# 计算SNR
def snr_db(signal, estimate):
    err = signal[:len(estimate)] - estimate[:len(signal)]
    return 10*np.log10(np.sum(signal**2)/(np.sum(err**2)+1e-10))

# 绘图
fig, axes = plt.subplots(2, 2, figsize=(12, 8))
axes[0,0].imshow(20*np.log10(np.abs(Zxx_noisy)+1e-6), aspect='auto', origin='lower',
                 extent=[0, duration, 0, fs/2])
axes[0,0].set_title(f'Noisy Speech Spectrogram (SNR={snr_db(clean, noisy):.1f}dB)')
axes[0,1].imshow(20*np.log10(mag_enhanced+1e-6), aspect='auto', origin='lower',
                 extent=[0, duration, 0, fs/2])
axes[0,1].set_title(f'谱减法增强 (SNR={snr_db(clean, enhanced):.1f}dB)')
t_plot = t[:len(enhanced)]
axes[1,0].plot(t_plot, clean[:len(enhanced)], 'b-', alpha=0.5, label='纯净语音')
axes[1,0].plot(t_plot, enhanced[:len(enhanced)], 'r-', alpha=0.7, label='谱减法')
axes[1,0].set_title('时域波形对比'); axes[1,0].legend()
axes[1,1].plot(t_plot, wiener_out[:len(enhanced)], 'g-', alpha=0.7, label='维纳Filtering')
axes[1,1].plot(t_plot, clean[:len(enhanced)], 'b-', alpha=0.5, label='纯净语音')
axes[1,1].set_title(f'维纳Filtering增强 (SNR={snr_db(clean, wiener_out):.1f}dB)'); axes[1,1].legend()
for ax in axes.flat:
    ax.set_xlabel('Time (s)')
plt.tight_layout()
plt.show()
```

**结果解释：**
- 语谱图对比可直观看到噪声被抑制的效果
- 谱减法会引入"音乐噪声"（语谱图上的残留纹理）
- 维纳滤波的结果通常比谱减法更平滑，音乐噪声更轻
- SNR提升幅度反映了去噪效果

## 6. 常见误区

**公式误解：**
- 谱减法不是"直接减噪声"——是对幅度谱做减法并取下限，相位仍用含噪语音的相位
- 维纳增益 $H = \xi/(\xi+1)$ 不是"信噪比越高增益越大"——它是后验估计，$\xi=0$时$H=0$（完全抑制），$\xi\to\infty$时$H=1$（完全保留）

**参数误解：**
- 过减因子 $\alpha$ 不是越大越好——过大会导致语音失真和"音乐噪声"
- 谱下限 $\beta$ 不是"残留噪声"——它是防止频谱完全为零的保底值

**适用条件误解：**
- 谱减法不能处理非平稳噪声（如突发性噪声）——需要实时更新噪声估计
- 语音增强不等于语音识别——增强后的语音人耳听着更清晰，但不一定提高识别率

**单位误解：**
- SNR的单位是dB，不是线性比例；0dB表示信号功率等于噪声功率

## 7. 启发问题与创新拓展

**一个反例或边界问题：**
如果噪声恰好与语音频谱完全重叠（如另一个说话人的语音），谱减法和维纳滤波都会失效——因为它们无法区分"语音"和"噪声"在频域上的重叠部分。这就是为什么"语音分离"比"语音增强"更难——纯加性噪声可以用幅度谱减法，但混叠语音需要更复杂的模型。

**一个跨学科迁移场景：**
地震信号去噪：地震勘探中的信号与语音信号有相似的时频结构——都是"有用信号+背景噪声"的叠加。语音增强中的谱减法和维纳滤波可以直接应用于地震数据的去噪处理，只需调整参数（如帧长、频率范围）。

**一个可做成课程项目的小想法：**
设计一个"语音增强方法渐进对比"实验：用同一段含噪语音，分别用谱减法、维纳滤波和简单的DNN掩模方法进行增强，对比PESQ分数和语谱图。特别关注：在低SNR（-5dB）和高SNR（10dB）条件下，三种方法的优劣是否反转？
